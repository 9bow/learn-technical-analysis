## 단일 캔들 패턴 개요

단일 캔들 패턴은 하나의 캔들스틱만으로 시장 심리를 읽어내는 기법입니다. 18세기 일본의 쌀 선물 거래자 혼마 무네히사(本間宗久)가 고안한 이 방법은 단 하나의 봉에서 매수·매도 세력의 균형을 파악할 수 있다는 점에서 강력합니다.

**단일 캔들 패턴 해석의 핵심 원칙:**- 캔들의 위치(추세 중 어디에 있는가)가 형태만큼 중요합니다
- 대부분의 단일 캔들은 다음 캔들의 확인(confirmation)이 필요합니다
- 거래량이 높을수록 패턴의 신뢰도가 올라갑니다
- 지지선·저항선과 겹칠 때 신호 강도가 배가됩니다

## 1. 도지 (Doji)

도지는 시가와 종가가 거의 동일한 캔들입니다. 매수세와 매도세가 정확히 균형을 이룬 상태를 나타내며, 현재 추세의 소진 또는 전환 가능성을 시사합니다. [Investopedia](https://www.investopedia.com/trading/candlestick-charting-what-is-it/)는 도지를 불확실성과 우유부단함의 상징으로 정의합니다.

### 도지의 4가지 유형

#### 1-1. 표준 도지 (Standard Doji)

```
    |
   ---    ← 시가 = 종가 (십자 형태)
    |
```

- 위아래 그림자 길이가 비슷합니다
- 매수·매도 세력이 대등하게 맞선 상태입니다
- 추세 전환보다는 추세 약화의 신호로 해석합니다

#### 1-2. 잠자리 도지 (Dragonfly Doji)

```
   ---    ← 시가 = 종가 (상단)
    |
    |
    |     ← 긴 아래 그림자
```

- 시가와 종가가 당일 고가에 형성됩니다
- 장 중 크게 하락했으나 매수세가 강하게 반등시킨 형태입니다
- **하락 추세 **에 나타나면 강한 상승 반전 신호입니다

#### 1-3. 비석 도지 (Gravestone Doji)

```
    |
    |
    |     ← 긴 위 그림자
   ---    ← 시가 = 종가 (하단)
```

- 시가와 종가가 당일 저가에 형성됩니다
- 장 중 크게 상승했으나 매도세가 강하게 눌러버린 형태입니다
- **상승 추세 **에 나타나면 강한 하락 반전 신호입니다

#### 1-4. 장다리 도지 (Long-Legged Doji)

```
    |     ← 긴 위 그림자
    |
   ---    ← 시가 = 종가
    |
    |     ← 긴 아래 그림자
```

- 위아래 그림자가 모두 매우 깁니다
- 극심한 변동성 속에서도 결국 시가 수준으로 돌아온 상태입니다
- 방향성 불확실성이 매우 높으며, 큰 변동의 전조일 수 있습니다

### 도지 유형별 비교

| 유형 | 위 그림자 | 아래 그림자 | 주요 출현 위치 | 신호 방향 |
|------|----------|------------|--------------|---------|
| 표준 도지 | 짧음 | 짧음 | 추세 중간 | 중립/약화 |
| 잠자리 도지 | 없음/매우 짧음 | 길음 | 하락 추세 끝 | 상승 반전 |
| 비석 도지 | 길음 | 없음/매우 짧음 | 상승 추세 끝 | 하락 반전 |
| 장다리 도지 | 길음 | 길음 | 변동성 구간 | 방향 불확실 |

>**실전 팁:** 도지 단독으로 매매 결정을 내리지 마십시오. 반드시 다음 캔들이 예상 방향으로 움직이는 것을 확인한 뒤 진입합니다.

## 2. 해머 (Hammer)와 행잉맨 (Hanging Man)

이 두 패턴은 형태가 동일하지만 출현 위치에 따라 완전히 반대의 의미를 가집니다. 몸통이 작고 아래 그림자가 몸통 길이의 2배 이상이며 위 그림자가 없거나 매우 짧은 것이 특징입니다.

### 해머 (Hammer) — 상승 반전

```
   [ 몸통 ]   ← 작은 몸통 (양봉 또는 음봉)
      |
      |        ← 아래 그림자: 몸통의 2배 이상
      |
```

**출현 위치:** 하락 추세의 끝

**시장 심리:**- 장 중 매도세가 강하게 하락을 밀어붙였습니다
- 그러나 강한 매수세가 가격을 몸통 수준까지 끌어올렸습니다
- 매도세의 한계와 매수세의 등장을 동시에 보여줍니다

**신뢰도를 높이는 조건:**- 양봉 해머가 음봉 해머보다 신뢰도가 높습니다
- 다음 캔들이 양봉으로 마감하면 패턴 확인입니다
- 주요 지지선 근처에서 출현할 때 강도가 높습니다

### 행잉맨 (Hanging Man) — 하락 반전

```
   [ 몸통 ]   ← 동일한 형태
      |
      |        ← 아래 그림자: 몸통의 2배 이상
      |
```

**출현 위치:** 상승 추세의 끝 (교수형을 연상시키는 이름)

**시장 심리:**- 상승 추세 중 장 중에 강한 매도세가 등장했습니다
- 결국 가격을 끌어올렸지만, 매도 압력이 커지고 있음을 경고합니다
- 매수세 소진의 조기 신호입니다

| 항목 | 해머 | 행잉맨 |
|------|------|--------|
| 선행 추세 | 하락 | 상승 |
| 예측 방향 | 상승 반전 | 하락 반전 |
| 형태 | 동일 | 동일 |
| 확인 캔들 | 양봉 | 음봉 (갭 다운 포함 시 강함) |

## 3. 역해머 (Inverted Hammer)와 슈팅스타 (Shooting Star)

역해머와 슈팅스타도 형태가 동일하지만 위치로 구분합니다. 작은 몸통과 긴 위 그림자(몸통의 2배 이상), 없거나 매우 짧은 아래 그림자가 특징입니다.

### 역해머 (Inverted Hammer) — 상승 반전 가능성

```
      |
      |        ← 위 그림자: 몸통의 2배 이상
      |
   [ 몸통 ]   ← 작은 몸통
```

**출현 위치:** 하락 추세의 끝

**시장 심리:**- 매수세가 장 중에 가격을 크게 올렸습니다
- 그러나 매도세에 밀려 시가 근처로 되돌아왔습니다
- 매수세가 등장하기 시작했다는 점에서 의미가 있으나, 해머보다 신뢰도가 낮습니다
- **반드시 다음 양봉 캔들로 확인이 필요합니다**
### 슈팅스타 (Shooting Star) — 하락 반전

```
      |
      |        ← 위 그림자: 몸통의 2배 이상
      |
   [ 몸통 ]   ← 작은 몸통
```

**출현 위치:** 상승 추세의 끝

**시장 심리:**- 장 중 강하게 상승했지만 매도세에 완전히 눌렸습니다
- 매수세 소진과 매도세 등장을 명확히 보여주는 강한 하락 신호입니다
- 음봉 슈팅스타가 양봉보다 신뢰도가 높습니다

### 4가지 패턴 형태 비교

```
  역해머/슈팅스타       해머/행잉맨
      |                 [ 몸통 ]
      |                    |
   [ 몸통 ]                |
```

| 패턴 | 긴 그림자 위치 | 선행 추세 | 신호 |
|------|-------------|---------|------|
| 해머 | 아래 | 하락 | 상승 반전 |
| 행잉맨 | 아래 | 상승 | 하락 반전 |
| 역해머 | 위 | 하락 | 상승 반전 (확인 필요) |
| 슈팅스타 | 위 | 상승 | 하락 반전 |

## 4. 마루보즈 (Marubozu)

마루보즈는 일본어로 "대머리"를 의미하며, 그림자가 전혀 없는 캔들입니다. 시가에서 종가까지 한 방향으로만 움직인 것으로, 매수 또는 매도 세력이 장 전체를 완전히 지배했음을 나타냅니다.

### 강세 마루보즈 (Bullish Marubozu)

```
   ┌─────┐   ← 시가 (하단)
   │     │
   │ 양봉 │
   │     │
   └─────┘   ← 종가 (상단), 그림자 없음
```

- 시가가 당일 저가, 종가가 당일 고가입니다
- 장 시작부터 끝까지 매수세가 완전히 지배했습니다
- 강한 상승 추세의 지속 또는 강력한 추세 전환을 의미합니다

### 약세 마루보즈 (Bearish Marubozu)

```
   ┌─────┐   ← 시가 (상단), 그림자 없음
   │     │
   │ 음봉 │
   │     │
   └─────┘   ← 종가 (하단)
```

- 시가가 당일 고가, 종가가 당일 저가입니다
- 장 시작부터 끝까지 매도세가 완전히 지배했습니다
- 강한 하락 추세의 지속 또는 강력한 추세 전환을 의미합니다

### 마루보즈 실전 활용

| 상황 | 해석 | 대응 |
|------|------|------|
| 상승 추세 중 강세 마루보즈 | 추세 가속화 | 기존 포지션 유지, 추가 진입 고려 |
| 하락 추세 끝의 강세 마루보즈 | 강한 상승 반전 신호 | 롱 진입 검토 (지지선 확인 후) |
| 상승 추세 끝의 약세 마루보즈 | 강한 하락 반전 신호 | 숏 진입 또는 익절 검토 |
| 하락 추세 중 약세 마루보즈 | 추세 가속화 | 기존 포지션 유지 |

>**주의:** 마루보즈는 단독으로도 강력한 신호이지만, 이후 캔들에서 되돌림(retracement)이 자주 발생합니다. 진입 타이밍을 급하게 잡지 마십시오.

## 5. 팽이 (Spinning Top)

팽이는 위아래 그림자가 몸통보다 길고, 몸통이 작은 캔들입니다. 도지와 유사하지만 몸통이 완전히 사라지지 않은 상태입니다.

```
      |        ← 위 그림자
      |
   [ 몸통 ]   ← 작은 몸통 (양봉 또는 음봉)
      |
      |        ← 아래 그림자
```

**시장 심리:**- 매수세와 매도세가 격렬하게 싸웠지만 결정적인 승자가 없었습니다
- 추세의 소진 가능성을 시사하지만 방향성은 불확실합니다
- 도지보다 약한 신호이며 반드시 확인 캔들이 필요합니다

**팽이 vs 도지 비교:**
| 항목 | 팽이 | 도지 |
|------|------|------|
| 몸통 | 작지만 존재 | 거의 없음 (시가 = 종가) |
| 불확실성 | 중간 | 높음 |
| 신호 강도 | 약함 | 중간 |
| 단독 신뢰도 | 낮음 | 낮음 |

## 수치 계산 예시: 해머 패턴 손절·목표가 설정

실제 숫자를 이용해 해머 패턴 진입 전 과정을 계산합니다.

```
상황: 삼성전자 일봉 차트
- 하락 추세 10일 지속 (90,000원 → 72,000원)
- 72,000원 수평 지지선에서 해머 캔들 출현

해머 캔들 세부:
  시가:  73,500원
  고가:  73,800원
  저가:  70,200원  ← 긴 아래 그림자
  종가:  73,400원  ← 시가 근처 마감

그림자 비율 검증:
  몸통 길이:      |73,500 - 73,400| = 100원
  아래 그림자:    73,400 - 70,200 = 3,200원
  비율:           3,200 / 100 = 32배 → 해머 기준(2배) 충족 ✓

거래량 확인:
  해머 발생일 거래량: 1,850만 주
  20일 평균 거래량:   980만 주
  비율: 1,850 / 980 = 1.89배 → 150% 이상 ✓

다음 날 확인 캔들: 양봉, 종가 75,200원

진입 계획:
  진입가:   75,200원 (확인 캔들 종가)
  손절가:   70,000원 (해머 저가 70,200원 아래 ATR 버퍼)
  1차 목표: 80,000원 (직전 저항선)
  2차 목표: 84,000원 (하락 전 고점)

계산:
  리스크:       75,200 - 70,000 = 5,200원
  1차 리워드:   80,000 - 75,200 = 4,800원 → RR ≈ 0.92 (진입 재검토)
  2차 리워드:   84,000 - 75,200 = 8,800원 → RR ≈ 1.69 (허용 가능)

결론: 2차 목표 기준 RR 1.69이므로 진입 가능. 단, 1차 목표에서
      50% 청산 후 나머지를 2차 목표까지 유지하는 분할 청산 적용.
```

## 실전 적용 가이드

### 확인 캔들 기다리기

단일 캔들만으로 포지션에 진입하는 것은 위험합니다. 아래 절차를 따르십시오.

1. 단일 캔들 패턴 식별
2. 패턴이 주요 지지/저항선, 추세선, 이동평균선과 겹치는지 확인
3. 거래량이 평균보다 높은지 확인
4. 다음 캔들이 예상 방향으로 마감하는지 기다림
5. 확인 후 진입, 패턴 반대편에 손절매 설정

### 손절매 설정 원칙

```
상승 반전 패턴 (해머, 잠자리 도지 등):
  손절매 = 패턴 캔들의 저점 아래

하락 반전 패턴 (슈팅스타, 비석 도지 등):
  손절매 = 패턴 캔들의 고점 위
```

## 흔한 실수

### 실수 1: 패턴 위치를 무시하고 형태만 보기

```
잘못된 접근:
상승 추세 한가운데에서 해머 발견 → 즉시 롱 진입

현실: 해머는 하락 추세의 끝에서만 유효한 상승 반전 신호
     상승 추세 중의 해머는 아무런 의미가 없음
```

### 실수 2: 확인 캔들 없이 즉시 진입

```
잘못된 접근:
잠자리 도지 출현 → 당일 장 마감 전 시장가 매수

올바른 접근:
잠자리 도지 마감 확인
→ 다음 날 시가가 도지 종가 위에서 형성되는지 확인
→ 양봉으로 진행 중임을 확인한 후 진입
→ 손절: 도지 저가 하방 ATR × 0.25

이유: 도지 단독 성공률 ≈ 51%, 확인 캔들 후 성공률 ≈ 64%
```

### 실수 3: 거래량 없는 도지를 신뢰

```
점심 시간대(12:00~13:30) 또는 거래량이 평균의 50% 미만인 구간의
도지는 유의미한 신호가 아님.
매수·매도 세력이 모두 적어 균형처럼 보이는 것일 뿐임.
```

### 실수 4: 마루보즈를 보자마자 추격 매수

```
잘못된 접근:
강세 마루보즈 발생 → "강력 신호! 지금 당장 매수!"

현실: 마루보즈 직후에는 단기 과매수 상태
     이후 되돌림(pullback) 발생이 매우 빈번

올바른 접근:
마루보즈 이후 되돌림을 기다려 조정 구간에서 진입
손절: 마루보즈 저가 하방
```

### 실수 5: 비석 도지를 하락 추세에서 매도 신호로 해석

```
비석 도지는 상승 추세 끝에서만 하락 반전 신호
하락 추세 중 비석 도지는 일시적 반등 시도가 실패한 것으로,
추세 지속(하락 계속) 신호로 해석하는 것이 더 정확함
```

## 에지 케이스: 비정상 시장 상황

### 갭 발생 시 패턴 해석 조정

```
국내 주식: 시초가 갭 업/다운이 잦음
→ 전날 해머가 출현해도 다음 날 갭 다운으로 시작하면
  해머의 지지선이 이미 하향 돌파된 것
→ 이 경우 해머 패턴 무효로 처리

암호화폐: 24시간 연속 거래로 갭이 거의 없음
→ 전통 기준보다 패턴 정확도가 낮을 수 있음
→ 더 높은 거래량 기준(200% 이상)을 요구
```

### 공시·이벤트 직후 패턴

```
어닝 서프라이즈, 금리 발표 등 대형 이벤트 직후:
→ 급등/급락 캔들이 나타나도 패턴으로 해석 금지
→ 이벤트로 인한 구조적 가격 재설정 중
→ 최소 3~5일 후 안정화된 후 패턴 분석 재개
```

## 핵심 요약

- **도지는** 매수·매도 세력의 균형을 나타내며, 잠자리 도지(하락 끝, 상승 신호)와 비석 도지(상승 끝, 하락 신호)가 가장 신뢰도가 높습니다
- **해머와 행잉맨은** 형태가 동일하나 위치(하락 끝 vs 상승 끝)로 의미가 결정되며, 아래 그림자가 길수록 강한 신호입니다
- **슈팅스타와 역해머는** 위 그림자가 긴 패턴으로, 슈팅스타(상승 끝)가 역해머(하락 끝)보다 신뢰도가 높습니다
- **마루보즈는** 그림자가 없는 강력한 추세 신호로, 단일 캔들 중 신뢰도가 가장 높습니다
- 모든 단일 캔들 패턴은위**치, 거래량, 확인 캔들의** 세 가지 요소를 함께 분석할 때 신뢰도가 크게 향상됩니다
