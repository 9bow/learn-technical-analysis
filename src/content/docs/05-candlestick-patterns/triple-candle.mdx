## 삼중 캔들 패턴 개요

삼중 캔들 패턴은 세 개의 연속된 캔들스틱으로 구성되며, 이중 캔들 패턴보다 더 많은 정보를 통해 추세 전환 또는 지속을 판단합니다. 패턴 형성에 더 많은 시간이 걸리는 만큼 신뢰도가 높고, 가격 움직임 이면의 시장 심리 변화를 명확하게 보여줍니다.

**삼중 캔들 패턴의 공통 특성:**- 세 캔들의 순서와 상대적 위치가 모두 중요합니다
- 단일·이중 캔들보다 형성 조건이 엄격합니다
- 거래량 변화가 패턴 확인의 핵심 증거입니다
- 일부 패턴(샛별, 석별)은 가운데 캔들과 양쪽 캔들 사이의 갭이 신뢰도를 높입니다

## 1. 샛별 (Morning Star)

샛별은 하락 추세 끝에 나타나는 강력한 상승 반전 패턴으로, [Investopedia](https://www.investopedia.com/terms/m/morningstar.asp)는 이를 "어둠 속에 나타나는 새벽별"에 비유합니다. 세 캔들이 각각 매도 지배, 불확실성, 매수 지배를 순서대로 표현합니다.

### 패턴 구조

```
  1번 캔들       2번 캔들       3번 캔들
  ┌───────┐
  │       │
  │  음봉  │       ┌───┐
  │(크고   │       │작은│      ┌───────┐
  │ 강함)  │       │캔들│      │       │
  └───────┘       └───┘      │  양봉  │
                             │(크고   │
                             │ 강함)  │
                             └───────┘

     -- 갭 다운 --           -- 갭 업 --
       (이상적)               (이상적)
```

**3단계 형성 과정:**
1.**1번 캔들 — 하락 지배:** 강한 음봉으로 하락 추세를 이어갑니다. 몸통이 클수록 좋습니다.
2.**2번 캔들 — 불확실성:** 작은 몸통의 캔들(도지, 팽이, 작은 음봉/양봉)이 등장합니다. 이상적으로는 1번 캔들과 갭 다운으로 시작합니다. 매도세가 약해졌음을 나타냅니다.
3.**3번 캔들 — 매수 지배:** 강한 양봉이 등장하며, 1번 캔들 몸통의 중간(50%) 이상까지 상승합니다. 이상적으로는 2번 캔들과 갭 업으로 시작합니다.

**신뢰도 체크리스트:**
- [ ] 명확한 하락 추세가 선행됨
- [ ] 2번 캔들 몸통이 1번보다 현저히 작음
- [ ] 3번 양봉이 1번 음봉 몸통의 50% 이상을 회복
- [ ] 3번 캔들의 거래량이 1번보다 많음
- [ ] 1번-2번, 2번-3번 사이에 갭이 존재 (필수 아님, 있으면 강함)

**거래량 패턴:**
| 캔들 | 이상적인 거래량 | 의미 |
|------|--------------|------|
| 1번 (음봉) | 높음 | 강한 매도세 확인 |
| 2번 (소형) | 낮음 | 매도세 소진 |
| 3번 (양봉) | 높음 | 강한 매수세 유입 확인 |

### 도지 샛별 (Morning Doji Star)

2번 캔들이 도지인 경우를 특별히 구분합니다. 2번 캔들에서 완전한 균형을 이루었다는 의미로 일반 샛별보다 강한 신호로 간주합니다.

```
  큰 음봉       도지         큰 양봉
  ┌───────┐      |          ┌───────┐
  │       │     ---         │       │
  │       │      |          │       │
  └───────┘                 └───────┘
```

## 2. 석별 (Evening Star)

석별은 샛별의 반대 패턴으로, 상승 추세 끝에 나타나는 강력한 하락 반전 신호입니다. "저녁별"이라는 이름처럼 상승 추세의 종말을 예고합니다.

### 패턴 구조

```
  1번 캔들       2번 캔들       3번 캔들
  ┌───────┐       ┌───┐
  │       │       │작은│
  │  양봉  │       │캔들│      ┌───────┐
  │(크고   │       └───┘      │       │
  │ 강함)  │                  │  음봉  │
  └───────┘                  │(크고   │
                             │ 강함)  │
                             └───────┘

     -- 갭 업 --             -- 갭 다운 --
       (이상적)               (이상적)
```

**형성 조건:**- 상승 추세 중 강한 양봉(1번)이 나타납니다
- 갭 업으로 시작하는 작은 캔들(2번)이 형성됩니다
- 갭 다운으로 시작하는 강한 음봉(3번)이 1번 몸통의 50% 이상을 잠식합니다

### 샛별 vs 석별 비교

| 항목 | 샛별 (Morning Star) | 석별 (Evening Star) |
|------|-------------------|-------------------|
| 선행 추세 | 하락 | 상승 |
| 1번 캔들 | 강한 음봉 | 강한 양봉 |
| 2번 캔들 | 작은 몸통 | 작은 몸통 |
| 3번 캔들 | 강한 양봉 | 강한 음봉 |
| 신호 방향 | 상승 반전 | 하락 반전 |
| 신뢰도 | 높음 | 높음 |

## 3. 적삼병 (Three White Soldiers)

적삼병은 세 개의 연속 양봉으로 구성된 강세 반전/지속 패턴입니다. [StockCharts](https://school.stockcharts.com/doku.php?id=chart_analysis:candlestick_pattern_dictionary)에 따르면 각 양봉이 이전 캔들의 몸통 안에서 시작하고 전날 고가를 넘어 마감해야 합니다.

### 패턴 구조

```
                             ┌─────┐
                             │     │
                   ┌─────┐   │ 3번 │
                   │     │   │양봉  │
         ┌─────┐   │ 2번 │   │     │
         │     │   │양봉  │   └─────┘
         │ 1번 │   │     │
         │양봉  │   └─────┘
         │     │
         └─────┘
    각 캔들이 이전 종가보다 높게 마감하며 계단식 상승
```

**엄격한 형성 조건:**- 세 캔들 모두 양봉이어야 합니다
- 각 양봉은 이전 캔들의 몸통 내부 또는 상단에서 시작합니다
- 각 양봉은 이전 캔들의 고가보다 높게 마감합니다
- 세 캔들의 몸통이 모두 충분히 커야 합니다 (그림자가 짧을수록 강함)
- 거래량이 세 캔들에 걸쳐 증가하면 더욱 신뢰도가 높습니다

**주의 사항:**
| 경고 신호 | 의미 |
|---------|------|
| 세 번째 캔들에서 거래량 급감 | 상승 모멘텀 약화 |
| 세 번째 캔들 위 그림자가 길어짐 | 저항에 부딪힘 |
| RSI 과매수 구간에서 발생 | 단기 과열, 되돌림 가능 |
| 주요 저항선 직전에서 발생 | 돌파 실패 가능성 |

## 4. 흑삼병 (Three Black Crows)

흑삼병은 적삼병의 반대 패턴으로, 세 개의 연속 음봉이 계단식으로 하락하는 강력한 하락 신호입니다. [BabyPips](https://www.babypips.com/learn/forex/three-candlestick-patterns)는 이를 하락 추세의 강력한 확인 신호로 분류합니다.

### 패턴 구조

```
         ┌─────┐
         │     │
         │ 1번 │
         │음봉  │
         │     │
         └─────┘   ┌─────┐
                   │     │
                   │ 2번 │
                   │음봉  │
                   │     │
                   └─────┘   ┌─────┐
                             │     │
                             │ 3번 │
                             │음봉  │
                             │     │
                             └─────┘
    각 캔들이 이전 시가보다 낮게 시작하고 저점을 낮추며 마감
```

**형성 조건:**- 세 캔들 모두 음봉이어야 합니다
- 각 음봉은 이전 캔들의 몸통 내부 또는 하단에서 시작합니다
- 각 음봉은 이전 캔들의 저가보다 낮게 마감합니다
- 아래 그림자가 짧을수록 강한 신호입니다

### 적삼병 vs 흑삼병 비교

| 항목 | 적삼병 | 흑삼병 |
|------|--------|--------|
| 캔들 색 | 3개 양봉 | 3개 음봉 |
| 선행 상황 | 하락 추세 끝 또는 횡보 | 상승 추세 끝 또는 횡보 |
| 신호 방향 | 강한 상승 | 강한 하락 |
| 신뢰도 | 높음 | 높음 |
| 거래량 이상적 패턴 | 3번으로 갈수록 증가 | 3번으로 갈수록 증가 |

## 5. 쓰리 인사이드 업/다운 (Three Inside Up / Three Inside Down)

쓰리 인사이드 패턴은 잉태형(Harami)을 확인해주는 3캔들 패턴입니다. 잉태형 2캔들 이후 세 번째 캔들이 패턴의 방향을 확인해주는 구조입니다.

### 쓰리 인사이드 업 (Three Inside Up)

```
  1번 캔들       2번 캔들       3번 캔들
  ┌───────┐
  │       │
  │  음봉  │    ┌───┐
  │(큰)    │    │양봉│           ┌─────┐
  │        │   └───┘           │ 양봉 │  -- 1번 음봉 고가 상향 돌파
  └───────┘    위치: 1번 안     └─────┘
```

**단계별 설명:** 1.**1번:** 강한 음봉 (하락 추세)
2.**2번:** 작은 양봉이 1번 몸통 안에 위치 (잉태형 형성)
3.**3번:** 양봉으로 마감하며 1번 음봉의 고가 이상까지 상승 (반전 확인)

### 쓰리 인사이드 다운 (Three Inside Down)

```
  1번 캔들       2번 캔들       3번 캔들
  ┌───────┐
  │       │
  │  양봉  │    ┌───┐
  │(큰)    │    │음봉│           ┌─────┐
  │        │   └───┘           │ 음봉 │  -- 1번 양봉 저가 하향 돌파
  └───────┘    위치: 1번 안     └─────┘
```

| 패턴 | 1번 | 2번 | 3번 | 신호 |
|------|-----|-----|-----|------|
| 쓰리 인사이드 업 | 큰 음봉 | 작은 양봉 (1번 안) | 양봉 (1번 고가 상향) | 상승 반전 |
| 쓰리 인사이드 다운 | 큰 양봉 | 작은 음봉 (1번 안) | 음봉 (1번 저가 하향) | 하락 반전 |

잉태형(2캔들)만으로는 전환 확신이 부족합니다. 세 번째 캔들이 이전 큰 캔들의 범위를 돌파함으로써 추세 전환을 확인하므로, 잉태형 단독보다 신뢰도가 높습니다.

## 6. 어밴던드 베이비 (Abandoned Baby)

어밴던드 베이비는 삼중 캔들 패턴 중 가장 희귀하고 강력한 패턴입니다. 도지 샛별/석별의 변형으로, 2번 캔들(도지)이 양쪽 캔들과 완전한 갭으로 분리되어 있는 것이 핵심입니다.

### 강세 어밴던드 베이비 (Bullish Abandoned Baby)

```
  1번 캔들                   3번 캔들
  ┌───────┐                 ┌───────┐
  │       │                 │       │
  │  음봉  │                 │  양봉  │
  │       │                 │       │
  └───────┘    2번 캔들      └───────┘
              (도지)
    -- 갭 다운 --  -- 갭 업 --
              ----
             (도지)
              ----
  완전한 갭: 2번 그림자가 1번, 3번과 겹치지 않음
```

**핵심 조건:**- 2번 캔들이 반드시 도지여야 합니다
- 1번과 2번 사이에 완전한 갭이 있어야 합니다 (그림자 포함)
- 2번과 3번 사이에도 완전한 갭이 있어야 합니다 (그림자 포함)
- 이 갭 조건으로 인해 주식보다 외환이나 선물 시장에서 드물게 발생합니다

### 약세 어밴던드 베이비 (Bearish Abandoned Baby)

```
  1번 캔들    2번 캔들      3번 캔들
  ┌───────┐              ┌───────┐
  │       │    ----      │       │
  │  양봉  │   (도지)     │  음봉  │
  │       │    ----      │       │
  └───────┘              └───────┘
    -- 갭 업 --  -- 갭 다운 --
      완전한 갭 (양방향)
```

**어밴던드 베이비가 강력한 이유:**
| 이유 | 설명 |
|------|------|
| 완전한 갭 | 이전 추세와 완전히 단절된 구조 |
| 도지의 존재 | 매수·매도 세력의 완전한 균형 지점 |
| 희소성 | 발생 빈도가 낮아 나타날 때 강한 의미 |
| 즉각적 반응 | 3번 캔들이 갭으로 반등/급락하며 강한 매수/매도 유입 |

## 삼중 캔들 패턴 신뢰도 종합

| 패턴 | 신뢰도 | 발생 빈도 | 확인 캔들 필요 |
|------|--------|---------|--------------|
| 어밴던드 베이비 | 매우 높음 | 매우 낮음 | 불필요 |
| 도지 샛별 / 도지 석별 | 매우 높음 | 낮음 | 권장 |
| 샛별 / 석별 | 높음 | 중간 | 권장 |
| 적삼병 / 흑삼병 | 높음 | 낮음 | 불필요 |
| 쓰리 인사이드 업/다운 | 중간~높음 | 중간 | 권장 |

## 거래량과 삼중 캔들 패턴

삼중 캔들 패턴에서 거래량 분석은 패턴의 신뢰도를 결정하는 핵심 변수입니다.

### 샛별/석별 이상적 거래량 패턴

```
거래량
 |  높음               높음
 |  ████               ████
 |  ████  낮음         ████
 |  ████  ████         ████
 +--1번---2번---3번--
   1번(높음) -> 2번(낮음) -> 3번(높음)
```

### 적삼병/흑삼병 이상적 거래량 패턴

```
거래량
 |               높음
 |         중간  ████
 |   낮음  ████  ████
 |   ████  ████  ████
 +--1번---2번---3번--
   거래량이 3번 캔들로 갈수록 증가 (신뢰도 상승)
```

## 수치 계산 예시: 샛별 패턴 손절·목표가 설정

실제 수치로 샛별 패턴의 진입·손절·목표가를 계산합니다.

**시나리오:** 카카오 주식이 2주간 연속 하락 후 52주 신저가 지지 구간에서 샛별 패턴 형성

| 항목 | 1번 캔들 (음봉) | 2번 캔들 (팽이) | 3번 캔들 (양봉) |
|------|--------------|--------------|--------------|
| 시가 | 54,000원 | 49,500원 | 50,200원 |
| 고가 | 54,300원 | 50,800원 | 56,800원 |
| 저가 | 49,200원 | 49,000원 | 50,000원 |
| 종가 | 49,800원 | 49,600원 | 56,500원 |
| 거래량 | 3,200,000주 | 980,000주 | 4,100,000주 |

**패턴 검증:**
$$\text{1번 음봉 몸통 중간} = \frac{54{,}000 + 49{,}800}{2} = 51{,}900원$$

$$\text{3번 양봉 종가}(56{,}500) > 중간값(51{,}900)\ \rightarrow\ 50\%\ 이상\ 회복\ \checkmark$$

거래량: 2번(980,000) < 1번(3,200,000), 3번(4,100,000) > 1번 → 이상적 패턴 확인

**손절·목표가 계산:**
$$\text{손절가} = 2번\ 캔들\ 저가 - 버퍼 = 49{,}000 - 500 = 48{,}500원$$

$$\text{리스크} = 진입가 - 손절가 = 56{,}500 - 48{,}500 = 8{,}000원$$

$$\text{1차 목표가} = 진입가 + 리스크 \times 2 = 56{,}500 + 16{,}000 = 72{,}500원$$

$$\text{2차 목표가} = 진입가 + 리스크 \times 3 = 56{,}500 + 24{,}000 = 80{,}500원$$

| 시나리오 | 손익 | 손익비 |
|---------|------|--------|
| 1차 목표 달성 | +16,000원/주 | 1:2.0 |
| 2차 목표 달성 | +24,000원/주 | 1:3.0 |
| 손절 | -8,000원/주 | — |

>**해석:** 52주 신저가 지지선에서 발생한 샛별이며, 3번 양봉의 거래량이 1번보다 많아 강한 매수 유입을 확인합니다. 손익비 1:2 이상 확보로 진입 조건 충족입니다.

## 실전 매매 전략

### 샛별 패턴 진입 예시

```
상황: 하락 추세 끝, 주요 지지선에서 샛별 형성

1. 패턴 확인: 3번 양봉 마감 시점에서 패턴 식별
2. 진입: 3번 캔들 종가 또는 4번 캔들 시가 부근
3. 손절매: 2번 캔들의 저가(또는 패턴 전체 저점) 아래
4. 목표가 1차: 최근 저항선
5. 목표가 2차: 패턴 이전 상승 추세의 되돌림 수준 적용
```

### 적삼병 활용 주의사항

적삼병은 강력한 패턴이지만 과매수 구간에서는 함정이 될 수 있습니다.

- RSI가 70 이상인 상태에서 완성되면 과매수 가능성을 경계하십시오
- 세 번째 양봉에서 거래량이 급감하면 모멘텀 약화 신호입니다
- 강한 저항선 직전의 적삼병은 돌파 실패 후 급락으로 이어질 수 있습니다

## 흔한 실수

**1. 2번 캔들 크기를 무시하고 샛별/석별 처리**2번 캔들의 몸통이 1번 캔들의 50% 이상일 때도 샛별로 분류하는 실수입니다. 2번 캔들은 1번 몸통 대비 현저히 작아야 하며, 이상적으로는 도지나 팽이 수준이어야 합니다. 몸통이 큰 2번 캔들은 샛별이 아니라 다른 패턴으로 해석해야 합니다.

**2. 3번 캔들의 50% 회복 조건 미달 시 진입샛**별/석별에서 3번 캔들이 1번 캔들 몸통의 50% 이상을 회복하지 못해도 "거의 반전됐다"는 판단으로 진입하는 경우입니다. 50% 미달 패턴은 관통형 수준의 약한 신호이므로 추가 확인이 필요합니다.

**3. 적삼병을 추세 초반이 아닌 과매수 구간에서 추격 매수적**삼병이 RSI 70 이상이거나 장기 저항선 직전에서 완성될 때 추격 매수하는 실수입니다. 이 경우 적삼병은 추세 지속이 아닌 단기 과열 후 급반전 신호일 수 있습니다. 적삼병의 의미는 추세 초기나 횡보 이탈 직후에 가장 강합니다.

**4. 흑삼병을 강한 상승 추세 중 되돌림과 혼동강**한 상승 추세 중 3일 연속 소폭 하락 음봉을 흑삼병으로 오인하는 경우입니다. 유효한 흑삼병은 각 음봉의 몸통이 충분히 크고, 각 캔들이 이전 캔들의 저가 아래로 마감해야 합니다. 추세 중 단순 조정 음봉과 혼동하지 마십시오.

**5. 어밴던드 베이비를 도지 샛별로 처리어**밴던드 베이비의 핵심은 2번 도지와 1번·3번 캔들 사이에 그림자까지 포함한 완전한 갭이 있어야 한다는 점입니다. 그림자가 겹치면 어밴던드 베이비가 아닌 도지 샛별/석별로 분류하며, 신호 강도도 낮게 평가해야 합니다.

## 에지 케이스: 비정상 시장 상황

**갭 없는 샛별 (한국·미국 주식 시장**)한국 주식 시장에서는 갭이 없는 샛별이 빈번합니다. 갭 조건이 없어도 3번 캔들이 1번 몸통의 50% 이상을 회복하고 거래량이 급증하면 유효한 패턴으로 인정합니다. 다만 갭 동반 패턴보다 신뢰도가 낮으므로 손절폭을 약간 넓게 설정하는 것이 안전합니다.

**거래량 데이터가 없는 크립토 현물 분석**DEX나 일부 크립토 집계 플랫폼에서는 거래량 신뢰도가 낮습니다. 이 경우 거래량 조건 대신 패턴의 형태(3번 캔들 크기, 갭 여부)에 더 집중하고, 온체인 데이터(거래소 순유입/유출)를 보조 확인 수단으로 활용하십시오.

**적삼병 직후 갭 업 발생적**삼병 완성 후 다음 캔들이 갭 업으로 시작하면 단기 과열 신호일 수 있습니다. 이미 포지션을 보유 중이라면 절반을 이익 실현하고, 신규 진입을 계획 중이라면 갭을 메우는 되돌림을 기다려 진입하는 전략이 리스크를 줄입니다.

## 핵심 요약

- **샛별과 석별은** 세 캔들이 각각 추세 지속, 불확실성, 반전 확인을 표현하는 고신뢰도 반전 패턴이며, 2번 캔들이 도지인 경우 신호가 더 강합니다
- **적삼병과 흑삼병은** 3개의 연속 양봉/음봉이 계단식으로 방향성을 보여주는 패턴으로, 각 캔들이 이전 캔들의 몸통 안에서 시작하고 고점/저점을 갱신해야 유효합니다
- **쓰리 인사이드 업/다운은** 잉태형 이후 세 번째 캔들이 추세 전환을 확인해주는 패턴으로, 잉태형 단독보다 신뢰도가 높습니다
- **어밴던드 베이비는** 2번 캔들(도지)이 양쪽과 완전한 갭으로 분리된 희귀 패턴으로, 발생 시 강력한 추세 전환 신호입니다
- 모든 삼중 캔들 패턴은거**래량 변화, 발생 위치(지지/저항), 기존 추세와의 관계를** 함께 분석할 때 매매 신뢰도가 극대화됩니다
