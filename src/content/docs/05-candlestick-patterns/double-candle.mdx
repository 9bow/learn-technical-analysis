## 이중 캔들 패턴 개요

<CandlestickDemo client:visible dataset="uptrend-stock" title="이중 캔들 패턴 차트" />

이중 캔들 패턴은 연속된 두 개의 캔들스틱 사이의 관계로 시장 심리의 전환을 포착합니다. 단일 캔들 패턴보다 더 많은 정보를 담고 있어 신뢰도가 높으며, 첫 번째 캔들이 현재 추세를, 두 번째 캔들이 그에 대한 반응을 보여줍니다.

**이중 캔들 패턴 해석의 핵심:**- 두 캔들의 상대적 크기와 위치 관계가 핵심입니다
- 두 번째 캔들이 패턴의 방향을 결정합니다
- 두 번째 캔들 마감 후 추가 확인 캔들을 기다리면 안전성이 높아집니다
- 패턴이 발생하는 추세와 가격대(지지·저항)가 신뢰도를 좌우합니다

---

## 1. 장악형 (Engulfing Pattern)

장악형은 이중 캔들 패턴 중 가장 강력하고 신뢰도 높은 패턴으로 꼽힙니다. [Investopedia](https://www.investopedia.com/terms/b/bullishengulfing.asp)에 따르면 두 번째 캔들의 몸통이 첫 번째 캔들의 몸통을 완전히 감싸는 구조입니다.

### 강세 장악형 (Bullish Engulfing)

```
  1번 캔들   2번 캔들
  ┌─────┐
  │ 음봉 │   ← 하락 추세의 연장
  └─────┘
          ┌───────┐
          │       │   ← 큰 양봉이 음봉 몸통을 완전히 감쌈
          │  양봉  │
          │       │
          └───────┘
```

**형성 조건:**- 명확한 하락 추세가 선행되어야 합니다
- 두 번째 캔들(양봉)의 시가가 첫 번째 캔들(음봉)의 종가 아래에서 시작합니다
- 두 번째 캔들의 종가가 첫 번째 캔들의 시가를 상향 돌파합니다
- 두 번째 캔들의 몸통이 클수록 신뢰도가 높습니다

**시장 심리:**- 하락 추세 중 매도세가 작은 음봉을 만들었습니다
- 다음 날 갭 다운으로 시작했으나 매수세가 폭발적으로 유입되었습니다
- 결국 전날 매도세를 완전히 압도하며 강한 양봉으로 마감했습니다
- 추세 전환의 강력한 신호입니다

### 약세 장악형 (Bearish Engulfing)

```
  1번 캔들   2번 캔들
          ┌─────┐
          │ 양봉 │   ← 상승 추세의 연장
          └─────┘
  ┌───────┐
  │       │   ← 큰 음봉이 양봉 몸통을 완전히 감쌈
  │  음봉  │
  │       │
  └───────┘
```

**형성 조건:**- 명확한 상승 추세가 선행되어야 합니다
- 두 번째 캔들(음봉)의 시가가 첫 번째 캔들(양봉)의 종가 위에서 시작합니다
- 두 번째 캔들의 종가가 첫 번째 캔들의 시가를 하향 돌파합니다

**신뢰도를 높이는 요소:**
| 요소 | 설명 |
|------|------|
| 두 번째 캔들 크기 | 클수록 강한 신호 |
| 거래량 | 두 번째 캔들의 거래량이 크게 증가할수록 신뢰도 상승 |
| 위치 | 주요 지지/저항선과 겹칠 때 더 강함 |
| 갭 | 두 번째 캔들이 갭을 동반하면 신호 강도 증가 |

## 2. 잉태형 (Harami Pattern)

잉태형은 장악형의 반대 구조로, 두 번째 캔들의 몸통이 첫 번째 캔들의 몸통 안에 완전히 들어가 있습니다. '하라미'는 일본어로 임신을 뜻하며, 큰 캔들 안에 작은 캔들이 품긴 모습에서 유래했습니다.

### 강세 잉태형 (Bullish Harami)

```
  1번 캔들   2번 캔들
  ┌───────┐
  │       │
  │  음봉  │   ← 큰 음봉 (하락 추세)
  │       │
  └───────┘
       ┌───┐
       │양봉│   ← 작은 양봉이 음봉 몸통 안에 위치
       └───┘
```

**형성 조건:**- 하락 추세 중 큰 음봉이 나타납니다
- 다음 캔들이 작은 양봉으로, 전날 음봉 몸통 범위 안에서 마감합니다

**시장 심리:**- 매도세가 지배하던 시장에서 첫날 큰 음봉 형성
- 둘째 날 매도세 약화 — 가격 변동폭이 크게 줄어듦
- 추세 소진의 초기 신호이지만, 반전 확인을 위해 추가 캔들이 필수입니다

### 약세 잉태형 (Bearish Harami)

```
  1번 캔들   2번 캔들
  ┌───────┐
  │       │
  │  양봉  │   ← 큰 양봉 (상승 추세)
  │       │
  └───────┘
       ┌───┐
       │음봉│   ← 작은 음봉이 양봉 몸통 안에 위치
       └───┘
```

**형성 조건:**- 상승 추세 중 큰 양봉 이후 작은 음봉이 나타납니다
- 두 번째 캔들의 몸통 전체가 첫 번째 캔들 몸통 범위 안에 있어야 합니다

### 잉태형 vs 장악형 비교

| 비교 항목 | 잉태형 (Harami) | 장악형 (Engulfing) |
|---------|---------------|-----------------|
| 두 번째 캔들 크기 | 첫 번째보다 작음 | 첫 번째보다 큼 |
| 구조 | 두 번째가 첫 번째 안에 포함 | 두 번째가 첫 번째를 포함 |
| 신호 강도 | 중간 (불확실성) | 강함 (명확한 전환) |
| 확인 캔들 필요 | 필수 | 강력 권장 |

### 잉태 십자형 (Harami Cross)

잉태형의 변형으로, 두 번째 캔들이 도지(Doji)인 경우입니다. 두 번째 캔들에서 매수·매도세가 완전한 균형을 이루었음을 의미하여 일반 잉태형보다 강한 신호로 간주합니다.

```
  큰 음봉    도지
  ┌───────┐
  │       │     |
  │       │    ---   ← 시가 = 종가 (잉태 범위 안)
  │       │     |
  └───────┘
```

## 3. 관통형 (Piercing Line)

관통형은 하락 추세에서 나타나는 상승 반전 패턴입니다. [BabyPips](https://www.babypips.com/learn/forex/two-candlestick-patterns)는 이 패턴이 전날 음봉의 절반 이상을 두 번째 양봉이 뚫고 올라와야 유효하다고 설명합니다.

```
  1번 캔들   2번 캔들
  ┌───────┐
  │       │
  │  음봉  │
  │       │          ← 2번 캔들 종가가 1번 몸통 중간(50%) 이상
  └───────┘   ┌───┐
              │양봉│
              └───┘  ← 2번 시가는 1번 종가 아래 (갭 다운)
```

**핵심 조건:**- 두 번째 양봉의 시가가 첫 번째 음봉의 종가 아래에서 시작합니다 (갭 다운)
- 두 번째 양봉의 종가가 첫 번째 음봉 몸통의 중간(50%) 이상을 상향 돌파합니다

**신뢰도 등급:**
| 관통 비율 | 신뢰도 |
|---------|--------|
| 50% 미만 | 패턴 불인정 |
| 50~75% | 낮음 — 확인 필수 |
| 75% 이상 | 중간 |
| 100% (완전 장악) | 높음 (장악형으로 분류) |

>**실전 팁:** 관통 비율이 50%를 겨우 넘긴 경우 신뢰도가 낮습니다. 75% 이상 관통했을 때 관심을 가지십시오.

## 4. 먹구름형 (Dark Cloud Cover)

먹구름형은 관통형의 반대 패턴으로, 상승 추세에서 나타나는 하락 반전 신호입니다.

```
  1번 캔들   2번 캔들
              ┌───┐  ← 2번 시가는 1번 종가 위 (갭 업)
              │음봉│
  ┌───────┐   └───┘  ← 2번 종가가 1번 몸통 중간(50%) 이하
  │       │
  │  양봉  │
  │       │
  └───────┘
```

**핵심 조건:**- 두 번째 음봉의 시가가 첫 번째 양봉의 종가 위에서 시작합니다 (갭 업)
- 두 번째 음봉의 종가가 첫 번째 양봉 몸통의 중간(50%) 이하로 하락합니다

**관통형 vs 먹구름형 비교:**
| 항목 | 관통형 | 먹구름형 |
|------|--------|---------|
| 선행 추세 | 하락 | 상승 |
| 1번 캔들 | 음봉 | 양봉 |
| 2번 캔들 | 양봉 | 음봉 |
| 2번 시작 | 갭 다운 | 갭 업 |
| 신호 방향 | 상승 반전 | 하락 반전 |
| 관통 기준 | 50% 이상 상향 | 50% 이상 하향 |

**먹구름형 강도를 높이는 조건:**- 두 번째 음봉의 마감이 첫 번째 양봉 몸통의 75% 이하까지 내려올 때
- 두 번째 음봉의 거래량이 첫 번째 양봉보다 많을 때
- 주요 저항선이나 과매수 구간에서 발생할 때

## 5. 핀셋 탑/바텀 (Tweezer Tops / Tweezer Bottoms)

핀셋 패턴은 연속된 두 캔들의 고가(탑) 또는 저가(바텀)가 거의 동일한 경우입니다. 해당 가격대에서 시장이 강한 저항 또는 지지를 받고 있음을 보여줍니다.

### 핀셋 탑 (Tweezer Tops)

```
   1번 고가    2번 고가
      |     |     |     |
      |  양봉  |  |  음봉  |
      |     |     |     |
            ← 두 캔들의 고가(그림자 끝)가 동일
```

**형성 조건:**- 상승 추세 중 발생합니다
- 두 캔들의 고가(위 그림자 끝)가 동일하거나 매우 유사합니다
- 첫 번째 양봉, 두 번째 음봉이 전형적인 형태입니다

**시장 심리:** 특정 가격대에서 두 차례 강한 매도세에 막혔음을 의미합니다. 해당 수준이 강한 저항선임을 나타냅니다.

### 핀셋 바텀 (Tweezer Bottoms)

```
            ← 두 캔들의 저가(그림자 끝)가 동일
      |     |     |     |
      |  음봉  |  |  양봉  |
      |     |     |     |
   1번 저가    2번 저가
```

**형성 조건:**- 하락 추세 중 발생합니다
- 두 캔들의 저가(아래 그림자 끝)가 동일하거나 매우 유사합니다
- 첫 번째 음봉, 두 번째 양봉이 전형적인 형태입니다

**핀셋 패턴 활용 팁:**
| 항목 | 내용 |
|------|------|
| 단독 신뢰도 | 낮음~중간 |
| 최적 활용 | 다른 반전 신호와 함께 사용 |
| 신뢰도 상승 조건 | 주요 지지/저항선에서 발생 |
| 강화 요소 | 두 번째 캔들이 도지 또는 마루보즈일 때 |

>**실전 팁:** 핀셋 탑/바텀이 RSI 과매수/과매도 구간, 볼린저밴드 외부, 또는 피보나치 되돌림 수준과 겹칠 때 신뢰도가 크게 올라갑니다.

| 강세 장악형 | 하락 | 상승 반전 | 높음 | 2번 양봉이 1번 음봉을 완전히 포함 |

| 약세 잉태형 | 상승 | 하락 가능 | 중간 | 2번 음봉이 1번 양봉 안에 포함 |
| 관통형 | 하락 | 상승 반전 | 중간 | 2번 양봉이 1번 음봉을 50% 이상 관통 |
| 먹구름형 | 상승 | 하락 반전 | 중간 | 2번 음봉이 1번 양봉을 50% 이상 침투 |
| 핀셋 탑 | 상승 | 하락 반전 | 낮음~중간 | 두 고가가 동일 |
| 핀셋 바텀 | 하락 | 상승 반전 | 낮음~중간 | 두 저가가 동일 |

## 수치 계산 예시: 강세 장악형 손절·목표가 설정

실제 수치로 강세 장악형의 진입·손절·목표가를 계산합니다.

**시나리오:** NAVER 주식이 3주간 하락 후 20일 이동평균선 부근 지지 구간에서 강세 장악형 발생

| 항목 | 1번 캔들 (음봉) | 2번 캔들 (양봉) |
|------|--------------|--------------|
| 시가 | 220,000원 | 215,000원 |
| 고가 | 221,500원 | 228,000원 |
| 저가 | 213,000원 | 214,500원 |
| 종가 | 214,000원 | 227,500원 |
| 거래량 | 850,000주 | 2,100,000주 (2.5배) |

**패턴 검증:**- 2번 양봉 시가(215,000) < 1번 음봉 종가(214,000)? → 아니오 (갭 업 없음, 바로 아래 시작)
- 2번 양봉 종가(227,500) > 1번 음봉 시가(220,000)? → 예 (완전 포함 확인)
- 거래량 증가 확인? → 예 (2.5배 급증)

**손절·목표가 계산:**
$$\text{손절가} = 2번\ 캔들\ 저가 - \text{버퍼} = 214{,}500 - 1{,}000 = 213{,}500원$$

$$\text{리스크} = 진입가 - 손절가 = 228{,}000 - 213{,}500 = 14{,}500원$$

$$\text{1차 목표가} = 진입가 + 리스크 \times 1.5 = 228{,}000 + 21{,}750 = 249{,}750원$$

$$\text{2차 목표가} = 진입가 + 리스크 \times 2.5 = 228{,}000 + 36{,}250 = 264{,}250원$$

| 시나리오 | 손익 | 손익비 |
|---------|------|--------|
| 1차 목표 달성 | +21,750원/주 | 1:1.5 |
| 2차 목표 달성 | +36,250원/주 | 1:2.5 |
| 손절 | -14,500원/주 | — |

>**해석:** 거래량이 2.5배 급증하며 2번 캔들이 1번을 완전히 장악했고, 손익비 1:1.5~2.5가 확보됩니다. 확인 캔들(3번째)이 양봉으로 마감하면 진입 조건 충족입니다.

## 실전 매매 전략

### 장악형 패턴 진입 예시

```
약세 장악형 발생 시:

1. 패턴 확인: 두 번째 음봉 종가에서 패턴 식별
2. 손절매 설정: 두 번째 음봉의 고가(또는 위 그림자 끝) 위
3. 목표가 설정: 직전 지지선 또는 패턴 높이의 1배~2배 수준
4. 진입: 세 번째 캔들이 음봉으로 시작하는 것을 확인 후 진입
```

### 패턴 실패 대응

이중 캔들 패턴도 실패합니다. 아래 상황에서는 포지션을 즉시 청산하거나 진입을 보류하십시오.

- 두 번째 캔들 마감 후 다음 캔들이 반대 방향으로 강하게 움직이는 경우
- 거래량이 두 캔들 모두에서 평균 이하인 경우
- 강한 추세 방향과 반대되는 패턴이 추세 중간에 발생하는 경우

## 흔한 실수

**1. 추세 없는 횡보 구간에서 패턴 매매장**악형이 명확한 추세 없이 박스권 중간에서 발생할 때 진입하는 실수입니다. 장악형은 선행 추세가 존재할 때만 유효합니다. 최소 3~5개의 연속 하락(상승) 캔들 이후 발생한 패턴을 대상으로 하십시오.

**2. 50% 미달 관통형을 유효 패턴으로 처리관**통형에서 두 번째 양봉이 첫 번째 음봉의 30~40%만 관통했음에도 "거의 50%"라는 이유로 진입하는 경우입니다. 50% 기준은 타협이 없습니다. 미달 패턴은 신호가 아닙니다.

**3. 장악형에서 몸통 vs 그림자 혼동장**악형의 조건은 몸통(시가~종가) 기준입니다. 두 번째 캔들의 몸통이 첫 번째 캔들의 그림자를 포함해야 한다고 잘못 이해하면 조건이 과도하게 까다로워집니다. 반대로 그림자만 포함되고 몸통은 미달이면 장악형이 아닙니다.

**4. 잉태형을 즉시 반전 신호로 처리잉**태형은 추세 소진의 초기 신호이지 확정 반전 신호가 아닙니다. 잉태형 발생 후 확인 캔들 없이 바로 진입하면 추세 재개 시 큰 손실이 납니다. 반드시 세 번째 캔들의 방향을 확인 후 진입하십시오.

**5. 핀셋 패턴을 단독 신호로 과신핀**셋 탑/바텀은 단독으로는 약한 신호입니다. RSI 과매수/과매도, 볼린저밴드 상단/하단, 피보나치 되돌림 등 최소 하나의 보조 신호와 함께 사용하지 않으면 오신호 비율이 높습니다.

## 에지 케이스: 비정상 시장 상황

**갭이 없는 장악형 (한국 주식**)한국 주식 시장에서는 갭 발생 빈도가 낮아 두 번째 양봉이 전날 종가와 거의 동일한 가격에서 시작하는 경우가 많습니다. 이때도 몸통이 완전히 포함되면 장악형으로 인정합니다. 단, 갭 동반 패턴보다 신뢰도가 약간 낮습니다.

**어닝 발표 직후 장악형실**적 서프라이즈로 인해 갭 업/다운이 발생하며 장악형이 형성되는 경우입니다. 이 패턴은 실적 모멘텀을 반영하므로 신뢰도가 높지만, 동시에 기대감이 이미 선반영된 경우 패턴 실패율도 높습니다. 거래량과 애널리스트 반응을 함께 확인하십시오.

**강한 추세 중 잉태형 발생강**한 상승 추세 중 나타나는 약세 잉태형은 대부분 단순 눌림목으로 끝납니다. 주봉이나 월봉의 추세가 강하게 살아있다면 일봉 이중 캔들 반전 신호의 신뢰도는 대폭 하락합니다. 상위 시간프레임 추세를 항상 먼저 확인하십시오.

**크립토 24시간 시장에서의 적용크**립토 시장은 캔들 마감 시간을 직접 설정할 수 있습니다. 일봉 기준으로 패턴을 분석할 때는 UTC 00:00 마감 캔들을 사용하는 것이 거래소 간 일관성을 유지합니다. 임의의 마감 시간 설정 시 패턴 형태가 달라질 수 있습니다.

## 핵심 요약

- **장악형은** 이중 캔들 중 신뢰도가 가장 높으며, 두 번째 캔들이 첫 번째 캔들을 완전히 포함할 때 발생합니다. 거래량 동반 시 강력한 반전 신호입니다
- **잉태형은** 장악형과 반대 구조로 추세 소진을 암시하지만 확인 캔들이 필수이며, 두 번째 캔들이 도지인 잉태 십자형이 더 강한 신호입니다
- **관통형과 먹구름형은** 두 번째 캔들이 첫 번째 캔들의 50% 이상을 관통해야 유효하며, 관통 비율이 높을수록 신뢰도가 상승합니다
- **핀셋 탑/바텀은** 단독 신뢰도가 낮으나 지지/저항선, 보조지표와 결합할 때 강력한 확인 신호가 됩니다
- 모든 이중 캔들 패턴은추**세 내 위치, 거래량, 주요 가격대와** 함께 분석하고 추가 확인 캔들을 기다리는 것이 원칙입니다
