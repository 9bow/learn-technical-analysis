## 캔들 패턴 조합 개요

<PatternHighlight client:visible dataset="double-bottom" title="캔들 패턴 조합 분석 차트" />

단독 캔들 패턴은 시장 심리의 스냅샷을 제공하지만, 그 자체만으로는 오신호(False Signal)가 빈번합니다. 통계적으로 단일 캔들 패턴의 신뢰도는 50~60% 수준에 머무르며, 이는 동전 던지기와 크게 다르지 않습니다.**패턴 조합(Pattern Combination**)은 캔들 패턴에 거래량, 지지·저항선, 이동평균선 등 독립적인 근거를 추가하여 신뢰도를 70% 이상으로 끌어올리는 방법론입니다.

**패턴 조합의 핵심 원칙:**- 독립적인 근거가 여러 개 동시에 나타날수록 신뢰도가 기하급수적으로 증가합니다
- 한 방향을 지지하는 근거가 2개 이상일 때만 진입을 고려합니다
- 조합 실패 시 손절매를 명확히 설정하는 것이 단독 패턴보다 쉽습니다

## 1. 캔들 패턴 + 거래량 확인

거래량은 캔들 패턴의진**정성(authenticity**)을 검증하는 가장 기초적인 도구입니다. 같은 모양의 패턴이라도 거래량에 따라 의미가 크게 달라집니다.

### 거래량 확인의 기본 원리

```
상승 반전 패턴 + 높은 거래량 = 강한 매수 압력 확인 → 신뢰도 상승
상승 반전 패턴 + 낮은 거래량 = 매수 참여 부족 → 오신호 가능성
하락 반전 패턴 + 높은 거래량 = 강한 매도 압력 확인 → 신뢰도 상승
하락 반전 패턴 + 낮은 거래량 = 매도 참여 부족 → 오신호 가능성
```

### 구체적 수치 기준

거래량 확인에는**20일 평균 거래량(20MA Volume**)을 기준으로 삼는 것이 일반적입니다.

| 거래량 수준 | 평균 대비 비율 | 패턴 신뢰도 조정 |
|------------|-------------|----------------|
| 매우 높음   | 200% 이상   | +2점 (강력한 확인) |
| 높음       | 150~200%   | +1점 (확인)     |
| 보통       | 80~150%    | 0점 (중립)      |
| 낮음       | 80% 미만   | -1점 (의심)     |

**실전 예시 — 더블 바텀에서의 강세 장악형:**
```
상황: 하락 추세 후 지지선 근처에서 강세 장악형 출현
1번 캔들 (음봉): 종가 48,200원, 거래량 85만 주 (평균의 90%)
2번 캔들 (양봉): 종가 50,800원, 거래량 230만 주 (평균의 245%)

분석:
- 두 번째 양봉이 음봉 몸통을 완전히 감쌈 (장악형 확인)
- 거래량이 평균의 245%로 폭발적 증가
- 결론: 강한 매수 압력 확인, 신뢰도 높음
```

### 클라이맥스 거래량 (Climax Volume)

반전 가능성이 가장 높은 거래량 패턴은클**라이맥스 거래량입**니다. 이는 추세의 끝에서 거래량이 폭발적으로 증가한 후 급격히 감소하는 패턴입니다.

```
하락 추세 말미:
1. 매도 클라이맥스: 거래량 500% 급증 + 급락 캔들
2. 다음 날: 거래량 급감 + 하락폭 축소 또는 소폭 상승
3. 해석: 모든 패닉 매도가 소진됨 → 반전 직전 신호
→ 이 시점의 상승 반전 캔들 패턴은 신뢰도가 크게 상승합니다
```

:::tip
거래량 스파이크(Spike) 다음 날의 캔들 패턴에 주목하십시오. 클라이맥스 거래량 직후 나타나는 도지(Doji)나 망치형(Hammer)은 추세 전환의 강력한 근거가 됩니다.
:::

## 2. 캔들 패턴 + 지지·저항선

지지선과 저항선은 과거 시장 참여자들이 집중적으로 매매한 가격대를 나타냅니다. 이 구간에서 반전 캔들 패턴이 출현하면두** 가지 독립적인 근거가** 같은 방향을 가리키는 것입니다.

### 지지선에서의 상승 반전 패턴

```
시나리오: 50,000원 지지선에서 망치형(Hammer) 출현

지지선 근거:
- 과거 3회 이상 50,000원 부근에서 반등 확인
- 해당 구간이 이전 고점(저항→지지 전환) 수준

캔들 패턴 근거:
- 망치형: 아래 꼬리 길이가 몸통의 2배 이상
- 위꼬리 없음, 작은 양봉 몸통

조합 결론:
- 두 독립 근거(지지선 + 망치형)가 상승 반전 지지
- 진입: 50,000원 (또는 다음 날 시가 확인 후)
- 손절: 49,000원 (지지선 하방 2% 수준)
- 목표: 55,000원 (다음 저항선)
```

### 저항선에서의 하락 반전 패턴

```
시나리오: 100,000원 저항선에서 유성형(Shooting Star) 출현

저항선 근거:
- 과거 2회 100,000원에서 강한 하락 발생
- 52주 신고가 근처 (심리적 저항)

캔들 패턴 근거:
- 유성형: 위꼬리 길이 몸통의 3배
- 시가 근처 마감, 작은 음봉 몸통

조합 결론:
- 두 독립 근거(저항선 + 유성형)가 하락 반전 지지
- 진입: 99,000원 (또는 다음 날 음봉 확인 후)
- 손절: 101,500원 (저항선 상방 1.5%)
- 목표: 93,000원 (이전 지지선)
```

### 지지·저항의 다중 검증

하나의 가격대가 여러 근거로 지지·저항이 확인될 때 그 강도가 증가합니다.

| 지지·저항 근거 | 예시 |
|-------------|------|
| 수평 지지·저항 | 이전 고점/저점 |
| 피보나치 되돌림 | 38.2%, 50%, 61.8% |
| 이동평균선 | 20일, 50일, 200일 이동평균 |
| 추세선 | 상승/하락 추세선 |
| 심리적 가격대 | 50,000원, 100,000원 단위 |

:::note
지지·저항 구간이 3개 이상의 근거로 확인될 때 이를컨**플루언스 존(Confluence Zone**)이라 합니다. 이 구간에서 출현하는 캔들 패턴의 신뢰도는 단독 패턴 대비 현저히 높습니다.
:::

## 3. 캔들 패턴 + 이동평균선

이동평균선은 추세의 방향과 강도를 나타내며, 이동평균선 근처에서의 캔들 패턴은 추세 추종과 반전 트레이딩 모두에 활용할 수 있습니다.

### 이동평균선 지지에서의 반등 패턴

상승 추세 중 20일 이동평균선(MA20)은 강한 동적 지지선 역할을 합니다.

```
패턴: 상승 추세 → 가격이 MA20까지 되돌림 → 반등 캔들 출현

구체적 조건:
- 50일 이동평균선이 우상향 중 (상승 추세 확인)
- 가격이 MA20까지 2~5% 이내로 하락
- MA20 근처에서 강세 장악형 또는 망치형 출현
- 거래량이 하락 과정에서 감소, 반등 캔들에서 증가

진입 전략:
- 반등 캔들 종가에서 진입
- 손절: MA20 하방 돌파 시 (MA20 - 1~2%)
- 목표: 직전 고점 또는 MA20 + 추세 각도 × 보유 기간
```

### 이동평균선 저항에서의 하락 패턴

하락 추세에서 200일 이동평균선(MA200)은 강한 동적 저항선입니다.

```
패턴: 하락 추세 → 반등 시도 → MA200 근처에서 하락 캔들 출현

구체적 조건:
- MA200이 우하향 중 (하락 추세 확인)
- 가격이 MA200 아래 5% 이내까지 반등
- MA200 근처에서 유성형, 약세 장악형, 먹구름형 출현
- 반등 거래량 감소, 반전 캔들에서 거래량 증가

진입 전략:
- 반전 캔들 종가에서 공매도 또는 청산
- 손절: MA200 상방 돌파 시 (MA200 + 1.5%)
- 목표: 이전 저점 또는 MA200 - 추세 폭
```

### 이동평균선 교차와 캔들 패턴

```
골든 크로스(MA50이 MA200 상향 돌파) + 상승 반전 패턴:
- 골든 크로스 발생 직후 또는 전 구간의 상승 반전 캔들
- 중장기 상승 추세 전환의 강력한 신호
- 예: MA50이 MA200 교차 직후 강세 장악형 출현

데드 크로스(MA50이 MA200 하향 돌파) + 하락 반전 패턴:
- 데드 크로스 발생 직후의 하락 반전 캔들
- 중장기 하락 추세 전환의 신호
```

:::caution
이동평균선 교차 신호는 후행(lagging) 지표입니다. 캔들 패턴과 조합할 때 교차 직후 나타나는 되돌림(pullback) 구간의 반전 캔들을 노리는 것이 실전에서 더 유리한 진입 타이밍을 제공합니다.
:::

## 4. 복수 캔들 시퀀스 분석

단일 패턴이 아닌 3개 이상의 캔들이 만드는시**퀀스(Sequence**)를 분석하면 추세 전환의 전체 흐름을 파악할 수 있습니다.

### 반전 시퀀스 패턴

**상승 반전 5단계 시퀀스:**
```
1단계 (하락 가속): 긴 음봉, 거래량 증가 → 패닉 매도 시작
2단계 (클라이맥스): 가장 큰 거래량 + 급락 캔들 → 매도 클라이맥스
3단계 (회복 시도): 짧은 양봉 또는 도지 → 매수세 첫 등장
4단계 (반전 신호): 강세 장악형 또는 망치형 → 매수 우위로 전환
5단계 (확인): 양봉 + 거래량 증가 → 반전 추세 시작 확인

→ 3~5단계 구간이 최적 매수 타이밍
```

**하락 반전 5단계 시퀀스:**
```
1단계 (상승 가속): 긴 양봉, 거래량 증가 → 매수 열기 최고조
2단계 (클라이맥스): 최대 거래량 + 급등 캔들 → 매수 클라이맥스
3단계 (약화 시도): 짧은 음봉 또는 도지 → 매도세 첫 등장
4단계 (반전 신호): 약세 장악형 또는 유성형 → 매도 우위로 전환
5단계 (확인): 음봉 + 거래량 증가 → 하락 추세 시작 확인

→ 3~5단계 구간이 최적 매도·청산 타이밍
```

### 횡보 이탈 시퀀스

```
박스권(횡보) 이탈 패턴:
1. 박스권 상단 저항: 수회 음봉으로 저항 확인
2. 매집 캔들: 작은 몸통의 캔들들이 저항선 근처에서 반복
3. 이탈 시도: 거래량 증가와 함께 저항선 돌파 캔들
4. 되돌림: 이탈 후 박스권 상단으로 되돌아와 지지 확인
5. 재상승: 지지 확인 후 상승 추세 시작

→ 4단계(되돌림) 구간의 양봉이 최고의 진입 타이밍
```

## 5. 패턴 신뢰도 점수 시스템

주관적 판단을 줄이고 일관성을 높이기 위해점**수 기반 필터링(Score-Based Filtering**)을 활용할 수 있습니다.

### 점수 배점표

| 확인 요소 | 충족 시 점수 |
|---------|-----------|
| 선행 추세 명확성 (3개월 이상) | +2 |
| 주요 지지·저항선과 일치 | +2 |
| 이동평균선 지지·저항과 일치 | +1 |
| 거래량 150% 이상 (반전 캔들) | +2 |
| 거래량 감소 중 패턴 출현 후 증가 | +1 |
| 피보나치 되돌림 구간과 일치 | +1 |
| 확인 캔들(3번째 캔들) 존재 | +2 |
| RSI 과매수·과매도 구간 (70 이상/30 이하) | +1 |
| 패턴 크기 (큰 몸통) | +1 |

**총점 해석:**
| 총점 | 신뢰도 등급 | 행동 |
|------|----------|------|
| 8점 이상 | A등급 (매우 높음) | 적극 진입 고려 |
| 5~7점 | B등급 (높음) | 진입 고려, 소규모 포지션 |
| 3~4점 | C등급 (보통) | 관망 또는 소극적 대응 |
| 2점 이하 | D등급 (낮음) | 진입 보류 |

### 점수 계산 예시

```
시나리오: 50,000원 지지선에서 강세 장악형 출현

항목별 점수:
+ 선행 하락 추세 3개월 이상: +2점
+ 50,000원 = 과거 3회 지지 확인된 수평 지지선: +2점
+ MA50 근처: +1점
+ 장악형 당일 거래량 220% (평균 대비): +2점
+ 패닉 매도 이후 거래량 감소 후 반등: +1점
+ 피보나치 61.8% 되돌림 구간과 일치: +1점
+ 다음 날 양봉으로 상승 확인: +2점

총점: 11점 → A등급 → 적극 진입 고려
손절: 48,800원 (지지선 하방 2.4%)
목표: 55,000원 (다음 저항선)
리스크/리워드: 1 : 4.2
```

## 실전 적용

### 매매 체크리스트 구축

점수 시스템을 바탕으로 매매 전 확인하는표**준 체크리스트를** 만들면 감정적 판단을 줄일 수 있습니다.

```
캔들 패턴 기반 매매 체크리스트
─────────────────────────────────
사전 조건 (모두 충족해야 진입 검토)
□ 선행 추세가 명확한가? (방향: _____)
□ 패턴이 발생한 위치가 주요 지지·저항선인가?
□ 패턴 유형은 무엇인가? (패턴명: _____)
□ 패턴 신뢰도 점수는 몇 점인가? (점수: _____)

신뢰도 강화 조건 (2개 이상 충족 시 진입)
□ 거래량이 150% 이상인가?
□ 이동평균선이 지지·저항 역할을 하는가?
□ 확인 캔들이 출현했는가?

리스크 관리 (진입 전 필수)
□ 손절가는 어디인가? (_____ 원)
□ 1차 목표가는? (_____ 원)
□ 리스크/리워드 비율은? (_____ : _____)

─────────────────────────────────
```

캔들 패턴 조합 전략은 시장의 특성에 맞게 조정이 필요합니다.

| 시장 | 특징 | 조합 전략 조정 |
|------|------|-------------|
| 국내 주식 | 9시 갭 업·다운 빈번 | 갭 발생 시 첫 30분 패턴 주목 |
| 미국 주식 | 프리·애프터마켓 존재 | 정규장 시가·종가 기준 패턴 우선 |
| 암호화폐 | 24시간 거래, 고변동성 | 거래량 기준 평균 24시간 사용 |
| 외환(Forex) | 갭 거의 없음 | 거래 세션 전환 시간대 주목 |

## 흔한 실수

### 실수 1: 과도한 조건 설정 (분석 마비)

너무 많은 조건을 요구하면 진입 기회를 놓칩니다.

```
잘못된 접근:
"다음 조건이 모두 충족될 때만 진입:
- 피보나치 61.8% + MA200 + RSI 30 + 거래량 200% +
  강세 장악형 + 볼린저밴드 하단 + MACD 골든크로스"
→ 이 모든 조건이 동시에 충족되는 경우는 1년에 1~2회

올바른 접근:
- 핵심 조건 2~3개 충족 시 진입 검토
- 추가 조건이 많을수록 보너스로 취급
```

### 실수 2: 강한 추세 중 역추세 패턴 진입

```
잘못된 접근:
상승 추세 중간에서 단기 음봉 2개 나타남
→ "이중 음봉이 나왔으니 하락 반전!"이라고 섣불리 공매도

올바른 접근:
- 강한 추세 중 나타나는 역추세 패턴의 신뢰도는 낮음
- 추세 방향과 반대 패턴은 점수를 -2점 차감
- 강한 추세 중에는 추세 추종 패턴(되돌림 반등)만 활용
```

### 실수 3: 확인 캔들 없이 즉시 진입

```
잘못된 접근:
망치형 캔들 출현 → 즉시 시장가 매수

올바른 접근:
망치형 캔들 마감 확인
→ 다음 캔들이 양봉으로 시작하는지 확인
→ 양봉 초반 진입 (확인 캔들 전략)
손절: 망치형 저가 하방

이유: 망치형의 단독 성공률은 약 53%이지만
      다음 날 양봉 확인 시 성공률은 약 68%로 상승
```

### 실수 4: 거래량 없는 패턴을 신뢰

```
고거래량 시간대(장 시작, 장 마감 30분) 이외의 시간에
나타나는 패턴은 거래량이 낮을 가능성이 높습니다.
특히 점심 시간대(12~13시)의 패턴은 참고용으로만 활용하고,
당일 종가 또는 다음 날 시가를 기준으로 최종 판단하십시오.
```

:::caution
패턴 점수가 높아도 시장 전체가 강한 하락 추세(예: 지수가 -3% 이상 하락)인 날에는 개별 종목의 상승 반전 패턴 신뢰도가 크게 낮아집니다. 시장 전체 흐름을 먼저 확인하는 것이 원칙입니다.
:::

## 에지 케이스: 시장별 조합 전략 조정

**크립토 시장의 거래량 기준 변경크**립토 거래소별로 거래량 집계 방식이 다릅니다. 바이낸스·업비트 기준으로 20일 평균을 계산할 때 주말 거래량이 포함되므로 한국 주식 시장보다 기준선이 낮게 형성됩니다. 크립토에서는 150% 기준 대신**200% 이상을** 강한 확인 기준으로 사용하는 것이 오신호를 줄입니다.

**급등락 뉴스 직후 패턴매**크로 이벤트(금리 결정, CPI 발표, 기업 실적 쇼크) 직후에는 패턴 신뢰도 점수를 2점 낮춰 적용하십시오. 뉴스 충격으로 형성된 캔들은 시장 심리 변화가 아닌 일시적 유동성 충격을 반영하므로, 점수가 높게 나와도 다음 날 방향이 뒤집히는 경우가 빈번합니다.

**박스권 말미에서의 패턴박**스권 상단/하단에서 클라이맥스 거래량 없이 패턴만 출현하는 경우는 이탈 실패로 끝나는 경우가 많습니다. 박스권 돌파 패턴의 최소 거래량 기준을 200% 이상으로 높이고, 되돌림 후 재이탈 확인 전략을 우선하십시오.

**복수 시간프레임 점수 불일치주**봉에서 점수가 낮은데 일봉에서 점수가 높게 나오는 경우, 상위 시간프레임의 판단을 우선합니다. 일봉 A등급 + 주봉 C등급이면 실질 신뢰도는 B등급 수준으로 낮춰 접근하십시오.

## 핵심 요약

-**거래량 확인은** 캔들 패턴 조합의 가장 기초적이고 필수적인 단계로, 반전 캔들의 거래량이 20일 평균의 150% 이상일 때 신뢰도가 유의미하게 상승합니다
-**지지·저항선과의 조합은** 두 가지 독립 근거가 같은 방향을 지지하므로 신뢰도를 크게 높이며, 컨플루언스 존(3개 이상 근거 겹침)에서의 패턴은 특히 강력합니다
-**이동평균선 조합에**서는 상승 추세 중 MA20 지지에서의 반등 패턴, 하락 추세 중 MA200 저항에서의 반전 패턴이 핵심 전략입니다
-**복수 캔들 시퀀스분**석으로 개별 패턴이 아닌 추세 전환의 전체 흐름을 파악하면 더 안전한 진입 타이밍을 잡을 수 있습니다
-**점수 시스템을** 활용하면 주관적 판단을 줄이고 일관성 있는 매매 결정을 내릴 수 있으며, 5점 이상(B등급)일 때 진입을 검토하는 것이 기본 원칙입니다
- 가장 흔한 실수는 강한 추세 중 역추세 패턴을 신뢰하거나, 거래량 확인 없이 패턴만으로 진입하는 것입니다
