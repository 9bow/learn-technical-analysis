변동성은 가격 움직임의 크기를 나타냅니다. 변동성이 낮으면 시장은 에너지를 축적하고, 높으면 에너지를 방출합니다. 이 사이클을 체계적으로 포착하는 네 가지 전략을 소개합니다.

## 전략 17: 볼린저 밴드 스퀴즈 (BB Squeeze)

### 개요

볼린저 밴드 스퀴즈는 **밴드 폭(Bandwidth)** 이 장기 최저 수준으로 수축한 뒤 가격이 한 방향으로 폭발적으로 움직이는 현상을 포착하는 전략입니다. 변동성 수축(스퀴즈) 후 반드시 확장이 뒤따른다는 원칙에 기반합니다.

<CandlestickDemo client:visible dataset="default" title="볼린저 밴드 스퀴즈 예시" />

### 핵심 지표

$$Bandwidth = \frac{Upper - Lower}{MA_{20}} \times 100$$

- $Upper = MA_{20} + 2\sigma$
- $Lower = MA_{20} - 2\sigma$
- $\sigma$: 20일 표준편차

Bandwidth가 낮을수록 변동성이 압축된 상태입니다. 일반적으로 **6개월 최저 Bandwidth**를 스퀴즈 기준으로 사용합니다.

### 진입 조건

| 조건 | 기준 |
|------|------|
| 스퀴즈 감지 | Bandwidth < 6개월 최저값 |
| 상단 돌파 | 종가 > Upper Band → 롱 진입 |
| 하단 돌파 | 종가 < Lower Band → 숏 진입 |
| 거래량 확인 | 돌파 캔들의 거래량이 20일 평균 대비 1.5배 이상 |

### 청산 조건

- **이익 실현**: 반대편 밴드 도달 시
- **손절**: 진입 캔들 저점(롱) 또는 고점(숏) 이탈

### 의사코드

```
[BB Squeeze 전략]

매일 장 마감 후:
  MA20 = 20일 단순이동평균
  sigma = 20일 표준편차
  Upper = MA20 + 2 * sigma
  Lower = MA20 - 2 * sigma
  BW = (Upper - Lower) / MA20 * 100
  BW_6M_min = 최근 126거래일 중 BW 최솟값

  스퀴즈 상태 = (BW < BW_6M_min * 1.05)  # 5% 여유 허용

  만약 스퀴즈 상태:
    만약 종가 > Upper AND 거래량 > 20일 평균거래량 * 1.5:
      롱 진입
      손절 = 진입 캔들 저점
      목표 = Lower Band (반대편)
    만약 종가 < Lower AND 거래량 > 20일 평균거래량 * 1.5:
      숏 진입
      손절 = 진입 캔들 고점
      목표 = Upper Band (반대편)

  포지션 보유 중:
    만약 롱 AND 종가 < 손절:
      롱 청산
    만약 롱 AND 종가 >= 목표:
      롱 청산
    만약 숏 AND 종가 > 손절:
      숏 청산
    만약 숏 AND 종가 <= 목표:
      숏 청산
```

### 보완 필터

```
추세 필터: 200일 이동평균 방향 확인
  → 200일 MA 위 → 롱 신호만 허용
  → 200일 MA 아래 → 숏 신호만 허용

모멘텀 확인: ADX > 20이어야 추세 돌파로 인정
  → ADX < 20: 돌파가 아닌 노이즈일 가능성 높음
```

:::tip
스퀴즈가 길수록(수축 기간이 길수록) 이후 확장의 폭이 큰 경향이 있습니다. 3개월 이상 지속된 스퀴즈는 특히 주목할 만합니다.
:::

### 장점

- 명확한 진입·청산 기준으로 규칙화가 용이합니다.
- 변동성 확장 초입에 진입하므로 수익 대비 리스크 비율이 좋습니다.
- 거래 빈도가 낮아 수수료 비용이 적습니다.

### 단점

- 스퀴즈 이후 방향이 예상과 반대로 움직일 수 있습니다(페이크아웃).
- 횡보장에서는 돌파 후 즉시 되돌아오는 경우가 빈번합니다.
- 빠른 시장에서는 돌파 확인 후 진입 시 슬리피지가 발생할 수 있습니다.

## 전략 18: ATR 기반 포지션 사이징 전략

### 개요

**ATR(Average True Range)** 을 이용해 현재 시장 변동성에 맞게 포지션 크기를 동적으로 조절하는 전략입니다. 변동성이 클 때는 포지션을 줄이고, 변동성이 낮을 때는 늘려 **위험 금액을 일정하게** 유지합니다.

### 핵심 공식

$$\text{포지션 크기} = \frac{\text{계좌자산} \times \text{리스크}\%}{ATR_N \times \text{배수}}$$

- $ATR_N$: N일 ATR (보통 14일 또는 20일)
- 배수: 손절 범위를 ATR의 몇 배로 설정할지 (보통 1.5~3배)
- 리스크%: 거래당 최대 손실 허용 비율 (보통 1~2%)

### 계산 예시

```
[포지션 크기 계산 예시]

계좌자산: 10,000,000원
리스크 비율: 1% (거래당 최대 손실 100,000원)
ATR_14: 50,000원 (주가 기준)
ATR 배수: 2배 (손절 폭 = 100,000원)

포지션 크기 = 10,000,000 × 0.01 / (50,000 × 2)
            = 100,000 / 100,000
            = 1주

→ 1주 매수 시 손절 범위(ATR×2) 내 최대 손실 = 100,000원 (자산의 1%)
```

### 변동성 상태별 포지션 조절

| ATR 상태 | 포지션 크기 | 설명 |
|---------|-----------|------|
| ATR < 20일 평균 ATR × 0.7 | 증가 | 저변동성, 더 많은 주수 매수 가능 |
| ATR ≈ 20일 평균 ATR | 기본 | 표준 포지션 |
| ATR > 20일 평균 ATR × 1.5 | 감소 | 고변동성, 포지션 축소 |
| ATR > 20일 평균 ATR × 2.0 | 최소화 | 극단적 변동성, 신규 진입 자제 |

### 의사코드

```
[ATR 포지션 사이징 전략]

매일 장 마감 후:
  ATR14 = 14일 ATR
  ATR_avg = ATR14의 20일 이동평균

  계좌자산 = 현재 계좌 총자산
  리스크_금액 = 계좌자산 * 0.01  # 1% 리스크
  손절_배수 = 2.0

  손절_폭 = ATR14 * 손절_배수
  포지션_크기 = 리스크_금액 / 손절_폭

  # 변동성 조정 필터
  만약 ATR14 > ATR_avg * 2.0:
    포지션_크기 = 포지션_크기 * 0.5  # 절반으로 축소
  만약 ATR14 < ATR_avg * 0.7:
    포지션_크기 = 포지션_크기 * 1.3  # 30% 확대 (상한 설정 필요)

  # 최대 포지션 상한 (자산의 10%)
  최대_포지션 = 계좌자산 * 0.10 / 현재가
  포지션_크기 = min(포지션_크기, 최대_포지션)

  진입 신호 발생 시:
    해당 포지션_크기만큼 진입
    손절가 = 진입가 - (ATR14 * 손절_배수)  # 롱 기준
    목표가 = 진입가 + (ATR14 * 손절_배수 * 2)  # 2:1 R/R
```

:::caution
포지션 사이징은 진입 신호와 별개의 레이어입니다. 아무리 강한 신호라도 계산된 포지션 크기를 초과해서는 안 됩니다. 이 원칙을 어기는 순간 리스크 관리가 무너집니다.
:::

### 장점

- 변동성이 높은 시장에서 자동으로 노출을 줄여 계좌를 보호합니다.
- 모든 종목과 전략에 적용 가능한 범용 리스크 관리 프레임워크입니다.
- 거래당 손실을 사전에 정의해 연속 손실 시에도 계좌 보존이 가능합니다.

### 단점

- ATR이 갑작스럽게 급등한 직후에는 포지션이 지나치게 작아질 수 있습니다.
- 단독으로는 진입 신호를 제공하지 않아 다른 전략과 반드시 결합해야 합니다.
- 종목별 ATR 수준 차이가 크므로 종목 간 비교 시 주의가 필요합니다.

## 전략 19: 변동성 사이클 전략 (Expansion/Contraction)

### 개요

시장 변동성은 **수축(Contraction)** 과 **확장(Expansion)** 사이클을 반복합니다. 이 전략은 수축 구간에서 대기하다가 확장이 시작되는 초입에 추세 방향으로 진입해 추세의 핵심 구간을 포착합니다.

### 핵심 공식

$$\text{상태} = \begin{cases} Contraction & ATR < ATR_{평균} \\ Expansion & ATR > ATR_{평균} \times 1.5 \end{cases}$$

- $ATR_{평균}$: ATR의 50일 이동평균
- Contraction(수축): 낮은 변동성, 에너지 축적 구간
- Expansion(확장): 높은 변동성, 추세 발생 구간

### 상태 전환 매트릭스

| 이전 상태 | 현재 상태 | 행동 |
|---------|---------|------|
| Contraction | Contraction | 대기 유지 |
| Contraction | Expansion | **진입 신호** — 추세 방향 확인 후 진입 |
| Expansion | Expansion | 포지션 유지 |
| Expansion | Contraction | 포지션 청산 준비 |

### 의사코드

```
[변동성 사이클 전략]

매일 장 마감 후:
  ATR14 = 14일 ATR
  ATR_avg50 = ATR14의 50일 이동평균
  MA50 = 50일 단순이동평균

  # 상태 결정
  만약 ATR14 > ATR_avg50 * 1.5:
    현재_상태 = Expansion
  만약 ATR14 < ATR_avg50:
    현재_상태 = Contraction
  그 외:
    현재_상태 = Neutral  # 전환 중간 구간

  # 진입 로직
  만약 이전_상태 == Contraction AND 현재_상태 == Expansion:
    # 추세 방향 결정 (MA50 활용)
    만약 종가 > MA50:
      롱 진입  # 상향 확장
      손절 = 최근 5일 최저가
    만약 종가 < MA50:
      숏 진입  # 하향 확장
      손절 = 최근 5일 최고가

  # 청산 로직
  만약 현재_상태 == Contraction AND 포지션 보유 중:
    포지션 청산  # 변동성 수축으로 추세 종료

  이전_상태 = 현재_상태
```

### 변동성 상태 확인 보조 지표

```
보조 확인 수단:
1. 볼린저 밴드 폭(Bandwidth): BW 상승 = Expansion 확인
2. 역사적 변동성(HV): HV_10 > HV_30 = 단기 변동성 가속
3. 켈트너 채널: 볼린저 밴드가 켈트너 채널 밖으로 돌출 = 강한 Expansion
```

:::note
변동성 사이클은 종목마다, 시장마다 사이클 길이가 다릅니다. 주식은 몇 주~몇 달, 암호화폐는 며칠~몇 주 단위가 일반적입니다. 백테스트로 해당 종목의 평균 사이클 길이를 확인하세요.
:::

### 장점

- 추세의 핵심 구간(확장 초입)에 집중해 효율적인 수익 포착이 가능합니다.
- 수축 구간에서 무의미한 거래를 자제해 수수료와 슬리피지를 절약합니다.
- ATR이라는 객관적 지표로 상태를 정의해 판단의 주관성을 제거합니다.

### 단점

- 확장 판정 후 진입하므로 이미 일부 움직임을 놓칩니다(후행성).
- 가짜 확장(한두 캔들만 튀고 다시 수축) 시 손절 발생이 잦습니다.
- 1.5배 임계값은 종목별로 최적화가 필요합니다.

## 전략 20: VIX 평균회귀 전략

### 개요

**VIX(CBOE Volatility Index)** 는 S&P 500 옵션 시장의 내재 변동성에서 산출되는 **공포 지수입**니다. VIX가 역사적 평균에서 크게 이탈하면 반드시 평균으로 회귀하는 특성을 이용해, VIX 급등(시장 공포 극단) 시 주식 시장을 매수하는 역추세 전략입니다.

### 핵심 공식

$$\text{신호} = VIX > \mu_{VIX} + 2\sigma_{VIX}$$

- $\mu_{VIX}$: VIX의 252거래일(1년) 이동평균
- $\sigma_{VIX}$: VIX의 252거래일 표준편차
- 신호 발생 시: S&P 500 ETF(SPY) 또는 코스피 관련 자산 매수

### VIX 레벨 해석표

| VIX 수준 | 시장 심리 | 전략적 의미 |
|---------|---------|-----------|
| 10 ~ 15 | 극단적 낙관 | 주식 매수 자제, 변동성 확대 준비 |
| 15 ~ 20 | 정상 | 중립 |
| 20 ~ 30 | 불안 | 주의 필요 |
| 30 ~ 40 | 공포 | 분할 매수 관심 구간 |
| 40 이상 | 극단적 공포 | **강한 매수 신호** — 평균회귀 기대 |

### 의사코드

```
[VIX 평균회귀 전략]

매일 장 마감 후:
  VIX_today = 당일 VIX 종가
  VIX_252 = 최근 252거래일 VIX 데이터

  mu_VIX = VIX_252의 평균
  sigma_VIX = VIX_252의 표준편차

  상단_임계값 = mu_VIX + 2 * sigma_VIX
  하단_임계값 = mu_VIX + 0.5 * sigma_VIX  # 청산 기준

  # 진입 조건 (분할 매수 전략)
  만약 VIX_today > 상단_임계값:
    1차 매수: SPY 포지션의 33%
  만약 VIX_today > 상단_임계값 * 1.2:
    2차 매수: SPY 포지션의 33% 추가
  만약 VIX_today > 상단_임계값 * 1.5:
    3차 매수: SPY 포지션의 34% 추가

  # 청산 조건
  만약 VIX_today < 하단_임계값 AND 포지션 보유 중:
    전량 청산  # VIX 정상화 = 공포 해소 = 수익 실현

  # 강제 손절 (VIX 지속 상승 시)
  만약 VIX_today > 상단_임계값 * 2.0:
    포지션 50% 축소  # 극단적 위기 시 노출 감소
```

### 역사적 VIX 스파이크 사례

```
주요 VIX 급등 사례 (매수 기회):
2008년 10월: VIX 89.5 (금융위기 극단)
  → 이후 1년 S&P 500 +60%

2020년 3월: VIX 82.7 (코로나 팬데믹)
  → 이후 1년 S&P 500 +77%

2022년 3월: VIX 36.5 (금리 인상 우려)
  → 이후 6개월 S&P 500 +10% (단기 반등)

패턴: VIX > 40인 시점은 역사적으로 중장기 매수 기회였습니다.
```

:::caution
VIX는 S&P 500 기반 지표입니다. 한국 주식(코스피)에는 VKOSPI를, 암호화폐에는 Bitcoin Volatility Index(BVIV)를 사용하세요. 시장별로 적절한 변동성 지수를 선택해야 합니다.
:::

### 성과 특성

```
전략 기대 성과 (역사적 분석 기반):
- 평균 보유 기간: 20~60거래일
- 승률: 약 70% (VIX > μ + 2σ 기준)
- 평균 수익: +15~25% (보유 기간 내)
- 최대 손실: -20~30% (VIX 지속 상승 시)
- 연간 거래 횟수: 3~8회

주의: 과거 성과가 미래를 보장하지 않습니다.
```

### 장점

- 시장 공포가 극단에 달한 시점, 즉 역사적으로 가장 좋은 매수 기회를 포착합니다.
- 단순한 규칙으로 감정적 매수 결정을 배제할 수 있습니다.
- VIX 데이터는 무료로 접근 가능하며 신뢰성이 높습니다.

### 단점

- VIX가 고점에서 더 높이 올라가는 동안 손실이 심화될 수 있습니다(2008년 사례).
- 진입 타이밍을 정확히 잡기 어려워 분할 매수가 필수입니다.
- S&P 500 이외 자산에 직접 적용하기 어렵고, 대체 지표의 신뢰도가 낮습니다.

## 핵심 요약

- **볼린저 스퀴즈는** 밴드 폭 수축 → 돌파 방향 추종 전략으로, 거래량 확인이 필수입니다.
- **ATR 포지션 사이징은** 진입 신호가 아닌 리스크 관리 도구로, 모든 전략에 결합 가능합니다.
- **변동성 사이클은** Contraction → Expansion 전환 초입 진입으로 추세의 핵심 구간을 포착합니다.
- **VIX 평균회귀는** 시장 공포 극단에서의 역추세 매수로, 분할 매수와 긴 보유 기간이 관건입니다.
- 변동성 전략들은 공통적으로 **"변동성은 회귀한다"**는 통계적 특성을 이용합니다.
