## 전략 21: 삼중 필터 시스템 (Triple Screen — Alexander Elder)

### 개요

**알렉산더 엘더(Alexander Elder)** 가 1986년 《Trading for a Living》에서 제안한 시스템입니다. 3개의 시간 프레임을 계층적으로 사용해 — 장기 추세 방향 확정(Filter 1) → 중기 역추세 조정 대기(Filter 2) → 단기 진입 타이밍(Filter 3) — 추세와 역추세를 동시에 활용하는 구조입니다.

<CandlestickDemo client:visible dataset="default" title="삼중 필터 시스템 — 3개 시간 프레임" />

### 시간 프레임 설정

시간 프레임 간 비율은 약 5배를 권장합니다.

| 필터 | 시간 프레임 | 목적 | 지표 |
|------|---------|------|------|
| Filter 1 (조류) | 주봉 | 큰 추세 방향 결정 | MACD 히스토그램 기울기 |
| Filter 2 (파도) | 일봉 | 역추세 조정 구간 대기 | 스토캐스틱 과매도/과매수 |
| Filter 3 (잔물결) | 시간봉 | 정밀 진입 타이밍 | 추적 매수(Trailing Buy Stop) |

### 핵심 공식

$$\text{Filter}_1: \text{주봉 MACD 히스토그램 기울기} > 0 \Rightarrow \text{상승 추세}$$

$$\text{Filter}_2: \text{일봉 스토캐스틱}(5,3,3) < 30 \Rightarrow \text{과매도 조정}$$

$$\text{Filter}_3: \text{시간봉 고점 돌파} \Rightarrow \text{롱 진입}$$

MACD 히스토그램 기울기:

$$\text{기울기} = MACD\text{\_Hist}_{오늘} - MACD\text{\_Hist}_{어제}$$

### 단계별 필터 로직

```
[Filter 1 — 주봉 추세 방향]

주봉 MACD(12, 26, 9) 히스토그램 계산:
  MACD_Line = EMA12(주봉) - EMA26(주봉)
  Signal = EMA9(MACD_Line)
  Histogram = MACD_Line - Signal

  히스토그램 기울기 = 이번 주 Histogram - 지난 주 Histogram

  만약 기울기 > 0:
    방향 = 상승  → 롱 신호만 탐색
  만약 기울기 < 0:
    방향 = 하락  → 숏 신호만 탐색
  그 외 (기울기 = 0):
    방향 = 중립  → 대기
```

```
[Filter 2 — 일봉 역추세 조정]

일봉 스토캐스틱(5, 3, 3) 계산:
  %K = (종가 - 5일 최저) / (5일 최고 - 5일 최저) × 100
  %D = %K의 3일 이동평균
  Slow %D = %D의 3일 이동평균

  만약 방향 == 상승:
    기다림: 일봉 Slow %D < 30 (과매도 조정 구간 도달)
  만약 방향 == 하락:
    기다림: 일봉 Slow %D > 70 (과매수 반등 구간 도달)
```

```
[Filter 3 — 시간봉 진입 타이밍]

방향 == 상승 AND 일봉 스토캐스틱 < 30일 때:
  당일 시간봉에서 추적 매수 설정:
    매수 트리거 = 전날 일봉 고점 + 1틱
    → 시가가 트리거 위에서 출발하면 시가에 매수
    → 가격이 트리거까지 상승하면 자동 매수

방향 == 하락 AND 일봉 스토캐스틱 > 70일 때:
  당일 시간봉에서 추적 매도 설정:
    매도 트리거 = 전날 일봉 저점 - 1틱
    → 시가가 트리거 아래에서 출발하면 시가에 매도
    → 가격이 트리거까지 하락하면 자동 매도
```

### 청산 전략

```
[청산 — 삼중 필터]

손절:
  롱: 진입일 당일 저점 아래 (시간봉 기준)
  숏: 진입일 당일 고점 위

이익 실현:
  1차 목표: 손절폭의 2배 (R/R 2:1)
  2차 목표: 채널 반대편 또는 주봉 저항선

추가 청산 신호:
  Filter 1 방향이 반전될 때 → 무조건 청산
  (주봉 MACD 히스토그램 기울기가 반대로 전환)
```

:::tip
삼중 필터에서 가장 흔한 실수는 Filter 1 없이 Filter 2와 3만 보는 것입니다. 큰 추세에 역행하는 스토캐스틱 신호는 무시해야 합니다. Filter 1이 하락을 가리키면 일봉 스토캐스틱 과매도 신호가 나와도 롱을 잡지 않습니다.
:::

### 실전 예시

```
[실전 적용 예시]

상황: 삼성전자 (일별 분석)

Filter 1 확인 (주봉):
  주봉 MACD 히스토그램: -200 → -150 (기울기 +50, 상승 전환 중)
  방향 = 상승 → 롱 신호 탐색 모드

Filter 2 확인 (일봉):
  일봉 스토캐스틱(5,3,3): Slow %D = 22
  기준(30) 아래 → 과매도 조정 완료 판정

Filter 3 실행 (시간봉):
  전날 일봉 고점: 72,500원
  매수 트리거 설정: 72,600원 (1틱 위)
  다음날 오전 72,600원 돌파 → 롱 진입

손절: 당일 시간봉 저점 71,800원
목표 1: 진입가 + (72,600 - 71,800) × 2 = 74,200원
```

### 장점

- 추세와 역추세를 동시에 활용해 추세 방향 진입 시 조정 구간 저점을 잡습니다.
- 3단계 필터로 오진입이 크게 줄어 신호의 신뢰도가 높습니다.
- 시간 프레임 비율 조정만으로 스윙부터 단타까지 다양하게 응용할 수 있습니다.
- 엘더의 원작에서 충분히 검증된 체계적 접근법입니다.

### 단점

- 3개 시간 프레임을 동시에 모니터링해야 해 운용이 복잡합니다.
- 강한 추세장에서는 스토캐스틱이 오버솔드로 내려오지 않아 진입 기회를 놓칩니다.
- 주봉 기반이므로 신호 발생까지 수 주를 기다려야 할 수 있습니다.
- 빠르게 변하는 시장(암호화폐 등)에서는 주봉 MACD가 너무 느립니다.

## 전략 22: 일목균형표 복합 전략 (Ichimoku Composite)

### 개요

**일목균형표(Ichimoku Kinko Hyo)** 는 가격의 균형, 추세, 지지/저항, 모멘텀을 단일 차트에 시각화하는 일본 기술적 분석 도구입니다. 5개의 선이 모두 같은 방향을 가리킬 때만 진입하는 **삼위일체(三位一體) 조건을** 핵심으로 삼습니다.

<CandlestickDemo client:visible dataset="default" title="일목균형표 복합 전략" />

### 5개 선 구성

$$\text{전환선(Tenkan-sen)} = \frac{\max(High_{9}) + \min(Low_{9})}{2}$$

$$\text{기준선(Kijun-sen)} = \frac{\max(High_{26}) + \min(Low_{26})}{2}$$

$$\text{선행스팬A(Senkou Span A)} = \frac{\text{전환선} + \text{기준선}}{2} \quad (\text{26일 선행})$$

$$\text{선행스팬B(Senkou Span B)} = \frac{\max(High_{52}) + \min(Low_{52})}{2} \quad (\text{26일 선행})$$

$$\text{후행스팬(Chikou Span)} = \text{당일 종가} \quad (\text{26일 후행})$$

### 구름(Kumo) 해석

| 구름 상태 | 의미 |
|---------|------|
| 가격 > 구름 (Span A > Span B) | 강세 — 구름이 지지 역할 |
| 가격 < 구름 (Span A < Span B) | 약세 — 구름이 저항 역할 |
| 가격 = 구름 내부 | 중립, 추세 불명확 |
| 구름 두께 얇음 | 돌파 용이, 변동성 확장 예고 |
| 구름 두께 두꺼움 | 강한 지지/저항, 돌파 어려움 |

### 삼위일체 진입 조건

```
[롱 진입 — 삼위일체 조건]

조건 1 (전환선/기준선):
  전환선(9일) > 기준선(26일)
  → 단기 모멘텀이 중기 균형을 상회

조건 2 (가격/구름):
  종가 > 구름 상단 (선행스팬 A와 B 중 높은 값)
  → 가격이 구름 위에 있어 강세 구조

조건 3 (후행스팬):
  후행스팬 > 26일 전 가격 (후행스팬이 과거 가격 위)
  → 현재 모멘텀이 과거 수준보다 강함

추가 확인 조건 (선택):
  미래 구름(26일 앞) = 녹색 (Span A > Span B)
  → 미래 구름도 강세 구조
```

```
[숏 진입 — 역방향 조건]

조건 1: 전환선 < 기준선
조건 2: 종가 < 구름 하단
조건 3: 후행스팬 < 26일 전 가격
추가:   미래 구름 = 적색 (Span B > Span A)
```

### 의사코드

```
[일목균형표 복합 전략]

매일 장 마감 후:
  # 5개 선 계산
  Tenkan = (최근 9일 고점 + 최근 9일 저점) / 2
  Kijun  = (최근 26일 고점 + 최근 26일 저점) / 2
  SpanA  = (Tenkan + Kijun) / 2  # 26일 전 값이 현재 구름
  SpanB  = (최근 52일 고점 + 최근 52일 저점) / 2  # 26일 전 값
  Chikou = 현재 종가  # 26일 후행 → 26일 전 가격과 비교

  구름_상단 = max(SpanA_26일전, SpanB_26일전)
  구름_하단 = min(SpanA_26일전, SpanB_26일전)

  # 롱 신호 확인
  c1_롱 = Tenkan > Kijun
  c2_롱 = 현재 종가 > 구름_상단
  c3_롱 = Chikou > 26일_전_종가

  만약 c1_롱 AND c2_롱 AND c3_롱:
    롱 진입
    손절 = Kijun (기준선) — 26일 균형을 이탈하면 청산
    목표 = 구름_상단 * 1.03 또는 채널 저항

  # 숏 신호 확인
  c1_숏 = Tenkan < Kijun
  c2_숏 = 현재 종가 < 구름_하단
  c3_숏 = Chikou < 26일_전_종가

  만약 c1_숏 AND c2_숏 AND c3_숏:
    숏 진입
    손절 = Kijun (기준선)
    목표 = 구름_하단 * 0.97 또는 채널 지지

  # 청산: 기준선 이탈
  만약 롱 포지션 AND 종가 < Kijun:
    롱 청산
  만약 숏 포지션 AND 종가 > Kijun:
    숏 청산
```

### 진입 강도 점수화

```
[일목 신호 강도 점수 (0~5점)]

전환선 > 기준선: +1점
가격 > 구름 상단: +1점
후행스팬 > 26일 전 가격: +1점
미래 구름 = 녹색: +1점
구름 두께 < 최근 3개월 평균: +1점 (얇은 구름 = 추세 전환 용이)

5점: 최강 신호 → 풀 포지션
4점: 강한 신호 → 75% 포지션
3점: 보통 신호 → 50% 포지션
2점 이하: 진입 자제
```

:::note
일목균형표는 일봉 기준으로 개발되었습니다. 9, 26, 52라는 숫자는 1주일 6거래일 기준의 일본 시장 관습에서 나왔습니다. 5거래일 기준인 현대 시장에는 10, 30, 60으로 조정하거나 원래 숫자를 그대로 유지하는 두 가지 접근이 모두 사용됩니다.
:::

### 크로스 확인: 전환선/기준선 교차

```
[TK 크로스 — 단기 진입 보조]

강세 TK 크로스:
  조건: 전환선이 기준선 아래에서 위로 교차
  조건 추가: 교차 발생 시 가격이 구름 위에 있어야 함
  의미: 단기 모멘텀 회복 → 조기 진입 기회

약세 TK 크로스:
  조건: 전환선이 기준선 위에서 아래로 교차
  조건 추가: 교차 발생 시 가격이 구름 아래에 있어야 함
  의미: 단기 모멘텀 악화 → 숏 조기 진입 기회
```

### 장점

- 5개 선이 단일 차트에서 추세, 모멘텀, 지지/저항, 시간 요소를 모두 제공합니다.
- 삼위일체 조건은 높은 신뢰도의 신호만 선별해 오진입을 최소화합니다.
- 기준선이 자연스러운 손절 기준이 되어 손절 설정이 직관적입니다.
- 미래 구름으로 앞으로의 지지/저항 레벨을 사전에 파악할 수 있습니다.

### 단점

- 5개 선을 동시에 해석해야 해 초보자에게 학습 곡선이 가파릅니다.
- 삼위일체 조건을 동시에 충족하는 경우가 드물어 거래 빈도가 매우 낮습니다.
- 주봉·월봉 등 상위 시간 프레임에서는 52주 데이터가 필요해 설정이 까다롭습니다.
- 빠르게 움직이는 시장에서는 기준선 손절이 너무 멀리 있어 R/R이 불리합니다.

## 전략 23: VWAP + RSI 데이트레이딩

### 개요

**VWAP(Volume Weighted Average Price, 거래량 가중 평균 가격)** 은 하루 중 실제 거래가 이루어진 평균 가격으로, 기관 투자자의 핵심 기준선입니다. VWAP 방향으로 추세를 확인하고, **RSI** 과매도 반등으로 정밀 진입 타이밍을 포착하는 인트라데이 전략입니다.

<CandlestickDemo client:visible dataset="volatile-crypto" title="VWAP + RSI 데이트레이딩" />

### 핵심 공식

$$VWAP = \frac{\sum_{t=1}^{n} (P_t \times V_t)}{\sum_{t=1}^{n} V_t}$$

- $P_t$: $t$ 시점의 대표 가격 $= \frac{High_t + Low_t + Close_t}{3}$ (Typical Price)
- $V_t$: $t$ 시점의 거래량
- $n$: 장 시작부터 현재까지의 누적

VWAP은 매일 장 시작 시 초기화됩니다. 전일 데이터를 이어받지 않습니다.

### VWAP 밴드

VWAP 단독보다 표준편차 밴드를 함께 사용합니다.

$$VWAP_{Upper1} = VWAP + 1\sigma_{VWAP}$$
$$VWAP_{Upper2} = VWAP + 2\sigma_{VWAP}$$
$$VWAP_{Lower1} = VWAP - 1\sigma_{VWAP}$$
$$VWAP_{Lower2} = VWAP - 2\sigma_{VWAP}$$

| 가격 위치 | 해석 |
|---------|------|
| VWAP 위 | 매수 우위 (기관 평단 위) |
| VWAP 아래 | 매도 우위 (기관 평단 아래) |
| Upper2 돌파 | 과매수 → 단기 되돌림 경계 |
| Lower2 이탈 | 과매도 → 단기 반등 경계 |

### 진입 조건 상세

```
[롱 진입 조건]

조건 1 (VWAP 방향):
  현재 VWAP > 30분 전 VWAP  (VWAP 상향 기울기)
  AND 현재가 > VWAP

조건 2 (RSI 반등):
  RSI(14, 1분 또는 5분) < 40 → 일시적 과매도
  직전 봉 RSI > 현재 RSI  → RSI 저점 형성 중
  현재 봉 RSI 반등 확인 (RSI 상승 전환)

조건 3 (가격 구조):
  VWAP Lower1 근처에서 지지 캔들 확인
  (긴 아랫꼬리 또는 도지 캔들)

조건 4 (거래량):
  직전 반등 캔들의 거래량 > 5분 평균 거래량 * 1.3

→ 4가지 조건 모두 충족 시 롱 진입
```

```
[숏 진입 조건]

조건 1: VWAP 하향 기울기 AND 현재가 < VWAP
조건 2: RSI(14) > 60 → 일시적 과매수 후 하락 전환
조건 3: VWAP Upper1 근처에서 저항 캔들 확인
조건 4: 직전 하락 캔들 거래량 > 5분 평균 * 1.3

→ 4가지 조건 모두 충족 시 숏 진입
```

### 의사코드

```
[VWAP + RSI 데이트레이딩 전략]

장 시작 시:
  누적_PV = 0  # 가격 × 거래량 합계
  누적_V  = 0  # 거래량 합계
  누적_PV2 = 0  # 가격² × 거래량 합계 (표준편차 계산용)

매 5분봉 마감 시:
  Typical_P = (High + Low + Close) / 3
  누적_PV  += Typical_P * Volume
  누적_V   += Volume
  누적_PV2 += Typical_P^2 * Volume

  VWAP = 누적_PV / 누적_V
  분산 = (누적_PV2 / 누적_V) - VWAP^2
  sigma = sqrt(분산)

  VWAP_U1 = VWAP + sigma
  VWAP_L1 = VWAP - sigma
  VWAP_L2 = VWAP - 2 * sigma

  RSI_14 = 최근 14개 5분봉 RSI 계산

  # 롱 조건
  c1 = 현재가 > VWAP AND VWAP > VWAP_30분전
  c2 = RSI_14 < 40 AND RSI_14 > RSI_14_직전봉  # RSI 반등
  c3 = 현재가 >= VWAP_L1 * 0.998 AND 현재가 <= VWAP_L1 * 1.002  # L1 근처
  c4 = 현재_거래량 > 5봉_평균거래량 * 1.3

  만약 c1 AND c2 AND c3 AND c4 AND 포지션_없음:
    롱 진입
    손절 = VWAP_L2  # 2시그마 밴드 이탈 시 손절
    목표1 = VWAP  # VWAP 복귀
    목표2 = VWAP_U1  # 상단 1시그마

  # 청산 조건
  만약 롱 포지션:
    만약 현재가 >= 목표1:
      50% 청산
    만약 현재가 >= 목표2:
      나머지 청산
    만약 현재가 <= 손절:
      전량 손절

  # 장 마감 30분 전 강제 청산 (데이트레이딩)
  만약 현재 시각 >= 장마감 30분 전:
    포지션 전량 청산
```

### 시간대별 VWAP 전략 강도

```
[장중 시간대별 신뢰도]

09:00 ~ 09:30 (장 초반):
  VWAP 신뢰도 낮음 (누적 데이터 부족)
  → 관망, 진입 자제

09:30 ~ 11:30 (오전 활성 구간):
  VWAP 방향성 형성
  → 신호 신뢰도 중상, 적극 탐색

11:30 ~ 13:30 (점심 구간):
  거래량 감소, 노이즈 신호 증가
  → 포지션 축소 또는 관망

13:30 ~ 14:30 (오후 활성 구간):
  거래량 회복, 기관 수급 마무리
  → 신호 신뢰도 높음

14:30 ~ 15:00 (마감 구간):
  급격한 방향 변화 가능
  → 신규 진입 금지, 기존 포지션 정리
```

:::tip
VWAP은 **인트라데이 전용** 지표입니다. 장 시작 시 리셋되므로 전일 VWAP과 비교하지 않습니다. 다만 여러 날에 걸친 **앵커드 VWAP(Anchored VWAP)** 을 사용하면 중요한 이벤트(실적 발표, 급락일 등) 기준의 VWAP을 중기 분석에 활용할 수 있습니다.
:::

### 실전 필터: 피해야 할 상황

```
[VWAP + RSI 진입 자제 상황]

1. 실적 발표일 또는 FOMC 당일:
   → 급격한 갭이 VWAP 계산을 왜곡

2. 거래량이 평소의 50% 미만인 날:
   → VWAP 기울기 의미 없음

3. 가격이 VWAP Upper2 또는 Lower2 이탈 후:
   → 극단적 편향 상태, 역추세 위험

4. VIX > 30인 날:
   → 시장 전체 변동성이 높아 VWAP 반등이 실패할 확률 증가

5. 오전 첫 30분:
   → 누적 거래량 부족으로 VWAP 오차 큼
```

### 장점

- VWAP은 기관 투자자의 실제 평균 비용이므로 기관의 매매 행동과 연계됩니다.
- RSI와 결합해 VWAP 지지에서의 반등 타이밍을 정밀하게 포착합니다.
- 데이트레이딩 특성상 오버나이트 리스크(뉴스, 갭)가 없습니다.
- 명확한 장 마감 청산 규칙으로 포지션 누적이 발생하지 않습니다.

### 단점

- 장중 상시 모니터링이 필요해 자동화 없이는 실행이 어렵습니다.
- 수수료와 슬리피지 비용이 스윙 트레이딩보다 크게 누적됩니다.
- 시장이 단방향으로 강하게 움직이는 날에는 VWAP 반등 전략이 손실을 냅니다.
- 종목별로 최적 RSI 임계값과 시간 프레임이 다르므로 개별 최적화가 필요합니다.

| 전략 | 시간 프레임 | 거래 빈도 | 자동화 난이도 | 핵심 강점 |
|------|---------|---------|------------|---------|
| 삼중 필터 | 주봉+일봉+시간봉 | 월 1~4회 | 중간 | 추세+역추세 결합 |
| 일목균형표 복합 | 일봉 중심 | 월 2~6회 | 낮음 | 단일 차트 종합 분석 |
| VWAP+RSI | 5분봉 (인트라데이) | 일 2~5회 | 높음 | 기관 수급 기준선 활용 |

## 다중 요소 전략의 공통 원칙

### 신호 계층 구조

```
[올바른 다중 필터 적용 순서]

1단계 (거시 필터): 시장 전체 방향 확인
  → 시장이 약세장이면 개별 종목 롱 신호 억제

2단계 (추세 필터): 해당 자산의 추세 방향 확인
  → 추세 방향과 일치하는 신호만 통과

3단계 (모멘텀 필터): 진입 시점의 모멘텀 확인
  → 추세 방향으로 모멘텀 회복 중인지 확인

4단계 (타이밍 필터): 정밀 진입 타이밍
  → 캔들, 패턴, 단기 지표로 최적 진입 시점 결정
```

### 신호 충돌 처리

```
[필터 간 신호 충돌 시 원칙]

상위 시간 프레임 > 하위 시간 프레임
  → 주봉이 하락을 가리키면 일봉 매수 신호 무효

필터 수 과반수 동의:
  → 4개 필터 중 3개 이상이 같은 방향을 가리킬 때만 진입

불일치 시 대기:
  → 필터 간 충돌이 있으면 진입하지 않음
  → 더 명확한 환경을 기다리는 것이 최선
```

:::caution
필터를 추가할수록 거래 빈도는 줄고 신호 신뢰도는 높아집니다. 하지만 너무 많은 필터는 과적합(Overfitting)을 초래해 백테스트에서만 잘 작동하는 전략이 될 수 있습니다. 필터는 3~5개 이내로 제한하고, 각 필터가 독립적인 관점을 제공하는지 확인하세요.
:::

## 핵심 요약

- **삼중 필터는** 주봉(추세) → 일봉(조정 대기) → 시간봉(진입)의 3단계 계층 구조로 추세 방향의 최적 진입 시점을 찾습니다.
- **일목균형표 복합은** 5개 선의 삼위일체 조건(전환>기준, 가격>구름, 후행>과거)을 동시에 충족할 때만 진입해 높은 신뢰도를 유지합니다.
- **VWAP+RSI**는 기관의 평균 매수 비용(VWAP)을 기준으로 RSI 반등 타이밍에 진입하는 인트라데이 전략으로, 장 마감 전 청산이 필수입니다.
- 다중 요소 전략의 핵심은 **필터 간 독립성** 확보입니다. 같은 정보를 다른 이름으로 중복 확인하는 것은 신뢰도를 높이지 않습니다.
- 필터 충돌 시에는 진입하지 않는 것이 최선입니다. 시장은 항상 다음 기회를 제공합니다.
