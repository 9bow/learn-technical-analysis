## 평균회귀 전략이란?

평균회귀(Mean Reversion)는 "가격은 장기 평균으로 돌아오려는 성질이 있다"는 통계적 원리에 기반합니다. 가격이 평균에서 크게 벗어났을 때 **반대 방향으로 포지션을 취해 평균 복귀 시 수익을** 얻는 전략입니다.

평균회귀 전략은 모멘텀 전략과 반대 관점을 취합니다:

| 전략 유형 | 시장 관점 | 최적 환경 |
|----------|----------|----------|
| **모멘텀** | 추세는 지속된다 | 강한 추세장 |
| **평균회귀** | 극단은 되돌아온다 | 횡보·범위 장세 |

평균회귀 전략이 유효한 근거:
- **통계적 정상성**: 많은 금융 시계열이 단기적으로 평균 복귀 성질을 가짐
- **과잉 반응**: 투자자들의 공포·탐욕에 의한 과도한 가격 이탈
- **시장 조성자**: 극단적 가격에서 시장 조성자의 반대 거래 증가

이 챕터에서는 볼린저 밴드 평균회귀, Z-Score 통계적 평균회귀, RSI+볼린저 복합 전략, 켈트너 채널 평균회귀 등 **4가지 실전 평균회귀 전략을** 다룹니다.

## 전략 5: 볼린저 밴드 평균회귀

### 원리

볼린저 밴드(Bollinger Bands)는 이동평균선(중간 밴드)을 중심으로 표준편차 배수만큼 떨어진 상·하단 밴드를 그립니다. 통계적으로 가격의 약 95%는 ±2 표준편차 범위 안에 존재하므로, **하단 밴드를 이탈하면 과매도, 상단 밴드를 이탈하면 과매수로** 판단합니다.

### 공식

$$Upper = MA_{20} + 2\sigma, \quad Lower = MA_{20} - 2\sigma$$

- $MA_{20}$: 20일 단순이동평균 (중간 밴드)
- $\sigma$: 20일 가격의 표준편차
- **상단 밴드**: 중간 밴드 + 2배 표준편차
- **하단 밴드**: 중간 밴드 - 2배 표준편차

**수치 예시**:

| 항목 | 값 |
|------|-----|
| 20일 MA | 50,000원 |
| 20일 표준편차 ($\sigma$) | 1,500원 |
| 상단 밴드 | 50,000 + 3,000 = **53,000원** |
| 하단 밴드 | 50,000 - 3,000 = **47,000원** |
| 현재 가격 | 46,800원 → 하단 이탈 → 매수 신호 |

### 진입·청산 규칙

| 신호 | 조건 | 행동 |
|------|------|------|
| **매수 진입** | 가격 < 하단 밴드 | 다음 봉 시가에 매수 |
| **매수 익절** | 가격 ≥ 중간 밴드 (MA20) | 익절 매도 |
| **공매도 진입** | 가격 > 상단 밴드 | 다음 봉 시가에 공매도 |
| **공매도 익절** | 가격 ≤ 중간 밴드 (MA20) | 익절 환매 |
| **손절** | 밴드 이탈 방향으로 2% 추가 이동 | 손절 |

**중간 밴드(MA20) 익절을** 사용하는 이유: 평균회귀 전략의 목표는 평균(중간선)으로의 복귀이므로, 중간 밴드 도달 시 익절하는 것이 논리적으로 일관됩니다.

### Python 의사코드

```python
# 볼린저 밴드 평균회귀 전략 의사코드
def bollinger_mean_reversion(prices, period=20, std_mult=2.0):
    ma = compute_sma(prices, period)
    std = compute_rolling_std(prices, period)
    upper_band = ma + std_mult * std
    lower_band = ma - std_mult * std

    position = None
    entry_price = None

    for i in range(period, len(prices)):
        price = prices[i]
        mid = ma[i]
        lower = lower_band[i]
        upper = upper_band[i]

        # 매수 진입: 하단 밴드 이탈
        if position is None and price < lower:
            position = 'long'
            entry_price = prices[i + 1]  # 다음 봉 시가
            stop_loss = entry_price * 0.98  # 2% 손절

        # 매수 청산: 중간 밴드 복귀
        elif position == 'long':
            if price >= mid:
                execute_sell(price)  # 중간 밴드 익절
                position = None
            elif price < stop_loss:
                execute_sell(price)  # 손절
                position = None

        # 공매도 진입: 상단 밴드 이탈
        elif position is None and price > upper:
            position = 'short'
            entry_price = prices[i + 1]
            stop_loss = entry_price * 1.02

        # 공매도 청산: 중간 밴드 복귀
        elif position == 'short':
            if price <= mid:
                execute_buy_cover(price)  # 중간 밴드 익절
                position = None
            elif price > stop_loss:
                execute_buy_cover(price)  # 손절
                position = None
```

### 장단점

**장점**:
- 횡보장에서 매우 높은 승률 — 통계적으로 가격의 95%가 밴드 내 존재
- 표준 파라미터(20일, 2배 표준편차)가 널리 검증됨
- 목표가(중간 밴드)와 손절 기준이 명확
- 변동성이 커질수록 밴드가 넓어져 자동으로 조건이 엄격해짐

**단점**:
- 강한 추세장에서 치명적 — 하단 밴드 이탈 후 계속 하락하는 경우 큰 손실
- 단독 사용 시 추세장 구별 불가
- 하단 밴드 터치 후 추가 이탈이 잦은 고변동성 시장에서 연속 손절 위험
- 주식 시장의 단방향 특성상 공매도 전략 적용에 제약

**개선 방법**: ADX < 20 조건(횡보장 필터)을 추가하여 추세가 없는 구간에서만 전략을 활성화합니다. 추세 강도가 높을 때는 모멘텀 전략으로 전환하는 체제 전환(regime switching) 접근이 효과적입니다.

## 전략 6: Z-Score 통계적 평균회귀

### 원리

Z-Score는 현재 가격이 평균에서 **몇 표준편차만큼 벗어났는지를** 나타내는 표준화 지표입니다. 통계적으로 Z-Score가 ±2를 넘을 확률은 약 4.6%에 불과하므로, 이 구간에서의 평균 복귀 가능성이 높습니다. 볼린저 밴드와 수학적으로 동일한 원리이나, **절댓값이 아닌 표준화된 수치로** 표현되어 서로 다른 자산을 동일 기준으로 비교할 수 있습니다.

### 공식

$$Z = \frac{P - \mu}{\sigma}$$

- $P$: 현재 가격
- $\mu$: 기준 기간의 평균 가격 (예: 20일 평균)
- $\sigma$: 기준 기간의 표준편차
- Z > 0: 평균 위 → 과매수 방향
- Z < 0: 평균 아래 → 과매도 방향

**수치 예시** (20일 기준):

| 항목 | 값 |
|------|-----|
| 20일 평균 ($\mu$) | 50,000원 |
| 20일 표준편차 ($\sigma$) | 2,000원 |
| 현재 가격 ($P$) | 54,200원 |
| Z-Score | (54,200 - 50,000) / 2,000 = **+2.1** |

→ Z = +2.1 → 평균 위 2.1 표준편차 → 과매수 → 공매도 신호

**Z-Score 해석 기준**:

| Z-Score 범위 | 통계적 의미 | 행동 |
|-------------|------------|------|
| Z < -2.5 | 극단적 과매도 (확률 1.2%) | 적극 매수 |
| -2.5 ≤ Z < -2.0 | 강한 과매도 (확률 4.6%) | 매수 |
| -2.0 ≤ Z < -1.5 | 보통 과매도 | 관찰 |
| -1.5 ≤ Z ≤ +1.5 | 정상 범위 | 중립 |
| +1.5 < Z ≤ +2.0 | 보통 과매수 | 관찰 |
| +2.0 < Z ≤ +2.5 | 강한 과매수 | 공매도 |
| Z > +2.5 | 극단적 과매수 | 적극 공매도 |

### Python 의사코드

```python
# Z-Score 평균회귀 전략 의사코드
def zscore_mean_reversion(prices, period=20, entry_z=2.0, exit_z=0.0):
    mu = compute_rolling_mean(prices, period)
    sigma = compute_rolling_std(prices, period)

    position = None

    for i in range(period, len(prices)):
        price = prices[i]
        z = (price - mu[i]) / sigma[i]  # 현재 Z-Score 계산

        # 과매도: Z < -entry_z → 매수
        if position is None and z < -entry_z:
            position = 'long'
            stop_loss_z = -3.5  # Z-Score 기반 손절 (극단 이탈 시)

        # 매수 청산: Z가 0(평균) 복귀
        elif position == 'long':
            if z >= exit_z:
                execute_sell(price)  # 평균 복귀 익절
                position = None
            elif z < stop_loss_z:
                execute_sell(price)  # 극단 이탈 손절
                position = None

        # 과매수: Z > entry_z → 공매도
        elif position is None and z > entry_z:
            position = 'short'
            stop_loss_z = 3.5

        # 공매도 청산: Z가 0(평균) 복귀
        elif position == 'short':
            if z <= exit_z:
                execute_buy_cover(price)  # 평균 복귀 익절
                position = None
            elif z > stop_loss_z:
                execute_buy_cover(price)  # 극단 이탈 손절
                position = None
```

### 장단점

**장점**:
- 표준화된 수치로 **서로 다른 자산의 과열 정도를 동일 기준으**로 비교 가능
- 페어 트레이딩(두 자산 가격 스프레드의 Z-Score)에 직접 적용 가능
- 진입/청산 기준(Z-Score 임계값)이 수학적으로 명확
- 롤링 기간 조정만으로 단기/장기 전략 전환 용이

**단점**:
- 볼린저 밴드와 수학적으로 동일 — 근본적인 한계도 동일
- 롤링 평균이 가격 추세를 따라가므로, 강한 추세에서는 Z-Score가 지속적으로 높은 값 유지
- 기준 기간(period) 선택에 따라 결과가 크게 달라짐
- 단순 가격 Z-Score는 자기상관(autocorrelation)을 고려하지 않음

## 전략 7: RSI + 볼린저 밴드 복합 전략

### 원리

RSI와 볼린저 밴드는 각각 **모멘텀과** **가격 위치라**는 다른 차원을 측정합니다. 두 조건을 동시에 충족(AND 필터)할 때만 진입하면 **거짓 신호를 크게 줄이고 신뢰도 높은 진입 시점을** 포착할 수 있습니다.

- **볼린저 밴드 하단 이탈**: 가격이 통계적 하한선 아래 — 위치 기반 신호
- **RSI < 30**: 모멘텀이 과매도 수준 — 강도 기반 신호

두 신호가 동시에 발생하면 과매도의 신뢰도가 훨씬 높아집니다.

### 진입·청산 규칙

**매수 진입 (AND 조건 — 모두 충족)**:

| 조건 번호 | 조건 | 설명 |
|----------|------|------|
| 조건 1 | 가격 < 볼린저 밴드 하단 | 가격 위치 과매도 |
| 조건 2 | RSI(14) < 30 | 모멘텀 과매도 |

**매수 청산**:
- 익절: 가격 ≥ 볼린저 밴드 중간선(MA20)
- 손절: 진입가 대비 -3% 또는 최근 5봉 최저점 하단

**공매도 진입 (AND 조건)**:

| 조건 번호 | 조건 | 설명 |
|----------|------|------|
| 조건 1 | 가격 > 볼린저 밴드 상단 | 가격 위치 과매수 |
| 조건 2 | RSI(14) > 70 | 모멘텀 과매수 |

### Python 의사코드

```python
# RSI + 볼린저 밴드 복합 전략 의사코드
def rsi_bollinger_combined(prices, bb_period=20, bb_std=2.0,
                            rsi_period=14, rsi_oversold=30, rsi_overbought=70):
    # 지표 계산
    ma = compute_sma(prices, bb_period)
    std = compute_rolling_std(prices, bb_period)
    upper_band = ma + bb_std * std
    lower_band = ma - bb_std * std
    rsi = compute_rsi(prices, rsi_period)

    position = None

    for i in range(max(bb_period, rsi_period), len(prices)):
        price = prices[i]
        current_rsi = rsi[i]
        lower = lower_band[i]
        upper = upper_band[i]
        mid = ma[i]

        # 매수 진입: AND 조건 (볼린저 하단 이탈 + RSI 과매도)
        if position is None:
            if price < lower and current_rsi < rsi_oversold:
                position = 'long'
                entry_price = prices[i + 1]
                stop_loss = entry_price * 0.97  # 3% 손절

        # 매수 청산
        elif position == 'long':
            if price >= mid:
                execute_sell(price)  # 중간 밴드 익절
                position = None
            elif price < stop_loss:
                execute_sell(price)  # 손절
                position = None

        # 공매도 진입: AND 조건 (볼린저 상단 이탈 + RSI 과매수)
        if position is None:
            if price > upper and current_rsi > rsi_overbought:
                position = 'short'
                entry_price = prices[i + 1]
                stop_loss = entry_price * 1.03

        # 공매도 청산
        elif position == 'short':
            if price <= mid:
                execute_buy_cover(price)  # 중간 밴드 익절
                position = None
            elif price > stop_loss:
                execute_buy_cover(price)  # 손절
                position = None
```

### 신호 발생 빈도 비교

AND 필터 적용으로 신호 수는 줄지만 품질이 향상됩니다:

| 전략 | 연간 신호 수 (예시) | 승률 (예시) |
|------|-----------------|----------|
| 볼린저 단독 | 24회 | 58% |
| RSI 단독 | 30회 | 55% |
| **RSI + 볼린저 복합** | **8회** | **72%** |

→ 신호 수는 1/3로 감소하지만 승률이 크게 향상됨

### 장단점

**장점**:
- 단일 전략 대비 거짓 신호 대폭 감소 — 두 독립적 지표의 동시 확인
- 높은 승률로 심리적 안정성 향상 — 연속 손절 횟수 감소
- 각 지표의 약점을 상호 보완 (RSI는 위치 정보가 없고, 볼린저는 모멘텀 정보가 없음)

**단점**:
- **신호 희소성** — AND 조건으로 인해 진입 기회가 크게 줄어들어 자본 활용 효율 저하
- 두 조건 중 하나만 충족하는 좋은 기회를 놓칠 수 있음
- 강한 추세장에서는 복합 조건을 충족해도 손실 발생 가능
- 파라미터 조합(BB 기간, RSI 기간, 임계값)이 4개로 늘어나 과최적화 위험 증가

**개선 방법**: 신호 희소성 문제는 여러 종목에 동시 적용(유니버스 스캔)하거나, 진입 조건을 한 단계 완화(예: RSI < 35 AND 볼린저 하단 근접)하는 방식으로 완화할 수 있습니다.

## 전략 8: 켈트너 채널 평균회귀

### 원리

켈트너 채널(Keltner Channel)은 볼린저 밴드와 유사하지만, **표준편차 대신 ATR(Average True Range)** 을 사용하여 밴드 폭을 결정합니다. ATR은 가격 변동성을 갭을 포함해 측정하므로, 켈트너 채널은 볼린저 밴드보다 **급격한 갭 움직임에 더 민감합**니다.

평균회귀 전략에서는 가격이 하단 채널을 이탈한 후 **중심선(EMA)으로 복귀하**는 움직임을 포착합니다.

### 공식

$$KC_{Upper} = EMA_{20} + 2 \times ATR_{10}$$

$$KC_{Lower} = EMA_{20} - 2 \times ATR_{10}$$

- $EMA_{20}$: 20일 지수이동평균 (중심선)
- $ATR_{10}$: 10일 ATR (평균 실제 변동 범위)
- 배수 2는 기본값 — 1.5(좁음)~2.5(넓음) 사이에서 조정 가능

**ATR 계산**:

$$TR = \max(H-L, |H-C_{prev}|, |L-C_{prev}|)$$

$$ATR_{10} = \frac{1}{10}\sum_{i=1}^{10} TR_i$$

- $TR$: True Range (당일 고가-저가, 전일 종가 기준 고가, 전일 종가 기준 저가 중 최대)
- $H$: 당일 고가, $L$: 당일 저가, $C_{prev}$: 전일 종가

**수치 예시**:

| 항목 | 값 |
|------|-----|
| EMA20 | 50,000원 |
| ATR10 | 1,200원 |
| 상단 채널 | 50,000 + 2×1,200 = **52,400원** |
| 하단 채널 | 50,000 - 2×1,200 = **47,600원** |
| 현재 가격 | 47,200원 → 하단 이탈 |
| 다음 봉 시가 | 47,400원 → **매수 진입** |

### 켈트너 vs 볼린저 채널 폭 비교

| 시장 상황 | 볼린저 밴드 | 켈트너 채널 |
|----------|-----------|-----------|
| 횡보 저변동 | 좁아짐 | 비교적 안정 |
| 급등·급락 | 일시적 확대 | 완만하게 확대 |
| 갭 발생 | 덜 반응 | **즉각 반응** |
| 용도 | 변동성 스퀴즈 탐지 | 안정적 채널 트레이딩 |

→ 켈트너 채널은 볼린저 밴드보다 **더 완만하고 안정적인 채널을** 형성하여 평균회귀 신호의 일관성이 높습니다.

### Python 의사코드

```python
# 켈트너 채널 평균회귀 전략 의사코드
def keltner_mean_reversion(high, low, close, ema_period=20, atr_period=10, mult=2.0):
    ema = compute_ema(close, ema_period)

    # ATR 계산
    tr = []
    for i in range(1, len(close)):
        true_range = max(
            high[i] - low[i],
            abs(high[i] - close[i - 1]),
            abs(low[i] - close[i - 1])
        )
        tr.append(true_range)
    atr = compute_sma(tr, atr_period)

    # 켈트너 채널
    upper_kc = ema[1:] + mult * atr
    lower_kc = ema[1:] - mult * atr

    position = None
    entered_below = False  # 하단 이탈 여부 추적

    for i in range(atr_period, len(close) - 1):
        price = close[i]
        lower = lower_kc[i]
        upper = upper_kc[i]
        center = ema[i]

        # 하단 이탈 감지
        if price < lower:
            entered_below = True

        # 하단 이탈 후 채널 내 복귀 시 매수 진입
        if position is None and entered_below and price > lower:
            position = 'long'
            entry_price = close[i + 1]  # 다음 봉 시가
            stop_loss = lower * 0.995  # 하단 채널 하방 0.5%
            entered_below = False

        # 매수 청산: 중심선(EMA) 복귀
        elif position == 'long':
            if price >= center:
                execute_sell(price)  # EMA 복귀 익절
                position = None
            elif price < stop_loss:
                execute_sell(price)  # 손절
                position = None

        # 상단 이탈 후 복귀 시 공매도
        if price > upper and position is None:
            entered_above = True
        if position is None and entered_above and price < upper:
            position = 'short'
            entry_price = close[i + 1]
            stop_loss = upper * 1.005
            entered_above = False

        elif position == 'short':
            if price <= center:
                execute_buy_cover(price)
                position = None
            elif price > stop_loss:
                execute_buy_cover(price)
                position = None
```

### 하단 이탈 후 복귀 진입의 중요성

단순히 "하단 이탈 시 매수"가 아니라 **"하단 이탈 후 채널 내로 복귀 시 매수"** 하는 것이 핵심입니다:

| 진입 방식 | 특징 | 위험 |
|----------|------|------|
| 하단 이탈 즉시 매수 | 가장 낮은 가격 목표 | 추가 하락 시 큰 손실 |
| **하단 이탈 후 복귀 시 매수** | **추세 반전 확인 후 진입** | **더 높은 가격, 낮은 위험** |
| 중심선 상향 돌파 시 매수 | 가장 안전한 확인 | 수익 기회 상당 부분 소멸 |

→ "이탈 후 복귀"는 **평균회귀가 실제로 시작됐음을 확인하**는 방식으로, 위험 대비 수익 비율을 개선합니다.

### 장단점

**장점**:
- ATR 기반 밴드는 볼린저 밴드보다 안정적 — 갑작스러운 변동성 급등에 덜 민감
- "이탈 후 복귀 시 진입" 규칙으로 추세 지속 종목 손실 감소
- EMA 중심선 사용으로 볼린저 밴드(SMA)보다 최근 가격에 더 빠르게 반응
- 채널 폭이 일정한 편이라 포지션 사이징 계산이 용이

**단점**:
- ATR 기간(10일)과 EMA 기간(20일) 두 가지 파라미터 조정 필요
- "이탈 후 복귀" 진입 조건이 복잡해 순수 가격 액션 트레이더에게 직관적이지 않음
- 볼린저 밴드 대비 인지도가 낮아 참고 자료 부족
- 횡보장에서 채널이 지나치게 좁아지는 경우 거짓 신호 증가 가능
