## 일목균형표란?

<IndicatorOverlay client:visible dataset="uptrend-stock" title="일목균형표 차트" availableIndicators={['sma20', 'sma50', 'ema20']} />

일목균형표(一目均衡表, Ichimoku Kinko Hyo)는 일본의 언론인 호소다 고이치(細田悟一, 필명 '일목산인')가 1930년대부터 연구를 시작해 1969년 공개한 기술적 분석 도구입니다. 이름의 뜻은 "한눈(一目)에 균형(均衡)을 보는 표(表)"로, 하나의 차트에서 추세, 지지·저항, 모멘텀을 동시에 파악하는 것이 목표입니다.

일목균형표는 단순한 지표가 아닌완**결된 트레이딩 시스템으**로, 동양의 독자적인 시장 분석 방법론입니다 [Investopedia, 2026].

---

## 5가지 구성 요소

### 1. 전환선 (Tenkan-sen, 轉換線)

단기 추세를 나타내는 선입니다.

$$\text{전환선} = \frac{\text{9봉 고가} + \text{9봉 저가}}{2}$$

- 기간: 9봉 (기본값)
- 이동평균이 아닌중**심값(midpoint**)으로 계산
- 가격의 단기 균형점을 나타냄

### 2. 기준선 (Kijun-sen, 基準線)

중기 추세와 시장의 균형을 나타냅니다.

$$\text{기준선} = \frac{\text{26봉 고가} + \text{26봉 저가}}{2}$$

- 기간: 26봉 (기본값)
- 지지·저항선으로 활용
- 가격이 기준선 위에 있으면 강세, 아래에 있으면 약세

### 3. 선행스팬 A (Senkou Span A, 先行스팬A)

구름대(cloud)의 상단 또는 하단을 구성합니다.

$$\text{선행스팬 A} = \frac{\text{전환선} + \text{기준선}}{2}$$

(26봉 앞으로 이동하여 표시)

- 전환선과 기준선의 평균값
- **26봉 미래에 표시되**므로 미래 지지·저항 예측 가능

### 4. 선행스팬 B (Senkou Span B, 先行스팬B)

구름대의 반대쪽 경계를 구성합니다.

$$\text{선행스팬 B} = \frac{\text{52봉 고가} + \text{52봉 저가}}{2}$$

(26봉 앞으로 이동하여 표시)

- 52봉(약 2개월) 기간의 중심값
- **26봉 미래에 표시**
### 5. 후행스팬 (Chikou Span, 遅行스팬)

현재 가격을 과거에 표시하여 비교합니다.

```
후행스팬 = 현재 종가를 26봉 과거에 표시
```

- 현재 가격이 과거 가격 대비 강한지 약한지 파악
- 확인 신호로 활용

---

## 구름대 (Kumo, 雲)

선행스팬 A와 선행스팬 B 사이의 영역을구**름대(Kumo**)라 합니다.

### 구름대 해석

| 상황 | 의미 |
|------|------|
| 가격이 구름 위에 있을 때 | 강세 추세, 구름이 지지 역할 |
| 가격이 구름 아래에 있을 때 | 약세 추세, 구름이 저항 역할 |
| 가격이 구름 안에 있을 때 | 횡보 또는 추세 전환 구간 |
| 선행스팬 A가 B보다 위에 있을 때 | 상승 구름 (녹색), 강세 신호 |
| 선행스팬 A가 B보다 아래에 있을 때 | 하락 구름 (적색), 약세 신호 |

### 구름 두께의 의미

구름이두**꺼울수록지**지·저항이 강하고,**얇을수록가**격이 쉽게 돌파할 가능성이 높습니다 [StockCharts, 2026].

## 주요 매매 신호

### 1. TK 크로스 (Tenkan-Kijun Cross)

전환선(Tenkan)과 기준선(Kijun)의 교차입니다.

```
강세 신호 (황금 크로스):
- 전환선이 기준선을 아래에서 위로 교차
- 구름 위에서 발생 시 가장 강한 신호

약세 신호 (죽음 크로스):
- 전환선이 기준선을 위에서 아래로 교차
- 구름 아래에서 발생 시 가장 강한 신호
```

| 크로스 발생 위치 | 신호 강도 |
|-----------------|-----------|
| 구름 위 | 강한 강세 신호 |
| 구름 안 | 중립 신호 (주의) |
| 구름 아래 | 강한 약세 신호 |

### 2. 가격-구름 관계

```
구름 돌파 상승: 가격이 구름을 아래에서 위로 통과 → 추세 전환 강세 신호
구름 돌파 하락: 가격이 구름을 위에서 아래로 통과 → 추세 전환 약세 신호
구름 지지 확인: 하락 후 구름에서 반등 → 추세 지속 신호
```

### 3. 후행스팬 (Chikou Span) 신호

후행스팬(현재 종가를 26봉 과거에 표시)이 26봉 전 가격보다 위에 있으면 강세, 아래에 있으면 약세입니다.

이 신호는확**인 신호(confirmation signal**)로 활용합니다. TK 크로스나 구름 돌파 신호가 발생했을 때 후행스팬이 동일한 방향을 지지하면 신뢰도가 높아집니다 [BabyPips, 2026].

## 완전한 매수 신호 체크리스트

일목균형표 트레이더들은 다음 조건이 모두 충족될 때 가장 강한 매수 신호로 봅니다.

```
1. 가격이 구름 위에 있음
2. 전환선이 기준선 위에 있음 (또는 상향 교차)
3. 구름이 상승 구름 (선행스팬 A가 B보다 위)
4. 후행스팬이 26봉 전 가격보다 위에 있음
```

반대로 4가지 조건이 모두 약세를 가리키면 강한 매도 신호입니다.

## 시간 이론

호소다 고이치는시**간이 가격만큼 중요하다고** 강조했습니다. 핵심 시간 주기는 다음과 같습니다.

| 기간 | 의미 |
|------|------|
| 9봉 | 단기 기본 단위 |
| 17봉 | 9 + 8 (일본 기본 주기) |
| 26봉 | 중기 기본 단위 (한 달) |
| 33봉 | 26 + 7 |
| 42봉 | 33 + 9 |
| 52봉 | 26 × 2, 장기 기본 단위 |
| 65봉 | 52 + 13 |

이 기간들은 중요한 추세 전환이 발생할 가능성이 높은 시점을 나타냅니다.

>**참고**: 기본값(9, 26, 52)은 일본의 주 6일 근무 시절에 맞춘 값입니다. 현대의 주 5일 시장에 맞게 (7, 22, 44)로 조정하는 트레이더도 있습니다 [Investopedia, 2026].

## 실전 적용 예시

일봉 차트에서 비트코인(BTC)을 분석한다고 가정합니다.

```
상황: BTC가 장기 하락 추세 후 반등 중

체크리스트:
1. 가격이 구름을 상향 돌파? → 확인
2. 전환선이 기준선 위로 교차? → 확인 (TK 골든 크로스)
3. 미래 구름이 상승 구름(녹색)으로 전환? → 확인
4. 후행스팬이 26봉 전 가격 위? → 확인

결론: 4가지 조건 모두 충족 → 강한 상승 전환 신호
진입: 구름 돌파 확인 후 매수
손절: 기준선 아래 (또는 구름 하단 아래)
목표: 이전 주요 저항 구역
```

## 일목균형표의 강점과 한계

### 강점
- 하나의 도구로 추세, 모멘텀, 지지·저항, 신호를 동시 파악
- 미래 구름을 통한 선행적 지지·저항 예측
- 시간 개념 포함

### 한계
- 구성 요소가 많아 초보자에게 복잡하게 보임
- 횡보 시장에서 잦은 허위 신호 발생
- 기본값(9, 26, 52)이 모든 시장·시간 프레임에 최적이 아닐 수 있음

## 수치 계산 예시: 전환선·기준선 직접 계산

실제 수치로 전환선과 기준선을 계산합니다.

**시나리오:** 삼성전자 일봉 기준, 최근 9일 및 26일 데이터

최근 9일 고가/저가:

| 일 | 고가 | 저가 |
|----|------|------|
| 최고가(9일 중) | 78,500원 | — |
| 최저가(9일 중) | — | 72,300원 |

$$\text{전환선} = \frac{78{,}500 + 72{,}300}{2} = \frac{150{,}800}{2} = 75{,}400원$$

최근 26일 고가/저가:

| 구분 | 값 |
|------|-----|
| 26일 최고가 | 82,000원 |
| 26일 최저가 | 69,800원 |

$$\text{기준선} = \frac{82{,}000 + 69{,}800}{2} = \frac{151{,}800}{2} = 75{,}900원$$

$$\text{선행스팬 A} = \frac{75{,}400 + 75{,}900}{2} = 75{,}650원 \quad \text{(26봉 후 표시)}$$

**신호 해석:**
- 현재가(76,200원) > 전환선(75,400) > 기준선(75,900)? → 기준선 > 전환선이므로 아직 TK 골든크로스 미완성
- 현재가(76,200원)가 구름대(선행스팬A 75,650 부근) 위에 위치 → 약한 강세
- 전환선이 기준선을 상향 돌파하려면 9일 고가/저가 중심값이 26일 중심값을 넘어야 함

>**해석:** 가격은 구름 위이지만 TK 크로스가 아직 발생하지 않아 완전한 매수 신호 조건 미충족. 전환선이 기준선을 상향 돌파하는 시점이 진입 타이밍.

## 흔한 실수

**1. 구름 안에서 TK 크로스 신호에 진입**TK 크로스가 구름 안에서 발생하면 중립 신호입니다. 구름 위 또는 아래에서 발생한 TK 크로스와 달리 구름 내부 크로스는 오신호 비율이 높습니다. 구름 위치를 반드시 확인하십시오.

**2. 후행스팬 신호를 독립 신호로 사용후**행스팬은 확인(confirmation) 신호이지 독립 진입 신호가 아닙니다. 후행스팬만 강세를 보인다고 매수에 진입하면 안 됩니다. TK 크로스나 구름 돌파 신호가 선행해야 합니다.

**3. 구름 두께를 무시한 돌파 추격 매수얇**은 구름은 쉽게 뚫리고 가짜 돌파도 많습니다. 두꺼운 구름을 돌파할 때는 강한 모멘텀이 필요합니다. 구름 돌파 직후 즉시 추격하기보다 구름 재테스트(되돌림 후 지지 확인) 시점을 노리는 것이 유리합니다.

**4. 기본값(9, 26, 52)을 모든 시장에 동일 적용원**래 기본값은 일본의 주 6일 거래를 기준으로 설계됐습니다. 주 5일 시장에서는 (7, 22, 44) 또는 (10, 30, 60) 조정을 고려하십시오. 크립토 24시간 시장에서는 기본값의 의미가 더욱 다릅니다.

**5. 횡보 시장에서 일목균형표 과신일**목균형표는 추세 시장에서 가장 효과적입니다. 박스권 횡보에서는 TK 크로스가 빈번하게 발생하며 오신호 비율이 높아집니다. ATR이나 ADX로 추세 강도를 먼저 확인 후 적용하십시오.

## 에지 케이스: 구름 두께와 돌파 신뢰도

**구름 두께와 지지·저항 강도**
$$\text{구름 두께} = |\text{선행스팬 A} - \text{선행스팬 B}|$$

구름 두께가 현재가의 3% 미만이면 얇은 구름, 5% 이상이면 두꺼운 구름으로 분류합니다. 얇은 구름 돌파는 추가 확인이 필요하고, 두꺼운 구름 돌파는 강한 모멘텀을 의미합니다.

**구름 색상 전환 지점 (선행스팬 A/B 교차**)미래 구름에서 선행스팬 A와 B가 교차하는 지점은 26봉 후 추세 전환 가능성이 높은 시점입니다. 이 교차점 전후로 가격 움직임에 주의를 기울이십시오.
