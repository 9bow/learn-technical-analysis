## 시장 구조란?

시장 구조(Market Structure)는 가격이 고점(swing high)과 저점(swing low)을 형성하는 패턴을 분석하여 현재 추세의 방향과 강도를 파악하는 개념입니다.

전통적인 추세 분석(더 높은 고점과 저점 = 상승 추세)을 기반으로 하되,**스마트 머니 개념(Smart Money Concepts, SMC**)은 기관 투자자(스마트 머니)의 행동 패턴을 중심으로 이를 더욱 정교하게 발전시킨 현대적 접근법입니다 [ICT, 2026].

## 핵심 개념: BOS와 CHoCH

### 시장 구조 돌파 (BOS, Break of Structure)

BOS**기존 추세가 지속됨을 확인하**는 신호입니다.

```
상승 추세에서의 BOS:
이전 고점(swing high)을 상향 돌파 → 상승 BOS
의미: 상승 추세 지속 확인

하락 추세에서의 BOS:
이전 저점(swing low)을 하향 돌파 → 하락 BOS
의미: 하락 추세 지속 확인
```

### 성격 변화 (CHoCH, Change of Character)

CHoCH**추세 전환의 첫 번째 신호입**니다.

```
상승 추세에서의 CHoCH:
이전 저점(swing low)을 하향 돌파 → 하락 CHoCH
의미: 상승 추세가 약화되고 있음, 추세 전환 가능성

하락 추세에서의 CHoCH:
이전 고점(swing high)을 상향 돌파 → 상승 CHoCH
의미: 하락 추세가 약화되고 있음, 추세 전환 가능성
```

### BOS vs CHoCH 비교

| 구분 | BOS | CHoCH |
|------|-----|-------|
| 의미 | 추세 지속 | 추세 전환 첫 신호 |
| 방향 | 추세 방향으로 돌파 | 추세 반대 방향으로 돌파 |
| 확신 | 강함 | 낮음 (추가 확인 필요) |
| 활용 | 추세 추종 진입 | 추세 전환 포착 |

## 오더 블록 (Order Block)

### 개념

오더 블록(OB)은 기관 투자자가대**규모 주문을 집행한 가격 영역입**니다. 기관은 한 번에 매수·매도할 수 없기 때문에 특정 가격대에 걸쳐 주문을 분산시키며, 이 영역이 이후에 강력한 지지·저항 구역으로 작동합니다 [Investopedia, 2026].

### 상승 오더 블록 (Bullish Order Block)

강한 상승 이동 직전의마**지막 하락 캔들**(또는 2~3개의 하락 캔들군)이 상승 오더 블록입니다.

```
식별 방법:
1. 강한 상승 이동(BOS 또는 충격 이동)을 찾습니다
2. 그 이동 직전의 마지막 하락 캔들을 식별합니다
3. 그 캔들의 고가~저가 구역이 상승 오더 블록입니다

활용:
가격이 해당 구역으로 되돌아올 때 매수 기회를 탐색합니다
```

### 하락 오더 블록 (Bearish Order Block)

강한 하락 이동 직전의마**지막 상승 캔들**(또는 2~3개의 상승 캔들군)이 하락 오더 블록입니다.

```
식별 방법:
1. 강한 하락 이동(BOS 또는 충격 이동)을 찾습니다
2. 그 이동 직전의 마지막 상승 캔들을 식별합니다
3. 그 캔들의 고가~저가 구역이 하락 오더 블록입니다

활용:
가격이 해당 구역으로 반등할 때 매도 기회를 탐색합니다
```

## 유동성 개념 (Liquidity Concepts)

### 매수 측 유동성 (Buy-Side Liquidity, BSL)

이전 고점 위에는매**도 포지션의 손절 주문(stop loss**)과 매수 돌파 주문이 집중되어 있습니다. 이를 매수 측 유동성이라 합니다.

```
위치: 명확한 스윙 고점, 더블 탑, 추세선 위
의미: 이 영역으로 가격이 움직이면 유동성을 '사냥(hunt)'하는 것
결과: 급격한 가격 상승 후 반전 가능성
```

### 매도 측 유동성 (Sell-Side Liquidity, SSL)

이전 저점 아래에는매**수 포지션의 손절 주문과** 매도 돌파 주문이 집중되어 있습니다.

```
위치: 명확한 스윙 저점, 더블 바텀, 지지선 아래
의미: 가격이 이 영역으로 하락하여 손절을 터치하고 반전
결과: 급격한 가격 하락 후 반전 가능성
```

### 유동성 사냥 (Liquidity Sweep)

기관 투자자가 소매 투자자의 손절을 유발하여유**리한 가격에 대량 주문을 채우는행**위입니다 [ICT, 2026].

```
패턴:
1. 가격이 명확한 지지·저항 레벨에 접근
2. 해당 레벨을 잠시 이탈 (손절 촉발)
3. 빠르게 반전하여 원래 방향으로 이동

예시 (상승 유동성 스윕):
- 명확한 저점 아래로 가격이 잠시 하락
- 소매 투자자의 매수 손절 주문 촉발
- 기관이 해당 가격에 매수 → 가격 급반등
```

## 공정 가치 갭 (FVG, Fair Value Gap)

### 개념

FVG(또는 Imbalance)는 강한 가격 이동 **유동성이 한쪽으로 치우쳐 형성된 가격 **입니다. 캔들 3개의 구조에서 첫 번째 캔들 고가와 세 번째 캔들 저가 사이의 공간이 FVG입니다.

```
상승 FVG 식별:
캔들 1 고가: 100
캔들 2 (강한 상승 캔들): 100 → 120
캔들 3 저가: 108

FVG 구역: 캔들 1 고가(100) ~ 캔들 3 저가(108)
이 구역은 이후 지지 구역으로 작용
```

### FVG 활용

| 유형 | 형성 상황 | 이후 역할 |
|------|-----------|-----------|
| 상승 FVG | 강한 상승 이동 | 되돌림 시 지지 구역 |
| 하락 FVG | 강한 하락 이동 | 반등 시 저항 구역 |

가격은 FVG 구역으로되**돌아와 '채우는(fill)'**경향이 있으며, 이 구역에서 반전 또는 추세 재개가 자주 발생합니다.

## SMC 기반 매매 시나리오

### 상승 시나리오 예시

```
1단계: 하락 추세에서 CHoCH 발생 확인
   → 이전 고점을 상향 돌파 (추세 전환 신호)

2단계: 첫 번째 상승 BOS 발생
   → 추세 전환 확인

3단계: 되돌림 시 상승 오더 블록 또는 FVG 구역 진입
   → 매수 구역 식별

4단계: 유동성 사냥 여부 확인
   → 매도 측 유동성 스윕 후 반전 시 진입 신뢰도 증가

5단계: 다음 매수 측 유동성(이전 고점) 위를 목표로 설정
```

## 수치 계산 예시: 오더 블록 식별과 진입

실제 수치로 상승 오더 블록의 구역과 매매 계획을 설정합니다.

**시나리오:**비트코인 4시간봉에서 상승 오더 블록 식별

```
가격 흐름:
1. 하락 추세 중 스윙 저점 28,400달러 형성
2. 마지막 하락 캔들: 시가 30,200 / 종가 28,600
3. 다음 캔들부터 강한 상승 시작 → 33,500달러까지 급등 (BOS 발생)
4. 이후 되돌림 발생
```

**오더 블록 구역:**$$\text{OB 하단} = \text{마지막 하락 캔들 저가} = 28{,}400달러$$
$$\text{OB 상단} = \text{마지막 하락 캔들 고가} = 30{,}200달러$$

**FVG 식별 (급등 구간):**- 1번 캔들 고가: 29,800달러
- 3번 캔들 저가: 31,200달러
- FVG 구역: 29,800 ~ 31,200달러

**매매 계획 (되돌림 시 OB + FVG 컨플루언스 진입):**
| 항목 | 가격 |
|------|------|
| 1차 진입 구역 (FVG 하단) | 31,200달러 |
| 2차 진입 구역 (OB 상단) | 30,200달러 |
| 손절 | 28,200달러 (OB 하단 아래) |
| 1차 목표 (BSL) | 33,500달러 (이전 고점) |
| 손익비 (FVG 진입 기준) | $(33{,}500 - 31{,}200) / (31{,}200 - 28{,}200) = 2{,}300 / 3{,}000 \approx 1:0.77$ |
| 손익비 (OB 진입 기준) | $(33{,}500 - 30{,}200) / (30{,}200 - 28{,}200) = 3{,}300 / 2{,}000 = 1:1.65$ |

>**해석:**OB 상단(30,200)에서 진입 시 손익비 1:1.65로 더 유리합니다. FVG에서 먼저 소량 진입 후 OB에서 주력 물량을 추가하는 분할 매수 전략이 효과적입니다.

## 흔한 실수

**1. 모든 음봉을 상승 OB로 처리상**승 오더 블록의 조건은 강한 상승 이동(BOS 또는 충격 이동) 직전의 마지막 하락 캔들입니다. 단순한 조정 음봉이나 횡보 중의 음봉을 OB로 처리하면 오신호가 많아집니다. OB를 유발한 상승 이동의 강도를 반드시 확인하십시오.

**2. CHoCH 발생 직후 즉시 진입**CHoCH는 추세 전환의 첫 번째 신호이지 확정 신호가 아닙니다. CHoCH 이후 BOS가 발생해야 추세 전환이 확인됩니다. CHoCH 직후 추격 진입하면 추세 재개 시 손실이 발생합니다. CHoCH → BOS → OB 되돌림 순서를 기다리십시오.

**3. 유동성 스윕을 추세 이탈로 오인가**격이 스윙 저점 아래로 잠깐 하락(유동성 스윕)할 때 추세가 바뀐 것으로 오인해 패닉 매도하는 경우입니다. 유동성 스윕의 핵심은 빠른 복귀입니다. 종가 기준으로 스윙 저점 아래에서 마감하는지 확인 후 판단하십시오.

**4. 미충전 FVG가 반드시 채워질 것이라는 기대**FVG가 이후 지지·저항으로 작동하는 경향이 있지만, 모든 FVG가 채워지지는 않습니다. 강한 추세에서는 FVG를 채우지 않고 계속 진행하는 경우가 많습니다. FVG는 잠재적 되돌림 구역으로 참고하되, 다른 근거(OB, 주요 지지·저항)와 결합해 활용하십시오.

**5. 낮은 시간프레임 OB를 높은 시간프레임과 동등하게 처리**15분봉 OB와 일봉 OB는 신뢰도가 크게 다릅니다. 상위 타임프레임(일봉, 4시간봉)의 OB가 더 강한 지지·저항 역할을 합니다. 하위 타임프레임 OB를 진입 타이밍으로만 사용하고, 상위 타임프레임 OB와 방향이 일치할 때만 주력 진입하십시오.

## 에지 케이스: OB 무효화

오더 블록은 가격이 해당 구역을완**전히 통과하면무**효화됩니다. 즉, OB 상단에서 진입했는데 가격이 OB 하단을 종가 기준으로 이탈하면 손절을 실행하고 해당 OB는 더 이상 유효한 구역이 아닙니다. 무효화된 OB를 계속 참고하면 반복 손실이 발생합니다.

## 핵심 요약

- BOS(시장 구조 돌파)는 추세 지속을, CHoCH(성격 변화)는 추세 전환의 첫 신호를 나타냅니다.
- 오더 블록은 기관이 대규모 주문을 집행한 구역으로, 이후 강력한 지지·저항 역할을 합니다.
- 유동성 개념은 스윙 고점 위(BSL)와 저점 아래(SSL)에 손절 주문이 집중되어 있으며, 기관이 이를 활용함을 이해하는 것입니다.
- 공정 가치 갭(FVG)은 강한 이동 후 형성된 불균형 구역으로, 가격이 되돌아와 채우는 경향이 있습니다.
