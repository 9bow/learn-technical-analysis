## 멀티 타임프레임 분석 개요

<CandlestickDemo client:visible dataset="uptrend-stock" title="멀티 타임프레임 추세 분석 차트" />

멀티 타임프레임 분석(Multi-Timeframe Analysis, MTFA)은 동일한 자산을 두 개 이상의 타임프레임에서 동시에 분석하여 매매 방향과 진입 타이밍의 정확도를 높이는 방법론입니다.

단일 타임프레임 분석의 가장 큰 문제는맥**락의 부재입**니다. 일봉 차트에서 명확한 매수 신호처럼 보이는 패턴이 주봉에서 보면 강한 하락 추세 중의 일시적 반등에 불과할 수 있습니다. 반대로 일봉 차트의 하락 패턴이 주봉 상승 추세의 되돌림 구간일 수 있습니다.

**멀티 타임프레임 분석의 핵심 원칙:**- 상위 타임프레임이 추세의 방향(Direction)을 결정합니다
- 중간 타임프레임이 패턴과 신호(Signal)를 식별합니다
- 하위 타임프레임이 정밀한 진입(Entry) 타이밍을 제공합니다
- 세 타임프레임의 방향이 일치할 때만 진입을 고려합니다

## 1. 탑다운 접근법 (Top-Down Approach)

탑다운 접근법은 가장 큰 그림(월봉)에서 시작하여 점진적으로 낮은 타임프레임(인트라데이)으로 내려가며 분석하는 계층적 방법입니다.

### 타임프레임 4계층 구조

```
[1계층] 월봉 (Monthly)
  목적: 장기 추세 방향 파악, 주요 지지·저항 확인
  분석 내용: 수년 단위 추세, 장기 이동평균(MA12, MA24)
  해당 투자자: 장기 투자자, 포지션 트레이더

[2계층] 주봉 (Weekly)
  목적: 중기 추세 방향 확인, 진입 구간 탐색
  분석 내용: 수개월 단위 추세, 이동평균(MA13, MA26)
  해당 투자자: 스윙 트레이더, 포지션 트레이더

[3계층] 일봉 (Daily)
  목적: 구체적 패턴과 신호 식별, 진입 타이밍 좁히기
  분석 내용: 수주 단위 패턴, MA20, MA50, MA200
  해당 투자자: 스윙 트레이더, 데이 트레이더

[4계층] 인트라데이 (1시간봉 / 15분봉)
  목적: 정밀한 진입·청산 타이밍 포착
  분석 내용: 당일 가격 흐름, 지지·저항, 캔들 패턴
  해당 투자자: 데이 트레이더, 스캘퍼
```

### 탑다운 분석의 실전 흐름

아래는 삼성전자 가상 시나리오를 통한 탑다운 분석 예시입니다.

```
[월봉 분석]
- 12개월 이동평균선(MA12)이 우상향 중
- 가격이 MA12 위에 위치: 장기 상승 추세 확인
- 결론: 매수 방향만 검토

[주봉 분석]
- 주봉 MA13이 우상향, MA26 위에 위치
- 최근 3주간 되돌림(Pullback) 발생 중
- RSI(14) = 45 (과매도는 아니나 하락 조정 구간)
- 결론: 되돌림 중, 반등 구간 포착 준비

[일봉 분석]
- MA20 근처(72,000원)에서 가격 지지 확인
- 강세 장악형 캔들 패턴 출현
- 거래량이 전일 대비 185% 증가
- 결론: 진입 신호 발생, 73,500원에서 매수 검토

[1시간봉 분석]
- 72,000원 지지 확인 후 첫 번째 상승 캔들
- 72,800원 돌파 시 단기 저항 없음
- 결론: 72,800원 돌파 시 진입, 손절 71,500원
```

:::tip
탑다운 분석에서 가장 중요한 단계는월**봉 방향 확인입**니다. 월봉이 하락 추세를 보이는 종목에서 일봉 매수 신호를 찾는 것은 조류를 거슬러 수영하는 것과 같습니다. 반드시 상위 타임프레임의 방향을 먼저 확인하십시오.
:::

## 2. 타임프레임 간 추세 정렬

추세 정렬(Trend Alignment)은 분석 대상의 모든 타임프레임이 같은 방향의 추세를 보이는 상태입니다. 이 상태에서의 매매 신호는 신뢰도가 가장 높습니다.

### 추세 정렬 판단 기준

각 타임프레임별 추세를 판단하는 기준을 표준화하면 주관적 판단을 줄일 수 있습니다.

| 타임프레임 | 상승 추세 조건 | 하락 추세 조건 |
|----------|------------|------------|
| 월봉 | MA12 우상향 + 가격 &gt; MA12 | MA12 우하향 + 가격 &lt; MA12 |
| 주봉 | MA13 우상향 + 가격 &gt; MA13 | MA13 우하향 + 가격 &lt; MA13 |
| 일봉 | MA20 우상향 + 가격 &gt; MA50 | MA20 우하향 + 가격 &lt; MA50 |
| 1시간봉 | MA20 우상향 + 고점·저점 상승 | MA20 우하향 + 고점·저점 하락 |

### 추세 정렬 시나리오 4가지

```
시나리오 A: 완전 정렬 (최고 신뢰도)
월봉 상승 + 주봉 상승 + 일봉 상승 → 매수 진입
월봉 하락 + 주봉 하락 + 일봉 하락 → 매도(공매도) 진입

시나리오 B: 상위 충돌 (진입 보류)
월봉 하락 + 주봉 상승 + 일봉 상승
→ 일봉의 상승 신호가 월봉 하락에 역행
→ 진입 보류 또는 소규모 포지션만 허용

시나리오 C: 중간 충돌 (선택적 진입)
월봉 상승 + 주봉 하락 + 일봉 상승
→ 주봉 되돌림 중의 일봉 반등
→ 조심스러운 매수 가능 (단기 포지션으로 제한)

시나리오 D: 전환점 탐색
월봉 하락 + 주봉 횡보 + 일봉 상승 시도
→ 장기 추세 전환 가능성 탐색 구간
→ 확증 신호 대기 (성급한 진입 금지)
```

### 추세 정렬 점수화

```
완전 정렬 (3/3 타임프레임 일치): 100점 → 적극 진입
부분 정렬 (2/3 타임프레임 일치): 60점 → 소규모 진입
불일치 (1/3 이하 일치): 20점 → 진입 보류
```

## 3. 엘더의 삼중 스크린 시스템

알렉산더 엘더(Alexander Elder) 박사가 1986년 개발한삼**중 스크린 시스템(Triple Screen System**)은 멀티 타임프레임 분석의 가장 체계적인 프레임워크 중 하나입니다 [Investopedia, 2026].

### 삼중 스크린의 구조

엘더는 3개의 타임프레임이**5배 비율로** 구성될 때 최적이라고 제안했습니다.

```
스크린 1 (1차 필터) — 최상위 타임프레임
  역할: 추세 방향 결정 (거래 방향을 제한하는 필터)
  도구: MACD 히스토그램, MA 방향
  예: 일봉 매매라면 → 주봉을 1차 스크린으로 사용

스크린 2 (2차 필터) — 중간 타임프레임
  역할: 추세 반대 방향의 진동(oscillation) 포착
  도구: 오실레이터 (스토캐스틱, RSI)
  예: 주봉 상승 추세 중 일봉 과매도 구간 탐색

스크린 3 (진입 신호) — 최하위 타임프레임
  역할: 정확한 진입 타이밍 포착
  도구: 가격 돌파, 캔들 패턴
  예: 15분봉 또는 1시간봉 돌파 진입
```

### 삼중 스크린 실전 적용 — 매수 시나리오

```
[스크린 1 — 주봉: 추세 방향 확인]
조건: 주봉 MACD 히스토그램이 0선 위에서 상승 중
      → 주봉 상승 추세 확인
      → 매수 방향으로만 거래하기로 결정

[스크린 2 — 일봉: 진입 구간 탐색]
조건: 일봉 스토캐스틱이 20 이하로 하락 (과매도)
      또는 일봉 RSI가 40 이하
      → 상승 추세 중의 되돌림 구간 확인
      → 매수 준비 신호

[스크린 3 — 1시간봉: 진입 타이밍 포착]
조건: 1시간봉에서 어제 고점 돌파
      또는 강세 캔들 패턴 출현 + 거래량 증가
      → 매수 실행
      손절: 일봉 과매도 저점 하방 1~2%
      목표: 주봉 상단 또는 이전 주봉 고점
```

### 삼중 스크린 — 매도(공매도) 시나리오

```
[스크린 1 — 주봉: 추세 방향 확인]
조건: 주봉 MACD 히스토그램이 0선 아래에서 하락 중
      → 주봉 하락 추세 확인
      → 매도(공매도) 방향으로만 거래하기로 결정

[스크린 2 — 일봉: 진입 구간 탐색]
조건: 일봉 스토캐스틱이 80 이상 (과매수)
      또는 일봉 RSI가 60 이상으로 반등
      → 하락 추세 중의 반등 구간 확인
      → 매도 준비 신호

[스크린 3 — 1시간봉: 진입 타이밍 포착]
조건: 1시간봉에서 직전 저점 하향 이탈
      또는 약세 캔들 패턴 출현 + 거래량 증가
      → 매도 실행
      손절: 일봉 과매수 고점 상방 1~2%
      목표: 주봉 하단 또는 이전 주봉 저점
```

:::note
엘더의 원래 시스템은 스크린 1에 MACD, 스크린 2에 스토캐스틱, 스크린 3에 추적 매수 호가(Trailing Buy Stop)를 사용했습니다. 현대적 적용에서는 도구를 바꾸어도**"상위 타임프레임 추세 방향 → 중간 타임프레임 역진동 → 하위 타임프레임 진입"**의 3단계 구조만 유지하면 동일한 원리가 작동합니다.
:::

## 4. 상위 TF 방향 + 하위 TF 진입

멀티 타임프레임 분석의 실전 핵심은방**향(Direction)과 진입(Entry)의 분리입**니다. 상위 타임프레임은 거래 방향을 결정하고, 하위 타임프레임은 최적의 진입 시점을 찾는 데 사용합니다.

### 되돌림 진입 전략 (Pullback Entry)

상위 타임프레임 상승 추세에서 하위 타임프레임 되돌림을 이용하는 전략입니다.

```
단계 1: 상위 TF 추세 확인
  - 주봉: MA13 위에서 3주 연속 양봉 마감
  - 추세 방향: 상승 → 매수 방향 설정

단계 2: 중간 TF 되돌림 식별
  - 일봉: 상승 추세 중 3~5일 하락 조정
  - 일봉 MA20 또는 이전 저점 수준까지 되돌림
  - RSI 45 이하로 과열 해소

단계 3: 하위 TF 반전 신호 포착
  - 4시간봉: 하락 모멘텀 감소 확인
  - 1시간봉에서 강세 캔들 패턴 출현
  - 거래량 증가 확인

단계 4: 진입 실행
  - 1시간봉 기준 직전 고점 상향 돌파 시 매수
  - 손절: 되돌림 저점 하방 1%
  - 목표: 주봉 직전 고점
```

### 구체적 수치 예시

```
코스피 종목 A 가상 시나리오:

주봉 분석:
- 주봉 MA13 = 85,000원, 현재가 = 92,000원
- MA13 위에 위치, 3주 연속 양봉 → 상승 추세

일봉 분석:
- 92,000원에서 87,500원까지 4일 조정 (-5.1%)
- 일봉 MA20 = 87,200원 (지지 구간)
- 일봉 RSI = 42 (과매도 근접)

1시간봉 분석:
- 87,500원에서 반등 시도
- 강세 장악형 출현, 거래량 전 시간 대비 +210%
- 88,300원 돌파 시 단기 저항 해소

실행:
- 매수 진입: 88,300원 돌파 시 88,400원
- 손절: 87,000원 (-1.6%)
- 1차 목표: 92,000원 (이전 고점, +4.1%)
- 2차 목표: 95,000원 (주봉 상단, +7.5%)
- 리스크/리워드: 1 : 2.6 (1차 목표 기준)
```

### 브레이크아웃 확인 전략 (Breakout Confirmation)

```
단계 1: 상위 TF에서 박스권 또는 패턴 확인
  - 주봉: 3개월간 85,000~95,000원 박스권
  - 상단 저항선: 95,000원

단계 2: 중간 TF에서 돌파 캔들 확인
  - 일봉: 95,000원 돌파 양봉, 거래량 폭발

단계 3: 하위 TF에서 되돌림 확인
  - 1시간봉: 돌파 후 95,000원으로 되돌림
  - 95,000원에서 지지 확인 (저항→지지 전환)

단계 4: 진입 실행
  - 95,000원 지지 확인 후 반등 캔들에서 매수
  - 손절: 94,000원
  - 목표: 박스권 높이(10,000원) + 95,000원 = 105,000원
```

## 5. 스윙·데이 트레이딩 워크플로

멀티 타임프레임 분석의 적용 방식은 거래 스타일에 따라 달라집니다.

### 스윙 트레이딩 워크플로 (보유 기간: 3~20일)

```
사용 타임프레임 조합:
  1차 스크린: 월봉 (추세 방향)
  2차 스크린: 주봉 (패턴·신호)
  3차 스크린: 일봉 (진입 타이밍)

일일 분석 루틴 (30~45분):
  [장 마감 후]
  1. 관심 종목 월봉 빠른 점검 (5분)
     → 장기 추세 변화 여부 확인
  2. 주봉 분석 (10분)
     → 이번 주 추세 방향, 패턴 식별
  3. 일봉 분석 (15분)
     → 구체적 진입 신호, 캔들 패턴 탐색
  4. 다음 날 주문 설정 (5분)
     → 진입가, 손절가, 목표가 입력
  5. 리스크 점검 (5분)
     → 포지션 크기, 총 계좌 리스크 계산

포지션 관리:
  - 진입: 일봉 신호 발생 후 다음 날 시가 또는 지정가
  - 손절: 일봉 기준 스윙 저점 하방 1~2%
  - 목표: 주봉 기준 다음 저항선
  - 부분 청산: 1차 목표 도달 시 50% 청산, 나머지 추세 추종
```

### 데이 트레이딩 워크플로 (보유 기간: 당일 청산)

```
사용 타임프레임 조합:
  1차 스크린: 일봉 (추세 방향)
  2차 스크린: 1시간봉 (패턴·신호)
  3차 스크린: 15분봉 (진입 타이밍)

장중 분석 루틴:
  [장 시작 전 (8:30~9:00)]
  1. 일봉 분석: 오늘 거래 방향 결정
  2. 어제 고점·저점 확인: 주요 레벨 설정
  3. 시가 갭 분석: 갭의 방향과 크기 확인

  [장 시작 후 (9:00~9:30)]
  4. 첫 30분 관망: 방향성 확인
  5. 1시간봉 캔들 흐름 파악

  [진입 구간 (9:30~14:30)]
  6. 15분봉 패턴 + 1시간봉 방향 일치 시 진입
  7. 진입 후 15분봉 기준 추적 손절

  [청산 (14:30 이후)]
  8. 종가 30분 전 모든 포지션 정리 원칙

포지션 관리:
  - 진입: 15분봉 직전 고점 돌파 시
  - 손절: 15분봉 기준 직전 저점 하방
  - 일일 최대 손실 한도: 계좌의 2%
```

### 타임프레임 조합 빠른 참조표

| 거래 스타일 | 1차 스크린 | 2차 스크린 | 3차 스크린 |
|----------|----------|----------|----------|
| 장기 투자 (수개월~수년) | 월봉 | 주봉 | 일봉 |
| 스윙 트레이딩 (3~20일) | 주봉 | 일봉 | 4시간봉 |
| 데이 트레이딩 (당일) | 일봉 | 1시간봉 | 15분봉 |
| 스캘핑 (수분~수십분) | 1시간봉 | 15분봉 | 1~5분봉 |

### 섹터와 지수를 1차 스크린으로 활용

개별 종목 매매 시 해당 종목이 속한 섹터 지수나 코스피·코스닥 전체 지수를 1차 스크린으로 활용하면 시장 맥락을 추가로 확보할 수 있습니다.

```
예시:

3차 스크린: 삼성전자 일봉/1시간봉 → 종목 진입 타이밍

세 방향이 일치할 때 가장 높은 신뢰도의 진입 기회
```

## 흔한 실수

### 실수 1: 상위 타임프레임 무시

```
잘못된 접근:
일봉에서 강한 매수 신호 발생
→ 주봉·월봉 확인 없이 즉시 매수
→ 주봉이 강한 하락 추세였음
→ 일봉 반등은 하락 추세의 일시적 되돌림
→ 수일 후 하락 재개로 손실

올바른 접근:
반드시 상위 타임프레임을 먼저 확인
하위 타임프레임 신호는 상위 타임프레임 방향과
일치할 때만 유효한 것으로 취급
```

### 실수 2: 타임프레임 전환 (Timeframe Hopping)

분석 결과가 마음에 들지 않을 때 원하는 신호가 나올 때까지 타임프레임을 바꾸는 행동입니다.

```
잘못된 접근:
주봉: 하락 추세 → "다른 타임프레임 보자"
일봉: 약한 상승 → "아직 확신이 안 서"
4시간봉: 상승 패턴 발견! → "여기서 매수하자"

이는 확증 편향(Confirmation Bias)의 전형적 사례
→ 원하는 신호가 나올 때까지 타임프레임을 검색하는 것
→ 분석 전에 사용할 타임프레임을 먼저 결정해야 함

올바른 접근:
분석 시작 전 타임프레임 조합을 고정
(예: "나는 스윙 트레이더이므로 주봉-일봉-4시간봉을 사용")
이후 해당 타임프레임 결과만 신뢰
```

### 실수 3: 타임프레임 충돌 시 진입

```
잘못된 접근:
월봉: 하락  주봉: 횡보  일봉: 상승
→ "일봉이 올라가고 있으니 매수!"

올바른 접근:
타임프레임 충돌 = 불확실성 증가
→ 신호의 신뢰도가 크게 낮아짐
→ 충돌 상황에서는 진입 보류
→ 상위 타임프레임 방향이 명확해질 때까지 대기
```

### 실수 4: 너무 많은 타임프레임 동시 분석

```
잘못된 접근:
월봉 + 주봉 + 일봉 + 4시간봉 + 1시간봉 + 15분봉 +
5분봉 + 1분봉을 모두 확인 후 결정

→ 정보 과부하(Information Overload)
→ 항상 충돌하는 신호가 나타남
→ 결정 장애로 진입 기회 놓침

올바른 접근:
3개 타임프레임 고정 사용 (5배 비율 권장)
스윙 트레이딩: 주봉 · 일봉 · 4시간봉
데이 트레이딩: 일봉 · 1시간봉 · 15분봉
```

:::caution
타임프레임 전환(Timeframe Hopping)은 손실의 가장 흔한 원인 중 하나입니다. 분석을 시작하기 전에 사용할 타임프레임 조합을 반드시 먼저 결정하고 일관성 있게 적용하십시오.
:::

## 에지 케이스: 타임프레임 충돌 특수 상황

**장기 추세 전환 구간월**봉 하락 + 주봉 상승 충돌이 수개월 지속되는 경우, 이는 장기 추세 전환 과도기일 수 있습니다. 이 구간에서는 포지션 크기를 평소의 50% 이하로 줄이고, 월봉이 명확한 방향을 잡을 때까지 관망하거나 단기 스윙만 허용하는 것이 안전합니다.

**크립토 24시간 시장의 타임프레임 선택크**립토는 주말·공휴일 없이 거래되므로 주봉 캔들의 의미가 주식 시장과 다릅니다. 비트코인의 경우 주봉(7일봉) 대신 3일봉을 중간 스크린으로 사용하거나, 4시간봉을 일봉 대체로 사용하는 방식이 실용적입니다. 단, 타임프레임 선택 후 일관성을 유지하십시오.

**시장 급변동 직후(뉴스 이후**)강한 매크로 이벤트 직후 단기 타임프레임(15분봉, 1시간봉)은 신호 신뢰도가 크게 낮아집니다. 이벤트 후 최소 2~3개의 일봉 캔들이 안정된 뒤 분석을 재개하고, 그 전까지는 상위 타임프레임(주봉) 방향만 참고하십시오.

## 핵심 요약

- **탑다운 접근법은** 월봉→주봉→일봉→인트라데이 순으로 분석하며, 상위 타임프레임이 거래 방향을 결정하고 하위 타임프레임이 정밀한 진입 타이밍을 제공합니다
- **추세 정렬이** 완전할 때(3개 타임프레임 방향 일치) 신뢰도가 가장 높으며, 타임프레임 충돌 시에는 진입을 보류하는 것이 원칙입니다
- **엘더의 삼중 스크린은** 1차(추세 방향) → 2차(역진동 포착) → 3차(진입 타이밍)의 3단계로 구성되며, 도구보다 구조를 유지하는 것이 핵심입니다
- **방향과 진입의 분리가** 멀티 타임프레임의 실전 핵심으로, 상위 TF로 방향을 정한 후 하위 TF의 되돌림 또는 돌파에서 진입하면 유리한 리스크/리워드를 확보할 수 있습니다
- **스윙 트레이딩은** 주봉-일봉-4시간봉,**데이 트레이딩은** 일봉-1시간봉-15분봉 조합이 표준이며 5배 비율 유지가 권장됩니다
- 가장 흔한 실수는 상위 타임프레임을 무시하거나, 원하는 신호가 나올 때까지 타임프레임을 바꾸는타**임프레임 전환행**동입니다
