## 이론의 배경

랠프 넬슨 엘리어트(Ralph Nelson Elliott, 1871-1948)는 1930년대에 75년치 주식 시장 데이터를 분석하여 가격이 반복적인 파동(wave) 패턴을 형성한다는 사실을 발견했습니다. 1938년 《파동 원리(The Wave Principle)》를 출간하며 이론을 체계화했습니다.

엘리어트는 시장이집**단 심리(crowd psychology**)의 반영이며, 낙관과 비관의 주기가 일정한 파동 구조를 만든다고 주장했습니다 [Investopedia, 2026]. 이후 로버트 프렉터(Robert Prechter)가 이 이론을 현대적으로 발전시켜 대중화했습니다.

## 기본 구조: 8파동 완전 사이클

<img src="/technical-analysis/images/diagrams/elliott-wave-structure.svg" alt="엘리어트 파동 구조: 5파동 충격파(1-2-3-4-5)와 3파동 조정파(A-B-C) 완전 사이클 다이어그램" style="width:100%;max-width:600px;margin:1rem auto;display:block;" />

엘리어트 파동의 완전한 사이클은**5파동 충격(impulse wave) + 3파동 조정(corrective wave**)으로 구성됩니다.

### 충격파 (Impulse Wave) - 파동 1~5

추세 방향으로 움직이는 5개의 파동입니다.

```
파동 1 (Wave 1): 새로운 추세의 시작, 대부분 인식하기 어려움
파동 2 (Wave 2): 파동 1의 되돌림 (조정), 파동 1 시작점 이하로 하락 불가
파동 3 (Wave 3): 가장 길고 강한 파동, 가장 높은 거래량
파동 4 (Wave 4): 두 번째 조정, 파동 1 고점과 겹치면 안 됨
파동 5 (Wave 5): 마지막 상승, 파동 3보다 짧은 경우가 많음
```

### 조정파 (Corrective Wave) - 파동 A, B, C

충격파 이후 반대 방향으로 움직이는 3개의 파동입니다.

```
파동 A: 이전 충격파 방향과 반대로 이동
파동 B: 파동 A의 부분 되돌림 (많은 트레이더가 추세 재개로 오해)
파동 C: 파동 A 방향으로 재개, 보통 파동 A 저점을 이탈
```

## 핵심 3대 규칙 (Inviolable Rules)

이 규칙들은 절대 위반될 수 없습니다. 위반 시 파동 카운팅이 잘못된 것입니다 [StockCharts, 2026].

### 규칙 1: 파동 2는 파동 1 시작점을 이탈할 수 없다

```
상승 추세 예시:
파동 1: 100 → 120 상승
파동 2: 120에서 조정
제약: 파동 2의 저점은 100 이상이어야 함
```

파동 2가 파동 1 시작점(100) 아래로 하락하면, 이는 파동 2가 아니라 새로운 하락 추세의 시작입니다.

### 규칙 2: 파동 3은 충격파 중 가장 짧을 수 없다

파동 1, 3, 5 중 파동 3이 가장 짧으면 안 됩니다. 파동 3은 보통 피보나치 1.618배 이상의 연장을 보입니다.

```
예시:
파동 1 길이: 20포인트
파동 3 길이: 최소 21포인트 이상이어야 함
파동 5 길이: 파동 1 또는 3보다 짧아도 무방
```

### 규칙 3: 파동 4는 파동 1 영역과 겹칠 수 없다

파동 4의 저점이 파동 1의 고점보다 낮아지면 안 됩니다 (상품 시장에서는 예외 존재).

```
상승 추세 예시:
파동 1: 100 → 120
파동 3: 115 → 160
파동 4: 160에서 조정
제약: 파동 4의 저점은 120(파동 1 고점) 이상이어야 함
```

## 가이드라인 (Guidelines)

규칙과 달리 가이드라인은 항상 적용되지 않지만, 높은 확률로 나타납니다.

| 가이드라인 | 내용 |
|-----------|------|
| 파동 교대(Alternation) | 파동 2와 4는 서로 다른 형태를 취함 (단순 vs 복잡) |
| 파동 3의 연장 | 충격파 중 파동 3이 가장 자주 연장됨 |
| 파동 4의 되돌림 | 보통 파동 3의 38.2% 되돌림 수준에서 지지 |
| 파동 5의 피보나치 관계 | 파동 5 ≈ 파동 1의 길이 (등폭) |
| 채널 기법 | 파동 2-4를 연결한 선과 평행한 채널이 파동 5의 목표 |

## 파동 등급 (Wave Degree)

엘리어트는 파동이 여러 시간 척도에서 동시에 나타난다고 보았습니다. 큰 파동은 작은 파동들로 구성됩니다.

| 등급 | 기간 범위 | 표기법 |
|------|-----------|--------|
| 그랜드 슈퍼사이클 (Grand Supercycle) | 수백 년 | [I], [II]... |
| 슈퍼사이클 (Supercycle) | 수십 년 | (I), (II)... |
| 사이클 (Cycle) | 1~수년 | I, II... |
| 프라이머리 (Primary) | 수개월~1년 | [1], [2]... |
| 인터미디에이트 (Intermediate) | 수주~수개월 | (1), (2)... |
| 마이너 (Minor) | 수일~수주 | 1, 2... |
| 미닛 (Minute) | 수시간~수일 | i, ii... |

이 개념을프**랙탈(fractal**)이라 하며, 어떤 파동도 하위 등급 파동들로 분해될 수 있습니다 [BabyPips, 2026].

## 조정파의 주요 유형

조정파(A-B-C)는 여러 형태로 나타납니다.

### 지그재그 (Zigzag)
가장 단순한 형태. A-B-C 각각 5-3-5 구조.

```
방향: 하락 A → 부분 반등 B → 추가 하락 C
특징: C 저점이 A 저점 아래로 하락
```

### 플랫 (Flat)
A-B-C가 3-3-5 구조. B파가 A의 시작점에 근접하게 되돌림.

```
특징: A와 B의 길이가 비슷, 옆으로 횡보하는 조정
```

### 삼각형 (Triangle)
A-B-C-D-E의 5개 파동으로 구성. 보통 파동 4나 파동 B 위치에서 나타남.

```
특징: 수렴하는 고점과 저점, 이후 강한 돌파 발생
```

## 피보나치와의 관계

엘리어트 파동과 피보나치 비율은 밀접하게 연관됩니다.

| 파동 관계 | 일반적 피보나치 비율 |
|-----------|---------------------|
| 파동 2 되돌림 | 파동 1의 50~61.8% |
| 파동 3 = 파동 1 × | 1.618배 |
| 파동 4 되돌림 | 파동 3의 23.6~38.2% |
| 파동 5 = 파동 1 × | 0.618~1.0배 |
| 파동 C = 파동 A × | 1.0~1.618배 |

## 실전 파동 카운팅 팁

1.**높은 시간 프레임에서 시작합**니다. 주봉 → 일봉 → 4시간 봉 순서로 큰 그림을 먼저 파악합니다.
2.**명확한 충격파를 먼저 찾습니다.**뚜렷한 5파동 구조를 먼저 확인하고 조정파를 분석합니다.
3.**대안 카운팅을 항상 준비합니다.**엘리어트 파동은 주관성이 높아, 하나의 카운팅만 고집하지 않습니다.
4.**3대 규칙 위반 시 즉시 카운팅을 재검토합니다.**5.**파동 3은 거래량이 가장 많아야 합니다.**파동 3 확인 시 거래량 동반 여부를 체크합니다.

>**주의**: 엘리어트 파동은해**석의 주관성이** 높습니다. 같은 차트를 보고 다른 카운팅이 나올 수 있으며, 이를 다른 기술적 도구로 보완하는 것이 중요합니다 [Investopedia, 2026].

## 수치 계산 예시: 파동 목표가 계산

실제 수치로 파동 3과 파동 5의 목표가를 계산합니다.

**시나리오:** 코스피 지수가 2,200p에서 바닥을 형성하고 파동 1이 시작된 경우

| 파동 | 시작 | 끝 | 길이 |
|------|------|-----|------|
| 파동 1 | 2,200 | 2,450 | 250p |
| 파동 2 되돌림 | 2,450 | 2,295 | 155p (62%) |
| 파동 3 목표 | 2,295 → ? | | |

**파동 2 검증:** $$\text{파동 2 되돌림 비율} = \frac{2{,}450 - 2{,}295}{2{,}450 - 2{,}200} = \frac{155}{250} = 62\%$$

62% 되돌림은 황금 되돌림(61.8%) 근방 → 규칙 1 준수 확인 (2,200 이탈 없음)

**파동 3 목표가 계산 (1.618배 연장):** $$\text{파동 3 목표} = 2{,}295 + 250 \times 1.618 = 2{,}295 + 404 = 2{,}699p$$

**파동 4 되돌림 예상 (파동 3의 38.2%):** $$\text{파동 4 저점} = 2{,}699 - (2{,}699 - 2{,}295) \times 0.382 = 2{,}699 - 154 = 2{,}545p$$

파동 4 검증: 2,545 > 파동 1 고점(2,450) → 규칙 3 준수 확인

**파동 5 목표가 (파동 1과 등폭):** $$\text{파동 5 목표} = 2{,}545 + 250 \times 1.0 = 2{,}795p$$

| 진입 포인트 | 가격 | 근거 |
|-----------|------|------|
| 파동 3 시작 | 2,295p | 파동 2 저점, 황금 되돌림 |
| 파동 5 시작 | 2,545p | 파동 4 저점, 38.2% 지지 |
| 손절 (파동 3 진입 시) | 2,190p | 파동 1 시작점 하방 |

## 흔한 실수

**1. 파동 3을 파동 1로 오인새**로운 상승의 첫 번째 강한 상승 파동을 파동 3으로 분류하지 않고 파동 1로 카운팅하는 실수입니다. 파동 1은 대부분 짧고 인식하기 어렵습니다. 거래량이 폭발적으로 증가하며 강한 상승을 보이면 파동 3일 가능성이 높습니다.

**2. 하나의 카운팅만 고집엘**리어트 파동은 해석이 주관적입니다. 같은 차트에서 동시에 여러 유효한 카운팅이 존재할 수 있습니다. 주 카운팅(Primary Count)과 대안 카운팅(Alternate Count)을 항상 병행하고, 3대 규칙이 위반될 때 즉시 대안으로 전환하십시오.

**3. 규칙 위반 카운팅 억지로 유지파**동 4가 파동 1 영역을 침범했을 때 "근소하게 겹쳤으니 허용 가능"으로 처리하는 경우입니다. 3대 규칙은 절대적입니다. 규칙이 위반되면 카운팅 자체가 틀린 것이므로 전체를 재검토해야 합니다.

**4. 조정파를 충격파로 혼동**A-B-C 조정파의 B파가 강하게 반등할 때 이를 새로운 충격파의 파동 1 또는 3으로 오인하는 실수입니다. 조정파 B는 충격파 구조(5파동)가 아닌 3파동 구조를 띱니다. 파동의 하위 구조를 확인하십시오.

**5. 소음 구간에서 파동 카운팅 시도거**래량이 극히 낮은 횡보 구간이나 갑작스러운 뉴스로 발생한 가격 급변에서 파동을 카운팅하는 실수입니다. 의미 있는 파동은 충분한 거래량을 동반합니다. 거래량 없는 소폭 움직임은 파동 카운팅 대상에서 제외하십시오.

## 에지 케이스: 확장파와 실패파

**파동 연장 (Extension**)충격파 중 하나(대부분 파동 3, 드물게 파동 1 또는 5)가 비정상적으로 길어지는 경우입니다. 파동 3이 파동 1의 2.618배 이상으로 연장되면 목표가 계산 시 2.618 확장 레벨을 적용합니다. 연장된 파동은 이후 조정폭도 커집니다.

**실패파 5 (Truncated Fifth**)파동 5의 고점이 파동 3의 고점을 넘지 못하는 경우입니다. 추세 소진의 신호로, 직후 급격한 하락이 이어지는 경우가 많습니다. 파동 5 진행 중 모멘텀 지표(RSI, MACD)에서 약세 다이버전스가 나타나면 실패파를 경계하십시오.

## 핵심 요약

- 엘리어트 파동은 시장이 5파동 충격(1-2-3-4-5) + 3파동 조정(A-B-C)의 반복적 구조로 움직인다는 이론입니다.
- 3대 절대 규칙(파동 2의 시작점 이탈 금지, 파동 3 최단 금지, 파동 4와 파동 1 영역 겹침 금지)은 파동 카운팅의 유효성을 검증합니다.
- 파동은 프랙탈 구조로, 큰 파동 안에 작은 파동이 존재하며 다양한 시간 프레임에서 동시에 나타납니다.
- 피보나치 비율과 결합하면 각 파동의 목표 구역을 정량적으로 예측할 수 있습니다.
