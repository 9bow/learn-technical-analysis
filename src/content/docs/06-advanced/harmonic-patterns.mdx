## 하모닉 패턴이란?

<PatternHighlight client:visible dataset="double-bottom" title="하모닉 패턴 차트 예시" />

하모닉 패턴(Harmonic Pattern)은 특정피**보나치 비율로** 정의된 가격 구조를 인식하여 잠재적 반전 구역을 예측하는 기술적 분석 방법입니다.

H.M. 가틀리(H.M. Gartley)가 1935년 《주식 시장의 이익(Profits in the Stock Market)》에서 처음 소개했으며, 이후 스콧 카니(Scott Carney)가 Butterfly, Bat, Crab 패턴을 추가하고 체계화했습니다 [Investopedia, 2026].

하모닉 패턴은**XABCD 구조로** 표현되며, 각 레그(leg) 사이의 피보나치 관계가 엄격하게 정의됩니다.

---

## 기본 구조: XABCD

모든 하모닉 패턴은 5개의 점(X, A, B, C, D)으로 구성된 4개의 레그를 갖습니다.

```
XA: 초기 이동 (기준 레그)
AB: XA의 되돌림
BC: AB의 되돌림
CD: BC의 확장 (패턴 완성점 D = PRZ)
```

패턴이 완성되는**D 지점****잠재 반전 구역(PRZ, Potential Reversal Zone**)입니다.

---

## 주요 하모닉 패턴

### 1. 가틀리 패턴 (Gartley 222)

가장 오래되고 널리 알려진 하모닉 패턴으로, 가틀리의 책 222페이지에 등장해 "222"라는 별명이 붙었습니다 [StockCharts, 2026].

| 레그 | 피보나치 비율 |
|------|--------------|
| AB | XA의 61.8% 되돌림 |
| BC | AB의 38.2% 또는 88.6% 되돌림 |
| CD | BC의 127.2% 또는 161.8% 확장 |
| AD | XA의 78.6% 되돌림 |

```
상승 가틀리 (Bullish Gartley):
X(저점) → A(고점) → B(되돌림 61.8%) → C(반등) → D(PRZ, XA의 78.6%)
D 지점에서 매수 기회
```

### 2. 버터플라이 패턴 (Butterfly Pattern)

가틀리와 유사하지만 D 지점이 X 너머까지 확장되는 것이 특징입니다.

| 레그 | 피보나치 비율 |
|------|--------------|
| AB | XA의 78.6% 되돌림 |
| BC | AB의 38.2% 또는 88.6% 되돌림 |
| CD | BC의 161.8% 또는 224% 확장 |
| XD | XA의 127.2% 또는 161.8% 확장 |

```
핵심 구별점:
- 가틀리: D가 X와 A 사이에 위치
- 버터플라이: D가 X를 넘어서 위치 (127.2~161.8% 확장)
```

### 3. 박쥐 패턴 (Bat Pattern)

스콧 카니가 2001년 발견한 패턴으로, 88.6%라는 독특한 비율을 사용합니다.

| 레그 | 피보나치 비율 |
|------|--------------|
| AB | XA의 38.2% 또는 50% 되돌림 |
| BC | AB의 38.2% 또는 88.6% 되돌림 |
| CD | BC의 161.8% 또는 261.8% 확장 |
| AD | XA의 88.6% 되돌림 |

```
박쥐 패턴의 핵심:
- AB 되돌림이 얕음 (38.2~50%)
- D 지점이 XA의 88.6% (매우 깊은 되돌림)
- 88.6%는 √0.786 (78.6%의 제곱근) 에서 파생
```

### 4. 크랩 패턴 (Crab Pattern)

스콧 카니가 2000년 발견한 패턴으로, 가장 정밀한 PRZ를 가지며 D 지점이 가장 멀리 확장됩니다.

| 레그 | 피보나치 비율 |
|------|--------------|
| AB | XA의 38.2% 또는 61.8% 되돌림 |
| BC | AB의 38.2% 또는 88.6% 되돌림 |
| CD | BC의 224% 또는 361.8% 확장 |
| XD | XA의 161.8% 확장 |

```
크랩 패턴의 핵심:
- D 지점이 XA의 161.8% 확장 (모든 패턴 중 가장 극단적)
- 높은 피보나치 정밀도 요구
- 반전 시 매우 강한 움직임 동반
```

## 패턴별 핵심 비율 비교

| 패턴 | AB / XA | AD / XA (또는 확장) |
|------|---------|---------------------|
| 가틀리 | 61.8% | 78.6% 되돌림 |
| 버터플라이 | 78.6% | 127.2~161.8% 확장 |
| 박쥐 | 38.2~50% | 88.6% 되돌림 |
| 크랩 | 38.2~61.8% | 161.8% 확장 |

---

## 잠재 반전 구역 (PRZ, Potential Reversal Zone)

PRZ는 여러 피보나치 레벨이밀**집된 구역으**로, 패턴 완성 시 가격이 반전될 가능성이 높은 영역입니다.

### PRZ 구성

PRZ는 일반적으로 다음 요소들로 이루어집니다.

```
1. CD 레그의 피보나치 확장 레벨
2. XA 레그에 대한 AD의 피보나치 되돌림 비율
3. BC 레그의 피보나치 확장
```

세 요소가 좁은 구역에 밀집될수록 PRZ의 신뢰도가 높아집니다 [HarmonicTrader.com, 2026].

### PRZ 예시 (상승 박쥐 패턴)

```
XA 이동: 100 → 150 (50포인트 상승)
B 지점: 150 → 130 (XA의 40% 되돌림) = 130
C 지점: 130 → 142 (AB의 61.8% 반등) = 142
D 지점 PRZ 계산:
  - XA의 88.6% 되돌림: 150 - (50 × 0.886) = 105.7
  - BC 확장 161.8%: 142 - (12 × 1.618) = 122.6 → 재계산 필요
PRZ 구역: 약 103 ~ 108 사이
```

## 매매 규칙

### 진입

- PRZ 구역에 도달했을 **즉시 진입하지 않습니다.**- 반전 확인 신호(confirmation)를 기다립니다.
  - 캔들 패턴: 핀 바(pin bar), 도지(doji), 엔걸핑(engulfing)
  - RSI 다이버전스(divergence)
  - 거래량 급증 후 감소

### 손절 (Stop Loss)

상승 패턴(bullish) 기준:
```
손절 위치: X 지점 아래 (패턴 무효화 기준)
```

D 지점을 크게 이탈하면 패턴이 실패한 것으로 간주합니다.

### 목표가 (Take Profit)

| 1차 목표 | 2차 목표 | 3차 목표 |
|----------|----------|----------|
| C 지점 | A 지점 | 피보나치 확장 |

일반적으로 CD 이동의 61.8%를 1차 목표로, A 지점을 2차 목표로 설정합니다.

## 하모닉 패턴 식별 시 주의사항

1.**비율 허용 오차**: 각 레그의 피보나치 비율은 ±2~3% 오차 범위 내에서 허용합니다.
2.**패턴 실패**: 하모닉 패턴도 실패합니다. 손절을 반드시 설정해야 합니다.
3.**시간 프레임**: 일봉 이상의 높은 시간 프레임에서 더 신뢰할 수 있는 신호가 나타납니다.
4.**컨플루언스**: 수평 지지·저항, 이동평균선, 구름대 등과 PRZ가 겹치는 구역의 신뢰도가 높습니다.
5.**자동화 도구**: TradingView 등의 자동 하모닉 패턴 인식 도구를 활용하되, 비율 검증은 직접 수행합니다.

## 수치 계산 예시: 상승 가틀리 PRZ 계산

실제 수치로 가틀리 패턴의 PRZ를 계산합니다.

**시나리오:**비트코인 일봉 차트에서 상승 가틀리 패턴 형성

| 점 | 가격 | 레그 |
|----|------|------|
| X | 25,000달러 | — |
| A | 32,000달러 | XA: 7,000달러 상승 |
| B | ? | AB = XA × 61.8% 되돌림 |
| C | ? | BC = AB × 38.2~88.6% |
| D (PRZ) | ? | AD = XA × 78.6% 되돌림 |

**B 지점 계산:**$$B = A - XA \times 0.618 = 32{,}000 - 7{,}000 \times 0.618 = 32{,}000 - 4{,}326 = 27{,}674달러$$

**C 지점 계산 (AB의 61.8% 반등):**$$AB = A - B = 32{,}000 - 27{,}674 = 4{,}326달러$$
$$C = B + AB \times 0.618 = 27{,}674 + 2{,}673 = 30{,}347달러$$

**D 지점 (PRZ) 계산:**$$D_1 = A - XA \times 0.786 = 32{,}000 - 7{,}000 \times 0.786 = 32{,}000 - 5{,}502 = 26{,}498달러$$

BC 확장 161.8% 확인:
$$BC = C - B = 30{,}347 - 27{,}674 = 2{,}673달러$$
$$D_2 = C - BC \times 1.618 = 30{,}347 - 4{,}325 = 26{,}022달러$$

$$\text{PRZ 구역} = 26{,}022 \sim 26{,}498달러$$

**매매 계획:**
| 항목 | 가격 |
|------|------|
| PRZ 진입 | 26,200달러 (PRZ 중간) |
| 손절 | 24,800달러 (X 아래) |
| 1차 목표 | 30,347달러 (C 지점) |
| 2차 목표 | 32,000달러 (A 지점) |
| 손익비 (1차) | 1:3.0 |

>**주의:**D 지점 도달 직후 즉시 진입하지 않습니다. PRZ 구역(26,022~26,498)에서 핀바, 장악형 등 반전 캔들 확인 후 진입합니다.

## 흔한 실수

**1. 비율 허용 오차를 과도하게 적용**"61.8% 근방"이라는 이유로 55%나 70% 되돌림을 가틀리 AB로 처리하는 실수입니다. 허용 오차는 ±2~3%가 기준이며, 그 이상 벗어나면 다른 패턴이거나 패턴 자체가 아닙니다. 비율 검증은 타협 없이 엄격하게 적용하십시오.

**2. PRZ 도달 즉시 시장가 진입**PRZ는 반전 구역이지 확정 반전 지점이 아닙니다. 가격이 PRZ를 통과해 계속 진행하는 경우가 자주 발생합니다. 반드시 반전 확인 신호(캔들 패턴, RSI 다이버전스, 거래량 변화)를 기다린 후 진입하십시오.

**3. X 지점 이상으로 손절 설정손**절을 PRZ 조금 아래, 또는 D 지점 하방 2% 수준으로 설정하는 실수입니다. 가격이 X를 이탈하면 패턴 자체가 무효화됩니다. 손절은 X 지점 아래에 설정하고, X와 PRZ 간 거리가 너무 크면 해당 패턴은 리스크 대비 리워드가 낮은 셋업으로 제외하십시오.

**4. 자동 인식 도구를 맹신**TradingView 등의 하모닉 패턴 자동 인식 지표는 비율 검증을 약간 느슨하게 적용하는 경우가 많습니다. 자동 인식 결과는 참고용으로만 사용하고, 핵심 비율(특히 AD/XA)을 직접 검증하는 습관을 유지하십시오.

**5. 낮은 시간프레임 패턴 과신**15분봉, 1시간봉의 하모닉 패턴은 노이즈가 많아 신뢰도가 낮습니다. 일봉 이상의 시간프레임에서 형성된 패턴을 우선하고, 단기 차트의 패턴은 보조 확인용으로만 활용하십시오.

## 에지 케이스: PRZ 실패 대응

**PRZ 초과 돌파 (Overshoot**)가격이 PRZ를 잠깐 초과한 뒤 반전하는 경우가 있습니다. X 지점까지 아직 여유가 있다면 손절이 유지되고 패턴이 여전히 유효할 수 있습니다. PRZ를 정확한 선이 아닌 구역으로 이해하고 X 지점 기준으로 손절을 설정한 이유가 여기 있습니다.

**크랩 패턴의 극단적 PR**Z크랩 패턴의 D 지점은 XA의 161.8% 확장으로 X를 훨씬 초과합니다. 이 경우 손절 설정이 모호해집니다. 크랩 패턴은 D 지점 반전 후 최소 A 지점까지 강한 반등이 동반될 때만 유효하게 처리하십시오.

## 핵심 요약

- 하모닉 패턴은 XABCD 구조와 엄격한 피보나치 비율로 정의되며, D 지점의 PRZ에서 가격 반전을 예측합니다.
- Gartley(AD=78.6%), Butterfly(XD=127~161.8%), Bat(AD=88.6%), Crab(XD=161.8%)는 각각 고유한 비율로 구별됩니다.
- PRZ에서 즉시 진입하는 것이 아니라 반전 확인 신호(캔들 패턴, RSI 다이버전스)를 기다린 후 진입합니다.
- 손절은 반드시 X 지점 아래에 설정하며, 목표는 단계적으로 C 지점, A 지점으로 설정합니다.
