## 차트의 역할

차트는 가격 데이터를 시각적으로 표현하여 추세, 패턴, 지지/저항 수준을 파악할 수 있게 해줍니다. 기술적 분석의 가장 기본적인 도구이며, 차트 유형에 따라 보여주는 정보의 범위와 깊이가 다릅니다.

## 1. 캔들스틱 차트 (Candlestick Chart)

가장 널리 사용되는 차트 유형으로, 각 봉(캔들)이 특정 기간의시**가(Open**),**고가(High**),**저가(Low**),**종가(Close**)4가지 가격 정보를 모두 보여줍니다
([Investopedia](https://www.investopedia.com/trading/candlestick-charting-what-is-it/)).

### 캔들의 구조

```
    │ ← 윗꼬리 (Upper Shadow/Wick)
    │
  ┌─┴─┐
  │   │ ← 몸통 (Real Body)
  │   │    양봉: 종가 > 시가 (보통 초록/흰색)
  │   │    음봉: 종가 < 시가 (보통 빨간/검정)
  └─┬─┘
    │
    │ ← 아래꼬리 (Lower Shadow/Wick)
```

- **몸통(Body**): 시가와 종가 사이의 범위. 양봉은 종가가 시가보다 높고, 음봉은 반대입니다.
- **꼬리(Shadow/Wick**): 해당 기간의 고가와 저가까지의 범위. 매수/매도 세력 간의 힘겨루기를 보여줍니다.
- **윗꼬리가 긴 경우**: 매도세가 가격을 밀어내렸음을 의미
- **아래꼬리가 긴 경우**: 매수세가 가격을 지탱했음을 의미

### 장점
- 가장 많은 가격 정보(OHLC)를 직관적으로 전달
- 캔들스틱 패턴을 통한 추가 분석 가능
- 매수/매도 심리를 시각적으로 파악

### 단점
- 초보자에게 처음에는 복잡해 보일 수 있음
- 많은 데이터 포인트를 한 화면에 표시할 때 가독성 저하

## 2. 라인 차트 (Line Chart)

각 기간의종**가(Close**)만을 연결한 가장 단순한 차트입니다
([BabyPips](https://www.babypips.com/learn/forex/types-of-charts)).

### 특징
- 종가만 표시하므로 추세의 전체적인 방향을 깔끔하게 보여줌
- 노이즈가 적어 장기 추세 파악에 유리
- 시가, 고가, 저가 정보는 확인 불가

### 활용
- 장기 추세 분석 시 개요 파악
- 여러 자산의 가격 비교
- 지지/저항선을 빠르게 식별

## 3. 바 차트 (Bar Chart / OHLC Chart)

캔들스틱과 동일한 OHLC 정보를 표시하지만, 형태가 다릅니다
([StockCharts](https://school.stockcharts.com/doku.php?id=chart_analysis:what_are_charts)).

### 바의 구조

```
    │  ← 고가 (High)
    │
  ──┤  ← 시가 (Open, 왼쪽 돌출)
    │
    ├──  ← 종가 (Close, 오른쪽 돌출)
    │
    │  ← 저가 (Low)
```

- 수직선: 해당 기간의 가격 범위(고가~저가)
- 왼쪽 수평선: 시가
- 오른쪽 수평선: 종가

### 캔들스틱과의 비교
- 동일한 정보를 제공하지만, 캔들스틱이 시각적으로 더 직관적
- 바 차트는 주로 서양 트레이더들이 전통적으로 사용
- 좁은 공간에 더 많은 바를 표시할 수 있어 장기 데이터 확인에 유리

## 4. 포인트 앤 피겨 차트 (Point & Figure Chart)

시간 축을 사용하지 않고가**격 변동의 방향만**을 기록하는 독특한 차트입니다.

### 특징
- ****X: 가격 상승 (설정된 박스 크기만큼)
- ****O: 가격 하락 (설정된 박스 크기만큼)
- 시간이 아닌 가격 변동에만 집중
- 반전 기준(Reversal): 보통 3박스 반전 사용

### 장점
- 시간의 노이즈를 제거하여 순수한 가격 움직임에 집중
- 명확한 지지/저항 수준 식별
- 가격 목표 산출에 유용

### 단점
- 거래량 정보 없음
- 실시간 분석에 적합하지 않음
- 현대 트레이딩에서 사용 빈도 낮음

## 차트 유형 비교표

| 차트 유형 | 표시 정보 | 시간 축 | 장점 | 적합한 용도 |
|-----------|----------|---------|------|-------------|
| 캔들스틱 | OHLC | 있음 | 가장 풍부한 정보, 패턴 분석 | 일반적 기술적 분석 |
| 라인 | 종가 | 있음 | 깔끔한 추세 파악 | 장기 추세, 비교 분석 |
| 바 (OHLC) | OHLC | 있음 | 공간 효율적 | 장기 데이터 분석 |
| 포인트앤피겨 | 가격 방향 | 없음 | 노이즈 제거 | 지지/저항, 목표가 |

## 권장 사항

대부분의 기술적 분석 학습에서는캔**들스틱 차트를** 기본으로 사용하는 것을 권장합니다:

1. 가장 많은 정보를 직관적으로 제공
2. 캔들스틱 패턴이라는 추가 분석 도구 활용 가능
3. 대부분의 트레이딩 플랫폼에서 기본 차트로 제공
4. 이 학습 사이트의 모든 예제도 캔들스틱 차트를 기준으로 설명

## 실전 적용

### 상황별 차트 유형 선택 기준

차트 유형을 선택할 때는 분석 목적과 타임프레임을 먼저 정해야 합니다.

**예시 1 — 단기 스윙 트레이딩 (1일~5일 보유**)
삼성전자 일봉 차트를 분석한다고 가정합니다. 주가가 70,000원에서 80,000원 사이를 횡보하는 구간입니다.

- **캔들스틱 선택 이유**: 각 캔들의 몸통과 꼬리를 보면 당일 매수/매도 세력 중 누가 이겼는지 즉시 파악 가능합니다. 예를 들어 장대 윗꼬리 캔들(위꼬리가 몸통의 2배 이상)이 저항선 75,000원 근처에서 나타나면 매도 압력이 강하다는 구체적인 신호입니다.
- 라인 차트로는 이 정보를 놓칩니다.

**예시 2 — 장기 추세 비교 (1년 이상**)
코스피, S&P500, 나스닥 3개 지수를 동시에 5년치로 비교할 때:

- **라인 차트 선택 이유**: 캔들스틱으로 3개 지수를 같은 차트에 겹치면 시각적으로 너무 복잡합니다. 라인 차트는 각 지수의 종가 흐름만 선으로 보여주므로 상관관계와 상대 성과를 즉시 비교할 수 있습니다.
- 캔들스틱으로는 이 비교가 불가능합니다.

**예시 3 — 20년치 월봉 데이터 분석**
한 화면에 240개월(20년) 데이터를 표시할 때:

- **바 차트 선택 이유**: 캔들스틱은 몸통의 색(양봉/음봉)을 구분하려면 일정 너비가 필요합니다. 바 차트는 더 좁은 공간에 표시되므로 같은 화면에 더 많은 기간을 담을 수 있습니다.
- 장기 고점/저점 구조 파악에 유리합니다.

**예시 4 — 지지/저항 목표가 계산**
비트코인의 다음 저항 목표가를 계산할 때:

- **포인트 앤 피겨 선택 이유**: 박스 크기 1,000달러, 3박스 반전 설정으로 만든 PnF 차트에서 X 컬럼의 개수를 세어 목표가를 수직으로 계산합니다. 예를 들어 X가 15칸이면 목표가 = 현재가 + (15 x 1,000달러) = 15,000달러 상승 목표. 시간 노이즈 없이 순수한 가격 움직임만 반영합니다.

:::tip[차트 유형 선택 요약]
- 일반적인 기술적 분석 → 캔들스틱
- 여러 자산 비교 또는 장기 추세 개요 → 라인
- 매우 많은 기간(수년치 일봉 등)을 한 화면에 → 바 차트
- 목표가 산출이나 순수 가격 방향 분석 → 포인트 앤 피겨
:::

## 흔한 실수

### 실수 1: 모든 상황에 캔들스틱만 고집한다

캔들스틱이 기본값이긴 하지만, 10년치 주간 데이터를 캔들스틱으로 보면 캔들 너비가 너무 좁아져 개별 패턴을 전혀 읽을 수 없습니다. 장기 분석에서는 라인이나 바 차트로 전환하는 것이 더 효과적입니다.

### 실수 2: 차트 유형이 달라도 정보가 같다고 착각한다

라인 차트와 캔들스틱 차트가 "같은 데이터를 다르게 보여준다"는 것은 반만 맞습니다. 라인 차트는 종가만 사용하므로 장중 변동성(고가-저가 범위)이 보이지 않습니다. 예를 들어 어떤 날 주가가 시가 100,000원에서 장중 85,000원까지 급락했다가 종가 99,000원으로 마감했다면, 라인 차트에서는 거의 변화가 없는 날처럼 보입니다. 캔들스틱에서는 긴 아래꼬리로 이 급락과 회복이 명확히 표시됩니다.

### 실수 3: 포인트 앤 피겨 차트의 시간 부재를 간과한다

포인트 앤 피겨 차트에는 시간 축이 없습니다. 한 컬럼이 하루일 수도, 한 달일 수도 있습니다. 따라서 "언제 매수해야 하나"보다 "어느 가격에서 매수해야 하나"를 판단하는 데 적합하며, 타이밍 분석에는 반드시 시간 축이 있는 차트와 병행해야 합니다.

### 실수 4: 차트를 바꿀 때 설정(스케일)을 확인하지 않는다

같은 데이터라도 산술 스케일(Linear)과 로그 스케일(Logarithmic)에 따라 차트 모양이 크게 달라집니다. 주가가 1,000원에서 10,000원으로 오른 것과 10,000원에서 100,000원으로 오른 것은 둘 다 10배 상승이지만, 산술 스케일에서는 후자의 변동이 훨씬 크게 보입니다. 장기 차트 분석에서는 로그 스케일을 사용하는 것이 더 정확합니다.

:::caution[혼합 사용 주의]
같은 분석 작업 내에서 차트 유형을 자주 바꾸면 착각이 생길 수 있습니다. 분석을 시작하기 전에 차트 유형을 정하고, 같은 유형으로 일관되게 분석을 완료하세요.
:::

## 엣지 케이스: 특수 시장 상황에서의 차트 해석

### 갭(Gap)이 발생한 경우

갭은 전일 종가와 당일 시가 사이에 가격 공백이 생기는 현상입니다. 차트 유형별로 갭이 다르게 표시됩니다:

- **캔들스틱**: 갭이 캔들 사이의 빈 공간으로 명확히 보임. 갭의 크기(전일 종가 vs 당일 시가)를 정확히 파악 가능
- **라인 차트**: 종가만 연결하므로, 갭 발생 여부와 크기를 쉽게 확인 가능하지만 장중 가격 정보는 없음
- **바 차트**: 캔들스틱과 동일하게 갭 확인 가능

가상자산 시장은 24시간 거래되므로 주식과 같은 갭이 거의 발생하지 않습니다. 단, 거래소 서버 오류나 점검 후 재개 시 인위적 갭이 생길 수 있습니다.

### 극단적 변동성 구간

하루 ±10% 이상 변동하는 극단적 구간에서는:

- 캔들의 몸통과 꼬리가 극단적으로 길어져 인접 캔들이 상대적으로 작아 보임
- 이런 구간 이후에는 차트 스케일이 바뀌어 이전 패턴이 납작하게 보임
- **로그 스케일 전환을 검토**: 특히 장기 차트에서 가격이 10배 이상 변한 구간은 로그 스케일이 더 실용적

### 유동성이 매우 낮은 자산

거래가 거의 없는 종목이나 시간대에서는:

- 캔들 사이에 긴 수평선이 나타남 (가격 변화 없이 시간만 경과)
- 소량의 거래로도 가격이 크게 움직여 차트가 왜곡될 수 있음
- 이런 차트에서 나타나는 패턴은 신뢰도가 낮음

## 핵심 요약

1. 캔들스틱 차트는 OHLC 4가지 가격을 직관적으로 표현하며 가장 널리 사용됩니다.
2. 라인 차트는 종가만 연결하여 전체 추세를 깔끔하게 보여줍니다.
3. 바 차트는 캔들스틱과 동일한 정보를 다른 형태로 표시합니다.
4. 차트 유형은 분석 목적과 상황에 따라 선택합니다.
