## 모멘텀 지표 (Momentum Indicators) 개요

모멘텀 지표는 가격 변화의 속도와 강도를 측정합니다. 추세의 강도가 약해지고 있는지, 과매수 또는 과매도 상태인지를 판단하는 데 사용됩니다.

**모멘텀 지표의 특성:**- 일정 범위 내에서 진동하는 오실레이터(Oscillator) 형태
- 추세 지표보다 빠른 신호를 제공하지만 오신호도 많음
- 횡보 시장에서 특히 유용하며 강한 추세 시장에서는 주의 필요
- 다이버전스(Divergence) 분석을 통해 추세 반전을 조기 포착

## 1. RSI (Relative Strength Index, 상대강도지수)

RSI는 J. Welles Wilder Jr.가 1978년에 개발한 지표로, 일정 기간의 상승 폭과 하락 폭을 비교하여 현재 가격의 상대적 강도를 0~100 범위로 나타냅니다. [Investopedia](https://www.investopedia.com/terms/r/rsi.asp)에 따르면 RSI는 가장 널리 사용되는 모멘텀 지표 중 하나입니다.

### RSI 공식

$$RSI = 100 - \frac{100}{1 + RS}$$

$$RS = \frac{\text{평균 상승폭}}{\text{평균 하락폭}}$$

- 평균 상승폭: N일간 상승한 날의 평균 상승폭
- 평균 하락폭: N일간 하락한 날의 평균 하락폭 (절댓값)

기본 기간:**14**
**단계별 계산 예시 (5일 단순화):**
| 날짜 | 종가 | 변화 | 상승 | 하락 |
|------|------|------|------|------|
| 1일차 | 100,000 | - | - | - |
| 2일차 | 102,000 | +2,000 | 2,000 | 0 |
| 3일차 | 101,000 | -1,000 | 0 | 1,000 |
| 4일차 | 103,000 | +2,000 | 2,000 | 0 |
| 5일차 | 102,500 | -500 | 0 | 500 |

$$\text{평균 상승폭} = \frac{2000 + 0 + 2000 + 0}{4} = 1000$$

$$\text{평균 하락폭} = \frac{0 + 1000 + 0 + 500}{4} = 375$$

$$RS = \frac{1000}{375} = 2.667$$

$$RSI = 100 - \frac{100}{1 + 2.667} = 100 - 27.3 = 72.7$$

> 예시: 주식 가격이 \$50,000에서 \$51,000으로 상승한 경우, 달러 기호는 수식과 구분됩니다.

### 과매수/과매도 해석

[StockCharts](https://school.stockcharts.com/doku.php?id=technical_indicators:relative_strength_index_rsi)는 RSI의 기준선을 다음과 같이 정의합니다.

| RSI 구간 | 해석 | 대응 전략 |
|---------|------|---------|
| 70 초과 | 과매수 (Overbought) | 매도 또는 매수 자제 |
| 50~70 | 강세 구간 | 추세 추종 매수 |
| 50 기준선 | 중립 | 추세 전환 관찰 |
| 30~50 | 약세 구간 | 추세 추종 매도 |
| 30 미만 | 과매도 (Oversold) | 매수 또는 매도 자제 |

**주의:**강한 상승 추세에서 RSI가 70 초과를 유지하는 경우가 많으므로 단순히 70 초과라고 매도하면 안 됩니다.

### RSI 다이버전스 (Divergence)

다이버전스는 가격과 RSI가 서로 반대 방향을 가리킬 때 발생하며, 추세 반전의 강력한 신호입니다.

**강세 다이버전스 (Bullish Divergence):**
- 가격: $\text{저점}_1 > \text{저점}_2$ (더 낮은 저점)
- RSI: $\text{저점}_1 < \text{저점}_2$ (더 높은 저점)
- → 하락 모멘텀 약화, 상승 반전 가능성

**약세 다이버전스 (Bearish Divergence):**
- 가격: $\text{고점}_1 < \text{고점}_2$ (더 높은 고점)
- RSI: $\text{고점}_1 > \text{고점}_2$ (더 낮은 고점)
- → 상승 모멘텀 약화, 하락 반전 가능성

## 2. 스토캐스틱 오실레이터 (Stochastic Oscillator)

스토캐스틱은 George Lane이 개발한 지표로, 현재 종가가 일정 기간의 가격 범위 내에서 어느 위치에 있는지를 측정합니다. [BabyPips](https://www.babypips.com/learn/forex/stochastic)에 따르면 "가격이 상승할 때 종가는 범위의 상단에, 하락할 때는 하단에 위치하는 경향"을 이용합니다.

### 공식

$$\%K = \frac{\text{현재 종가} - \text{N일 최저가}}{\text{N일 최고가} - \text{N일 최저가}} \times 100$$

$$\%D = \%K\text{의 M일 이동평균 (신호선)}$$

기본 설정:**%K = 14일, %D = 3일 SM**A
**예시 계산:**
| 항목 | 값 |
|------|-----|
| 현재 종가 | 108,000원 |
| 14일 최저가 | 100,000원 |
| 14일 최고가 | 115,000원 |

$$\%K = \frac{108000 - 100000}{115000 - 100000} \times 100 = \frac{8000}{15000} \times 100 = 53.3$$

### Fast vs Slow 스토캐스틱

| 구분 | 설명 | 특성 |
|------|------|------|
| Fast Stochastic | 원래 %K와 %D | 민감, 오신호 많음 |
| Slow Stochastic | Fast %D를 새 %K로, 이를 다시 평활화 | 완만, 신뢰도 높음 |
| Full Stochastic | 모든 매개변수 사용자 정의 가능 | 유연성 최고 |

### 신호 해석

| 신호 유형 | 조건 | 의미 |
|---------|------|------|
| 과매수 | %K 또는 %D가 80 초과 | 매도 고려 |
| 과매도 | %K 또는 %D가 20 미만 | 매수 고려 |
| 상향 교차 | %K가 %D를 아래에서 위로 교차 | 매수 신호 |
| 하향 교차 | %K가 %D를 위에서 아래로 교차 | 매도 신호 |
| 불리시 다이버전스 | 가격 저점 하락, 스토캐스틱 저점 상승 | 반등 가능성 |

## 3. CCI (Commodity Channel Index, 상품채널지수)

CCI는 Donald Lambert가 1980년에 개발한 지표로, 현재 가격이 통계적 평균에서 얼마나 떨어져 있는지를 측정합니다.

### 공식

$$CCI = \frac{\text{Typical Price} - SMA}{0.015 \times \text{Mean Deviation}}$$

$$\text{Typical Price} = \frac{\text{고가} + \text{저가} + \text{종가}}{3}$$

$$\text{Mean Deviation} = \frac{\sum |\text{Typical Price} - SMA|}{N}$$

- $0.015$: 약 70~80%의 값이 $-100 \sim +100$ 범위에 들어오도록 하는 상수

기본 기간:**20**
### CCI 해석

| CCI 구간 | 해석 |
|---------|------|
| 100 초과 | 과매수, 새로운 강한 상승 추세 시작 가능 |
| 0~100 | 강세 구간 |
| 0 기준선 | 중립 |
| -100~0 | 약세 구간 |
| -100 미만 | 과매도, 새로운 강한 하락 추세 시작 가능 |

**CCI의 독특한 해석:**- RSI/스토캐스틱과 달리, CCI에서 100 초과는 단순 과매수가 아닌강**한 추세의 시작으**로 해석하기도 합니다
- -100 미만도 강한 하락 추세의 시작으로 볼 수 있습니다

## 4. Williams %R

Williams %R은 Larry Williams가 개발한 지표로, 스토캐스틱과 유사하지만 스케일이 반전되어 있습니다. 0에서 -100 사이에서 움직입니다.

### 공식

$$\%R = \frac{\text{N일 최고가} - \text{현재 종가}}{\text{N일 최고가} - \text{N일 최저가}} \times (-100)$$

기본 기간:**14**
### Williams %R 해석

| %R 구간 | 해석 |
|--------|------|
| -20 초과 (0에 가까울수록) | 과매수 |
| -20 ~ -80 | 중립 구간 |
| -80 미만 (-100에 가까울수록) | 과매도 |

**스토캐스틱 %K와의 관계:**
$$\%R = \%K - 100$$

따라서 $\%R = -30$은 스토캐스틱 $\%K = 70$과 동일합니다.

## 5. 모멘텀 지표 비교

| 지표 | 범위 | 과매수 | 과매도 | 기본 기간 | 특징 |
|------|------|--------|--------|---------|------|
| RSI | 0~100 | 70 초과 | 30 미만 | 14일 | 가장 보편적 |
| 스토캐스틱 | 0~100 | 80 초과 | 20 미만 | 14일 | 신호선(%D) 활용 |
| CCI | 무제한 | 100 초과 | -100 미만 | 20일 | 추세 시작 신호 |
| Williams %R | -100~0 | -20 초과 | -80 미만 | 14일 | 스토캐스틱의 반전 |

## 실전 적용

### 시나리오 A: RSI로 과매도 반등 포착

**상황:**코스닥 종목이 3주간 18% 하락, RSI가 27로 진입

```
단계별 접근:

1. RSI 과매도 확인: RSI = 27 (30 미만)

2. 가격 구조 확인
   → 장기 지지선(50,000원) 근처에서 과매도 발생
   → 지지선 + 과매도 = 신뢰도 상승

3. 진입 타이밍
   → RSI가 30을 상향 돌파하는 시점 확인 (27 → 32)
   → 스토캐스틱도 20 이하에서 상향 교차 확인

4. 진입: 51,000원
   손절: 49,500원 (지지선 아래)
   목표: 57,000원 (직전 저항선)

5. 리스크/리워드: 6,000 / 1,500 = 1:4
```

### 시나리오 B: 스토캐스틱 교차로 단기 반등 포착

**상황:**외환(USD/KRW) 일봉, 스토캐스틱 과매도 구간 교차

| 항목 | 값 |
|------|-----|
| %K | 8 → 14 (상승) |
| %D | 11 (아직 하강) |
| 교차 위치 | 20 이하 ✓ |
| RSI | 28 (과매도 확인) |
| 진입가 | 1,340원 |
| 손절 | 1,330원 (%K가 다시 %D 아래로 교차 시) |
| 목표 | 1,360원 (직전 저항) |

### 시나리오 C: CCI로 강한 새 추세 진입 포착

**상황:**원자재 ETF가 장기 횡보 후 CCI +100 돌파

```
CCI 해석:
- CCI가 -100~+100 구간에서 횡보 중 (정상 범위)
- CCI가 +100을 강하게 돌파 → 새로운 강한 상승 추세 시작 신호
- 거래량 증가 동반 → 신뢰도 높음

진입 전략:
- CCI +100 돌파 확인 후 진입
- 손절: CCI가 다시 +100 이하로 하락 시
- 목표: CCI +200 또는 직전 고점
```

## 흔한 실수

### 실수 1: 과매수에서 즉시 역방향 진입

RSI 70을 넘자마자 공매도를 진입합니다. 강한 상승 추세에서 RSI는 70~80 구간에서 수주간 유지됩니다. 과매수는 반전 신호가 아닌강**한 추세의 확인입**니다. RSI 70 초과 시 추가 매수를 자제하되, 공매도 진입은 다이버전스, 페일러 스윙, 가격 구조 이탈 등 추가 신호를 확인 후 합니다.

### 실수 2: 단일 모멘텀 지표만으로 진입

RSI가 30 이하라는 이유만으로 매수합니다. 하락 추세에서 RSI는 30 이하에 수주간 머물 수 있습니다. 항상두** 가지 이상의 지표 확인이** 필요합니다. RSI + 스토캐스틱, 또는 RSI + 가격 지지선 조합이 단일 지표보다 훨씬 신뢰도가 높습니다.

### 실수 3: Williams %R과 스토캐스틱을 동시에 사용

두 지표 모두 화면에 올려놓고 "두 지표가 동의할 때만 진입"합니다. Williams %R은 스토캐스틱의 수학적 변형(공식 관계: %R = %K - 100)이므로 사실상 같은 정보입니다. 동일 계열 지표를 중복 사용하면 정보가 추가되지 않고 확신만 증가합니다.**서로 다른 원리의 지표**(모멘텀 + 추세 + 거래량)를 조합합니다.

### 실수 4: CCI의 +100/-100을 과매수/과매도로만 해석

CCI +100 초과를 RSI 70처럼 즉각적인 역방향 신호로 해석합니다. CCI는 RSI와 달리 +100 돌파가강**한 새 추세의 시작을** 의미할 수 있습니다. CCI를 개발한 Lambert의 원래 의도는 +100 초과를 강세 추세 진입, -100 미만을 약세 추세 진입 신호로 사용하는 것이었습니다.

### 실수 5: 다이버전스 신호 후 즉시 진입

RSI 약세 다이버전스를 확인하고 바로 공매도 진입합니다. 다이버전스는 추세 약화의경**고이**지 즉각적인 반전 신호가 아닙니다. 강한 추세에서 다이버전스가 수주간 유지되다 갑자기 해소되는 경우가 많습니다. 다이버전스 확인 후 반드시가**격 구조의 확인**(지지선 이탈, 넥라인 돌파 등)을 기다립니다.

## 엣지 케이스: 이상 시장 상황

### 모든 모멘텀 지표가 동시에 과매도인 경우

RSI, 스토캐스틱, CCI, Williams %R이 모두 극단적 과매도를 나타내는 경우입니다.

**해석:**이는 일반적으로패**닉 매도(Capitulation**)의 신호입니다. 단기적으로 반등 가능성이 높지만, 패닉 매도가 추세 전환의 신호인지 추가 하락의 시작인지 즉시 판단하기 어렵습니다.

**대응:**패닉 매도 직후 1~2일 관망 후, 반등 캔들이 나타나고 거래량이 감소하면 단기 매수를 고려합니다. 강한 포지션 진입보다 소규모 테스트 진입을 권장합니다.

### 모멘텀 지표 간 신호 충돌

RSI는 매수 신호(30 이하), 스토캐스틱은 과매수(80 이상)를 동시에 나타내는 경우입니다.

**원인:**각 지표의 기간 설정이 다르거나, 시장이 복잡한 구조를 형성 중

**대응:**신호 충돌 시 진입하지 않습니다. 더 짧은 기간 설정의 지표가 먼저 방향을 결정하므로, 신호 충돌이 해소될 때까지 대기합니다.

## 핵심 요약

-**RS**I는 0~100 범위에서 70 초과를 과매수, 30 미만을 과매도로 정의하며, 가격과 RSI의 방향이 불일치하는 다이버전스는 강력한 추세 반전 신호입니다
-**스토캐스틱 오실레이터는** %K와 %D 두 선의 교차 신호를 사용하며, Slow 스토캐스틱이 Fast보다 오신호가 적고 실전에서 더 많이 활용됩니다
-**CC**I는 현재 가격과 통계적 평균의 차이를 측정하며, -100~+100 범위를 벗어날 때 과매수/과매도가 아닌 강한 새 추세의 시작으로 해석할 수 있습니다
- 모든 모멘텀 지표는 강한 추세 시장에서 오신호가 많으므로추**세 지표와 함께사**용하고, 횡보 시장에서 더 신뢰도 높은 신호를 제공합니다
