## 개요

스토캐스틱 오실레이터(Stochastic Oscillator)는 George Lane이 1950년대에 개발한 모멘텀 지표입니다. 핵심 아이디어는 간단합니다.**"가격이 상승할 때 종가는 최근 범위의 상단에, 하락할 때는 하단에 모이는 경향이 있다."**
이 관찰을 수치화하면, 현재 종가가 최근 N일 범위에서 어느 위치에 있는지를 0~100 사이의 값으로 나타낼 수 있습니다. 이것이 스토캐스틱의 %K입니다.

기초 모멘텀 챕터에서 %K와 %D의 기본 공식을 다뤘습니다. 이 챕터에서는단**계별 수치 계산**,**Fast/Slow/Full 스토캐스틱 구분**,**교차 신호의 세부 활용법**,**다이버전스**,**RSI와의 결합**, 그리고**Lane의 원래 방법론을** 깊이 있게 다룹니다.

## 1. %K와 %D: 단계별 계산

### %K 공식

$$\%K = \frac{\text{현재 종가} - \text{N일 최저가}}{\text{N일 최고가} - \text{N일 최저가}} \times 100 \quad (\text{기본 } N = 14)$$

%K는 현재 가격이 최근 14일 가격 범위에서 상대적으로 어느 위치에 있는지를 나타냅니다.
- %K = 100: 14일 최고가에서 종가 형성
- %K = 0: 14일 최저가에서 종가 형성
- %K = 50: 14일 범위 정중앙에서 종가 형성

### 단계별 수치 계산 예시

아래 예시는 최근 14일 가격 데이터입니다.

| 날짜 | 고가 | 저가 | 종가 |
|------|------|------|------|
| 1일 | 52,000 | 49,000 | 50,500 |
| 2일 | 53,500 | 50,000 | 52,000 |
| 3일 | 54,000 | 51,000 | 51,500 |
| 4일 | 53,000 | 49,500 | 50,000 |
| 5일 | 51,500 | 47,000 | 47,500 |
| 6일 | 49,000 | 45,000 | 45,500 |
| 7일 | 47,000 | 43,000 | 43,500 |
| 8일 | 45,000 | 41,500 | 42,000 |
| 9일 | 44,000 | 40,000 | 41,000 |
| 10일 | 43,000 | 39,000 | 39,500 |
| 11일 | 42,000 | 38,000 | 38,500 |
| 12일 | 41,000 | 37,000 | 37,500 |
| 13일 | 40,500 | 36,500 | 37,000 |
|**14**|**41,00**0|**36,00**0|**38,00**0|

```
14일 기간:
  최고가 = 54,000원 (3일차)
  최저가 = 36,000원 (14일차)
  현재 종가 = 38,000원 (14일차)

%K = ((38,000 - 36,000) / (54,000 - 36,000)) x 100
%K = (2,000 / 18,000) x 100
%K = 11.1
```

%K = 11.1은 현재 가격이 14일 범위의 하단 11.1% 위치에 있음을 의미합니다. 이는 강한 약세 모멘텀을 나타냅니다.

### %D 계산: %K의 이동평균

%D는 %K를 평활화한 신호선입니다. 기본 설정은 %K의 3일 단순이동평균(SMA)입니다.

| 날짜 | %K | %D (3일 SMA) |
|------|-----|------------|
| 12일 | 15.3 | - |
| 13일 | 13.2 | - |
| 14일 | 11.1 | (15.3 + 13.2 + 11.1) / 3 =**13.**2|

$$\%D = \frac{\%K_{t-2} + \%K_{t-1} + \%K_{t}}{3} = \frac{15.3 + 13.2 + 11.1}{3} = 13.2$$

**신호선 해석:**- %K가 %D보다 위에 있으면 → 모멘텀 상승 중
- %K가 %D보다 아래에 있으면 → 모멘텀 하락 중
- %K가 %D를 교차할 때 → 교차 신호 발생

## 2. Fast, Slow, Full 스토캐스틱

### Fast 스토캐스틱

Fast 스토캐스틱은 Wilder의 원래 공식을 그대로 사용합니다.

```
Fast %K = 위에서 계산한 원래 %K
Fast %D = Fast %K의 3일 SMA
```

**특성:** 매우 민감하여 가격 변화에 빠르게 반응합니다. 그러나 노이즈가 많아 오신호가 빈번합니다. 실전에서는 거의 단독 사용하지 않습니다.

### Slow 스토캐스틱

Slow 스토캐스틱은 Fast 스토캐스틱을 한 번 더 평활화합니다.

```
Slow %K = Fast %D (Fast의 3일 SMA를 새로운 %K로 사용)
Slow %D = Slow %K의 3일 SMA
```

즉, Fast %D가 Slow %K가 되고, 이를 다시 3일 평균 낸 것이 Slow %D입니다.

**수치 비교 예시:**
| 날짜 | Fast %K | Fast %D = Slow %K | Slow %D |
|------|---------|------------------|---------|
| 12일 | 15.3 | - | - |
| 13일 | 13.2 | - | - |
| 14일 | 11.1 | 13.2 | - |
| 15일 | 14.5 | 12.9 | - |
| 16일 | 18.2 | 14.6 | (13.2 + 12.9 + 14.6) / 3 =**13.**6|

**특성:** Fast에 비해 완만하고 신뢰도가 높습니다. 실전에서 가장 많이 사용되는 설정입니다.

### Full 스토캐스틱

Full 스토캐스틱은 모든 매개변수를 사용자가 직접 지정합니다.

```
Full %K = 원래 %K를 P기간 SMA로 평활화
Full %D = Full %K의 Q기간 SMA
```

기본 설정: (14, 3, 3) → %K 기간 14일, %K 평활화 3일, %D 3일

**일반적인 Full 스토캐스틱 설정:**
| 설정 | 기간 | 특성 | 적합한 상황 |
|------|------|------|-----------|
| (5, 3, 3) | 단기 | 매우 민감 | 단기 트레이딩 |
| (14, 3, 3) | 표준 | 균형 | 스윙 트레이딩 |
| (21, 5, 5) | 장기 | 완만 | 중장기 투자 |

:::note
실전에서는 대부분**Slow 스토캐스틱 (14, 3**)을 표준으로 사용합니다. 차트 소프트웨어의 기본값이 Slow 스토캐스틱인 경우가 많습니다. Fast는 노이즈가 너무 많아 실전 활용도가 낮습니다.
:::

## 3. 과매수/과매도 구간

### 기본 해석 기준

| 구간 | 해석 | 신호 강도 |
|------|------|---------|
| 80 초과 | 과매수 (Overbought) | 매도 고려 |
| 50~80 | 강세 구간 | 추세 추종 |
| 50 기준선 | 중립 | 추세 전환 관찰 |
| 20~50 | 약세 구간 | 추세 추종 |
| 20 미만 | 과매도 (Oversold) | 매수 고려 |

### 강세/약세 시장에서의 조정

RSI와 마찬가지로, 스토캐스틱도 시장 국면에 따라 다르게 해석해야 합니다.

**강세 시장:**- 스토캐스틱이 80 이상에서 오래 머무는 것이 정상입니다
- 20으로 내려오는 경우가 드물며, 40~50 구간이 실질적인 지지 역할
- 80 초과를 보자마자 매도하면 상승 추세를 역행하게 됩니다

**약세 시장:**- 스토캐스틱이 20 이하에서 오래 머무는 것이 정상입니다
- 80으로 올라오는 경우가 드물며, 50~60 구간이 실질적인 저항 역할
- 20 미만을 보자마자 매수하면 하락 추세를 역행하게 됩니다

:::caution
**강한 추세 시장에서 과매수/과매도 신호를 역방향 진입으로 사용하는 것은 매우 위험합니다.**추세가 강할수록 과매수 구간이 장기간 유지될 수 있습니다. 항상 추세 방향을 먼저 확인하고, 추세와 같은 방향의 신호만 사용하세요.
:::

## 4. 교차 신호 (Crossover Signals)

교차 신호는 %K와 %D가 교차할 때 발생합니다. 단순하지만 가장 많이 활용되는 신호 유형입니다.

### 과매도 구간 상향 교차 (매수 신호)

```
조건:
1. %K와 %D 모두 20 이하 과매도 구간 진입
2. %K가 %D를 아래에서 위로 교차
3. 교차가 20 이하에서 발생할수록 신호 강도 높음
```

**수치 예시:**
| 날짜 | %K | %D | 상태 |
|------|-----|-----|------|
| D-2 | 12.3 | 15.8 | 과매도, %K 아래 |
| D-1 | 10.5 | 13.2 | 과매도, %K 아래 |
| D | 16.8 | 13.5 |**상향 교차 발생 → 매수 신호**|

교차 당일: %K(16.8)가 %D(13.5)를 위로 돌파

### 과매수 구간 하향 교차 (매도 신호)

```
조건:
1. %K와 %D 모두 80 이상 과매수 구간 진입
2. %K가 %D를 위에서 아래로 교차
3. 교차가 80 이상에서 발생할수록 신호 강도 높음
```

**수치 예시:**
| 날짜 | %K | %D | 상태 |
|------|-----|-----|------|
| D-2 | 87.5 | 84.2 | 과매수, %K 위 |
| D-1 | 85.3 | 86.1 | 과매수, %K 아래로 하락 중 |
| D | 79.8 | 83.7 |**하향 교차 발생 → 매도 신호**|

교차 당일: %K(79.8)가 %D(83.7) 아래로 돌파

### 교차 신호의 필터링

과매수/과매도 구간 외부에서 발생하는 교차 신호는 신뢰도가 낮습니다.

| 교차 위치 | 신뢰도 | 권장 여부 |
|---------|--------|---------|
| 20 이하에서 상향 | 높음 | 권장 |
| 20~50 사이에서 상향 | 낮음 | 비권장 |
| 80 이상에서 하향 | 높음 | 권장 |
| 50~80 사이에서 하향 | 낮음 | 비권장 |

:::tip
교차 신호의 신뢰도를 높이는 간단한 방법:** 교차가 극단 구간(20 이하 또는 80 이상)에서 발생하**고,**교차 직후 %K가 방향성 있게 움직일 **만 진입하세요. 교차 직후 다시 반전되면 무시합니다.
:::

## 5. 스토캐스틱 다이버전스

다이버전스는 스토캐스틱에서도 강력한 반전 신호를 제공합니다. RSI 다이버전스와 동일한 원리로 작동합니다.

### 강세 다이버전스 (Bullish Divergence)

```
가격: 저점1 --하락→ 저점2  (저점2 < 저점1, 더 낮은 저점)
스토캐스틱: 저점1 --상승→ 저점2  (저점2 > 저점1, 더 높은 저점)
→ 하락 모멘텀 약화, 상승 반전 가능성
```

**수치 예시:**
| 항목 | 저점 1 (3주 전) | 저점 2 (현재) |
|------|--------------|------------|
| 가격 | 45,000원 | 42,000원 (더 낮음) |
| 스토캐스틱 %K | 12 | 18 (더 높음) |
| 해석 | 하락 모멘텀이 약화되고 있음 |

### 약세 다이버전스 (Bearish Divergence)

```
가격: 고점1 --상승→ 고점2  (고점2 > 고점1, 더 높은 고점)
스토캐스틱: 고점1 --하락→ 고점2  (고점2 < 고점1, 더 낮은 고점)
→ 상승 모멘텀 약화, 하락 반전 가능성
```

**수치 예시:**
| 항목 | 고점 1 (2주 전) | 고점 2 (현재) |
|------|--------------|------------|
| 가격 | 58,000원 | 62,000원 (더 높음) |
| 스토캐스틱 %K | 88 | 78 (더 낮음) |
| 해석 | 상승 모멘텀이 약화되고 있음 |

### RSI 다이버전스 vs 스토캐스틱 다이버전스

| 항목 | RSI 다이버전스 | 스토캐스틱 다이버전스 |
|------|-------------|------------------|
| 민감도 | 중간 | 높음 (더 자주 발생) |
| 신뢰도 | 높음 | 중간 (오신호 빈번) |
| 유효 구간 | 30/70 기준 | 20/80 기준 |
| 추천 용도 | 주요 추세 전환 | 단기 반전 포착 |

## 6. RSI와 스토캐스틱의 결합

RSI와 스토캐스틱은 같은 모멘텀 계열이지만 계산 방식이 달라서**로 보완적으**로 작동합니다. 두 지표가 동시에 같은 신호를 보낼 때 신뢰도가 크게 높아집니다.

### 결합 원리

| 지표 | 계산 기반 | 민감도 | 강점 |
|------|---------|--------|------|
| RSI | 기간 내 가격 변화의 크기 | 중간 | 안정적, 다이버전스에 강함 |
| 스토캐스틱 | 현재 종가의 범위 내 위치 | 높음 | 과매수/과매도 반응 빠름 |

### 이중 확인 전략 (Dual Confirmation)

**매수 신호 확인:**
| 조건 | RSI | 스토캐스틱 | 신뢰도 |
|------|-----|----------|--------|
| 둘 다 과매도 구간 | 30 이하 | 20 이하 | 높음 |
| RSI만 과매도 | 30 이하 | 20 이상 | 중간 |
| 스토캐스틱만 과매도 | 30 이상 | 20 이하 | 낮음 |

**실전 매수 신호 예시:**
```
시나리오: 코스피 종목 일봉

1. 가격이 3주간 15% 하락
2. RSI: 27 (과매도)
3. 스토캐스틱 %K: 14, %D: 18 (과매도)
4. 스토캐스틱 %K가 %D를 상향 교차 (14→22)
5. RSI가 30을 상향 돌파 (27→32)

→ 양쪽 모두 신호 발생: 강한 매수 신호
```

### StochRSI: 두 지표의 결합

StochRSI는 RSI에 스토캐스틱 공식을 적용한 파생 지표입니다.

$$StochRSI = \frac{RSI - RSI_{min}}{RSI_{max} - RSI_{min}} \quad (\text{기본 } N = 14)$$

StochRSI는 RSI의 변화를 더 세밀하게 측정하여, RSI가 과매수/과매도에 있는지를 더 빠르게 파악합니다. 그러나 매우 민감하여 오신호가 많으므로 보조 확인 도구로만 활용하는 것을 권장합니다.

:::note
StochRSI의 기본값은 0~1 또는 0~100으로 표시됩니다. 일부 차트 소프트웨어는 0.8을 과매수, 0.2를 과매도 기준으로 사용하고, 다른 소프트웨어는 80과 20을 사용합니다. 설정을 확인 후 사용하세요.
:::

## 7. George Lane의 원래 방법론

George Lane은 스토캐스틱을 단순한 과매수/과매도 지표로 만들지 않았습니다. 그의 원래 의도는**"가격이 아닌 모멘텀을 먼저 반전시킨다"**는 개념을 포착하는 것이었습니다.

### Lane의 핵심 철학

Lane의 유명한 말:** "스토캐스틱은 가격이 아닌 모멘텀을 따른다. 모멘텀은 가격보다 먼저 방향을 바꾼다."**
이 철학이 다이버전스 개념의 근거입니다. 가격이 계속 상승하더라도 모멘텀(스토캐스틱)이 먼저 하락하기 시작하면, 곧 가격도 반전될 가능성이 높습니다.

### Lane의 4가지 신호 우선순위

Lane은 신호의 신뢰도를 다음 순서로 평가했습니다.

| 순위 | 신호 유형 | 설명 |
|------|---------|------|
| 1위 | 다이버전스 | 가격과 스토캐스틱의 불일치 |
| 2위 | 과매수/과매도 구간 교차 | 80 이하 하향 또는 20 이상 상향 교차 |
| 3위 | 극단 구간 내 교차 | 과매수/과매도 구간 내부에서의 교차 |
| 4위 | 50 기준선 교차 | 50을 상향/하향 돌파 |

### Lane의 설정 권장

Lane은 실제로 다양한 기간 설정을 실험했습니다.

| 설정 | 원래 명칭 | 특성 |
|------|---------|------|
| (5, 3) | Lane's Fast | 단기 사이클 포착 |
| (14, 3) | 표준 | 가장 보편적 |
| (21, 5) | 장기 | 노이즈 감소 |

Lane 본인은단**기 사이클 포착에** (5, 3) 설정을 주로 사용했다고 알려져 있습니다.

### 가격 사이클과 스토캐스틱

Lane은 스토캐스틱이 주로**28일 주기의** 가격 사이클을 가장 잘 추적한다고 주장했습니다. 이는 달(Month)의 절반에 해당하는 주기로, 상품 선물 시장에서 자주 관찰되는 패턴입니다.

현대에는 시장 특성에 따라 주기가 달라지므로, 기계적으로 적용하기보다는해**당 자산의 특성에 맞게 최적화하**는 것을 권장합니다.

## 실전 적용

### 하락 추세에서 스토캐스틱 활용 시나리오

아래는 하락 추세에 있는 자산에서 스토캐스틱을 활용하는 실전 시나리오입니다.

**배경:** 한 종목이 3개월간 지속적으로 하락 중. 200일 이동평균선 아래에서 거래 중.

```
1. 주봉 스토캐스틱: 22 (과매도 구간)
   → 하락 추세 확인, 약세 시장

2. 일봉 스토캐스틱:
   - %K = 8, %D = 12 (극단적 과매도)
   - 강세 다이버전스 형성: 가격 저점 낮아졌으나 스토캐스틱 저점 높아짐

3. 일봉 스토캐스틱 상향 교차:
   - %K(14)가 %D(11)를 상향 교차
   - 교차가 20 이하에서 발생 → 고신뢰 매수 신호

4. RSI 확인:
   - RSI = 28 (과매도)
   - RSI도 30 구간에서 상향 반전 조짐

→ 공매도 포지션 청산 또는 단기 반등 매수 타이밍
→ 단, 하락 추세이므로 강한 반등보다 단기 반등에 그칠 가능성 고려
```

### 자산 유형별 최적 설정

| 자산 유형 | 권장 설정 | 이유 |
|---------|---------|------|
| 주식 (스윙) | (14, 3, 3) Slow | 표준, 균형 잡힌 신호 |
| 주식 (단기) | (5, 3, 3) Fast | 단기 사이클 빠른 포착 |
| 암호화폐 | (21, 5, 5) 또는 (14, 5, 5) | 높은 변동성으로 노이즈 감소 필요 |
| 외환 (Forex) | (14, 3, 3) Slow | 표준 |
| 지수 ETF | (21, 5, 5) | 장기 사이클 추적 |

## 흔한 실수

### 실수 1: 횡보 구간에서 모든 교차 신호 진입

**잘못된 접근:** 스토캐스틱이 과매수와 과매도를 빠르게 오갈 때 모든 교차 신호에 진입

**올바른 접근:** 횡보 시장에서는 오신호가 매우 많습니다. 볼린저 밴드, ADX 등 다른 지표로 횡보 구간임을 먼저 확인하고, 교차 신호를 필터링하세요. ADX 20 이하라면 추세가 없는 구간으로, 역추세 전략(지지/저항 매매)이 더 적합합니다.

### 실수 2: Fast 스토캐스틱을 실전에 사용

**잘못된 접근:** 빠른 신호를 위해 Fast 스토캐스틱 단독 사용

**올바른 접근:** Fast %K는 노이즈가 너무 많아 실전에서 사용하기 어렵습니다. Slow 스토캐스틱(Fast %D를 %K로 사용)이 훨씬 신뢰도가 높습니다.

### 실수 3: 추세 방향 무시하고 역방향 진입

**잘못된 접근:** 강한 하락 추세에서 스토캐스틱 20 이하를 매수 신호로 사용

**올바른 접근:** 하락 추세에서 스토캐스틱 20 이하는 빈번하게 발생합니다. 추세와 반대 방향의 신호는 무시하거나, 단기 반등 트레이딩에만 제한적으로 사용하세요. 추세 방향 확인을 위해 이동평균선 또는 ADX를 먼저 참조하세요.

### 실수 4: StochRSI를 주요 진입 신호로 사용

**잘못된 접근:** StochRSI 교차를 주요 진입 신호로 사용

**올바른 접근:** StochRSI는 너무 민감하여 하루에도 여러 번 신호가 발생합니다. 이를 주요 신호로 사용하면 과도한 매매와 높은 수수료로 이어집니다. RSI와 스토캐스틱의 보조 확인 도구로만 사용하세요.

### 실수 5: 두 저점/고점 사이 간격 무시

**잘못된 접근:** 2~3 캔들 간격의 다이버전스를 강한 신호로 해석

**올바른 접근:** 스토캐스틱 다이버전스는 두 저점/고점 사이에최**소 5~10 캔들 이상의** 간격이 있어야 의미 있는 신호로 볼 수 있습니다. 너무 짧은 간격의 다이버전스는 노이즈일 가능성이 높습니다.

## 엣지 케이스: 이상 시장 상황

### 스토캐스틱이 과매수/과매도 구간에서 장기 고착

강한 추세 시장에서 스토캐스틱이 80 이상 또는 20 이하에서 수 주간 머무는 경우입니다.

**잘못된 대응:** 80 초과를 확인하고 매도 진입 → 강한 상승 추세를 역행하여 손실

**올바른 대응:** RSI와 마찬가지로 강세 시장에서는 과매수 구간 장기 유지가 정상입니다. ADX로 추세 강도를 확인하고, ADX 30 이상이면 과매수 역방향 진입을 삼갑니다. 스토캐스틱이 80에서 80 아래로 내려오는하**향 교차만** 신호로 활용합니다.

### 교차 신호가 너무 빈번하게 발생

횡보 시장에서 스토캐스틱이 20~80을 빠르게 오가며 교차 신호가 하루에 여러 번 발생하는 경우입니다.

**발생 원인:** 시장 변동성이 낮고 방향성이 없는 상태

**대응:**- 기간 설정을 늘립니다 (14 → 21)
- 교차 신호를 무시하고 극단 구간(20 이하/80 이상)의 교차만 활용합니다
- ADX로 횡보 시장 확인 후 범위 매매 전략으로 전환합니다

### 스토캐스틱 다이버전스가 오래 지속

다이버전스가 5~10캔들 이상 지속되는데 반전이 오지 않는 경우입니다.

**대응:** 스토캐스틱 다이버전스는 RSI 다이버전스보다 민감하여 오래 지속될 수 있습니다. 다이버전스 신호로 진입했다면 손절매를 반드시 유지하고, 가격이 명확한 반전 구조를 만들 때까지 추가 진입을 자제합니다.

### 갭 발생 시 스토캐스틱 왜곡

갭이 발생하면 스토캐스틱의 최고가/최저가 기준이 왜곡됩니다. 상승 갭 발생 시 %K가 갑자기 90 이상으로 급등하거나, 하락 갭 발생 시 10 이하로 급락하는 현상입니다.

**대응:** 갭 발생 직후의 스토캐스틱 신호는 신뢰도가 낮습니다. 갭 이후 3~5 캔들 후 스토캐스틱이 안정화될 때까지 신호 활용을 자제합니다.

## 핵심 요약

- **%K 계산:** ((현재 종가 - N일 최저) / (N일 최고 - N일 최저)) × 100으로, 현재 가격이 최근 N일 범위에서 어느 위치인지를 나타냅니다
- **Fast vs Slow:** Fast는 원래 %K와 %D, Slow는 Fast %D를 새 %K로 사용하여 한 번 더 평활화합니다. 실전에서는 Slow 스토캐스틱(14, 3)이 표준입니다
- **교차 신호:** 과매도 구간(20 이하)에서의 상향 교차와 과매수 구간(80 이상)에서의 하향 교차가 신뢰도 높은 신호이며, 극단 구간 밖의 교차는 신뢰도가 낮습니다
- **다이버전스:** 가격과 스토캐스틱이 반대 방향으로 움직일 때 발생하며, 두 저점/고점 간 최소 5캔들 이상 간격이 있어야 유효합니다
- **RSI와 결합:** RSI(안정적, 추세 기반)와 스토캐스틱(민감한, 범위 기반)이 동시에 같은 신호를 보낼 때 신뢰도가 크게 높아집니다
- **Lane의 철학:** "모멘텀이 가격보다 먼저 반전한다"는 원칙으로, 다이버전스가 가장 신뢰도 높은 신호이며 단순 과매수/과매도보다 우선합니다
