## 개요

RSI(Relative Strength Index, 상대강도지수)는 J. Welles Wilder Jr.가 1978년 저서 *New Concepts in Technical Trading Systems*에서 발표한 지표입니다. 기초 모멘텀 챕터에서 RSI의 기본 해석을 다뤘다면, 이 챕터에서는계**산 방식의 세부 원리**,**다이버전스의 4가지 유형**,**페일러 스윙**,**시장 국면별 RSI 범위 재설정**,**멀티 타임프레임 분석**, 그리고**Andrew Cardwell의 고급 방법론을** 깊이 있게 다룹니다.

RSI를 진정으로 이해하면 단순한 과매수/과매도 신호를 넘어, 추세의 강도와 지속 가능성, 반전 가능성을 정밀하게 평가할 수 있습니다.

## 1. RSI 계산 원리: Wilder 스무딩 방식

기초 챕터에서는 단순 평균으로 RSI를 계산했습니다. 실제 RSI**Wilder의 지수 스무딩(Wilder Smoothing**)방식을 사용합니다. 이 차이가 중요한 이유는 지표 값의 정확한 해석에 영향을 주기 때문입니다.

### 1단계: 초기값 계산 (첫 14일)

첫 14일 동안은 단순 평균으로 초기 RS를 구합니다.

| 날짜 | 종가 | 변화 | 상승 | 하락 |
|------|------|------|------|------|
| 1일 | 100,000 | - | - | - |
| 2일 | 102,500 | +2,500 | 2,500 | 0 |
| 3일 | 101,000 | -1,500 | 0 | 1,500 |
| 4일 | 103,800 | +2,800 | 2,800 | 0 |
| 5일 | 103,200 | -600 | 0 | 600 |
| 6일 | 105,000 | +1,800 | 1,800 | 0 |
| 7일 | 104,500 | -500 | 0 | 500 |
| 8일 | 106,000 | +1,500 | 1,500 | 0 |
| 9일 | 107,200 | +1,200 | 1,200 | 0 |
| 10일 | 106,800 | -400 | 0 | 400 |
| 11일 | 108,000 | +1,200 | 1,200 | 0 |
| 12일 | 107,500 | -500 | 0 | 500 |
| 13일 | 109,000 | +1,500 | 1,500 | 0 |
| 14일 | 110,000 | +1,000 | 1,000 | 0 |

```
14일 합산:
상승 합계: 2,500 + 2,800 + 1,800 + 1,500 + 1,200 + 1,200 + 1,500 + 1,000 = 13,500
하락 합계: 1,500 + 600 + 500 + 400 + 500 = 3,500

초기 평균 상승폭 = 13,500 / 14 = 964.3
초기 평균 하락폭 = 3,500 / 14 = 250.0

RS = 964.3 / 250.0 = 3.857
RSI = 100 - (100 / (1 + 3.857)) = 100 - 20.6 = 79.4
```

### 2단계: 이후 날들의 Wilder 스무딩

14일 이후부터는 단순 평균이 아닌**Wilder 스무딩 공식을** 적용합니다.

$$\text{평균 상승폭}_t = \frac{\text{평균 상승폭}_{t-1} \times 13 + \text{당일 상승폭}}{14}$$

$$\text{평균 하락폭}_t = \frac{\text{평균 하락폭}_{t-1} \times 13 + \text{당일 하락폭}}{14}$$

15일차에 종가가 109,000으로 하락했다면(하락폭 = 1,000):

```
새 평균 상승폭 = (964.3 x 13 + 0) / 14 = 12,535.9 / 14 = 895.4
새 평균 하락폭 = (250.0 x 13 + 1,000) / 14 = 4,250 / 14 = 303.6

RS = 895.4 / 303.6 = 2.949
RSI = 100 - (100 / (1 + 2.949)) = 100 - 25.3 = 74.7
```

:::note
Wilder 스무딩은 EMA의 α = 1/14와 동일합니다. 이 방식 덕분에 RSI는 과거 데이터를 모두 반영하며, 최근 변화에는 빠르게 반응하되 급격한 노이즈는 걸러냅니다.
:::

### 공식 요약

$$RSI = 100 - \frac{100}{1 + RS}$$

$$RS = \frac{\text{평균 상승폭}}{\text{평균 하락폭}} \quad (\text{Wilder 스무딩})$$

- 초기값: 14일 단순 평균
- 이후: $\frac{\text{이전 평균} \times 13 + \text{당일값}}{14}$

## 2. 다이버전스 (Divergence): 4가지 유형

<img src="/technical-analysis/images/diagrams/divergence-types.svg" alt="RSI 다이버전스 4가지 유형: 일반 강세/약세 다이버전스와 히든 강세/약세 다이버전스 비교 다이어그램" style="width:100%;max-width:600px;margin:1rem auto;display:block;" />

다이버전스는 가격 움직임과 RSI 움직임이 불일치할 때 발생합니다. 기초 챕터에서 일반 다이버전스를 다뤘다면, 이 챕터에서는히**든 다이버전스까**지 포함한 4가지 유형 전체를 다룹니다.

### 유형 1: 일반 강세 다이버전스 (Regular Bullish Divergence)

```
가격: 저점1 --하락→ 저점2  (더 낮은 저점: 저점2 < 저점1)
RSI:  저점1 --상승→ 저점2  (더 높은 저점: RSI저점2 > RSI저점1)
```

**의미:**가격은 더 낮은 저점을 만들었지만, RSI는 그만큼 하락하지 않았습니다. 하락 모멘텀이 약화되고 있으며, 상승 반전 가능성이 높습니다.

**활용 조건:**- RSI가 30 이하 과매도 구간에서 발생할 때 신뢰도 높음
- 두 저점 사이에 충분한 시간 간격 필요 (최소 5~10 캔들)
- 지지선, 추세선 등 다른 기술적 레벨과 일치할 때 강력

**예시:**
| 항목 | 저점 1 | 저점 2 |
|------|--------|--------|
| 가격 | 50,000원 | 47,000원 (더 낮음) |
| RSI | 28 | 32 (더 높음) |
| 해석 | 하락 모멘텀 약화 → 반전 준비 |

### 유형 2: 일반 약세 다이버전스 (Regular Bearish Divergence)

```
가격: 고점1 --상승→ 고점2  (더 높은 고점: 고점2 > 고점1)
RSI:  고점1 --하락→ 고점2  (더 낮은 고점: RSI고점2 < RSI고점1)
```

**의미:**가격은 신고점을 갱신했지만 RSI는 그만큼 오르지 못했습니다. 상승 모멘텀이 약화되고 있으며, 하락 반전 가능성이 높습니다.

**예시:**
| 항목 | 고점 1 | 고점 2 |
|------|--------|--------|
| 가격 | 110,000원 | 115,000원 (더 높음) |
| RSI | 72 | 65 (더 낮음) |
| 해석 | 상승 모멘텀 약화 → 조정 가능성 |

### 유형 3: 히든 강세 다이버전스 (Hidden Bullish Divergence)

```
가격: 저점1 --상승→ 저점2  (더 높은 저점: 저점2 > 저점1)
RSI:  저점1 --하락→ 저점2  (더 낮은 저점: RSI저점2 < RSI저점1)
```

**의미:**상승 추세 중 조정 국면에서 나타납니다. 가격의 저점은 높아지고 있는데(상승 추세 유지), RSI의 저점은 낮아집니다. 이는추**세 지속 신호입**니다.

:::tip
히든 다이버전스는 추세 추종 트레이더에게 매우 유용합니다. 상승 추세의 조정 구간에서 히든 강세 다이버전스가 나타나면, 추세가 재개될 가능성이 높으므로 추가 매수 기회로 활용할 수 있습니다.
:::

**예시:**
| 항목 | 저점 1 | 저점 2 |
|------|--------|--------|
| 가격 | 48,000원 | 51,000원 (더 높음: 상승추세 조정 저점) |
| RSI | 38 | 33 (더 낮음) |
| 해석 | 상승 추세 지속 신호 → 매수 기회 |

### 유형 4: 히든 약세 다이버전스 (Hidden Bearish Divergence)

```
가격: 고점1 --하락→ 고점2  (더 낮은 고점: 고점2 < 고점1)
RSI:  고점1 --상승→ 고점2  (더 높은 고점: RSI고점2 > RSI고점1)
```

**의미:**하락 추세 중 반등 국면에서 나타납니다. 가격의 고점은 낮아지고 있는데(하락 추세 유지), RSI의 고점은 높아집니다. 이는하**락 추세 지속 신호입**니다.

### 4가지 다이버전스 비교표

| 유형 | 가격 | RSI | 신호 | 사용 국면 |
|------|------|-----|------|---------|
| 일반 강세 | 저점 낮아짐 | 저점 높아짐 | 반전 (상승) | 하락 후 저점 |
| 일반 약세 | 고점 높아짐 | 고점 낮아짐 | 반전 (하락) | 상승 후 고점 |
| 히든 강세 | 저점 높아짐 | 저점 낮아짐 | 추세 지속 (상승) | 상승 추세 조정 |
| 히든 약세 | 고점 낮아짐 | 고점 높아짐 | 추세 지속 (하락) | 하락 추세 반등 |

:::caution
다이버전스 신호만으로 즉시 진입하는 것은 위험합니다. 다이버전스가 형성된 후에도 추세가 한동안 지속될 수 있습니다. 항상 가격의 구조적 확인(추세선 돌파, 지지/저항 레벨 등)과 함께 사용하세요.
:::

## 3. 페일러 스윙 (Failure Swing)

페일러 스윙은 Wilder가 직접 제시한 개념으로, 다이버전스보다 더 순수하게**RSI 자체의 움직임만으로반**전을 판단합니다. 가격을 전혀 참조하지 않습니다.

### 강세 페일러 스윙 (Bullish Failure Swing)

**조건:**1. RSI가 30 이하 과매도 구간 진입 (저점 A)
2. RSI가 반등하여 30 위로 올라옴 (고점 B)
3. RSI가 다시 하락하지만 30 위에서 멈춤 (저점 C, C &gt; A)
4. RSI가 고점 B를 상향 돌파 (확인점 D)

```
RSI 움직임:
   B         D
   ^         ^
   |   C     |
   | /   \   |
   |/     \→/
   A (30 이하)
```

**D 지점이 진입 신호입니다.**이때 가격이 어떻게 움직이든 RSI만의 구조가 강세 전환을 나타냅니다.

**수치 예시:**
| 단계 | RSI 값 | 설명 |
|------|--------|------|
| A (저점) | 25 | 과매도 진입 |
| B (고점) | 42 | 반등 |
| C (저점) | 33 | 재하락하지만 30 위에서 지지 |
| D (돌파) | 43 초과 | B 수준 돌파 → 매수 신호 |

### 약세 페일러 스윙 (Bearish Failure Swing)

**조건:**1. RSI가 70 이상 과매수 구간 진입 (고점 A)
2. RSI가 하락하여 70 아래로 내려옴 (저점 B)
3. RSI가 반등하지만 70 아래에서 멈춤 (고점 C, C &lt; A)
4. RSI가 저점 B를 하향 돌파 (확인점 D)

```
RSI 움직임:
   A (70 이상)
   ^
   | \     C
   |  \   / \
   |   B /   \→D
```

**D 지점이 매도 신호입니다.**
**수치 예시:**
| 단계 | RSI 값 | 설명 |
|------|--------|------|
| A (고점) | 78 | 과매수 진입 |
| B (저점) | 62 | 하락 |
| C (고점) | 68 | 반등하지만 70 미달 |
| D (돌파) | 61 이하 | B 수준 하향 돌파 → 매도 신호 |

:::tip
페일러 스윙은 다이버전스와 달리**RSI 내부 구조만 봅니다**. 이 때문에 노이즈에 덜 취약하고, 신호가 명확합니다. Wilder 본인은 페일러 스윙을 다이버전스보다 신뢰도 높은 신호로 간주했습니다.
:::

## 4. 강세/약세 시장의 RSI 범위 조정

Andrew Cardwell의 가장 중요한 기여 중 하나는**RSI가 시장 국면에 따라 다른 범위에서 작동한다는** 발견입니다.

### 강세 시장 (Bull Market) RSI 범위: 40-80

상승 추세가 강한 시장에서 RSI는 40~80 구간에서 진동합니다.

| 구간 | 의미 | 대응 |
|------|------|------|
| 80 초과 | 과매수 (단기 고점) | 일부 익절 |
| 40~80 | 정상 강세 구간 | 추세 추종 |
| 40 근처 | 매수 기회 (지지선) | 매수 진입 |
| 40 이하 | 경고: 추세 약화 가능 | 주의 필요 |

**실전 적용:**강세 시장에서 RSI가 40~50까지 눌릴 때는추**세 조정 매수 기회입**니다. 과거라면 과매도처럼 보일 수 있는 수준(40~45)이 실제로는 지지 구간입니다.

### 약세 시장 (Bear Market) RSI 범위: 20-60

하락 추세가 강한 시장에서 RSI는 20~60 구간에서 진동합니다.

| 구간 | 의미 | 대응 |
|------|------|------|
| 60 근처 | 매도 기회 (저항선) | 매도 진입 |
| 20~60 | 정상 약세 구간 | 추세 추종 (하락) |
| 20 미만 | 과매도 (단기 저점) | 일부 매도 청산 |
| 60 초과 | 경고: 추세 반전 가능 | 주의 필요 |

:::note
이 범위 이론은 RSI의 과매수/과매도 기준이 고정된 것이 아니라는 것을 보여줍니다. 현재 시장이 강세인지 약세인지 먼저 파악하고, 그에 맞게 RSI 해석 기준을 조정해야 합니다.
:::

### 시장 국면 판단 방법

1.**200일 이동평균선 기준:**가격이 200일 MA 위 → 강세 시장, 아래 → 약세 시장
2.**RSI 50 기준선 관찰:**RSI가 50 위에서 지지받으면 강세, 아래에서 저항받으면 약세
3.**최근 고점/저점 구조:**고점과 저점이 모두 높아지면 강세, 낮아지면 약세

## 5. 멀티 타임프레임 RSI 분석

단일 타임프레임에서 RSI를 보는 것은 전체 그림의 일부만 보는 것과 같습니다. 멀티 타임프레임 분석은 더 높은 타임프레임의 방향성과 더 낮은 타임프레임의 진입 타이밍을 결합합니다.

### 3단계 타임프레임 구조

| 역할 | 타임프레임 예시 | 목적 |
|------|--------------|------|
| 상위 (방향 결정) | 주봉 / 일봉 | 전체 추세와 RSI 시장 국면 파악 |
| 중위 (설정 확인) | 일봉 / 4시간봉 | 다이버전스 또는 페일러 스윙 확인 |
| 하위 (진입 타이밍) | 4시간봉 / 1시간봉 | 정밀 진입 시점 포착 |

### 실전 멀티 타임프레임 시나리오

**매수 설정 예시:**
```
1. 주봉 RSI: 55 (강세 시장 범위 40-80 중간)
   → 상승 추세 확인, 강세 시장

2. 일봉 RSI: 42 (강세 시장 지지 구간)
   → 조정 국면, 일반 강세 다이버전스 형성 중

3. 4시간봉 RSI: 강세 페일러 스윙 완성
   → 정밀 진입 신호
```

이 세 조건이 모두 정렬될 때 진입하면 오신호 가능성이 크게 줄어듭니다.

:::caution
타임프레임 간 충돌이 있을 때는항**상 상위 타임프레임을 우선합**니다. 일봉이 하락 추세인데 1시간봉에서 강세 다이버전스가 나타난다면, 이는 하락 추세의 일시적 반등에 불과할 가능성이 높습니다.
:::

## 6. Andrew Cardwell의 RSI 방법론

Andrew Cardwell은 Wilder에게 RSI를 직접 배웠으며, 수십 년의 실전 경험으로 RSI를 대폭 발전시켰습니다. 그의 핵심 기여는 다음 세 가지입니다.

### 6-1. RSI 범위 분석 (이미 위에서 다룸)

강세 시장 40-80, 약세 시장 20-60.

### 6-2. 포지티브 리버설 (Positive Reversal)

**조건:**- 일반 강세 다이버전스처럼 보이지만반**전이 아닌 추세 지속을** 의미
- RSI가 이전 저점보다 낮은 저점을 만들었지만, 가격은 더 높은 저점을 형성

```
가격: 저점1 --상승→ 저점2  (저점2 > 저점1, 상승 추세 내 조정)
RSI:  저점1 --하락→ 저점2  (RSI 저점2 < RSI 저점1)
→ 히든 강세 다이버전스와 동일한 구조
```

**목표 가격 계산:**Cardwell은 포지티브 리버설 후의 목표가를 다음과 같이 설정했습니다.

$$\text{목표가} = \text{직전 고점} + (\text{직전 고점} - \text{조정 저점})$$

예: 직전 고점 110,000, 조정 저점 104,000이면
목표가 = 110,000 + (110,000 - 104,000) = 116,000원

### 6-3. 네거티브 리버설 (Negative Reversal)

**조건:**- 가격이 이전 고점보다 낮은 고점을 형성 (하락 추세 내 반등)
- RSI는 이전 고점보다 높은 고점을 형성
- 히든 약세 다이버전스와 동일한 구조이며, 하락 추세 지속 신호

**목표 가격 계산:**$$\text{목표가} = \text{직전 저점} - (\text{반등 고점} - \text{직전 저점})$$

### 핵심 통찰: 다이버전스의 재해석

Cardwell의 가장 혁명적인 주장은일**반 강세 다이버전스가 하락 추세에서는 반전이 아닌 추세 지속 신호일** 수 있다는 것입니다.

| 시장 국면 | 일반 강세 다이버전스의 의미 |
|---------|------------------------|
| 하락 추세 | 반전 신호가 아닌, 하락 지속 전 반등일 수 있음 |
| 상승 추세 | 조정 후 추세 재개 신호 (포지티브 리버설) |

:::tip
Cardwell의 방법론은 단순히 다이버전스를 보는 것이 아니라,**현재 시장이 어느 추세에 있는지를 먼저 판단하**고 그 맥락에서 RSI 신호를 해석해야 한다는 것을 강조합니다.
:::

## 실전 적용

### 체크리스트: RSI 심화 신호 확인

RSI 다이버전스 또는 페일러 스윙 발생 시 다음을 확인하세요.

**1단계: 시장 국면 파악**- [ ] 상위 타임프레임 추세 방향 확인 (200일 MA 등)
- [ ] RSI가 강세 범위(40-80)와 약세 범위(20-60) 중 어느 구간에서 작동 중인지 확인

**2단계: 다이버전스/페일러 스윙 유형 확인**- [ ] 일반 다이버전스인지 히든 다이버전스인지 구분
- [ ] 페일러 스윙 구조가 완성되었는지 확인 (돌파 포인트 D 형성)

**3단계: 멀티 타임프레임 정렬**- [ ] 상위 타임프레임과 신호 방향 일치 여부
- [ ] 하위 타임프레임에서 진입 타이밍 신호 확인

**4단계: 지지/저항 레벨 검증**- [ ] 신호가 주요 지지/저항 레벨 근처에서 발생하는지 확인
- [ ] 볼린저 밴드, 피보나치 레벨 등과 일치 여부

### 실전 예시: 상승 추세 진입

```
시나리오: 비트코인 일봉

1. 주봉 RSI: 62 → 강세 시장 범위(40-80), 상승 추세 확인
2. 일봉 RSI: 44 (저점 A), 가격 조정으로 48,000달러
   → RSI 강세 범위 하단인 40에 근접
3. 일봉 RSI: 47 (저점 C), 가격 52,000달러
   → 저점 A(44)보다 높은 저점 형성: 히든 강세 다이버전스 구조
4. 4시간봉에서 페일러 스윙 B 돌파 확인
   → 진입 신호

리스크 관리:
- 손절: 조정 저점(48,000달러) 아래 2~3%
- 목표: 이전 고점(55,000달러) 또는 포지티브 리버설 목표가
```

## 흔한 실수

### 실수 1: 과매수에서 무조건 매도

**잘못된 접근:**RSI 70 초과를 보자마자 공매도 진입

**올바른 접근:**강세 시장에서 RSI 70~80은 정상 범위입니다. 단순히 70을 초과했다는 이유로 매도하면 상승 추세를 역행하는 위험한 거래가 됩니다.

### 실수 2: 다이버전스를 즉시 실행 신호로 사용

**잘못된 접근:**일반 약세 다이버전스 발견 즉시 공매도

**올바른 접근:**다이버전스는 경고 신호입니다. 실제 반전 확인을 위해 지지선 이탈, 이동평균선 하향 돌파, 페일러 스윙 완성 등의 추가 확인이 필요합니다.

### 실수 3: 단일 타임프레임 분석

**잘못된 접근:**15분봉 RSI 다이버전스만 보고 진입

**올바른 접근:**상위 타임프레임이 하락 추세라면 15분봉의 강세 다이버전스는 단기 반등에 불과합니다. 항상 상위 타임프레임과 방향을 먼저 확인하세요.

### 실수 4: RSI 기간 변경 없이 모든 자산에 14 적용

**잘못된 접근:**모든 자산, 모든 타임프레임에 RSI 14 고정

**올바른 접근:**- 단기 트레이딩: RSI 9 (더 민감)
- 스윙 트레이딩: RSI 14 (표준)
- 장기 투자: RSI 21~25 (노이즈 감소)
- 암호화폐처럼 변동성이 큰 자산: RSI 21 이상 권장

### 실수 5: 시장 국면 고려 없는 다이버전스 해석

**잘못된 접근:**모든 강세 다이버전스를 반전 신호로 해석

**올바른 접근:**Cardwell의 방법론에 따라, 하락 추세에서의 강세 다이버전스는 반전이 아닌 추세 지속의 포지티브 리버설일 수 있습니다.

## 엣지 케이스: 이상 시장 상황

### 다이버전스 실패 (Divergence Failure)

강세 다이버전스가 완성됐는데도 가격이 계속 하락하는 경우입니다.

**발생 원인:**- 하락 추세가 매우 강할 때: Cardwell이 지적했듯 하락 추세에서의 강세 다이버전스는 반전이 아닌 추세 지속 신호일 수 있습니다
- 거시경제 충격: 예상치 못한 악재가 기술적 신호를 무력화
- 시간 불일치: 다이버전스는 올바르지만 반전 시점이 수주~수개월 후에 나타남

**대응:**다이버전스 신호만으로 진입하지 않고, 가격 구조(지지선 방어, 추세선 회복)를 함께 확인합니다. 다이버전스는 "경보"이지 "즉시 진입 신호"가 아닙니다.

### RSI가 70~80 구간에 수주간 고착되는 경우

강세 시장에서 RSI가 70 이상에서 장기간 머무는 현상입니다. 이는 Cardwell의 강세 시장 범위(40-80) 이론과 일치하며, 정상적인 강세 시장의 특성입니다.

**잘못된 대응:**70 초과를 보고 계속 공매도 시도 → 손실 누적
**올바른 대응:**강세 시장 범위(40-80) 기준으로 재설정하여 80 초과에서만 단기 조정을 고려하고, 추세 추종 포지션을 유지합니다.

### 페일러 스윙 연속 실패

페일러 스윙 신호가 연속으로 2~3회 발생하는 경우입니다. 이는 강한 추세 시장에서 RSI가 과매수/과매도 구간에서 장기간 머물 때 나타납니다.

**대응:**상위 타임프레임 추세 방향을 재확인합니다. 추세 방향과 반대인 페일러 스윙은 무시하고, 같은 방향의 페일러 스윙만 활용합니다.

### 암호화폐에서의 RSI 극단값 지속

암호화폐는 변동성이 높아 RSI가 80~90 구간 또는 10~20 구간에서 수주간 머무는 경우가 빈번합니다.

**대응:**암호화폐에는 기본 14일 RSI보다 21일 이상의 긴 기간을 사용하고, 시장 국면별 범위 조정을 더욱 엄격하게 적용합니다. 단기 교차 신호보다 다이버전스와 페일러 스윙에 집중합니다.

## 핵심 요약

-**RSI 계산:**단순 평균이 아닌 Wilder 스무딩 방식(이전값 × 13 + 당일값) / 14 을 사용하며, 이는 α = 1/14인 EMA와 동일합니다
-**4가지 다이버전스:**일반 강세/약세(반전 신호)와 히든 강세/약세(추세 지속 신호)를 구분하여 적용해야 합니다
-**페일러 스윙:**RSI 자체 구조만으로 반전을 판단하는 방법으로, Wilder가 다이버전스보다 신뢰도 높게 평가했으며 B 지점 돌파(D 포인트)가 진입 신호입니다
-**시장 국면별 범위:**강세 시장에서는 RSI 40-80, 약세 시장에서는 20-60 범위로 과매수/과매도 기준을 조정해야 합니다
-**멀티 타임프레임:**상위 타임프레임으로 방향을 결정하고, 하위 타임프레임으로 진입 타이밍을 포착하며, 타임프레임 간 충돌 시 상위를 우선합니다
-**Cardwell 방법론:**포지티브/네거티브 리버설은 히든 다이버전스의 개념을 확장하며, 다이버전스의 의미는 시장 국면에 따라 달라진다는 핵심 통찰을 제공합니다
