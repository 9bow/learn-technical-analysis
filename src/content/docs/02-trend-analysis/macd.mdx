## MACD란?

MACD(Moving Average Convergence Divergence, 이동평균 수렴·확산)는 Gerald Appel이 1979년에 개발한추**세추종형 모멘텀 지표입**니다. 두 이동평균선의 관계를 통해 추세의 방향, 강도, 전환점을 파악합니다
([Investopedia](https://www.investopedia.com/terms/m/macd.asp)).

## 구성 요소

MACD는 3가지 요소로 구성됩니다
([StockCharts](https://school.stockcharts.com/doku.php?id=technical_indicators:moving_average_convergence_divergence_macd)):

### 1. MACD선 (MACD Line)

$$MACD = EMA(12) - EMA(26)$$

- 12일 EMA에서 26일 EMA를 뺀 값
- 두 이동평균의차**이(Gap**)를 수치화
- 양수: 단기 MA가 장기 MA 위 → 강세
- 음수: 단기 MA가 장기 MA 아래 → 약세

### 2. 시그널선 (Signal Line)

$$Signal = EMA(MACD,\ 9)$$

- MACD선의 9일 EMA
- MACD선보다 더 부드러운 움직임
- MACD선과의 교차가 매매 신호

### 3. 히스토그램 (Histogram)

$$Histogram = MACD - Signal$$

- MACD선과 시그널선의 차이를 막대그래프로 표시
- 양수: MACD선이 시그널선 위 (강세 모멘텀)
- 음수: MACD선이 시그널선 아래 (약세 모멘텀)
- 막대의 크기 변화가 모멘텀의 가속/감속을 보여줌

## 기본 설정값

| 파라미터 | 기본값 | 의미 |
|---------|--------|------|
| 빠른 EMA | 12 | 단기 추세 (약 2주) |
| 느린 EMA | 26 | 장기 추세 (약 1개월) |
| 시그널 EMA | 9 | 매매 신호 라인 (약 1.5주) |

이 값은 Gerald Appel의 원래 설정이며, 대부분의 플랫폼에서 기본값으로 사용됩니다. 단기 매매자는 (5, 13, 1), 장기 투자자는 (24, 52, 18) 등으로 조정하기도 합니다.

## 매매 신호

### 1. 시그널선 교차 (Signal Line Crossover)

가장 기본적인 MACD 매매 신호입니다
([BabyPips](https://www.babypips.com/learn/forex/macd)):

-**강세 교차**: MACD선이 시그널선을위**로 교차**→ 매수 신호
-**약세 교차**: MACD선이 시그널선을아**래로 교차**→ 매도 신호

히스토그램으로 보면:
- 음수 → 양수 전환: 강세 교차
- 양수 → 음수 전환: 약세 교차

### 2. 제로선 교차 (Zero Line Crossover)

-**MACD가 0 위로**: 12일 EMA > 26일 EMA → 상승 추세 확인
-**MACD가 0 아래로**: 12일 EMA < 26일 EMA → 하락 추세 확인

시그널선 교차보다 느리지만 더 확정적인 신호입니다.

### 3. 히스토그램 반전

히스토그램이 극값(최대/최소)에서 줄어들기 시작하면:
- 양수 히스토그램 감소: 강세 모멘텀 약화 → 조정 또는 전환 가능
- 음수 히스토그램 감소(0에 가까워짐): 약세 모멘텀 약화 → 반등 가능

## MACD 다이버전스

다이버전스(Divergence)는 가격과 MACD**반대 방향으**로 움직이는 현상으로, 추세 전환의 조기 경고입니다:

### 강세 다이버전스 (Bullish Divergence)

-**가격**: 저점을 낮춤 (Lower Low)
-**MAC**D: 저점을 높임 (Higher Low)
-**의미**: 하락 모멘텀 약화 → 상승 반전 가능

### 약세 다이버전스 (Bearish Divergence)

-**가격**: 고점을 높임 (Higher High)
-**MAC**D: 고점을 낮춤 (Lower High)
-**의미**: 상승 모멘텀 약화 → 하락 반전 가능

### 다이버전스 주의사항

- 다이버전스는경**고 신호이**지 즉각적인 매매 신호가 아님
- 강한 추세에서는 다이버전스가 여러 번 나타나도 추세가 지속될 수 있음
- 반드시 가격의 실제 반전 확인(추세선 돌파, 패턴 완성 등) 후 행동

## MACD의 장단점

### 장점
- 추세 방향과 모멘텀을 동시에 파악
- 시각적으로 직관적 (히스토그램)
- 대부분의 시장과 타임프레임에서 활용 가능
- 다이버전스를 통한 조기 경고 기능

### 단점
-**후행 지표**: 이동평균 기반이므로 신호가 늦음
-**횡보장 취약**: 추세 없는 시장에서 잦은 거짓 교차
-**절대적 과매수/과매도 기준 없음**: RSI와 달리 고정된 상한/하한이 없음

## 실전 활용 팁

1.**MACD는 추세장에서 사용**: 횡보 구간에서는 다른 오실레이터(RSI, 스토캐스틱) 활용
2.**상위 타임프레임 확인**: 주봉 MACD로 대추세 확인 후 일봉 MACD로 진입
3.**다른 지표와 조합**: 지지/저항, 캔들스틱 패턴 등과 함께 사용
4.**히스토그램 주목**: 히스토그램의 크기 변화가 교차보다 먼저 모멘텀 변화를 알려줌

## 실전 적용

### 단계별 MACD 계산 워크스루

실제 숫자로 MACD를 단계별로 계산합니다. 설명을 단순하게 하기 위해 EMA 초기값은 SMA로 대체하고 짧은 기간 예시를 사용합니다.

**예시 데이터 (10일 종가**)
| 일차 | 종가 |
|------|------|
| 1 | 50,000 |
| 2 | 51,500 |
| 3 | 50,800 |
| 4 | 52,200 |
| 5 | 53,000 |
| 6 | 52,500 |
| 7 | 54,000 |
| 8 | 55,200 |
| 9 | 54,800 |
| 10 | 56,000 |

**1단계: EMA(3) 계산**(빠른 EMA — 실제는 12일이지만 예시는 3일)

승수: $k = \frac{2}{3 + 1} = 0.5$

초기값(1~3일 SMA): $\frac{50{,}000 + 51{,}500 + 50{,}800}{3} = 50{,}767$

| 일차 | 종가 | EMA(3) |
|------|------|--------|
| 3 | 50,800 | 50,767 |
| 4 | 52,200 | 50,767 × 0.5 + 52,200 × 0.5 = 51,483 |
| 5 | 53,000 | 51,483 × 0.5 + 53,000 × 0.5 = 52,242 |
| 6 | 52,500 | 52,242 × 0.5 + 52,500 × 0.5 = 52,371 |
| 7 | 54,000 | 52,371 × 0.5 + 54,000 × 0.5 = 53,186 |

**2단계: EMA(5) 계산**(느린 EMA — 실제는 26일이지만 예시는 5일)

승수: $k = \frac{2}{5 + 1} = 0.333$

초기값(1~5일 SMA): $\frac{50{,}000 + 51{,}500 + 50{,}800 + 52{,}200 + 53{,}000}{5} = 51{,}500$

| 일차 | 종가 | EMA(5) |
|------|------|--------|
| 5 | 53,000 | 51,500 |
| 6 | 52,500 | 51,500 × 0.667 + 52,500 × 0.333 = 51,833 |
| 7 | 54,000 | 51,833 × 0.667 + 54,000 × 0.333 = 52,722 |

**3단계: MACD선 계산**
`MACD(7일차) = EMA(3) - EMA(5) = 53,186 - 52,722 = 464`

MACD 값이 양수(+464)이므로 단기 추세가 장기 추세보다 위에 있고, 현재 상승 모멘텀이 존재합니다.

**4단계: 시그널선 계산**(MACD의 EMA(3) — 실제는 9일)

MACD 값이 누적되면 시그널선을 계산합니다. 시그널선은 MACD선의 평활화된 버전으로, MACD가 시그널을 위로 교차하면 매수 신호입니다.

:::tip
실제 트레이딩 플랫폼은 이 모든 계산을 자동으로 수행합니다. 직접 계산하는 목적은 MACD가 "두 EMA의 차이"라는 본질을 이해하기 위함입니다. 이를 알면 MACD가 왜 느린지, 왜 횡보장에서 거짓 신호를 내는지 직관적으로 파악됩니다.
:::

### MACD 다이버전스 식별 가이드

다이버전스는 가격과 MACD가 반대 방향으로 움직이는 현상입니다. 발견하는 절차를 정리합니다.

**약세 다이버전스 식별 절차**
```
1. 가격 차트에서 신고점(Higher High) 2개 이상 찾기
   예: 1차 고점 58,000원 → 2차 고점 61,000원

2. 같은 시점의 MACD 고점 확인
   예: 1차 MACD 고점 +250 → 2차 MACD 고점 +180

3. 가격은 올라갔는데(61,000 > 58,000) MACD는 낮아졌다면(180 < 250) → 약세 다이버전스

4. 추세선이나 지지선 이탈 등 가격의 실제 반전 신호 확인 후 대응
```

**강세 다이버전스 식별 절차**
```
1. 가격 차트에서 신저점(Lower Low) 2개 이상 찾기
   예: 1차 저점 45,000원 → 2차 저점 42,000원

2. 같은 시점의 MACD 저점 확인
   예: 1차 MACD 저점 -300 → 2차 MACD 저점 -180

3. 가격은 낮아졌는데(42,000 < 45,000) MACD는 높아졌다면(-180 > -300) → 강세 다이버전스

4. 하락 추세선 돌파나 저항선 상향 돌파 등 가격의 실제 반전 신호 확인 후 매수 검토
```

**다이버전스 신뢰도를 높이는 조건**
| 조건 | 신뢰도 영향 |
|------|-----------|
| 두 고점/저점의 거리가 충분히 멀수록 (최소 2~4주) | 신뢰도 증가 |
| 상위 타임프레임에서도 같은 다이버전스 확인 | 신뢰도 증가 |
| 주요 지지/저항 구간 근처에서 발생 | 신뢰도 증가 |
| 직전 추세가 강할수록 | 다이버전스 의미 약화 (강한 추세는 오래 지속됨) |

## 흔한 실수

### 실수 1: 다이버전스 발생 즉시 역방향 진입

약세 다이버전스가 보인다고 해서 즉시 매도하면, 강한 추세가 이어지며 손실이 커질 수 있습니다. 다이버전스는 "모멘텀이 약해지고 있다"는 신호이지 "지금 당장 뒤집힌다"는 신호가 아닙니다. 가격의 실제 반전 확인 후에 행동해야 합니다.

:::caution
강한 상승 추세에서는 약세 다이버전스가 3~5번 연속으로 나타나도 추세가 계속될 수 있습니다. 다이버전스만 보고 역방향 진입하면 여러 번 손절을 맞을 수 있습니다.
:::

### 실수 2: 횡보 구간에서 시그널선 교차 추종

가격이 횡보할 때 MACD선과 시그널선이 자주 교차합니다. 이 교차를 모두 매매 신호로 삼으면 수수료만 낭비하게 됩니다. MACD 교차 신호는 ADX가 25 이상인 추세 구간에서만 활용해야 합니다.

### 실수 3: MACD 기본값(12, 26, 9)을 무조건 변경

"나만의 최적 설정"을 찾으려고 기간을 바꾸면 과거 데이터에 과적합될 위험이 있습니다. Gerald Appel이 설계한 기본값(12, 26, 9)은 수십 년간 검증된 값입니다. 특별한 이유 없이 변경하지 않는 것이 좋습니다.

### 실수 4: 제로선 교차와 시그널선 교차를 동시에 기다림

두 신호가 동시에 발생하는 경우는 드물며, 기다리는 동안 큰 움직임을 놓칠 수 있습니다. 시그널선 교차(빠른 신호)와 제로선 교차(확정 신호) 중 자신의 전략에 맞는 하나를 선택하여 일관되게 사용하는 것이 낫습니다.

### 실수 5: 히스토그램의 절대값에 집착

히스토그램 값이 +500이라고 해서 +50일 때보다 반드시 더 강한 추세라고 볼 수 없습니다. 자산의 가격 수준에 따라 히스토그램의 절대값이 달라집니다. 히스토그램에서 중요한 것은절**대값의 크기가** 아니라방**향의 변화**(증가에서 감소로 전환)입니다.

## 엣지 케이스: MACD 분석이 어려워지는 상황

### 가격 급등·급락 후 MACD 극단값

가격이 하루 ±10% 이상 급등락하면 MACD가 극단적인 값에 도달합니다:

-**급등 **: MACD 히스토그램이 역사적 최대값에 도달. 이후 히스토그램이 줄어드는 것은 상승 모멘텀 약화이지 즉시 하락 신호가 아님
-**급락 **: MACD가 극단적 음수값. 이 상태에서 시그널선 교차가 발생해도 V자 반등인지 데드캣 바운스인지 구분하기 어려움
-**대응**: 극단적 MACD 값 이후에는 히스토그램의 방향 변화를 관찰하되, 반드시 가격 패턴과 거래량으로 추가 확인

### 가격 스케일 차이로 인한 MACD 비교 오류

서로 다른 자산의 MACD 히스토그램 절대값을 비교하는 것은 의미가 없습니다:

- 삼성전자(주가 약 80,000원)의 MACD 히스토그램 값과 비트코인(약 50,000달러)의 값은 직접 비교 불가
- 같은 자산이라도 가격이 크게 변한 전후의 MACD 절대값 비교도 주의 필요
-**대응**: MACD는 절대값이 아닌 방향과 추세(증가/감소)로만 해석

### 신규 상장 자산의 MACD

상장 초기 26일이 지나지 않은 자산에서는:

- EMA(26)를 계산할 데이터가 부족하여 MACD 값이 불안정
- 초기 30~50일간의 MACD 신호는 신뢰도가 낮음
-**대응**: 최소 50~60일 이상의 거래 데이터가 쌓인 후 MACD를 활용

### 파라미터 변경 시 주의

단기 매매를 위해 MACD 파라미터를 (5, 13, 1)처럼 축소하면:

- 신호가 빨라지지만 거짓 신호도 비례하여 증가
- 히스토그램이 너무 자주 방향을 바꿔 패턴 인식이 어려워짐
- 시그널 기간을 1로 설정하면 사실상 시그널선이 없어져 교차 신호를 사용할 수 없음

## 핵심 요약

1. MACD는 12일 EMA와 26일 EMA의 차이로 계산되며, 추세와 모멘텀을 동시에 보여줍니다.
2. 시그널선 교차, 제로선 교차, 히스토그램 반전이 주요 매매 신호입니다.
3. 다이버전스는 추세 전환의 조기 경고 신호이지만, 단독으로 매매하면 위험합니다.
4. MACD는 추세장에서 효과적이며, 횡보장에서는 다른 지표를 병행해야 합니다.
