## 추세선이란?

추세선(Trendline)은 차트에서연**속적인 고점 또는 저점을 직선으로 연결한** 것입니다. 추세의 방향과 속도를 시각적으로 표현하며, 동적 지지/저항 역할을 합니다
([Investopedia](https://www.investopedia.com/terms/t/trendline.asp)).

## 추세선 그리기

### 상승 추세선

**최소 2개의 Higher Low를 연결하**여 아래에서 위로 우상향하는 선을 그립니다:

```
          /
         / HH₂
        /
   HH₁/
      /
     / HL₂   ← 두 번째 접점
    /
HL₁/          ← 첫 번째 접점
──/───────────── 상승 추세선
```

- 저점(HL)을 연결 (캔들의 꼬리 또는 몸통)
- 최소 2개 접점 필요,**3개 이상이면 신뢰도 높음**- 추세선 위에 가격이 있으면 상승 추세 유효

### 하락 추세선

**최소 2개의 Lower High를 연결하**여 위에서 아래로 우하향하는 선을 그립니다:

```
──\───────────── 하락 추세선
LH₁\          ← 첫 번째 접점
    \
     \ LH₂   ← 두 번째 접점
      \
       \
   LL₁  \ LL₂
          \
```

- 고점(LH)을 연결
- 추세선 아래에 가격이 있으면 하락 추세 유효

### 추세선 작성 규칙

1.**최소 2개 접점**: 1개는 점, 2개는 선, 3개 이상은 확인된 추세선
2.**꼬리 vs 몸통**: 꼬리 끝을 연결하는 것이 일반적이지만, 일관성이 중요
3.**관통 허용**: 약간의 관통(Wick Penetration)은 허용 — 종가 기준으로 판단
4.**각도**: 45도에 가까운 추세선이 가장 지속가능. 너무 가파르면 곧 깨질 가능성

## 채널 (Channel)

채널은 추세선과평**행선을** 함께 그려 가격이 움직이는 범위를 나타냅니다
([StockCharts](https://school.stockcharts.com/doku.php?id=chart_analysis:trend_lines)).

### 상승 채널

```
        ╱─── 채널 상단 (저항, 이익실현 영역)
       ╱  /
      ╱  /
     ╱  /
    ╱  /
   ╱──── 채널 하단 = 상승 추세선 (지지, 매수 영역)
```

- **하단**: 상승 추세선 (HL 연결) → 지지 역할 → 매수 영역
- **상단**: 추세선과 평행한 선 (HH 연결) → 저항 역할 → 이익실현 영역

### 하락 채널

```
   ╲──── 채널 상단 = 하락 추세선 (저항, 매도 영역)
    ╲  \
     ╲  \
      ╲  \
       ╲  \
        ╲─── 채널 하단 (지지, 이익실현 영역)
```

### 채널 매매 전략

| 위치 | 상승 채널 | 하락 채널 |
|------|----------|----------|
|**채널 하단 도달**| 매수 기회 | 이익실현 (숏 포지션) |
|**채널 상단 도달**| 이익실현 또는 부분 매도 | 매도(숏) 기회 |
|**채널 상단 돌파**| 추세 가속 → 추격 매수 주의 | 추세 전환 가능 |
|**채널 하단 이탈**| 추세 전환 가능 | 추세 가속 → 추격 매도 주의 |

## 추세선 돌파

추세선이 돌파되면 추세의 약화 또는 전환을 시사합니다:

### 유효한 돌파의 조건

1.**종가 기준**: 일중 관통이 아닌 종가 기준으로 추세선 아래(위)에서 마감
2.**거래량 증가**: 돌파 시 평균 이상의 거래량이 동반
3.**후속 움직임**: 돌파 후 1~3일간 돌파 방향으로 지속
4.**되돌림 테스트**: 돌파 후 이전 추세선으로 돌아와 테스트(지지↔저항 전환 확인)

### 거짓 돌파 (False Breakout)

- 일시적으로 추세선을 관통했다가 다시 원래 추세로 복귀
- 거래량이 낮거나, 종가가 추세선 안쪽에서 마감
- 거짓 돌파 후 원래 추세가 더 강하게 재개되는 경우도 많음

## 추세선의 기울기 변화

추세가 진행되면서 기울기가 변할 수 있습니다:

### 팬 원리 (Fan Principle)

추세가 가속되거나 감속되면 새로운 추세선을 그릴 수 있습니다:

1.**첫 번째 추세선**: 초기 추세의 기울기
2.**두 번째 추세선**: 추세 가속 시 더 가파른 기울기
3.**세 번째 추세선**: 추가 가속, 그러나 지속 불가능한 기울기

세 번째 추세선이 깨지면 추세 전체의 반전을 시사합니다
([BabyPips](https://www.babypips.com/learn/forex/trend-lines)).

## 실전 적용

### 추세선 그리기 규칙: 접점 수와 각도의 의미

추세선의 신뢰도는 접점 수와 기울기(각도)에 크게 좌우됩니다.

**접점 수와 신뢰도**
| 접점 수 | 신뢰도 | 설명 |
|---------|--------|------|
| 2개 | 낮음 | 추세선 후보. 아직 확인 단계 |
| 3개 | 보통 | 확인된 추세선. 실전 활용 가능 |
| 4개 이상 | 높음 | 강한 추세선. 돌파 시 큰 움직임 예상 |

접점 수가 많을수록 해당 추세선을 시장 참여자들이 주목하고 있다는 의미입니다. 4번 이상 지지/저항으로 작용한 추세선이 돌파될 때는 더 강한 방향성이 나타나는 경우가 많습니다.

**각도(기울기)의 의미**
| 각도 범위 | 의미 | 지속 가능성 |
|----------|------|------------|
| 15~30도 | 완만한 추세 | 높음 — 장기간 유지 가능 |
| 30~50도 | 건강한 추세 | 중간 — 조정이 올 수 있지만 추세 지속 |
| 50~70도 | 가파른 추세 | 낮음 — 단기간에 과열, 조정 임박 가능 |
| 70도 이상 | 급등/급락 | 매우 낮음 — 거의 반드시 깨짐 |

:::tip
45도에 가까운 추세선이 가장 이상적이라고 알려져 있지만, 실제로는 30~50도 범위면 건강한 추세로 볼 수 있습니다. 너무 가파른 추세선(60도 이상)은 재조정 추세선이 필요한 신호입니다.
:::

**실제 가격 예시로 각도 확인**
```
상승 추세선 예시:
  - 1월 1일 접점: 50,000원
  - 2월 1일 접점: 55,000원 (31일 후 +10%)
  - 3월 1일 접점: 60,000원 (31일 후 +9.1%)

→ 월 약 10% 상승률의 추세선, 건강한 각도 범위
→ 만약 3월 1일 접점이 75,000원이라면 너무 가팔라서 지속 가능성 낮음
```

### 채널 돌파 매매 전략

채널에서 돌파가 발생할 때의 매매 전략을 구체적인 규칙과 함께 설명합니다.

**상승 채널에서 상단 돌파**
상승 채널의 상단 저항선을 강하게 돌파하면 추세가 가속됩니다.

| 단계 | 행동 | 구체적 기준 |
|------|------|-----------|
| 돌파 확인 | 관망 | 종가 기준 채널 상단 + 0.5% 이상에서 마감 |
| 거래량 확인 | 조건 검토 | 최근 20일 평균 거래량의 1.5배 이상 |
| 진입 | 매수 | 돌파 다음날 시가 또는 채널 상단 되돌림 테스트 시 |
| 손절 | 설정 | 채널 상단 아래로 종가 기준 재진입 시 청산 |
| 목표가 | 계산 | 채널 폭만큼 상방 투영: 목표가 = 돌파 지점 + 채널 폭 |

**채널 폭 계산 예시**
- 채널 하단(추세선): 3월 1일 기준 50,000원
- 채널 상단(평행선): 3월 1일 기준 58,000원
- 채널 폭: $58{,}000 - 50{,}000 = 8{,}000$원

상단 돌파 발생 (돌파 지점 58,500원):

$$\text{목표가} = 58{,}500 + 8{,}000 = 66{,}500\text{원}$$

**상승 채널에서 하단 이탈**
상승 채널의 추세선(하단)이 이탈되면 추세 전환 가능성이 높습니다.

| 단계 | 행동 | 구체적 기준 |
|------|------|-----------|
| 이탈 확인 | 경계 | 종가 기준 채널 하단 이하에서 마감 |
| 매도/손절 | 롱 포지션 청산 | 이탈 확인 즉시 또는 채널 하단 되돌림 저항 확인 후 |
| 공매도 진입 | 선택적 | 채널 하단을 되돌림 후 저항 확인 시 |
| 손절 | 설정 | 채널 하단 위로 종가 기준 재진입 시 |

:::caution
채널 하단 이탈이 항상 추세 전환을 의미하지는 않습니다. 거짓 이탈(False Breakdown) 후 채널 안으로 돌아오는 경우도 있습니다. 반드시 종가 기준, 거래량 동반 여부를 확인하고 대응하세요.
:::

## 흔한 실수

### 실수 1: 2개의 접점으로 확정된 추세선 취급

2개의 점은 직선을 그리기에 충분하지만, 그것이 시장이 인식하는 지지/저항선임을 증명하지는 않습니다. 2개 접점 추세선은 "가설"이고, 3번째 접점에서 반응이 나올 때 비로소 "확인된" 추세선이 됩니다. 2개 접점만으로 강하게 베팅하지 마세요.

### 실수 2: 너무 많은 점을 억지로 연결

원하는 추세선을 미리 머릿속에 그린 후 그에 맞는 점들만 선택하는 확증 편향입니다. 추세선은 자연스럽게 연결되어야 합니다. 특정 고점이나 저점을 무시해야만 그릴 수 있는 추세선은 신뢰하기 어렵습니다.

:::note
추세선 분석에는 어느 정도 주관이 개입됩니다. 여러 분석가가 동일한 차트를 보고 다른 추세선을 그릴 수 있습니다. 중요한 것은일**관된 기준을** 유지하는 것입니다.
:::

### 실수 3: 일중 꼬리 관통을 추세선 이탈로 판단

일중에 추세선을 잠시 뚫었다가 종가는 추세선 안쪽에서 마감하는 경우가 많습니다. 이를 이탈로 판단하고 포지션을 청산하면 좋은 기회를 놓치게 됩니다. 추세선 이탈은 반드시종**가 기준으**로 판단해야 합니다.

### 실수 4: 채널 상단에서 추가 매수

상승 채널의 상단에 도달했을 때 "추세가 강하니 더 오를 것"이라는 생각으로 추가 매수하는 것은 고점에서 매수하는 전형적인 실수입니다. 채널 상단은 이익실현 구간이고, 채널 하단이 매수 구간입니다. 채널 안에서는 하단 근처에서 매수하고 상단 근처에서 매도하는 원칙을 지켜야 합니다.

### 실수 5: 가파른 추세선 이탈 후 즉시 역방향 진입

가파른 추세(각도 60도 이상)가 깨지면 급격한 하락이 올 것으로 기대하고 즉시 공매도를 하는 경우가 있습니다. 그러나 가파른 추세선이 깨져도 더 완만한 추세선이 형성되며 상승이 이어지는 경우가 많습니다(팬 원리). 추세선 이탈 자체보다 전체적인 추세 구조(HH/HL 패턴)가 무너지는지 확인 후 역방향 전략을 고려하세요.

## 엣지 케이스: 추세선·채널 분석이 어려워지는 상황

### 갭을 동반한 추세선 돌파

가격이 갭을 만들며 추세선을 돌파하는 경우:

- **갭 하향 이탈**: 상승 추세선을 갭으로 이탈하면 일반적인 종가 기준 이탈보다 강한 하락 신호. 갭의 상단이 새로운 저항이 될 수 있음
- **갭 상향 돌파**: 하락 추세선을 갭으로 상향 돌파하면 강한 매수 신호. 되돌림 테스트 없이 상승이 가속될 가능성
- **대응**: 갭 돌파는 거래량 확인 조건을 완화해도 유효한 신호로 해석 가능. 단, 소진형 갭(Exhaustion Gap)인지 돌파형 갭(Breakaway Gap)인지 확인 필요

### 추세선 각도가 70도 이상인 급등/급락

- 매우 가파른 추세선(70도 이상)은 지속 가능성이 낮지만, 깨질 때는 급격한 반대 방향 움직임이 아닌 더 완만한 추세선으로 재조정되는 경우가 많음 (팬 원리)
- **수치 예시**: 30일간 100% 상승한 추세선(월 33% 수익률)이 깨지면 즉각 하락이 아니라 15~20도의 완만한 새 추세선이 형성되며 상승 지속

### 채널이 매우 좁은 경우 (타이트 채널)

채널 폭이 가격의 2~3% 이내로 매우 좁을 때:

- 채널 내 매매(하단 매수·상단 매도)의 수익이 수수료·슬리피지를 감당하기 어려움
- 타이트 채널은 에너지 축적 구간으로 볼린저 밴드 수축처럼 큰 움직임이 임박했을 가능성
- **대응**: 타이트 채널에서는 내부 매매보다 돌파 방향으로의 진입을 준비

### 로그 스케일과 산술 스케일의 추세선 차이

- 가격이 2배 이상 변한 장기 차트에서는 산술 스케일로 그린 추세선과 로그 스케일로 그린 추세선의 각도가 크게 다름
- 비트코인처럼 수천 % 상승한 자산의 장기 추세선은 반드시 로그 스케일로 그려야 의미 있는 추세선을 얻을 수 있음
- **대응**: 장기 차트(1년 이상)는 로그 스케일을, 단기 차트(수주 이내)는 산술 스케일을 사용

:::note[관련 챕터]

## 핵심 요약

1. 상승 추세선은 HL을, 하락 추세선은 LH를 연결합니다.
2. 3개 이상의 접점을 가진 추세선이 더 신뢰할 수 있습니다.
3. 채널은 추세선 + 평행선으로 매매 범위를 제공합니다.
4. 추세선 돌파는 종가, 거래량, 후속 움직임으로 유효성을 확인합니다.
