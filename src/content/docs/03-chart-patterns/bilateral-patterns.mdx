## 양방향 패턴 (Bilateral Patterns) 개요

양방향 패턴은 삼각형(Triangle) 형태의 차트 패턴으로, 이론적으로 상향 또는 하향 어느 방향으로도 돌파될 수 있습니다. 그러나 각 삼각형 유형은 통계적으로 특정 방향으로 돌파되는 편향성을 보입니다.

[StockCharts](https://school.stockcharts.com/doku.php?id=chart_analysis:chart_patterns:triangle_continuation_pattern)에 따르면 삼각형 패턴은 가격 변동폭이 점차 수렴하는 형태로, 패턴이 완성될수록 에너지가 응축되어 결국 강한 돌파로 이어집니다.

**삼각형 패턴의 공통 특성:**- 두 개의 수렴하는 추세선으로 구성됩니다
- 패턴 내에서 거래량이 점차 감소합니다 (수렴 현상)
- 최소 4개의 접점(각 추세선에 2개씩)이 필요합니다
- 일반적으로 삼각형 완성의 1/2~3/4 지점에서 돌파가 발생합니다
- 목표가는 삼각형의 최대 높이를 돌파 지점에 적용합니다

## 1. 상승 삼각형 (Ascending Triangle)

상승 삼각형은 수평 저항선과 상승하는 지지선으로 구성됩니다. 저점이 점차 높아지면서 매수 압력이 증가하고 있음을 나타냅니다.

### 패턴 구조

```
저항선: ----+-------+-------+-------돌파→
           |       |       |      /
           |   /   |   /   |   /
지지선:  /   /       /       /
       (상승하는 저점들)
```

**구성 요소:**-**수평 저항선**: 같은 수준에서 2회 이상 반발한 고점들을 연결
- **상승 지지선**: 점차 높아지는 저점들을 연결
- **수렴점**: 두 선이 만나는 지점 (일반적으로 돌파는 그 전에 발생)

### 편향성과 돌파 확률

[Investopedia](https://www.investopedia.com/terms/t/triangle.asp)에 따르면 상승 삼각형은상**향 돌파 편향성을** 가집니다.

| 돌파 방향 | 확률 | 의미 |
|---------|------|------|
| 상향 돌파 | 약 75% | 매수 압력이 저항을 극복 |
| 하향 돌파 | 약 25% | 예상치 못한 약세 반전 |

**상승 삼각형이 상향 편향인 이유:**- 저점이 높아진다는 것은 매도자들이 점차 낮은 가격에서 매도를 포기함을 의미합니다
- 매수자들이 저항 수준까지 가격을 밀어올리는 의지를 반복적으로 보여줍니다
- 결국 저항선을 돌파할 에너지가 축적됩니다

### 거래량 특성

```
패턴 시작 → 거래량 많음
패턴 진행 → 거래량 점진적 감소 (수렴)
돌파 직전 → 거래량 최저점
돌파 시   → 거래량 급증 (확인)
```

**거래량 수렴 현상**: 패턴이 진행될수록 시장 참여자들의 관심이 집중되면서 가격 변동폭은 좁아지지만, 내부적으로 에너지가 응축됩니다. 이 응축된 에너지가 돌파 시 방출됩니다.

### 목표가 측정

$$\text{목표가} = \text{돌파가} + \text{삼각형 최대 높이}$$

$$\text{삼각형 최대 높이} = \text{저항선} - \text{첫 번째 저점}$$

**예시:**
| 항목 | 가격 |
|------|------|
| 수평 저항선 | 55,000원 |
| 삼각형 시작 저점 | 50,000원 |
| 삼각형 높이 | 5,000원 |
| 상향 돌파가 | 55,500원 |
| 목표가 | 60,500원 |

## 2. 하락 삼각형 (Descending Triangle)

하락 삼각형은 상승 삼각형의 반대 형태로, 수평 지지선과 하락하는 저항선으로 구성됩니다. 고점이 점차 낮아지면서 매도 압력이 증가하고 있음을 나타냅니다.

### 패턴 구조

```
저항선:  \   \       \       \
          \   \   /   \   /   \
           \  |   \   |   \   |
지지선: ----+-------+-------+-------돌파↓
```

**구성 요소:**-**수평 지지선**: 같은 수준에서 2회 이상 지지된 저점들을 연결
- **하락 저항선**: 점차 낮아지는 고점들을 연결

### 편향성과 돌파 확률

하락 삼각형은하**향 돌파 편향성을** 가집니다.

| 돌파 방향 | 확률 | 의미 |
|---------|------|------|
| 하향 돌파 | 약 72% | 매도 압력이 지지를 극복 |
| 상향 돌파 | 약 28% | 예상치 못한 강세 반전 |

**하락 삼각형이 하향 편향인 이유:**- 고점이 낮아진다는 것은 매수자들이 점차 낮은 가격에서 매수를 포기함을 의미합니다
- 매도자들이 지지 수준까지 가격을 밀어내리는 압력이 지속됩니다
- 결국 지지선을 하향 돌파하는 에너지가 축적됩니다

### 상승 vs 하락 삼각형 비교

| 비교 항목 | 상승 삼각형 | 하락 삼각형 |
|---------|----------|----------|
| 수평선 위치 | 저항선 (상단) | 지지선 (하단) |
| 기울어진 선 | 지지선 (상승) | 저항선 (하락) |
| 극값 방향 | 저점 상승 | 고점 하락 |
| 주요 편향 | 상향 돌파 | 하향 돌파 |
| 시장 심리 | 매수자 우세 | 매도자 우세 |

### 하락 삼각형 목표가

$$\text{목표가} = \text{돌파가} - \text{삼각형 최대 높이}$$

$$\text{삼각형 최대 높이} = \text{첫 번째 고점} - \text{수평 지지선}$$

**예시:**
| 항목 | 가격 |
|------|------|
| 삼각형 시작 고점 | 50,000원 |
| 수평 지지선 | 45,000원 |
| 삼각형 높이 | 5,000원 |
| 하향 돌파가 | 44,500원 |
| 목표가 | 39,500원 |

## 3. 대칭 삼각형 (Symmetrical Triangle)

대칭 삼각형은 하락하는 저항선과 상승하는 지지선이 수렴하는 패턴입니다. 두 추세선이 대칭적으로 수렴하므로 방향성 편향이 상대적으로 적습니다. [BabyPips](https://www.babypips.com/learn/forex/triangle-chart-patterns)에 따르면 대칭 삼각형은 진정한 양방향 패턴으로, 기존 추세가 있는 경우 지속 패턴으로, 추세 전환점에서는 반전 패턴으로 기능할 수 있습니다.

### 패턴 구조

```
  \                        /
   \     /\     /\        /  돌파 후 상승
    \   /  \   /  \      /
     \ /    \ /    \    /
      X      X      \  /
     / \    / \      \/
    /   \  /   \
   /     \/     \
  /               \  또는 돌파 후 하락
```

### 대칭 삼각형의 돌파 방향 판단

방향성 편향이 적은 대칭 삼각형에서 돌파 방향을 예측하는 방법:

| 신호 | 해석 |
|------|------|
| 기존 상승 추세에서 형성 | 상향 돌파 확률 높음 (지속 패턴) |
| 기존 하락 추세에서 형성 | 하향 돌파 확률 높음 (지속 패턴) |
| 추세 전환점에서 형성 | 방향 불명확, 돌파 확인 필요 |
| 거래량이 한 방향에서 증가 | 해당 방향으로 돌파 가능성 |

**돌파 전 신호 확인법:** 1. 패턴 내 스윙 고점에서 거래량 관찰
2. 스윙 저점에서 거래량 관찰
3. 어느 쪽에서 거래량이 더 크게 나타나는지 확인
4. 거래량이 많은 방향으로 돌파 가능성이 높음

### 대칭 삼각형 거래량 특성

```
접점 1: 거래량 높음
접점 2: 거래량 감소
접점 3: 거래량 더 감소
접점 4: 거래량 최저
돌파:   거래량 급증 (방향 확인)
```

### 돌파 타이밍

삼각형 패턴에서 돌파는 일반적으로 삼각형 완성의**1/2~3/4 지점사**이에서 발생합니다.

```
삼각형 완성 비율: |----1/2----|----3/4----|----수렴점----|
이상적 돌파 구간:       [--------돌파 발생 구간--------]
너무 늦은 돌파:                                    [위험]
```

- 수렴점 이후에도 돌파가 발생하지 않으면 패턴 신뢰도 감소
- 수렴점 이전 1/4 구간에서의 조기 돌파도 신뢰도 감소

### 목표가 측정

$$\text{목표가} = \text{돌파가} \pm (\text{첫 번째 고점} - \text{첫 번째 저점})$$

## 4. 수치 계산 워크스루: 상승 삼각형 완전 예시

실제 데이터를 사용하여 상승 삼각형 패턴의 전 과정을 단계별로 계산합니다.

**배경:** 코스닥 종목의 상승 삼각형 형성 (가상 데이터)

| 접점 | 날짜 | 가격 | 종류 | 거래량 |
|------|------|------|------|--------|
| 접점 1 (저항) | 1월 10일 | 55,000원 | 저항 고점 | 1,800만 주 |
| 접점 2 (지지) | 1월 24일 | 50,000원 | 상승 저점 1 | 900만 주 |
| 접점 3 (저항) | 2월 8일 | 55,000원 | 저항 고점 | 1,200만 주 |
| 접점 4 (지지) | 2월 20일 | 52,000원 | 상승 저점 2 | 650만 주 |
| 접점 5 (저항) | 3월 5일 | 55,000원 | 저항 고점 | 900만 주 |
| 접점 6 (지지) | 3월 15일 | 53,500원 | 상승 저점 3 | 450만 주 |
| 돌파 | 3월 22일 | 55,800원 | 상향 돌파 | 4,200만 주 |

**단계 1: 삼각형 높이 측정**
$$\text{삼각형 높이} = \text{수평 저항선} - \text{첫 번째 저점} = 55{,}000 - 50{,}000 = 5{,}000\text{원}$$

**단계 2: 수렴점 계산**
저점들의 상승 기울기: 50,000 → 52,000 → 53,500 (거래일 기준 선형 보간)

상승 지지선이 55,000원(저항선)과 만나는 지점이 수렴점입니다. 이 예시에서는 약 4월 10일경으로 추정됩니다. 돌파는 3월 22일, 즉 수렴점 3주 전 발생 →삼**각형 완성의 약 70% 시점에서 돌파 (이상적**)✓

**단계 3: 목표가 계산**
$$\text{목표가} = \text{돌파가} + \text{삼각형 높이} = 55{,}800 + 5{,}000 = 60{,}800\text{원}$$

**단계 4: 거래량 수렴 확인**
| 구간 | 고점 거래량 | 저점 거래량 | 수렴 여부 |
|------|-----------|-----------|---------|
| 1차 고점/저점 | 1,800만 | 900만 | 기준 |
| 2차 고점/저점 | 1,200만 | 650만 | 33% 감소 ✓ |
| 3차 고점/저점 | 900만 | 450만 | 50% 감소 ✓ |
| 돌파 | 4,200만 | - | 기준 대비 2.3배 급증 ✓ |

**단계 5: 손절매 설정**
$$\text{손절매} = \text{삼각형 직전 저점} - \text{버퍼} = 53{,}500 - 1{,}000 = 52{,}500\text{원}$$

$$RR = \frac{60{,}800 - 55{,}800}{55{,}800 - 52{,}500} = \frac{5{,}000}{3{,}300} \approx 1:1.5$$

## 5. 실전 적용

### 시나리오 A: 상승 삼각형 매수 진입

**상황:** 반도체 섹터 ETF가 9개월 상승 추세 중 상승 삼각형 형성

```
진입 결정 흐름:

1. 패턴 확인
   → 수평 저항선 55,000원에서 3회 반발 확인
   → 상승 저점 3개 이상 확인
   → 거래량 수렴 진행 중

2. 돌파 조건 설정
   → 종가 기준 55,000원 상향 돌파
   → 돌파 당일 거래량 > 20일 평균의 1.5배

3. 진입 방법
   - 즉시 진입: 55,800원 (돌파 캔들 종가)
   - 보수적 진입: 돌파 후 55,000원 되돌림(pull-back) 지지 확인 후 진입

4. 손절/목표
   - 손절: 직전 저점(53,500원) 아래 52,500원
   - 1차 목표: 58,000원 (높이 50%) → 절반 청산
   - 2차 목표: 60,800원 (높이 100%) → 잔여 청산
```

### 시나리오 B: 하락 삼각형 하향 돌파 매매

**상황:** 은행주가 6개월 하락 추세 중 하락 삼각형 형성

| 항목 | 수치 |
|------|------|
| 삼각형 시작 고점 | 48,000원 |
| 수평 지지선 | 42,000원 |
| 삼각형 높이 | 6,000원 |
| 하향 돌파가 | 41,500원 |
| 목표가 | 41,500 - 6,000 =**35,500**|
| 손절매 | 직전 고점 위: 44,000원 |

**편향성 확인:** 하락 삼각형은 약 72% 확률로 하향 돌파. 하락 추세 중 형성이므로 편향이 더욱 강화됩니다.

### 시나리오 C: 대칭 삼각형 방향 불확실 시 대응

**상황:** 상승 추세 없이 대칭 삼각형 형성 → 방향 불명확

```
방향 판단 체크리스트:
□ 직전 추세 방향: 없음 (추세 전환점)
□ 상승 캔들 거래량 vs 하락 캔들 거래량 비교
   → 상승 구간 평균 거래량: 850만 주
   → 하락 구간 평균 거래량: 1,200만 주
   → 하락 구간 거래량이 더 많음 → 하향 돌파 편향

대응:
- 하향 돌파 대기
- 상향 돌파 시에도 즉시 진입하지 않고 1~2거래일 확인 후 진입
- 방향성이 불확실한 경우 포지션 크기를 표준의 50% 이하로 제한
```

## 6. 흔한 실수

### 실수 1: 접점이 부족한 삼각형 확인

고점 1개, 저점 1개만 연결하고 "삼각형이 형성됐다"고 판단합니다. 유효한 삼각형은각** 추세선에 최소 2개씩, 총 4개 이상의 접점이** 필요합니다. 접점이 부족하면 그것은 삼각형이 아니라 단순한 수렴 채널의 시작일 뿐입니다.

### 실수 2: 수렴점에서 돌파가 없어도 계속 대기

삼각형 수렴점(두 추세선이 만나는 지점)이 지났는데도 돌파가 없습니다. 수렴점 이후에는 패턴의 에너지가 소진되어 신호 신뢰도가 크게 낮아집니다. 수렴점을 20% 이상 지난 패턴은 포기하고 새로운 패턴을 탐색합니다.

### 실수 3: 대칭 삼각형을 항상 지속 패턴으로 해석

"대칭 삼각형은 추세 지속 패턴"이라는 공식을 외운 뒤, 하락 추세 중 형성된 대칭 삼각형에서도 상향 돌파를 기대합니다. 대칭 삼각형은기**존 추세가 있을 때는 지속 패턴이**지만,**추세 전환점에서는 반전 패턴이** 될 수 있습니다. 항상 맥락을 확인합니다.

### 실수 4: 상승 삼각형에서 하향 돌파 시 당황

상승 삼각형에서 25% 확률로 하향 돌파가 발생합니다. 이미 상향 돌파를 기대하며 매수했다면 손절매를 즉시 실행하지 않고 "곧 회복하겠지"하며 버팁니다.**모든 패턴에는 실패 확률이 존재하**므로 손절매 원칙을 사전에 설정하고 철저히 따릅니다.

### 실수 5: 저거래량 돌파 즉시 진입

삼각형 상단을 돌파했지만 거래량이 평균 수준입니다. 그래도 "돌파는 돌파"라며 진입합니다. 삼각형 패턴에서 특히 거래량 확인이 중요합니다. 패턴 내 거래량이 크게 수렴했기 때문에 돌파 시 거래량 급증이 없다면에**너지 방출이 없는 거짓 돌파일** 가능성이 높습니다.

## 7. 엣지 케이스: 이상 시장 상황

### 거짓 돌파 (False Breakout) 후 역방향 폭락

저거래량으로 상향 돌파 후 즉시 패턴 내로 복귀하고, 이후 반대 방향으로 강하게 돌파하는 경우입니다. 이를 "역거짓 돌파(Failed Breakout → Breakdown)"라고 합니다.

**식별:** 돌파 캔들이 긴 위 꼬리를 형성하거나, 다음 캔들이 바로 패턴 내로 복귀한다면 경계합니다.

**대응:** 거짓 돌파 후 반대 방향으로의 돌파는 오히려 강력한 신호입니다. 처음 돌파 방향과 반대로 진입하는 전략을 고려할 수 있습니다.

### 삼각형이 너무 작은 경우 (마이크로 패턴)

1~2주 이내 단기간에 형성된 소규모 삼각형은 신뢰도가 낮습니다. 삼각형은 최소**3~4주 이상형**성되어야 의미 있는 신호를 제공합니다. 단기 소규모 패턴은 단순 노이즈일 가능성이 높으므로 더 상위 타임프레임에서 패턴을 확인합니다.

### 시장 급등락 중 패턴 붕괴

삼각형 형성 중 시장 전체의 급등락(뉴스, 이벤트)으로 패턴이 붕괴되는 경우입니다. 이 경우 새로운 패턴이 형성될 때까지 대기합니다. 기존 패턴 분석에 집착하면 잘못된 판단으로 이어집니다.

## 9. 거래량 수렴 현상 심층 분석

거래량 수렴은 삼각형 패턴의 핵심 특성입니다. 이 현상은 단순한 거래량 감소가 아니라 시장의 심리적 변화를 반영합니다.

### 수렴의 심리적 해석

| 단계 | 거래량 | 시장 심리 |
|------|--------|---------|
| 패턴 초기 | 높음 | 방향 불확실, 매수/매도 모두 적극적 |
| 패턴 중기 | 감소 | 추세 불확실, 관망 증가 |
| 패턴 후기 | 최저 | 에너지 응축, 결정적 움직임 대기 |
| 돌파 시 | 급증 | 누적된 에너지 방출, 방향 결정 |

### 거짓 돌파 (False Breakout) 식별

거래량 확인 없는 돌파는 거짓 돌파일 가능성이 높습니다.

**거짓 돌파의 특징:**- 돌파 시 거래량이 평균 이하
- 돌파 후 바로 패턴 내로 되돌아옴
- 돌파 캔들의 몸통이 작고 꼬리가 긺

**진짜 돌파의 특징:**- 돌파 시 거래량이 평균의 1.5배 이상
- 돌파 후 패턴 외부에서 지지/저항 형성
- 돌파 캔들의 몸통이 크고 꼬리가 짧음

## 5. 세 삼각형 패턴 종합 비교

| 항목 | 상승 삼각형 | 하락 삼각형 | 대칭 삼각형 |
|------|----------|----------|----------|
| 상단 추세선 | 수평 | 하락 | 하락 |
| 하단 추세선 | 상승 | 수평 | 상승 |
| 주요 편향 | 상향 (75%) | 하향 (72%) | 기존 추세 방향 |
| 시장 특성 | 매수 우세 | 매도 우세 | 균형 |
| 주요 용도 | 지속/반전 | 지속/반전 | 지속/반전 |
| 거래량 수렴 | 있음 | 있음 | 있음 |

---

## 핵심 요약

- **상승 삼각형은** 수평 저항선과 상승 지지선으로 구성되며, 약 75%의 확률로 상향 돌파하는 편향성을 가집니다
- **하락 삼각형은** 수평 지지선과 하락 저항선으로 구성되며, 약 72%의 확률로 하향 돌파하는 편향성을 가집니다
- **대칭 삼각형은** 두 추세선이 대칭적으로 수렴하는 진정한 양방향 패턴으로, 기존 추세 방향으로 돌파되는 경향이 있으며 거래량 수렴 후 급증으로 돌파 방향을 확인합니다
- **거래량 수렴은** 삼각형 패턴의 핵심 특성으로, 패턴 내 거래량 감소 후 돌파 시 급증이 나타나면 신뢰도가 높으며, 거래량 확인 없는 돌파는 거짓 돌파 위험이 있습니다
