## 반전 패턴 (Reversal Patterns) 개요

반전 패턴은 현재 추세가 끝나고 반대 방향으로 전환될 가능성을 시사하는 차트 구조입니다. 상승 추세 끝에 나타나는 패턴은 하락 반전을, 하락 추세 끝에 나타나는 패턴은 상승 반전을 예고합니다.

**반전 패턴의 공통 특성:**- 명확한 기존 추세가 선행되어야 합니다
- 패턴 완성에는 시간이 필요하며, 형성 기간이 길수록 이후 움직임이 큰 경향이 있습니다
- 거래량 (Volume)이 패턴 확인의 핵심 증거입니다
- 넥라인 (Neckline) 또는 핵심 지지/저항 돌파로 패턴이 최종 확인됩니다

## 1. 헤드앤숄더 (Head and Shoulders)

<img src="/technical-analysis/images/diagrams/head-shoulders.svg" alt="헤드앤숄더 패턴: 왼쪽 어깨, 머리, 오른쪽 어깨와 넥라인 구조 다이어그램" style="width:100%;max-width:600px;margin:1rem auto;display:block;" />

헤드앤숄더는 기술적 분석에서 가장 신뢰도 높은 반전 패턴 중 하나로, 상승 추세 종료 후 하락 전환을 예고합니다. [Investopedia](https://www.investopedia.com/terms/h/head-shoulders.asp)에 따르면 이 패턴은 세 개의 연속적인 고점으로 구성됩니다.

### 패턴 구성 요소

```
     Head (머리)
      /\
     /  \
    /    \
   /  LS  \  RS
  /   /\   /\
 /   /  \ /  \
------Neckline------
```

| 구성 요소 | 설명 |
|----------|------|
| 왼쪽 어깨 (Left Shoulder) | 첫 번째 고점, 거래량이 높음 |
| 머리 (Head) | 세 고점 중 가장 높은 점, 거래량이 왼쪽 어깨보다 낮을 수 있음 |
| 오른쪽 어깨 (Right Shoulder) | 세 번째 고점, 머리보다 낮으며 거래량이 가장 적음 |
| 넥라인 (Neckline) | 두 저점을 연결한 선, 돌파 시 패턴 확인 |

### 거래량 요구사항

1.**왼쪽 어깨 형성**: 높은 거래량으로 고점 형성
2.**머리 형성**: 거래량이 감소하면서 더 높은 고점 형성 (약세 신호)
3.**오른쪽 어깨 형성**: 가장 적은 거래량으로 고점 형성 (매수 압력 소진)
4.**넥라인 돌파**: 높은 거래량을 동반한 넥라인 하향 돌파로 패턴 확인

> 거래량이 패턴과 역행한다면 (머리에서 거래량이 급등하는 경우) 패턴 신뢰도가 낮아집니다.

### 목표가 측정법

헤드앤숄더의 목표가는 패턴의 높이를 넥라인 돌파 지점에서 차감하여 계산합니다.

$$\text{목표가} = \text{넥라인 돌파가} - (\text{머리 고점} - \text{넥라인})$$

**예시 계산:**- 머리 고점: 105,000원
- 넥라인: 100,000원
- 패턴 높이: 5,000원
- 넥라인 돌파가: 99,500원
- 목표가: 99,500 - 5,000 =**94,500**
### 패턴 확인 조건 체크리스트

- [ ] 명확한 상승 추세가 선행됨
- [ ] 세 고점 중 머리가 가장 높음
- [ ] 오른쪽 어깨가 왼쪽 어깨보다 낮음
- [ ] 거래량이 왼쪽 어깨 → 머리 → 오른쪽 어깨 순으로 감소
- [ ] 넥라인이 수평 또는 완만하게 기울어짐
- [ ] 거래량 증가를 동반한 넥라인 하향 돌파

## 2. 역 헤드앤숄더 (Inverse Head and Shoulders)

역 헤드앤숄더는 헤드앤숄더를 뒤집은 형태로, 하락 추세 이후 상승 반전을 예고합니다. [StockCharts](https://school.stockcharts.com/doku.php?id=chart_analysis:chart_patterns)는 이를 하락 추세에서 가장 강력한 반전 신호 중 하나로 분류합니다.

```
------Neckline------
   \   \/   \/
    \  RS   LS
     \      /
      \    /
       \  /
        \/
      Head (머리)
```

### 역 헤드앤숄더 거래량 특성

| 구간 | 거래량 패턴 | 의미 |
|------|------------|------|
| 왼쪽 어깨 저점 | 높음 | 매도 압력이 강함 |
| 머리 저점 | 중간 (감소 추세) | 매도 압력 약화 시작 |
| 오른쪽 어깨 저점 | 낮음 | 매도 압력 소진 |
| 넥라인 돌파 | 급등 | 강한 매수 신호 확인 |

### 목표가 계산

$$\text{목표가} = \text{넥라인 돌파가} + (\text{넥라인} - \text{머리 저점})$$

**예시:**- 머리 저점: 45,000원
- 넥라인: 50,000원
- 패턴 높이: 5,000원
- 목표가: 50,500 + 5,000 =**55,500**

## 3. 더블 탑 (Double Top)과 더블 바텀 (Double Bottom)

<img src="/technical-analysis/images/diagrams/double-top-bottom.svg" alt="더블 탑(M자형)과 더블 바텀(W자형) 반전 패턴 구조 및 넥라인 돌파 다이어그램" style="width:100%;max-width:600px;margin:1rem auto;display:block;" />

더블 탑은 "M자형", 더블 바텀은 "W자형"으로도 불립니다. [BabyPips](https://www.babypips.com/learn/forex/chart-patterns)에 따르면 이 패턴들은 외환 시장에서 특히 자주 관찰됩니다.

### 더블 탑 (Double Top)

```
  Peak1    Peak2
   /\      /\
  /  \    /  \
 /    \  /    \
/      \/      \
    Neckline (지지선)
```

**형성 과정:**1. 상승 추세에서 첫 번째 고점 형성
2. 지지선까지 하락 (저점 형성)
3. 첫 번째 고점과 유사한 수준까지 반등
4. 두 번째 고점 형성 후 하락
5. 지지선(넥라인) 하향 돌파로 패턴 확인

**핵심 조건:**- 두 고점의 가격이 유사해야 합니다 (±3% 허용)
- 두 고점 사이 저점이 명확해야 합니다
- 두 번째 고점에서 거래량이 첫 번째보다 낮아야 합니다
- 넥라인 돌파 시 거래량 증가가 필요합니다

**목표가:**
$$\text{목표가} = \text{넥라인} - (\text{고점} - \text{넥라인})$$

### 더블 바텀 (Double Bottom)

```
    Neckline (저항선)
\      /\      /
 \    /  \    /
  \  /    \  /
   \/      \/
 Trough1  Trough2
```

더블 바텀은 하락 추세의 종료를 알리며 상승 반전을 예고합니다.

**확인 조건 비교:**
| 항목 | 더블 탑 | 더블 바텀 |
|------|--------|---------|
| 선행 추세 | 상승 추세 | 하락 추세 |
| 패턴 형태 | M자형 | W자형 |
| 넥라인 돌파 방향 | 하향 | 상향 |
| 거래량 (돌파 시) | 증가 | 증가 |
| 두 번째 극값 거래량 | 첫 번째보다 적음 | 첫 번째보다 많을 수 있음 |

**더블 바텀 목표가:**
$$\text{목표가} = \text{넥라인} + (\text{넥라인} - \text{저점})$$

## 4. 트리플 탑 (Triple Top)과 트리플 바텀 (Triple Bottom)

트리플 탑/바텀은 더블 탑/바텀보다 형성 기간이 길고 신뢰도가 높은 패턴입니다. 세 번의 시도 끝에도 돌파에 실패했다는 것은 해당 수준의 저항/지지가 매우 강하다는 의미입니다.

### 트리플 탑 (Triple Top)

```
  P1        P2        P3
  /\        /\        /\
 /  \      /  \      /  \
/    \    /    \    /    \
      \  /      \  /
       \/        \/
      ---Neckline---
```

**트리플 탑의 특징:**- 세 고점이 거의 같은 가격대를 형성합니다
- 각 반등 시 거래량이 점차 감소합니다
- 두 개의 저점을 연결한 넥라인을 하향 돌파 시 확인됩니다
- 더블 탑보다 형성 기간이 길어 더 강한 반전 신호로 여겨집니다

### 트리플 바텀 (Triple Bottom)

**트리플 바텀 특징:**
| 특성 | 내용 |
|------|------|
| 선행 조건 | 명확한 하락 추세 |
| 저점 수 | 3개 (거의 같은 가격대) |
| 거래량 추세 | 각 저점마다 감소 |
| 확인 신호 | 넥라인 상향 돌파 + 거래량 증가 |
| 목표가 | 넥라인 + 패턴 높이 |

### 트리플 탑 vs 더블 탑 비교

| 비교 항목 | 더블 탑 | 트리플 탑 |
|---------|--------|---------|
| 형성 기간 | 상대적으로 짧음 | 상대적으로 길음 |
| 신호 강도 | 중간 | 높음 |
| 발생 빈도 | 높음 | 낮음 |
| 목표가 신뢰도 | 중간 | 높음 |

## 5. 수치 계산 워크스루: 헤드앤숄더 완전 예시

실제 주가 데이터를 사용하여 헤드앤숄더 패턴의 전 과정을 단계별로 계산합니다.

**배경:**코스피 종목이 6개월간 상승 후 헤드앤숄더 형성 (가상 데이터)

| 구간 | 날짜 | 가격 | 거래량 |
|------|------|------|--------|
| 왼쪽 어깨 고점 | 1월 15일 | 82,000원 | 2,500만 주 |
| 첫 번째 저점 (넥라인 좌) | 2월 3일 | 76,000원 | 1,200만 주 |
| 머리 고점 | 2월 28일 | 89,000원 | 1,800만 주 |
| 두 번째 저점 (넥라인 우) | 3월 20일 | 76,500원 | 1,100만 주 |
| 오른쪽 어깨 고점 | 4월 10일 | 81,000원 | 900만 주 |
| 넥라인 돌파 | 4월 25일 | 75,200원 | 3,100만 주 |

**단계 1: 넥라인 설정**
두 저점(76,000원, 76,500원)을 연결합니다. 넥라인이 완만하게 우상향하므로 돌파 시점의 넥라인 수준은 약 76,500원입니다.

**단계 2: 패턴 높이 측정**
$$\text{패턴 높이} = \text{머리 고점} - \text{넥라인} = 89{,}000 - 76{,}500 = 12{,}500\text{원}$$

**단계 3: 목표가 계산**
$$\text{목표가} = \text{넥라인 돌파가} - \text{패턴 높이} = 75{,}200 - 12{,}500 = 62{,}700\text{원}$$

**단계 4: 거래량 검증**
| 구간 | 거래량 | 평가 |
|------|--------|------|
| 왼쪽 어깨 | 2,500만 주 | 기준선 |
| 머리 | 1,800만 주 | 왼쪽 어깨보다 28% 감소 ✓ |
| 오른쪽 어깨 | 900만 주 | 왼쪽 어깨보다 64% 감소 ✓ |
| 넥라인 돌파 | 3,100만 주 | 20일 평균 대비 2.4배 급증 ✓ |

모든 거래량 조건 충족 → 패턴 신뢰도 높음

**단계 5: 손절매 및 리스크/리워드 계산**
$$\text{손절매} = \text{오른쪽 어깨 고점} \times 1.01 = 81{,}000 \times 1.01 = 81{,}810\text{원}$$

$$RR = \frac{75{,}200 - 62{,}700}{81{,}810 - 75{,}200} = \frac{12{,}500}{6{,}610} \approx 1:1.9$$

## 6. 실전 적용

### 시나리오 A: 헤드앤숄더 공매도 진입

**상황:**코스피 종목이 12개월 상승 추세 후 헤드앤숄더 완성 직전

```
의사결정 흐름:
1. 패턴 형성 확인
   → 왼쪽 어깨·머리·오른쪽 어깨 고점 식별
   → 거래량 감소 추세 확인 (왼쪽 어깨 → 머리 → 오른쪽 어깨)

2. 넥라인 설정 및 이탈 대기
   → 두 저점 연결하여 넥라인 75,000원 설정
   → 종가 기준 넥라인 하향 이탈 확인

3. 진입 방법 선택
   - 공격적: 이탈 당일 장 후반 진입
   - 보수적: 넥라인으로의 풀백(pull-back) 반등 시 저항 확인 후 진입
     → 가격이 75,000원으로 되돌아온 뒤 거부당하면 진입

4. 손절/목표 설정
   - 손절: 오른쪽 어깨 고점 위 (81,810원)
   - 1차 목표: 패턴 높이의 50% 지점 (약 68,950원) → 절반 청산
   - 2차 목표: 패턴 높이 100% (62,700원) → 잔여 청산

5. 포지션 관리
   - 1차 목표 도달 시 손절을 진입가로 이동 (손익분기 보호)
   - 잔여 포지션은 트레일링 스탑 적용
```

### 시나리오 B: 더블 바텀 매수 진입

**상황:**하락 추세 7개월 후 W자형 더블 바텀 형성

| 항목 | 수치 |
|------|------|
| 첫 번째 저점 | 38,000원 |
| 넥라인 (중간 고점) | 44,500원 |
| 두 번째 저점 | 38,400원 (첫 번째와 유사, +1% 허용) |
| 패턴 높이 | 44,500 - 38,000 = 6,500원 |
| 넥라인 돌파가 | 45,200원 |
| 목표가 | 45,200 + 6,500 =**51,700**|
| 손절매 | 두 번째 저점 아래: 37,500원 |

**진입 조건 체크리스트:**- [x] 7개월 하락 추세 선행
- [x] 두 저점이 ±3% 이내 유사 수준
- [x] 두 번째 저점의 거래량이 첫 번째보다 감소 (매도 압력 소진)
- [x] 넥라인 돌파 시 거래량 평균 1.5배 이상
- [x] RSI가 40 이상으로 회복 중 (모멘텀 전환 확인)

## 7. 흔한 실수

### 실수 1: 넥라인 이탈 전 조기 진입

오른쪽 어깨처럼 보이는 고점에서 "패턴이 완성됐다"며 넥라인 이탈 전에 진입합니다. 실제로는 새로운 상승의 시작일 수 있습니다.**패턴은 반드시 넥라인 종가 이탈로만 확인하**고 진입합니다.

### 실수 2: 어깨 높이의 완벽한 대칭을 요구함

오른쪽 어깨가 왼쪽 어깨보다 약간 높다는 이유로 패턴을 포기합니다. 현실에서 완전 대칭은 드뭅니다.**오른쪽 어깨가 머리 고점보다 낮으면패**턴은 유효합니다. 두 어깨 높이 차이가 머리 높이의 30% 이내면 허용됩니다.

### 실수 3: 손절매를 넥라인 바로 위에 설정

넥라인 돌파 후 가격이 잠시 되돌아오는 풀백은 매우 흔한 현상입니다. 손절매가 넥라인 바로 위에 있으면 정상적인 되돌림에서 청산됩니다. 손절매는오**른쪽 어깨 고점 **에 설정합니다.

### 실수 4: 거래량 없는 넥라인 이탈에 진입

넥라인이 이탈됐지만 거래량이 평균 수준이거나 이하인 경우 진입합니다. 거래량이 동반되지 않은 이탈은 거짓 돌파(False Breakout) 가능성이 높습니다.**돌파 시 거래량이 20일 평균의 1.3배 이상이**어야 신뢰합니다.

### 실수 5: 상승 추세 중반부에서 억지로 반전 패턴 탐색

"이제 꼭지겠지"라는 편향으로 상승 추세 중반부에서 헤드앤숄더를 찾으려 합니다. 반전 패턴은 추세가 성숙되어 모멘텀이 약화될 때 의미 있습니다. 상승 추세가 최소**3~6개월 지속되**고 거래량 감소가 동반될 때 탐색합니다.

---

## 8. 엣지 케이스: 이상 시장 상황

### 거짓 돌파 (False Breakout) 대응

넥라인을 일시 이탈 후 가격이 다시 패턴 안으로 복귀하는 경우입니다.

**식별 방법:**- 이탈 시 거래량이 평균 이하
- 이탈 캔들의 몸통이 작고 꼬리가 긺
- 다음 캔들이 바로 넥라인 위로 복귀

**대응:**종가 기준 이탈이 확인되지 않으면 진입하지 않습니다. 이미 진입했다면 손절매 원칙을 철저히 따릅니다.

### 기울어진 넥라인

넥라인이 15° 이상 기울어지면 패턴 신뢰도와 목표가 정확도가 낮아집니다. 우상향 넥라인의 헤드앤숄더는 하향 넥라인보다 약한 신호입니다. 목표가를 보수적으로 조정하고(패턴 높이의 70~80% 적용) 진입 크기를 줄입니다.

### 강한 시장 호재 속 패턴 실패

패턴 형성 중 시장 전체에 강한 호재(금리 인하, 정책 발표)가 발생하면 패턴이 실패할 수 있습니다. 패턴 매매는 항상시**장 전체 맥락과** 함께 평가해야 합니다. 패턴 형성 중 시장 상황이 급변하면 포지션 크기를 줄이거나 대기합니다.

### 트리플 탑에서 더블 탑으로의 진화

두 번째 고점에서 더블 탑으로 매도했는데, 세 번째 고점이 추가로 형성되는 경우입니다. 이는 패턴 실패가 아니라트**리플 탑으로의 발전이**므로, 손절매 수준을 세 번째 고점 위로 조정하여 포지션을 유지할 수 있습니다. 단, 사전에 손절매 조정 계획을 세워두어야 합니다.

## 10. 패턴 실패와 함정

반전 패턴이 항상 성공하는 것은 아닙니다. 패턴 실패를 인식하는 것이 중요합니다.

### 일반적인 실패 원인

1.**거짓 돌파 (False Breakout**): 넥라인을 돌파했다가 되돌아오는 현상
2.**거래량 불일치**: 돌파 시 거래량이 증가하지 않는 경우
3.**시장 전체 추세**: 전체 시장이 강한 추세를 보일 때 역방향 패턴은 실패 확률이 높음
4.**조급한 진입**: 넥라인 돌파 확인 전 진입

### 리스크 관리

```
손절매 위치:
- 헤드앤숄더: 오른쪽 어깨 고점 위
- 더블 탑: 두 번째 고점 위
- 트리플 탑: 세 번째 고점 위
```

---

## 핵심 요약

-**헤드앤숄더는** 세 개의 고점(왼쪽 어깨-머리-오른쪽 어깨)으로 구성되며, 넥라인 돌파와 거래량 감소 패턴이 신뢰도의 핵심입니다
-**더블 탑/바텀은** M자 또는 W자 형태로 두 번의 극값 이후 지지/저항선 돌파로 확인되며, 외환 시장에서 특히 자주 관찰됩니다
-**트리플 탑/바텀은** 더블 탑/바텀보다 형성 기간이 길고 신뢰도가 높으며, 세 번의 극값을 통해 강한 지지/저항을 확인합니다
-**목표가 측정은** 패턴 높이(극값과 넥라인의 차이)를 돌파 지점에 적용하는 방식으로, 거래량 확인과 함께 사용할 때 신뢰도가 높아집니다
