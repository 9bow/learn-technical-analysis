## 지속 패턴 (Continuation Patterns) 개요

지속 패턴은 기존 추세 중 일시적인 가격 조정 또는 횡보 이후 추세가 재개될 가능성을 시사하는 차트 구조입니다. [StockCharts](https://school.stockcharts.com/doku.php?id=chart_analysis:chart_patterns:flag_pennant)에 따르면 이 패턴들은 "추세 내 휴식(rest within a trend)"으로 이해할 수 있습니다.

**지속 패턴의 공통 특성:**- 강한 기존 추세(폴 또는 마스트)가 선행됩니다
- 패턴 형성 중 거래량이 감소하는 경향이 있습니다
- 돌파 시 거래량이 급증하며 추세를 재개합니다
- 목표가는 선행 폴(Pole)의 길이를 기준으로 측정합니다

## 1. 깃발 패턴 (Flag Pattern)

깃발 패턴은 강한 수직 상승 또는 하락(폴, Pole) 이후 평행한 채널 내에서 반대 방향으로 소폭 조정되는 패턴입니다. [Investopedia](https://www.investopedia.com/terms/f/flag.asp)는 이를 가장 신뢰도 높은 지속 패턴 중 하나로 분류합니다.

### 상승 깃발 (Bull Flag)

```
      |  (폴: 강한 상승)
      |
      |    /--\
      |   /    \   (깃발: 하향 채널)
      |  /      \--\
      | /            \
      |/         돌파 지점
```

**형성 과정:**1. 폴 (Pole): 짧은 기간의 강한 상승 (이상적으로 거래량 급증 동반)
2. 깃발 (Flag): 하향 기울기의 평행 채널 내 조정 (거래량 감소)
3. 돌파: 채널 상단을 상향 돌파 (거래량 재증가)

### 하락 깃발 (Bear Flag)

```
      |\         진입 지점
      | \    /--/
      |  \  /  /   (깃발: 상향 채널)
      |   \/  /
      |       (폴: 강한 하락)
      |
```

### 깃발 패턴 특성 비교

| 항목 | 상승 깃발 | 하락 깃발 |
|------|---------|---------|
| 선행 폴 방향 | 상승 | 하락 |
| 깃발 기울기 | 하향 (역방향) | 상향 (역방향) |
| 형성 기간 | 1~4주 | 1~4주 |
| 거래량 (패턴 중) | 감소 | 감소 |
| 돌파 방향 | 상향 | 하향 |

### 목표가 측정

$$\text{상승 깃발 목표가} = \text{돌파가} + \text{폴의 길이}$$

$$\text{하락 깃발 목표가} = \text{돌파가} - \text{폴의 길이}$$

**예시:**- 폴 시작: 80,000원, 폴 끝: 100,000원 → 폴 길이 = 20,000원
- 깃발 형성 후 돌파가: 95,000원
- 목표가: 95,000 + 20,000 =**115,000**

## 2. 페넌트 패턴 (Pennant Pattern)

페넌트는 깃발과 유사하나, 조정 구간이 평행 채널이 아닌 수렴하는 삼각형 형태를 보입니다. [BabyPips](https://www.babypips.com/learn/forex/continuation-chart-patterns)에 따르면 페넌트는 단기 지속 패턴으로 주로 수일~수주에 걸쳐 형성됩니다.

### 상승 페넌트 (Bullish Pennant)

```
      |   (폴)
      |  /\
      | /  \  (수렴 삼각형)
      |/    \/\
      |         \
      |      돌파 후 상승
```

**깃발 vs 페넌트 비교:**
| 비교 항목 | 깃발 | 페넌트 |
|---------|------|-------|
| 조정 형태 | 평행 채널 | 수렴 삼각형 |
| 추세선 수 | 2개 (평행) | 2개 (수렴) |
| 형성 기간 | 1~4주 | 1~3주 (더 짧은 경향) |
| 거래량 감소 | 점진적 감소 | 급격한 감소 |
| 목표가 방법 | 폴 길이 적용 | 폴 길이 적용 |

### 페넌트 거래량 특성

1.**폴 형성**: 거래량 급증 (추세 강도 확인)
2.**페넌트 형성**: 거래량이 빠르게 감소 (관심 감소, 조정 중)
3.**돌파 직전**: 거래량이 극도로 낮아짐 (에너지 응축)
4.**돌파 **: 거래량 급증 (추세 재개 확인)

## 3. 쐐기형 패턴 (Wedge Patterns)

쐐기형은 두 추세선이 같은 방향으로 수렴하는 패턴입니다. 중요한 점은쐐**기형은 지속 패턴일 수도, 반전 패턴일 수도 있다는** 점입니다. 방향에 따라 의미가 다릅니다.

### 상승 쐐기 (Rising Wedge)

```
            /   /
           / / /
          / / /
         / / /
        / / /
       /   /
      수렴점 (상방)
```

**상승 쐐기의 이중적 성격:**
| 위치 | 의미 |
|------|------|
| 상승 추세 중 | 지속 패턴 (일시적 조정 후 하락 돌파는 드묾) |
| 하락 추세 중 반등 | 반전 패턴 (하향 돌파로 하락 재개) |
| 상승 추세 말기 | 반전 패턴 (하향 돌파 후 하락 전환) |

**상승 쐐기 핵심 특성:**- 두 추세선 모두 상향이지만, 하단 추세선의 기울기가 더 가파름
- 두 추세선이 위쪽에서 수렴
- 거래량이 패턴 내에서 지속 감소
- 일반적으로 하향 돌파 (추세와 반대 방향)

### 하락 쐐기 (Falling Wedge)

```
      \   \
       \ \ \
        \ \ \
         \ \ \
          \ \ \
           \   \
            수렴점 (하방)
```

**하락 쐐기 특성:**
| 특성 | 내용 |
|------|------|
| 두 추세선 방향 | 모두 하향, 상단이 더 가파름 |
| 수렴 방향 | 아래쪽으로 수렴 |
| 거래량 | 패턴 내 지속 감소 |
| 돌파 방향 | 상향 돌파 |
| 전형적 의미 | 하락 추세 중: 반전 / 상승 추세 중: 지속 |

### 쐐기형 목표가

$$\text{목표가} = \text{돌파가} \pm \text{쐐기 시작점의 높이}$$

## 4. 직사각형 패턴 (Rectangle Pattern)

직사각형 패턴은 가격이 수평 지지선과 저항선 사이에서 횡보하는 구간입니다. [StockCharts](https://school.stockcharts.com/doku.php?id=chart_analysis:chart_patterns)에 따르면 직사각형은 추세 중 나타나는 통합(consolidation) 구간으로, 돌파 방향에 따라 추세 지속 또는 반전을 시사합니다.

### 직사각형 구조

```
저항선: ----+------+------+-----
           |      |      |
지지선: ----+------+------+-----
      진입   고점1  저점1  고점2  돌파
```

**특성:**- 2회 이상의 고점이 같은 수준의 저항을 형성합니다
- 2회 이상의 저점이 같은 수준의 지지를 형성합니다
- 패턴 내에서 거래량이 불규칙하게 변동합니다
- 저항선 돌파 시 거래량 급증이 나타납니다

### 거래량 특성

| 구간 | 거래량 | 해석 |
|------|--------|------|
| 패턴 형성 중 | 불규칙 또는 감소 | 방향성 불확실 |
| 저항 접근 시 | 증가 가능 | 매도 압력 |
| 지지 접근 시 | 증가 가능 | 매수 압력 |
| 상향 돌파 시 | 급증 | 강한 매수 신호 |
| 하향 돌파 시 | 급증 | 강한 매도 신호 |

### 직사각형 목표가

$$\text{목표가} = \text{돌파가} \pm (\text{저항} - \text{지지})$$

**예시:**- 저항선: 52,000원
- 지지선: 48,000원
- 직사각형 높이: 4,000원
- 상향 돌파가: 52,500원
- 목표가: 52,500 + 4,000 =**56,500**

## 5. 수치 계산 워크스루: 상승 깃발 완전 예시

실제 데이터를 사용하여 상승 깃발 패턴의 전 과정을 단계별로 계산합니다.

**배경:**코스피 종목이 강한 상승 폴(Pole) 형성 후 깃발 구간 진행 중

| 구간 | 날짜 | 가격 | 거래량 |
|------|------|------|--------|
| 폴 시작 (저점) | 1월 5일 | 72,000원 | 800만 주 |
| 폴 끝 (고점) | 1월 19일 | 92,000원 | 3,200만 주 |
| 깃발 상단 | 1월 22일 | 90,000원 | 1,100만 주 |
| 깃발 하단 | 1월 30일 | 85,000원 | 400만 주 |
| 깃발 상단 | 2월 5일 | 88,500원 | 350만 주 |
| 깃발 하단 | 2월 10일 | 83,500원 | 280만 주 |
| 돌파 시점 | 2월 14일 | 89,500원 | 2,800만 주 |

**단계 1: 폴의 길이 측정**
$$\text{폴 길이} = 92{,}000 - 72{,}000 = 20{,}000\text{원}$$

$$\text{폴 상승률} = \frac{20{,}000}{72{,}000} \times 100 = 27.8\%$$

이상적인 폴 상승률: 15~50% (단기간 강한 상승)

**단계 2: 깃발의 기울기 확인**
깃발 구간의 고점과 저점이 모두 하향 기울기를 형성하는지 확인합니다.

- 고점: 90,000원 → 88,500원 (하향 ✓)
- 저점: 85,000원 → 83,500원 (하향 ✓)
- 두 추세선이 평행 ✓

**단계 3: 거래량 수렴 확인**
| 기간 | 거래량 | 폴 대비 |
|------|--------|---------|
| 폴 형성 | 3,200만 주 | 기준 |
| 깃발 초반 | 1,100만 주 | 66% 감소 |
| 깃발 중반 | 400만 주 | 88% 감소 |
| 깃발 후반 | 280만 주 | 91% 감소 |
| 돌파 시 | 2,800만 주 | 10배 급증 ✓ |

**단계 4: 목표가 계산**
$$\text{목표가} = \text{돌파가} + \text{폴 길이} = 89{,}500 + 20{,}000 = 109{,}500\text{원}$$

**단계 5: 손절매 설정**
$$\text{손절매} = \text{깃발 하단} - \text{버퍼} = 83{,}500 - 1{,}000 = 82{,}500\text{원}$$

$$RR = \frac{109{,}500 - 89{,}500}{89{,}500 - 82{,}500} = \frac{20{,}000}{7{,}000} \approx 1:2.9$$

## 6. 실전 적용

### 시나리오 A: 상승 깃발 매수 진입

**상황:**IT 섹터 ETF가 실적 발표 후 강한 폴을 형성하고 현재 깃발 패턴 진행 중

```
진입 결정 흐름:

1. 폴 강도 확인
   → 2주간 28% 상승, 거래량 평균 3배 이상 → 유효한 폴 ✓

2. 깃발 구조 확인
   → 평행 채널, 하향 기울기 10~15° ✓
   → 거래량 꾸준히 감소 ✓

3. 돌파 대기 및 진입
   - 깃발 상단: 88,500원
   - 돌파 확인: 종가 기준 상단 돌파 + 거래량 평균 2배 이상
   - 진입가: 89,500원 (돌파 익일 시가)

4. 포지션 크기 결정
   - 손절: 82,500원 (7,000원 리스크)
   - 총 자산 1,000만원, 리스크 2% = 200,000원 허용 손실
   - 매수 수량: 200,000 ÷ 7,000 ≈ 28주

5. 목표가 분할 청산
   - 1차: 폴 길이의 50% = 99,500원 → 50% 청산
   - 2차: 폴 길이 100% = 109,500원 → 잔여 청산
```

### 시나리오 B: 하락 깃발 공매도 진입

**상황:**금융주가 2주간 급락(폴) 후 상향 채널에서 반등 중

| 항목 | 수치 |
|------|------|
| 폴 시작 (고점) | 65,000원 |
| 폴 끝 (저점) | 50,000원 |
| 폴 길이 | 15,000원 |
| 깃발 형성 기간 | 10거래일 (상향 채널) |
| 돌파가 (하향) | 51,500원 |
| 목표가 | 51,500 - 15,000 =**36,500**|
| 손절매 | 깃발 상단 위: 54,500원 |

**핵심 확인 사항:**깃발이 폴 길이의 38.2~61.8% 수준에서 형성되었는지 확인합니다. 깃발이 폴 길이의 70% 이상 되돌림이 일어나면 깃발이 아닌 추세 전환 가능성을 고려해야 합니다.

### 시나리오 C: 직사각형 돌파 매매

**상황:**코스닥 종목이 6주간 박스권(52,000~56,000원) 형성

```
박스권 분석:
- 저항선: 56,000원 (3회 저항 확인)
- 지지선: 52,000원 (3회 지지 확인)
- 박스권 높이: 4,000원
- 거래량: 박스권 내 불규칙

돌파 시 매매:
- 돌파 확인: 56,500원에 거래량 2배 이상으로 저항 돌파
- 진입가: 57,000원
- 손절: 55,500원 (저항 → 지지 전환 실패 시)
- 목표가: 57,000 + 4,000 = 61,000원
- 리스크/리워드: 4,000 / 1,500 ≈ 1:2.7
```

## 7. 흔한 실수

### 실수 1: 폴 없이 깃발/페넌트를 찾으려 함

강한 선행 폴이 없는 상태에서 단순히 채널이나 삼각형이 보인다고 깃발/페넌트라고 판단합니다. 깃발과 페넌트의 핵심은강**한 폴이 선행되**어야 한다는 점입니다. 폴이 없으면 깃발이 아니라 단순 채널 패턴입니다.

### 실수 2: 깃발의 되돌림이 너무 클 때 계속 보유

깃발 구간에서 폴 길이의 50% 이상 되돌림이 일어나면 깃발 패턴 실패 가능성이 높습니다. 이상적인 깃발의 되돌림은 폴 길이의**23.6~38.2%**수준입니다. 깊은 되돌림은 추세 전환 신호일 수 있으므로 손절매를 준수합니다.

### 실수 3: 거래량 없는 돌파에 진입

깃발 상단을 돌파했지만 거래량이 여전히 낮습니다. 거래량 없는 돌파는 세력 부재를 의미하며 거짓 돌파 가능성이 높습니다.**돌파 시 거래량이 깃발 구간 평균의 2배 이상이**어야 신뢰할 수 있습니다.

### 실수 4: 쐐기형의 방향 혼동

상승 쐐기를 보고 "위로 수렴하니까 상향 돌파"라고 판단합니다.**상승 쐐기는 주로 하향 돌파**,**하락 쐐기는 주로 상향 돌파합**니다. 쐐기형은 추세와 반대 방향으로 돌파하는 특성을 가집니다.

### 실수 5: 직사각형에서 방향 예단

박스권 돌파 전에 "상승 추세였으니 당연히 위로 돌파하겠지"라고 예단하고 미리 매수합니다. 직사각형은 상향, 하향 어느 방향으로도 돌파할 수 있습니다.**반드시 돌파 방향을 확인한 **진입합니다.

## 8. 엣지 케이스: 이상 시장 상황

### 깃발/페넌트의 거짓 돌파

깃발 상단을 돌파한 뒤 다시 깃발 내부로 복귀하는 경우입니다.

**발생 원인:**- 저거래량 돌파
- 전체 시장 급락
- 섹터 악재

**대응:**거짓 돌파 확인 즉시 손절합니다. 재진입은 패턴이 다시 완성되고 두 번째 돌파가 고거래량을 동반할 때만 고려합니다.

### 쐐기형의 수렴점 이후 돌파 실패

삼각형이나 쐐기형이 수렴점을 지나도 돌파가 발생하지 않는 경우입니다. 수렴점 이후에는 패턴의 에너지가 소진되어 신호 신뢰도가 크게 낮아집니다. 수렴점을 지난 패턴은 무시하고 새로운 패턴 형성을 기다립니다.

### 채찍질(Whipsaw) 현상

직사각형 패턴에서 돌파 방향이 빠르게 전환되며 연속 손실이 발생하는 경우입니다. 이는 횡보 시장의 특성으로, ADX가 20 미만이면 직사각형 돌파 매매보다 박스권 내 범위 매매(지지에서 매수, 저항에서 매도)가 더 적합합니다.

## 핵심 요약

-**깃발과 페넌트는** 강한 폴(Pole) 이후 형성되는 단기 지속 패턴으로, 깃발은 평행 채널, 페넌트는 수렴 삼각형 형태를 보이며 목표가는 폴의 길이를 돌파 지점에 적용합니다
-**쐐기형은** 두 추세선이 같은 방향으로 수렴하는 패턴으로, 상승 쐐기는 주로 하향 돌파, 하락 쐐기는 주로 상향 돌파하며 추세 위치에 따라 지속 또는 반전 의미를 가집니다
-**직사각형은** 수평 지지/저항 사이의 횡보 구간으로, 기존 추세 방향으로 돌파 시 지속 패턴, 반대 방향 돌파 시 반전 패턴으로 해석합니다
- 모든 지속 패턴에서거**래량 감소 후 돌파 시 급증이** 나타나면 패턴 신뢰도가 높아지며, 거래량 확인 없는 돌파는 거짓 돌파 위험이 있습니다
