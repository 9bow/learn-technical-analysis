## 손절의 중요성

손절(Stop Loss)은 트레이드가 예상과 반대 방향으로 움직일 **미리 정해진 가격에서 포지션을 청산하**여 손실을 제한하는 주문입니다.

손절 없이 트레이딩하는 것은 안전장치 없이 비행하는 것과 같습니다. 손실이 크게 날수록 회복에 필요한 수익률이 기하급수적으로 늘어납니다.

```
손실 회복에 필요한 수익률:
10% 손실 → 11.1% 수익 필요
20% 손실 → 25.0% 수익 필요
50% 손실 → 100% 수익 필요
80% 손실 → 400% 수익 필요
```

## 방법 1: 고정 손절 (Fixed Stop Loss)

### 개념

진입가에서고**정된 비율 또는 금액으**로 손절을 설정합니다. 가장 단순한 방법입니다.

```
고정 비율 손절 예시:
진입가: 100,000원, 손절 비율: 2%
손절가: 100,000 × (1 - 0.02) = 98,000원
```

### 장단점

| 장점 | 단점 |
|------|------|
| 계산이 간단 | 시장 변동성 무시 |
| 일관성 있음 | 지지·저항 구조 무시 |
| 포지션 사이징 계산 쉬움 | 잦은 손절 유발 가능 |

---

## 방법 2: ATR 기반 손절 (ATR-Based Stop)

### ATR이란?

평균 진폭 범위(Average True Range, ATR)는 J. 웰스 와일더(J. Welles Wilder)가 1978년 개발한 변동성 지표입니다 [Investopedia, 2026].

$$TR = \max\left(\text{당일 고가} - \text{당일 저가},\ \left|\text{당일 고가} - \text{전일 종가}\right|,\ \left|\text{당일 저가} - \text{전일 종가}\right|\right)$$

$$ATR = \text{TR의 N일 이동평균 (기본값: 14일)}$$

### ATR 기반 손절 계산

시장의 현재 변동성에 맞게 손절을 설정합니다.

$$\text{손절가 (매수)} = \text{진입가} - (ATR \times \text{배수})$$

$$\text{손절가 (매도)} = \text{진입가} + (ATR \times \text{배수})$$

```
매수 포지션:
  보수적: 진입가 - ATR × 1.5
  일반:   진입가 - ATR × 2.0
  공격적: 진입가 - ATR × 3.0
```

### 예시

```
BTC 일봉 ATR = 2,000,000원 (14일 평균)
진입가: 50,000,000원
배수: 2.0

매수 손절가: 50,000,000 - (2,000,000 × 2.0) = 46,000,000원
트레이드 리스크: 4,000,000원 (8%)
```

ATR 배수는 전략과 시간 프레임에 따라 조정합니다. 짧은 시간 프레임(1시간봉)에서는 낮은 배수(1.0~1.5), 긴 시간 프레임(주봉)에서는 높은 배수(2.0~3.0)를 사용합니다.

## 방법 3: 구조적 손절 (Structure-Based Stop)

### 개념

기술적지**지·저항 구조를** 기반으로 손절을 설정합니다. 시장 구조가 무효화되는 지점에 손절을 둡니다 [StockCharts, 2026].

```
매수 포지션의 구조적 손절:
- 직전 스윙 저점 아래
- 주요 지지 구역 아래
- 오더 블록 하단 아래

매도 포지션의 구조적 손절:
- 직전 스윙 고점 위
- 주요 저항 구역 위
- 오더 블록 상단 위
```

### 예시

```
시나리오: 상승 추세에서 61.8% 되돌림 구간 매수
진입가: 45,000원 (피보나치 61.8% 레벨)
직전 스윙 저점: 43,000원
구조적 손절: 43,000원 아래 → 42,500원 설정 (약간 여유)
트레이드 리스크: 45,000 - 42,500 = 2,500원
```

### 장단점

| 장점 | 단점 |
|------|------|
| 시장 구조에 논리적으로 부합 | 손절 범위가 커질 수 있음 |
| 가짜 이탈에 덜 민감 | 포지션 크기 계산이 복잡 |
| 높은 승률 경향 | 손익비가 낮아질 수 있음 |

## 방법 4: 추적 손절 (Trailing Stop)

추적 손절은 가격이 유리한 방향으로 움직일 때 손절도 함께 이동하여이**익을 보호하**는 방법입니다.

### 유형 1: 고정 비율 추적 손절

```
진입 후 2% 아래에 손절 유지
가격이 100 → 110으로 상승하면 손절: 107.8 (110 × 0.98)
가격이 110 → 105로 하락하면 손절 이동 없음 (107.8 유지)
105가 107.8 아래로 하락 시 청산
```

### 유형 2: ATR 기반 추적 손절 (Chandelier Exit)

$$\text{Chandelier Exit} = \text{최근 N봉 최고가} - ATR \times \text{배수}$$

```
N = 22봉, ATR = 2,000, 배수 = 3.0
최근 22봉 최고가: 60,000원
추적 손절: 60,000 - (2,000 × 3.0) = 54,000원
```

가격이 새로운 고점을 형성할 때마다 손절이 상향됩니다.

### 유형 3: 이동평균 기반 추적 손절

```
20일 이동평균 아래로 일봉 종가 기준 청산
또는
50일 이동평균 아래로 주봉 종가 기준 청산
```

장기 추세 추종 트레이더에게 적합합니다.

## 손절 설정의 흔한 실수

### 1. 손절 없는 트레이딩 ("반드시 반등한다")

```
잘못된 사고: "조금만 기다리면 오를 거야"
결과: 작은 손실이 계좌 파괴로 이어짐
```

### 2. 손절을 너무 좁게 설정

```
일반 변동성 범위 안에 손절 설정
→ 정상적인 가격 노이즈에 의해 청산
→ 방향이 맞아도 손실
```

### 3. 손절을 임의로 이동

```
손절에 도달하자 손절을 더 멀리 이동
→ 원래 손절의 의미 상실
→ 더 큰 손실로 이어짐
```

### 4. 심리적 손절만 가지고 있음

실제 주문이 아닌 마음속으로만 손절을 정하는 것입니다. 감정이 개입되면 대부분 손절을 지키지 못합니다.**항상 실제 주문으**로 설정합니다.

## 실전 적용

### 지지 구조 기반 손절 배치

손절은 단순히 "몇 퍼센트"가 아니라시**장 구조가 무효화되는 지점 바로 아래(매수) 또는 위(매도**)에 배치해야 합니다.

```
[매수 진입 시 손절 배치 예시]

상황: 비트코인이 50,000,000원 지지 구간에서 반등 확인 후 진입

직전 스윙 저점: 49,200,000원
ATR(14일): 1,500,000원
버퍼: ATR × 0.25 = 375,000원

구조적 손절: 49,200,000 - 375,000 = 48,825,000원
→ 스윙 저점 아래 ATR의 25%만큼 여유를 두어
  정상적인 가격 노이즈로 인한 손절을 방지
```

:::tip
버퍼(여유 폭)를 ATR의 20~30%로 설정하면 가짜 이탈(Fakeout)에 의한 손절을 줄일 수 있습니다. 단, 버퍼가 커질수록 포지션 크기를 줄여 리스크 금액을 일정하게 유지해야 합니다.
:::

### ATR 기반 손절 — 시간 프레임별 배수 기준

| 시간 프레임 | 권장 ATR 배수 | 이유 |
|------------|-------------|------|
| 5분봉 | 1.0 ~ 1.5 | 노이즈가 적음, 빠른 손절 |
| 1시간봉 | 1.5 ~ 2.0 | 일반적인 스캘핑/데이트레이딩 |
| 4시간봉 | 2.0 ~ 2.5 | 스윙 트레이딩 기준 |
| 일봉 | 2.0 ~ 3.0 | 구조적 움직임 반영 |
| 주봉 | 3.0 이상 | 장기 추세 추종 |

```
[4시간봉 ATR 손절 계산 예시]

종목: 삼성전자
현재 4시간봉 ATR: 1,200원
진입가: 72,000원 (매수)
배수: 2.0

손절가: 72,000 - (1,200 × 2.0) = 69,600원
트레이드 리스크: 2,400원 (진입가의 3.3%)

포지션 크기 계산 (계좌 1,000만원, 리스크 1%):
계좌 리스크: 100,000원
포지션: 100,000 / 2,400 ≈ 41주
총 투자금: 41 × 72,000 = 2,952,000원
```

### 추적 손절 전환 시점

초기 손절로 진입했다가 이익이 쌓이면 추적 손절로 전환하여 이익을 보호합니다.

```
[이익 보호 전환 예시]

진입: 50,000원, 손절: 47,500원 (ATR × 2)
1차 목표(RR 1:1): 52,500원 → 절반 청산
→ 나머지 포지션 손절을 진입가(50,000원)로 이동 (본전 손절)

2차 목표(RR 1:2): 55,000원 도달 시
→ 손절을 51,500원 (최근 고점 - ATR × 1.5)으로 상향

결과: 최악의 경우에도 나머지 포지션은 손실 없음
```

:::note
본전 손절(Break-Even Stop) 전환은 심리적 안정을 주지만, 너무 빨리 이동하면 정상적인 되돌림에 청산될 수 있습니다. 최소 RR 1:1 달성 후 전환을 권장합니다.
:::

## 손절 방법 비교 요약

| 방법 | 변동성 반영 | 구조 반영 | 복잡도 | 적합 상황 |
|------|------------|----------|--------|-----------|
| 고정 손절 | 없음 | 없음 | 낮음 | 빠른 스캘핑 |
| ATR 기반 | 있음 | 없음 | 중간 | 모든 전략 |
| 구조적 | 없음 | 있음 | 중간 | 스윙 트레이딩 |
| 추적 손절 | 선택 | 선택 | 높음 | 추세 추종 |

## 에지 케이스: 갭과 슬리피지

**갭 하락 시 손절 미체결한**국 주식 시장에서 장 시작 시 갭 하락이 발생하면 설정한 손절가보다 훨씬 낮은 가격에 체결될 수 있습니다(슬리피지). 예를 들어 손절을 48,000원으로 설정했는데 45,000원에 갭 오픈하면 45,000원에 체결됩니다. 이를 대비해 손절폭 계산 시 ATR의 20~30% 버퍼를 추가하거나, 갭 리스크가 높은 이벤트(실적 발표, 공시) 직전에는 포지션 크기를 줄이십시오.

**크립토 24시간 시장의 손절 운용크**립토는 갭이 없는 대신 극단적 변동성(단기 20~30% 급락)이 발생할 수 있습니다. 거래소의 청산 메커니즘과 손절 주문이 충돌할 수 있으므로, 레버리지 포지션에서는 청산가보다 반드시 위에 손절을 설정하고, 손절가와 청산가 사이의 간격을 최소 ATR × 1.5 이상 확보하십시오.

## 핵심 요약

- 손절은 선택이 아닌 필수이며, 작은 손실이 계좌를 보호하고 다음 기회를 만들어 줍니다.
- ATR 기반 손절은 현재 변동성을 반영하여 정상적인 가격 노이즈에 의한 불필요한 청산을 줄입니다.
- 구조적 손절은 시장 구조가 무효화되는 지점(스윙 저점 아래)에 설정하여 논리적 근거를 갖습니다.
- 추적 손절은 이익이 쌓일수록 손절을 이동하여 손실 없이 청산하거나 이익을 확정하는 도구입니다.
