## 손익비란?

손익비(Risk-Reward Ratio, RR)**한 트레이드에서 감수하는 리스크 대비 기대하는 보상의 비율입**니다. 트레이딩 전략의 수익성을 평가하는 핵심 지표입니다.

<BeforeAfter client:visible dataset="uptrend-stock" title="손익비 분석 차트" beforeLabel="진입 구간" afterLabel="목표 달성 확인" />

$$\text{손익비} = \frac{\text{목표 수익}}{\text{최대 손실}}$$

```
예시:
손절 범위: 100,000원 (리스크)
목표 수익: 200,000원 (리워드)
손익비: 200,000 / 100,000 = 2 (또는 1:2로 표기)
```

---

## 손익비 계산

### 기본 계산

```
진입가: E
손절가: SL
목표가: TP

리스크 = |E - SL|
리워드 = |TP - E|

RR = 리워드 / 리스크
```

### 예시: 매수 포지션

```
진입가: 50,000원
손절가: 48,000원
목표가: 56,000원

리스크: 50,000 - 48,000 = 2,000원
리워드: 56,000 - 50,000 = 6,000원
손익비: 6,000 / 2,000 = 3.0 (1:3)
```

### 예시: 매도 포지션

```
진입가: 100,000원
손절가: 103,000원
목표가: 91,000원

리스크: 103,000 - 100,000 = 3,000원
리워드: 100,000 - 91,000 = 9,000원
손익비: 9,000 / 3,000 = 3.0 (1:3)
```

---

## 최소 손익비 기준

일반적으로 권장되는 최소 손익비는**1:** 2입니다 [Investopedia, 2026].

| 손익비 | 평가 |
|--------|------|
| 1:1 미만 | 비권장 (손실이 이익보다 큼) |
| 1:1 | 승률 50% 이상이어야 손익분기 |
| 1:1.5 | 최소 수준, 승률 40% 이상 필요 |
| 1:2 | 권장 최소 기준, 승률 33% 이상이면 이론상 수익 |
| 1:3 | 양호, 승률 25% 이상이면 이론상 수익 |
| 1:5 이상 | 우수, 낮은 승률로도 수익 가능 |

## 기대값 (Expectancy)

### 개념

기대값(Expectancy)은 트레이드당평**균적으로 기대할 수 있는 수익입**니다. 전략의 장기 수익성을 하나의 숫자로 표현합니다 [Investopedia, 2026].

$$Expectancy = (\text{승률} \times \text{평균 이익}) - (\text{패율} \times \text{평균 손실})$$

또는 정규화 버전:

$$Expectancy = (W \times R) - (1 - W)$$

- $W$ = 승률 (Win Rate)
- $R$ = 평균 손익비 (Average RR)

### 계산 예시

```
시나리오 A:
승률 40%, 평균 손익비 2.5

Expectancy = (0.40 × 2.5) - (0.60 × 1)
Expectancy = 1.0 - 0.6
Expectancy = 0.4

의미: 트레이드당 리스크 금액의 40%를 평균 기대
1트레이드 리스크 10만원 → 트레이드당 평균 4만원 기대 이익
```

```
시나리오 B:
승률 60%, 평균 손익비 1.0

Expectancy = (0.60 × 1.0) - (0.40 × 1.0)
Expectancy = 0.6 - 0.4
Expectancy = 0.2

기대값은 양수이지만 A보다 낮음
```

### 기대값 해석

| 기대값 | 의미 |
|--------|------|
| 0 초과 | 장기적으로 수익성 있는 전략 |
| 0 | 손익분기, 장기적으로 제로 수익 |
| 0 미만 | 장기적으로 손실이 누적되는 전략 |

---

## 승률과 손익비 관계 테이블

같은 기대값(Expectancy = 0)을 만드는 승률과 손익비 조합입니다. 이 표의 값보다 높은 승률이면 수익성이 있습니다.

| 손익비 (RR) | 손익분기 승률 |
|------------|--------------|
| 1:1 | 50.0% |
| 1:1.5 | 40.0% |
| 1:2 | 33.3% |
| 1:2.5 | 28.6% |
| 1:3 | 25.0% |
| 1:4 | 20.0% |
| 1:5 | 16.7% |

$$BE\ Win\ Rate = \frac{1}{1 + RR}$$

```
RR = 2일 때: 1 / (1 + 2) = 33.3%
```

## 실전 사례: 전략 비교

두 가지 가상 전략을 비교합니다.

| 항목 | 전략 A | 전략 B |
|------|--------|--------|
| 승률 | 65% | 40% |
| 평균 손익비 | 1:1 | 1:3 |
| 기대값 | 0.30 | 0.80 |
| 100트레이드 후 결과 | 리스크의 30배 | 리스크의 80배 |

승률이 높아도 손익비가 낮으면 기대값이 낮을 수 있습니다.**전략 B가 훨씬 더 효율적입**니다.

> 많은 초보 트레이더가 "승률이 높으면 좋은 전략"이라고 생각하지만, 기대값 관점에서는 낮은 승률이라도 높은 손익비를 가진 전략이 더 수익적일 수 있습니다 [BabyPips, 2026].

## R-배수 (R-Multiple) 개념

### R이란?

**R(Risk Unit**)은 하나의 트레이드에서 감수하는 리스크 금액을 기본 단위로 삼는 개념입니다. 반 타프(Van Tharp) 박사가 트레이딩 성과를 금액이 아닌리**스크 단위로 표준화하**기 위해 도입했습니다.

```
1R = 해당 트레이드의 리스크 금액

예시:
진입가: 50,000원, 손절가: 48,000원
1R = 50,000 - 48,000 = 2,000원

결과별 R-배수:
- 손절 청산:          -1R  (2,000원 손실)
- 목표가 1:2 달성:    +2R  (4,000원 이익)
- 목표가 1:3 달성:    +3R  (6,000원 이익)
- 손절 전 조기 청산:  -0.5R (1,000원 손실)
- 본전 청산:           0R
```

:::tip
R-배수로 성과를 기록하면 계좌 잔고와 무관하게 전략의 기대값을 객관적으로 평가할 수 있습니다. 계좌가 100만원이든 1억이든, 트레이드당 평균 +0.5R이면 동일한 전략 효율입니다.
:::

### 트레이딩 일지 R-배수 기록 예시

| 날짜 | 종목 | 진입가 | 손절가 | 목표가 | 청산가 | R-배수 |
|------|------|--------|--------|--------|--------|--------|
| 3/1 | BTC | 85,000 | 83,000 | 91,000 | 91,000 | +3R |
| 3/3 | ETH | 3,200 | 3,100 | 3,500 | 3,100 | -1R |
| 3/5 | BTC | 87,000 | 85,500 | 91,000 | 88,500 | +1R |
| 3/8 | SOL | 140 | 133 | 161 | 155 | +2.1R |
| 3/10 | ETH | 3,350 | 3,200 | 3,800 | 3,200 | -1R |

```
5트레이드 합산 R-배수: +3 - 1 + 1 + 2.1 - 1 = +4.1R
평균 R-배수: 4.1 / 5 = +0.82R per trade

계좌 리스크 10만원 기준 실현 손익:
+0.82R × 10만원 = 트레이드당 평균 82,000원 기대 이익
```

:::note
R-배수 일지를 20~30트레이드 쌓으면 자신의 전략 기대값을 신뢰도 있게 계산할 수 있습니다. 소수의 결과로는 편향이 크므로, 최소 30트레이드 이상 데이터를 활용하세요.
:::

## 실전 적용

### 손익분기 승률 계산 — 전략 설계의 출발점

트레이드를 설계할 때 먼저 "이 손익비에서 몇 퍼센트 이상 이겨야 수익이 나는가"를 계산합니다.

$$BE\ Win\ Rate = \frac{1}{1 + RR}$$

```
RR별 손익분기 승률:
RR 1:1   → 1 / (1+1) = 50.0%
RR 1:1.5 → 1 / (1+1.5) = 40.0%
RR 1:2   → 1 / (1+2) = 33.3%
RR 1:2.5 → 1 / (1+2.5) = 28.6%
RR 1:3   → 1 / (1+3) = 25.0%
RR 1:4   → 1 / (1+4) = 20.0%
RR 1:5   → 1 / (1+5) = 16.7%
```

:::caution
손익분기 승률은 거래 비용(수수료, 슬리피지)을 포함하지 않습니다. 실제로는 손익분기 승률보다 2~5%p 높은 승률이 필요합니다.

예: RR 1:2, 수수료 0.1% × 2 = 0.2% → 실질 손익분기 ≈ 34~35%
:::

### 전략별 기대값 시뮬레이션

100트레이드 시뮬레이션으로 전략의 장기 수익성을 확인합니다. 계좌 리스크 10만원 기준:

```
[전략 A] 승률 45%, RR 1:2
손익분기 승률: 33.3% → 45% 초과 → 수익성 있음
기대값: (0.45 × 2) - (0.55 × 1) = 0.9 - 0.55 = 0.35R
100트레이드: +35R = +350만원

[전략 B] 승률 60%, RR 1:1
손익분기 승률: 50% → 60% 초과 → 수익성 있음
기대값: (0.60 × 1) - (0.40 × 1) = 0.6 - 0.4 = 0.2R
100트레이드: +20R = +200만원

[전략 C] 승률 30%, RR 1:3
손익분기 승률: 25% → 30% 초과 → 수익성 있음
기대값: (0.30 × 3) - (0.70 × 1) = 0.9 - 0.7 = 0.2R
100트레이드: +20R = +200만원
```

전략 A가 가장 높은 기대값을 보입니다. 같은 기대값이라도 승률과 RR 조합이 다르면심**리적 경험이** 크게 달라집니다. 전략 C는 70%가 손실이라 심리적 유지가 어렵습니다.

## 흔한 실수

### 1. 목표가를 설정하지 않고 진입

```
잘못된 접근: "오르면 팔지"
결과: 이익이 나자마자 두려움에 조기 청산
      → 평균 이익이 평균 손실보다 작아짐
      → 기대값 음수 가능성

올바른 접근: 진입 전 목표가와 손절가를 동시에 설정
```

### 2. 손익비만 보고 승률을 무시

```
극단 사례: RR 1:10이지만 승률 5%
기대값: (0.05 × 10) - (0.95 × 1) = 0.5 - 0.95 = -0.45R
→ 손익비가 높아도 승률이 너무 낮으면 손실 전략
```

:::caution
"손익비만 높으면 된다"는 오해가 많습니다. 반드시 승률과 함께 기대값으로 평가하세요.
:::

### 3. 목표가에서 욕심을 부려 청산 지연

```
시나리오: RR 1:3 목표가 도달
행동: "조금 더 갈 것 같아" → 청산 미루기
결과: 가격 반전 → 이익 반납 → RR 1:0.5로 마감
영향: 실제 기대값이 계획보다 크게 하락
```

### 4. 승률에 집착하여 손절을 미룸

```
심리: "이 트레이드만큼은 손실 내면 안 돼"
행동: 손절가 도달 후 손절 이동
결과: -1R이 될 손실이 -3R, -5R로 확대
영향: 한 번의 큰 손실이 여러 번의 이익을 삭제
```

### 5. 과거 승률로 미래를 확신

```
오류: "지난달 승률 70%였으니 이번 달도 70%"
현실: 시장 환경 변화, 샘플 수 부족으로 편향된 통계
대응: 최소 100트레이드 이상 데이터 누적 후 평가
      시장 환경(추세/횡보) 구분하여 전략별 통계 분리
```

## 손익비 개선 방법

### 1. 진입 타이밍 개선

더 정확한 진입으로 손절 범위를 줄이면 동일한 목표가에서 손익비가 높아집니다.

```
개선 전: 진입 50,000, 손절 46,000, 목표 58,000 → RR = 2.0
개선 후: 진입 50,000, 손절 48,000, 목표 58,000 → RR = 4.0
```

### 2. 단계적 목표가 설정

1차 목표에서 일부 청산하고, 나머지는 더 높은 목표를 향해 유지합니다.

```
포지션 100주:
1차 목표(RR 1:1.5): 50주 청산
2차 목표(RR 1:3): 나머지 50주 청산

평균 RR = (50 × 1.5 + 50 × 3) / 100 = 2.25
```

### 3. 추적 손절 활용

이익이 쌓이면 손절을 이동하여 최소 손익비를 확보합니다.

## 에지 케이스: 비대칭 수수료와 실질 손익비

**크립토 선물 수수료 적용바**이낸스 등 크립토 선물에서 메이커 수수료 0.02%, 테이커 수수료 0.05%를 적용하면 왕복 수수료가 0.1%에 달합니다. 손익비 1:2 전략에서 리스크 1%라면, 실질 기대값 계산 시 수수료를 반드시 포함해야 합니다.

$$\text{실질 기대값} = (W \times RR) - (L \times 1) - \text{수수료} \times 2$$

예: 승률 40%, RR 1:2, 수수료 0.1% × 2 = 0.2%
$$= (0.4 \times 2) - (0.6 \times 1) - 0.002 = 0.8 - 0.6 - 0.002 = +0.198R$$

수수료 무시 시 0.2R로 계산되지만 실질은 0.198R로 소폭 낮습니다. 스캘핑처럼 트레이드 수가 많을수록 수수료 누적 영향이 커집니다.

## 핵심 요약

- 손익비는 목표 수익을 최대 손실로 나눈 값으로, 최소 1:2 이상을 권장합니다.
- 기대값(Expectancy) = (승률 × 평균 손익비) - (패율 × 1) 공식으로 전략의 장기 수익성을 평가합니다.
- 손익분기 승률은 1 / (1 + RR)으로 계산하며, 손익비가 높을수록 낮은 승률로도 수익이 가능합니다.
- 승률보다 기대값이 중요하며, 낮은 승률과 높은 손익비 조합이 높은 승률과 낮은 손익비보다 수익성이 높을 수 있습니다.
