## 포지션 사이징이란?

포지션 사이징(Position Sizing)**하나의 트레이드에 얼마를 투자할 것인지를** 결정하는 과정입니다. 기술적 분석에서 진입 시점을 찾는 것만큼, 혹은 그 이상으로 중요한 리스크 관리의 핵심입니다.

반 타프(Van Tharp) 박사는 "트레이딩 성공의 대부분은 포지션 사이징에서 결정된다"고 강조했습니다 [Van Tharp Institute, 2026]. 최고의 진입 기술도 잘못된 포지션 사이징으로 파산할 수 있고, 평범한 전략도 올바른 사이징으로 꾸준한 수익을 낼 수 있습니다.

## 핵심 개념: 계좌 리스크 vs 트레이드 리스크

포지션 사이징에서 두 가지 리스크를 명확히 구분해야 합니다.

| 구분 | 정의 | 예시 |
|------|------|------|
| 계좌 리스크 (Account Risk) | 전체 계좌 대비 이번 트레이드에서 잃을 수 있는 최대 금액 | 계좌 1,000만원의 1% = 10만원 |
| 트레이드 리스크 (Trade Risk) | 진입가와 손절가 사이의 가격 차이 | 진입 50,000원, 손절 48,000원 = 2,000원 |

$$\text{포지션 크기} = \frac{\text{계좌 리스크 금액}}{\text{트레이드 리스크 (1주당)}}$$

---

## 방법 1: 고정 비율법 (Fixed Fractional Method)

### 개념

전체 계좌 잔고의고**정된 비율(%**)만 위험에 노출시키는 방법입니다. 가장 단순하고 널리 사용되는 방법입니다 [Investopedia, 2026].

### 1~2% 규칙

전문 트레이더들이 권장하는 표준 리스크 비율입니다.

```
보수적 트레이더: 계좌의 0.5~1% 위험
일반 트레이더:   계좌의 1~2% 위험
공격적 트레이더: 계좌의 2~3% 위험 (비권장)
```

2% 이상을 지속적으로 투자하면 연속 손실 시 계좌가 빠르게 감소합니다.

### 계산 예시

계좌 잔고: 1,000만원, 리스크 비율: 1%

```
계좌 리스크 금액: 1,000만원 × 1% = 10만원

시나리오 1 - 주식 거래:
진입가: 50,000원
손절가: 48,000원
트레이드 리스크: 50,000 - 48,000 = 2,000원 (1주당)
포지션 크기: 100,000원 / 2,000원 = 50주

시나리오 2 - 비트코인 거래:
진입가: 50,000,000원
손절가: 48,500,000원
트레이드 리스크: 1,500,000원 (1BTC당)
포지션 크기: 100,000원 / 1,500,000원 ≈ 0.067 BTC
```

### 연속 손실 시 계좌 감소 비교

| 연속 손실 횟수 | 리스크 1% | 리스크 5% | 리스크 10% |
|----------------|-----------|-----------|------------|
| 5연속 손실 | 95.1% 잔존 | 77.4% 잔존 | 59.0% 잔존 |
| 10연속 손실 | 90.4% 잔존 | 59.9% 잔존 | 34.9% 잔존 |
| 20연속 손실 | 81.8% 잔존 | 35.8% 잔존 | 12.2% 잔존 |

1% 리스크에서 10연속 손실을 당해도 계좌의 90%가 남습니다. 10% 리스크라면 35%만 남습니다.

## 방법 2: 켈리 공식 (Kelly Criterion)

### 개념

켈리 공식은 수학자 존 켈리(John Kelly)가 1956년 Bell Labs에서 개발한 공식으로,**장기적으로 자본 성장을 극대화하는 최적 베팅 비율을** 계산합니다 [Investopedia, 2026].

### 공식

$$Kelly\% = W - \frac{1 - W}{R}$$

- $W$ = 승률 (Win Rate)
- $R$ = 평균 손익비 (Average Win / Average Loss)

### 계산 예시

```
승률: 55% (W = 0.55)
평균 이익: 150만원, 평균 손실: 100만원
R = 150 / 100 = 1.5

Kelly% = 0.55 - (1 - 0.55) / 1.5
Kelly% = 0.55 - 0.45 / 1.5
Kelly% = 0.55 - 0.30
Kelly% = 0.25 (25%)
```

### 하프 켈리 (Half Kelly) 사용

순수 켈리 공식이 제시하는 25%**너무 공격적입**니다. 실제 트레이딩에서는하**프 켈리(켈리% / 2**)또는쿼**터 켈리(켈리% / 4**)를 사용하는 것이 일반적입니다.

```
하프 켈리: 25% / 2 = 12.5%
쿼터 켈리: 25% / 4 = 6.25%
```

>**주의**: 켈리 공식은 과거 통계가 미래에도 동일하게 유지된다는 가정 하에 작동합니다. 시장 조건이 변하면 입력값도 바뀌므로 정기적으로 재계산해야 합니다.

## 방법 3: 고정 금액법 (Fixed Dollar Amount)

가장 단순한 방법으로, 매 트레이드마다동**일한 금액을** 투자합니다. 계좌 크기가 달라져도 투자 금액이 변하지 않아 하락 국면에서는 과도한 위험, 상승 국면에서는 낮은 활용도가 발생할 수 있어 비권장됩니다.

## 포지션 사이징 계산기 (종합)

```
입력값:
- 계좌 잔고: A
- 리스크 비율: R% (권장: 1~2%)
- 진입가: E
- 손절가: SL

계산:
1. 계좌 리스크 금액: A × R / 100
2. 1단위당 리스크: |E - SL|
3. 포지션 크기(단위): 계좌 리스크 금액 / 1단위당 리스크
4. 총 투자 금액: 포지션 크기 × E

예시:
A = 5,000,000원, R = 1%, E = 100,000원, SL = 95,000원
1. 50,000원
2. 5,000원
3. 50,000 / 5,000 = 10주
4. 10 × 100,000 = 1,000,000원 (총 투자)
```

## 방법 4: 변동성 기반 사이징 (Volatility-Based Sizing)

### 개념

변동성 기반 사이징은자**산의 현재 변동성(ATR)을 포지션 크기 결정에 직접 반영하**는 방법입니다. 변동성이 높은 시기에는 포지션을 줄이고, 변동성이 낮은 시기에는 포지션을 늘려 리스크를 일정하게 유지합니다.

$$\text{포지션 크기} = \frac{\text{계좌} \times \text{리스크 비율}}{\text{ATR} \times \text{배수}}$$

```
예시 1: 저변동성 구간
계좌: 1,000만원, 리스크 비율: 1%
BTC ATR(14일): 1,000,000원, 배수: 2.0

포지션 크기 = 100,000 / (1,000,000 × 2.0) = 0.05 BTC
총 투자금 = 0.05 × 50,000,000 = 2,500,000원

예시 2: 고변동성 구간 (ATR 2배 증가)
BTC ATR(14일): 2,000,000원, 배수: 2.0

포지션 크기 = 100,000 / (2,000,000 × 2.0) = 0.025 BTC
총 투자금 = 0.025 × 50,000,000 = 1,250,000원
→ 변동성이 2배 높아지면 포지션 크기도 절반으로 자동 조정
```

:::tip
변동성 기반 사이징의 핵심은 "항상 ATR × 배수만큼의 손실을 계좌 리스크 비율과 일치시키는 것"입니다. 이 방법은 손절 위치를 별도로 고민하지 않아도 되므로 추세 추종 전략에 특히 유용합니다.
:::

### 고정 비율법 vs 변동성 기반 비교

| 항목 | 고정 비율법 | 변동성 기반 |
|------|------------|------------|
| 손절 설정 | 별도로 결정 | ATR 배수로 자동 결정 |
| 시장 환경 반영 | 없음 | 있음 (변동성 자동 반영) |
| 계산 복잡도 | 낮음 | 중간 |
| 적합 전략 | 모든 전략 | 추세 추종, 스윙 |

## 실전 적용: 포지션 사이징 계산기 워크스루

### 완전한 계산 절차 (단계별)

실제 트레이드를 앞두고 포지션 크기를 결정하는 전체 과정을 단계별로 따라가 보겠습니다.

```
[상황]
종목: 카카오(035720)
현재가: 55,000원
진입 계획: 눌림목 매수, 20일 이평 지지 확인 후 진입
계좌 잔고: 5,000만원
목표 리스크 비율: 1%

[Step 1] 계좌 리스크 금액 계산
계좌 리스크 = 5,000만원 × 1% = 50만원

[Step 2] 손절가 결정
직전 스윙 저점: 52,500원
ATR(14일): 1,200원
버퍼: 1,200 × 0.25 = 300원
손절가: 52,500 - 300 = 52,200원

[Step 3] 1주당 리스크 계산
1주당 리스크 = 55,000 - 52,200 = 2,800원

[Step 4] 포지션 크기 계산
포지션 크기 = 50만원 / 2,800원 = 178.57 → 178주

[Step 5] 총 투자금 확인
총 투자금 = 178 × 55,000 = 9,790,000원 (계좌의 약 19.6%)

[Step 6] 손익비 확인
목표가: 61,000원 (직전 고점)
리워드: 61,000 - 55,000 = 6,000원
RR = 6,000 / 2,800 = 2.14 → RR 1:2.1 (적정)

[Step 7] 최종 검증
- 계좌 리스크: 50만원 (1%) ✓
- 총 투자금: 계좌의 20% 이하 ✓
- 손익비: 1:2 이상 ✓
→ 진입 승인
```

:::caution
총 투자금이 계좌의 30%를 넘으면 레버리지 효과 없이도 포지션 자체의 변동성 리스크가 커집니다. 손절 범위가 넓어 총 투자금이 과도해지는 경우, 포지션 크기를 줄이고 리스크 비율을 0.5%로 낮추는 것을 고려하세요.
:::

### 동시 다중 포지션 관리

여러 포지션을 동시에 보유할 때는총** 계좌 리스크를** 합산하여 관리합니다.

```
[예시: 3개 포지션 동시 보유]

포지션 A (삼성전자): 리스크 1% = 50만원
포지션 B (카카오):   리스크 1% = 50만원
포지션 C (BTC):      리스크 0.5% = 25만원

총 계좌 노출 리스크: 2.5% = 125만원
→ 권장 한도 3~5% 이내이므로 적정

추가 포지션 D 진입 시:
남은 리스크 한도: 5% - 2.5% = 2.5% = 125만원
→ 이 범위 내에서만 신규 진입 허용
```

## 흔한 실수

### 1. 포지션 크기를 감(感)으로 결정

```
잘못된 방식: "이 종목은 좋아 보이니 많이 사야겠다" → 계좌의 30% 투자
결과: 예상과 다른 방향으로 움직이면 계좌에 치명적 타격

올바른 방식: 반드시 손절가를 먼저 설정 → 계좌 리스크 금액을 트레이드 리스크로 나누어 계산
```

### 2. 연속 손실 후 베팅 크기 증가

```
오류: "5연속 손실 = 다음에는 반드시 이긴다" → 2배 포지션 진입
현실: 각 트레이드는 독립 사건, 이전 결과는 다음에 영향 없음
결과: 더 큰 손실로 계좌 급감

대응: 연속 손실 시 포지션 크기를 줄이거나(50%로 축소) 트레이딩 일시 중단
```

:::caution
마틴게일(Martingale) 전략처럼 손실 후 베팅을 배로 늘리는 방식은 단기적으로 회복처럼 보이지만, 연속 손실이 길어지면 계좌 전액을 소멸시킬 수 있습니다.
:::

### 3. 레버리지 적용 시 실제 리스크 계산 착오

```
잘못된 계산:
레버리지 10배, 계좌 100만원, 포지션 1,000만원
"손절 1% = 10만원 손실 = 계좌의 10%"

올바른 계산:
포지션 1,000만원의 1% = 10만원 = 계좌의 10%
→ 레버리지 없을 때보다 훨씬 큰 비율이 위험에 노출

리스크 비율을 계좌 대비로 계산하면:
계좌 1% = 1만원 → 포지션 1,000만원에서 0.1%만 하락해도 계좌 1% 손실
```

### 4. 켈리 공식을 풀 켈리로 적용

```
켈리 공식 결과: 25%
풀 켈리 적용: 계좌의 25%를 매 트레이드에 리스크
결과: 극단적 변동성, 10연속 손실 시 계좌 94% 소멸

올바른 적용:
하프 켈리: 12.5%
쿼터 켈리: 6.25%
실전 권장: 켈리 결과의 25~50%만 적용
```

### 5. 포지션 크기 계산 후 감정적으로 조정

```
계산 결과: 50주
생각: "50주는 너무 많아 불안해" → 20주로 줄임
또는: "수익이 나면 더 크게 벌어야지" → 100주로 늘림

문제: 시스템 기반 사이징의 의미 상실
      장기적으로 기대값과 실제 결과의 괴리 발생
해결: 계산된 크기를 그대로 실행하는 기계적 규칙 수립
```

## 레버리지와 포지션 사이징

레버리지(leverage) 상품 거래 시 포지션 사이징에 더욱 주의가 필요합니다.

```
레버리지 10배, 계좌 100만원:
실제 통제 금액: 1,000만원

1% 리스크 적용:
계좌 리스크 금액: 100만원 × 1% = 1만원
손절 범위: 진입가의 0.1% (1만원 / 1,000만원)
```

레버리지가 높을수록 손절 범위가 좁아지므로, 가격 변동성에 맞는 손절 설정이 필수입니다.

## 에지 케이스: 극단적 레버리지와 청산 리스크

**고레버리지 크립토 선물 거래레**버리지 20배 이상의 크립토 선물에서는 1% 리스크 규칙을 그대로 적용하면 손절 범위가 지나치게 좁아집니다. 예를 들어 레버리지 20배로 계좌 100만원에서 1만원(1%)을 리스크로 설정하면, 포지션 2,000만원에서 0.05% 하락만으로 손절이 발동됩니다. 고레버리지 환경에서는 리스크 비율을 0.2~0.5%로 낮추고, 손절 범위를 ATR 기반으로 설정하는 것이 현실적입니다.

**드로다운 중 포지션 크기 조정계**좌가 20% 이상 드로다운된 상황에서 원금 회복을 위해 포지션 크기를 늘리는 것은 위험합니다. 드로다운이 깊을수록 오히려 리스크 비율을 줄여(예: 2% → 0.5%) 추가 손실을 제한하고, 회복 후 원래 수준으로 복귀하는 단계적 접근이 안전합니다.

## 핵심 요약

- 포지션 사이징은 진입 기술만큼 중요한 리스크 관리의 핵심으로, 계좌 리스크를 트레이드 리스크로 나누어 포지션 크기를 결정합니다.
- 고정 비율법(1~2% 규칙)은 가장 실용적인 방법으로, 10연속 손실에도 계좌의 90%를 보존합니다.
- 켈리 공식은 승률과 손익비를 기반으로 최적 투자 비율을 계산하며, 실전에서는 하프 켈리(Kelly / 2)를 사용합니다.
- 레버리지 상품에서는 실질 포지션 크기를 기준으로 리스크를 계산하여 과도한 노출을 방지합니다.
