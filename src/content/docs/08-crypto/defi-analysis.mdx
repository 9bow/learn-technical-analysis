## 개요

DeFi(Decentralized Finance, 탈중앙화 금융)는 블록체인 위에서 스마트 컨트랙트로 구현된 금융 서비스 생태계입니다. 은행, 거래소, 대출 기관 없이 코드가 금융 로직을 자동 실행합니다.

DeFi 분석은 전통적 기술적 분석과 온체인 분석을 결합하되,**프로토콜 고유의 경제 지표를** 추가로 이해해야 합니다. 가격 차트만 보는 것으로는 DeFi 자산의 내재 가치를 판단하기 어렵습니다.

위 차트는 DeFi 토큰의 전형적인 고변동성 패턴을 보여줍니다. 프로토콜 펀더멘털을 이해하면 이러한 변동성 속에서 내재 가치 대비 가격 괴리를 판단할 수 있습니다.

:::note
DeFi 분석은 기술적 분석을 대체하지 않습니다. 프로토콜 펀더멘털 분석 + 온체인 데이터 + 가격 차트를 종합하는 접근이 필요합니다.
:::

## TVL (Total Value Locked) 분석

### TVL의 정의

TVL은 특정 DeFi 프로토콜의 스마트 컨트랙트에예**치된 자산의 총 USD 환산 가치입**니다. 프로토콜의 사용 규모와 신뢰도를 가늠하는 1차 지표입니다.

```
TVL = 유동성 풀 예치금 + 대출 프로토콜 담보금 + 스테이킹 자산
      (모두 현재 시점 USD 환산)
```

**예시:**
```
Uniswap V3 유동성 풀:
ETH 풀: 500,000 ETH × 3,200 USD = 1,600,000,000 USD
USDC 풀: 800,000,000 USDC
기타 토큰 풀: 200,000,000 USD

TVL = 1,600,000,000 + 800,000,000 + 200,000,000
    = 2,600,000,000 USD (26억 달러)
```

### TVL 활용 지표

**TVL 비율 (TVL Ratio = 시가총액 / TVL**)
```
프로토콜 X:
시가총액: 500,000,000 USD
TVL: 2,000,000,000 USD
TVL 비율: 500,000,000 / 2,000,000,000 = 0.25

프로토콜 Y:
시가총액: 800,000,000 USD
TVL: 400,000,000 USD
TVL 비율: 800,000,000 / 400,000,000 = 2.0
```

| TVL 비율 | 해석 |
|---------|------|
| 1 미만 | 시가총액이 TVL보다 작음 — 상대적 저평가 가능성 |
| 1 ~ 2 | 일반적 범위 |
| 3 이상 | 시가총액이 TVL 대비 높음 — 고평가 검토 필요 |

:::caution
TVL은 토큰 가격에 연동됩니다. 토큰 가격이 하락하면 TVL도 동반 하락하므로, TVL 변화가 실제 사용자 이탈인지 가격 변동의 결과인지를 구분해야 합니다.
:::

### TVL 추세 분석

```
TVL 상승 + 토큰 가격 상승 → 유기적 성장, 긍정 신호
TVL 상승 + 토큰 가격 하락 → 예치량 증가, 추가 매도 압력 경계
TVL 하락 + 토큰 가격 상승 → 사용자 이탈, 지속 가능성 점검 필요
TVL 하락 + 토큰 가격 하락 → 전반적 자본 유출, 부정 신호
```

## Yield Farming APY 계산

### APR vs APY

DeFi 수익률은 APR(Annual Percentage Rate, 단순 연 수익률)과 APY(Annual Percentage Yield, 복리 적용 연 수익률)로 표기됩니다.

$$APY = \left(1 + \frac{APR}{n}\right)^n - 1$$

- $n$ = 연간 복리 횟수
- $APR$ = 복리 없는 단순 연 수익률

**예시: 일별 복리(n=365**)
```
APR: 50%

APY = (1 + 0.50 / 365)^365 - 1
    = (1 + 0.001370)^365 - 1
    = 1.6487 - 1
    = 0.6487
    = 64.87%

→ APR 50%가 일별 복리 적용 시 실질 APY 64.87%
```

### 복리 주기별 APY 비교

APR 100% 기준:

```
복리 주기          APY
연 1회 (n=1)  →  100.00%
월 1회 (n=12) →  161.30%
일 1회 (n=365)→  171.46%
시간별 (n=8760)→  171.81%
연속 복리       →  171.83% (e^1 - 1)
```

:::note
APR 100%라도 복리 효과로 APY는 최대 약 171.8%까지 증가합니다. DeFi 플랫폼의 표기 방식을 항상 확인하십시오.
:::

### 실제 수익 계산

```
조건:
원금: 10,000 USDC
풀 APR: 80% (일별 복리 재투자)
투자 기간: 90일

일 수익률: 80% / 365 = 0.2192%

90일 후 자산 = 10,000 × (1 + 0.002192)^90
             = 10,000 × 1.2189
             = 12,189 USDC

90일 수익: 2,189 USDC (21.89%)
연환산 APY: (1.002192)^365 - 1 = 122.1%
```

### Yield Farming의 실질 수익 감소 요인

표기된 APY가 실제 수익과 다를 수 있는 이유:

```
1. 토큰 인플레이션
   보상 토큰 발행 증가 → 토큰 가격 하락
   APY 100% 보상받았으나 토큰 가격 -60% → 실질 수익 -20%

2. TVL 증가로 인한 수익률 희석
   풀에 자금이 몰릴수록 동일 보상을 더 많이 나눔
   → APY 자동 하락

3. 거래 수수료 및 가스비
   이더리움 기반 풀: 재투자마다 가스비 발생
   소액 예치에서는 가스비가 수익을 초과할 수 있음
```

## 비영구적 손실 (Impermanent Loss)

### 개념

비영구적 손실(IL)은 AMM(Automated Market Maker) 유동성 풀에 자산을 예치했을 때,**단순 보유(HODL) 대비 발생하는 가치 손실입**니다. 가격이 초기 예치 시점에서 변동할수록 손실이 커집니다 [Binance Academy, 2026].

"비영구적"이라는 이름은 가격이 초기 비율로 돌아오면 손실이 사라지기 때문입니다. 그러나 실제로는 가격이 돌아오지 않는 경우가 많아실**현 손실로** 굳어질 수 있습니다.

### IL 계산 공식

Uniswap V2 방식 50:50 풀 기준:

$$IL = \frac{2\sqrt{r}}{1 + r} - 1$$

- $r$ = 현재 가격 / 초기 가격 (가격 변동 배수)

**단계별 계산 예시:**
```
초기 예치:
ETH 1개 (가격: 2,000 USDC)
USDC 2,000개
총 예치 가치: 4,000 USDC

→ ETH 가격이 3,000 USDC로 50% 상승한 경우

r = 3,000 / 2,000 = 1.5

IL = 2 × sqrt(1.5) / (1 + 1.5) - 1
   = 2 × 1.2247 / 2.5 - 1
   = 2.4495 / 2.5 - 1
   = 0.9798 - 1
   = -0.0202
   = -2.02%
```

**풀 내 실제 자산 변화:**
```
AMM은 항상 x × y = k 공식 유지 (k = 2,000)
초기: 1 ETH × 2,000 USDC = 2,000

ETH 가격 3,000 USDC 시:
새 ETH 수량 = sqrt(k / 새가격) = sqrt(2,000 / 3,000) = 0.8165 ETH
새 USDC 수량 = sqrt(k × 새가격) = sqrt(2,000 × 3,000) = 2,449.5 USDC

풀 내 가치: 0.8165 × 3,000 + 2,449.5 = 2,449.5 + 2,449.5 = 4,899 USDC

단순 보유 시 가치: 1 ETH × 3,000 + 2,000 = 5,000 USDC

비영구적 손실: 4,899 - 5,000 = -101 USDC (-2.02%)
```

### 가격 변동별 IL 크기

ETH/USDC 풀, ETH 가격 변동 기준:

```
가격 변동    IL
+25%     →  -0.6%
+50%     →  -2.0%
+100%    →  -5.7%
+200%    →  -13.4%
+400%    →  -25.5%
-25%     →  -0.6%
-50%     →  -5.7%
-75%     →  -25.5%
```

:::caution
두 자산 모두 변동성이 높은 풀(예: ETH/BTC)은 가격 상관관계에 따라 IL이 더 커질 수 있습니다. IL이 수수료 수입을 초과하는 경우 유동성 공급이 손해입니다.
:::

### IL 완화 전략

```
1. 안정화폐 풀 (USDC/USDT): IL 거의 없음, 수수료 낮음
2. 높은 수수료 풀 선택: IL을 수수료로 상쇄
3. 단기 예치: 가격 변동 노출 시간 최소화
4. 집중 유동성(V3): 특정 가격 범위에 집중 → 수수료 효율 향상
```

## DEX 유동성 풀 메커니즘

### AMM의 가격 결정 원리

탈중앙화 거래소(DEX)는 오더북 대신자**동화 시장 조성자(AMM**)알고리즘으로 가격을 결정합니다.

$$x \times y = k$$

- $x$ = 토큰 A 수량
- $y$ = 토큰 B 수량
- $k$ = 상수 (풀 불변량)

**거래 실행 예시:**
```
풀 상태: ETH 100개, USDC 200,000개
k = 100 × 200,000 = 20,000,000
현재 ETH 가격: 200,000 / 100 = 2,000 USDC

사용자가 ETH 1개 구매:
새 ETH 수량: 100 - 1 = 99개
새 USDC 수량: 20,000,000 / 99 = 202,020.2 USDC

지불 USDC: 202,020.2 - 200,000 = 2,020.2 USDC
실제 ETH 가격: 2,020.2 USDC (슬리피지 포함)

→ 수량이 적을수록 슬리피지(가격 미끄러짐) 증가
```

### 슬리피지와 풀 깊이

```
풀 규모별 슬리피지 (ETH 10개 구매 기준):

소형 풀 (ETH 100개):
구매 후 ETH 수량: 90개
k = 20,000,000
USDC 수량: 20,000,000 / 90 = 222,222 USDC
지불 금액: 22,222 USDC → ETH당 2,222.2 USDC
슬리피지: +11.1%

대형 풀 (ETH 10,000개):
k = 2,000,000,000
구매 후 ETH: 9,990개
USDC: 2,000,000,000 / 9,990 = 200,200 USDC
지불: 200 USDC 증가 → ETH당 2,002 USDC
슬리피지: +0.1%
```

풀이 클수록 슬리피지가 낮고, 대규모 거래도 가격에 미치는 영향이 적습니다.

### 수수료 구조

유동성 공급자(LP)는 거래 수수료의 일정 비율을 수익으로 받습니다.

```
Uniswap V2: 거래액의 0.30%
Uniswap V3: 0.01% / 0.05% / 0.30% / 1.00% (풀별 상이)
Curve: 0.04% (안정화폐 특화, 낮은 슬리피지)
```

## DeFi 프로토콜 밸류에이션

### P/S 비율 (Price-to-Sales Ratio)

전통 금융의 주가매출비율과 유사하게, 프로토콜 시가총액을 연간 수수료 수익으로 나눈 값입니다.

```
P/S = 시가총액 / 연간 프로토콜 수수료 수익

예시:
프로토콜 X:
시가총액: 1,000,000,000 USD
일평균 수수료: 1,000,000 USD
연간 수수료: 365,000,000 USD
P/S = 1,000,000,000 / 365,000,000 = 2.74
```

| P/S 비율 | 판단 참고 |
|---------|----------|
| 1 미만 | 수수료 대비 시가총액 낮음 — 잠재적 저평가 |
| 1 ~ 5 | 일반적 범위 |
| 10 이상 | 성장 기대가 가격에 선반영 — 고평가 가능성 |

### P/E 유사 지표 (프로토콜 수익 기준)

프로토콜에 따라 수수료 전체가 아닌토**큰 홀더 또는 재무(Treasury)에 귀속되는 수익만**을 사용하는 경우도 있습니다.

```
프로토콜 수익 = 총 수수료 - LP 배분 수수료
(실제로 프로토콜/토큰 홀더에게 귀속되는 부분)

P/E(유사) = 시가총액 / 연간 프로토콜 순수익
```

**예시:**
```
Uniswap V3 (단순화):
총 수수료: 연 3억 USD
LP 배분: 100% (현재 프로토콜 수수료 없음)
프로토콜 순수익: 0

→ P/E 계산 불가 (수익이 토큰 홀더에 귀속되지 않음)

Curve Finance (단순화):
총 수수료: 연 1억 USD
LP 배분: 50%
프로토콜 귀속: 50% = 5,000만 USD
시가총액: 5억 USD
P/E(유사) = 500,000,000 / 50,000,000 = 10
```

### 주요 DeFi 지표 종합

| 지표 | 계산식 | 의미 |
|------|--------|------|
| TVL | 예치 자산 총계 (USD) | 프로토콜 규모 |
| TVL 비율 | 시가총액 / TVL | 밸류에이션 수준 |
| P/S | 시가총액 / 연 수수료 | 수익 대비 가격 |
| 일 수수료 | 24h 수수료 합계 | 실제 사용 수요 |
| 활성 사용자 | 일간 고유 지갑 수 | 실제 이용자 규모 |
| 수수료 / TVL | 연 수수료 / TVL | 자본 효율성 |

**수수료/TVL 계산 예시:**
```
프로토콜 A:
연간 수수료: 100,000,000 USD
TVL: 2,000,000,000 USD
수수료/TVL = 5.0% → 예치 자본의 연 5% 수수료 창출

프로토콜 B:
연간 수수료: 50,000,000 USD
TVL: 500,000,000 USD
수수료/TVL = 10.0% → 자본 효율성이 A의 2배
```

## 실전 적용: DeFi 프로토콜 비교 분석

두 가상의 DEX 프로토콜을 비교합니다:

```
           프로토콜 A    프로토콜 B
시가총액     20억 USD      5억 USD
TVL         80억 USD      8억 USD
TVL 비율    0.25          0.625
일 수수료    200만 USD     80만 USD
연 수수료    7.3억 USD     2.92억 USD
P/S         2.74          1.71
수수료/TVL  9.1%          36.5%
```

**해석:**
- 프로토콜 A는 절대 규모(TVL, 수수료)가 크지만 P/S와 TVL 비율에서 B보다 비싸게 거래됩니다.
- 프로토콜 B는 TVL 대비 수수료 효율(36.5%)이 A(9.1%)보다 월등히 높습니다. 소규모이지만 자본 효율성이 뛰어납니다.
- 투자 관점: 성장 잠재력 중시 시 B, 안정성 중시 시 A.

:::tip
단일 지표로 판단하지 말고, TVL + P/S + 수수료/TVL + 활성 사용자를 함께 검토하십시오. 지표 간 상충이 있을 경우 그 이유를 파악하는 것이 핵심입니다.
:::

## 흔한 실수

### 1. 표기 APY를 실제 수익으로 오인

```
문제: "APY 500%" 표기 → 1,000 USDC 예치 → 1년 후 5,000 USDC 기대

현실:
- 보상 토큰 가격이 -90% 하락
- 실질 수익: 500% × (1 - 0.9) = 50%
- 가스비 차감 후: 약 30~40%
```

보상 토큰의 가격 지속 가능성을 반드시 검토하십시오.

---

### 2. 비영구적 손실 과소평가

```
문제: "수수료 APY 30%면 손해 없겠지"

현실:
ETH가 2배 상승 시 IL = -5.7%
ETH가 4배 상승 시 IL = -25.5%

→ 강세장에서 유동성 공급이 오히려 손해
→ IL이 수수료 수입을 초과하는지 주기적 계산 필요
```

---

### 3. 스마트 컨트랙트 리스크 무시

DeFi의 본질적 리스크는코**드 취약점 익스플로잇입**니다:

```
2021~2026년 DeFi 해킹 주요 사례:
- 플래시론 공격
- 오라클 가격 조작
- 재진입(Reentrancy) 취약점
- 거버넌스 토큰 집중 악용
```

신규 프로토콜은 최소감**사(Audit) 완료 여부**, 버그 바운티 프로그램 운영 여부, TVL 대비 보험 커버리지를 확인하십시오.

---

### 4. 거버넌스 리스크 과소평가

DeFi 프로토콜은 거버넌스 토큰 보유자가 프로토콜 변경을 결정합니다.

```
리스크 시나리오:
- 소수 고래가 거버넌스 토큰 다수 보유
- 악의적 제안 통과 → 자금 유출 가능
- 수수료 구조 변경으로 LP 수익 감소
```

거버넌스 토큰 분산도(Gini 계수), 제안 통과 정족수를 사전 확인하십시오.

## 에지 케이스

### 1. TVL 급감이 항상 악재는 아닌 경우

```
시나리오: 프로토콜 TVL 100억 USD → 40억 USD (-60%)

원인 A (악재): 해킹·익스플로잇 → 자금 강제 유출
원인 B (중립): 인센티브 프로그램 종료 → 용병 자본 이탈
원인 C (호재 가능): 거버넌스 개편으로 유동성 집중화 → 효율 상승

→ TVL 변화만으로 판단 금지
→ 수수료/TVL 비율이 TVL 감소 후 오히려 상승했다면 자본 효율성 개선 신호
→ 활성 사용자 수와 수수료 절대값도 동시에 확인
```

### 2. 극단적 APY의 폰지 구조 감별

```
경고 신호:
- 보상 토큰이 거버넌스·수수료 외 실질 가치 없음
- TVL 대비 일 배출 보상 > 일 수수료 × 10 이상
- 출시 2주 이내 APY 1,000% 이상

계산 예시:
보상 토큰 일 배출량: 10,000개 (가격: 10 USD) = 10만 USD/일
프로토콜 일 수수료: 5,000 USD

→ 배출 비용이 수익의 20배 → 지속 불가능한 구조
→ 초기 진입자가 보상을 덤핑하면 토큰 가격 폭락 → 실질 APY 급락
```

### 3. 집중 유동성(Uniswap V3) 범위 이탈

```
상황: ETH/USDC 풀에 ETH 1,800~2,200 범위로 집중 유동성 공급

ETH 가격이 2,500으로 상승 시:
→ 포지션이 유동성 범위 이탈 → 수수료 수입 0
→ 전체 포지션이 USDC로 전환 (ETH 상승분 이익 포기)
→ ETH를 보유했을 때보다 불리한 결과

대응:
- 변동성이 높을수록 범위를 넓게 설정 (수수료 효율 감소)
- 가격 이탈 시 리밸런싱 비용(가스비)과 수익 비교 후 판단
```

### 4. 오라클 가격 조작 공격 중 분석 오류

```
시나리오:
플래시론 공격자가 DEX 풀 가격을 10배 조작 (블록 내 일시적)
온체인 데이터 도구가 해당 블록의 극단적 가격을 기록

→ 분석 차트에 스파이크 발생 → 실제 시장 움직임으로 오인 가능
→ 공격 후 TVL 급감과 함께 나타나면 익스플로잇 징후
→ 이상 스파이크 발생 시 블록 탐색기에서 해당 블록 직접 확인
```

## 핵심 요약

- TVL은 프로토콜 예치 자산 총계로, TVL 비율(시가총액/TVL)이 1 미만이면 상대적 저평가 신호일 수 있습니다.
- Yield Farming APY는 단순 표기값이 아닌 보상 토큰 가격 변동, 수수료, TVL 증가에 따른 희석을 반영한 실질 수익으로 계산해야 합니다.
- 비영구적 손실은 AMM 풀의 구조적 특성으로, 가격이 2배 상승 시 약 -5.7%, 4배 상승 시 약 -25.5%가 발생하며 수수료 수입으로 상쇄해야 합니다.
- DEX의 x × y = k 공식은 풀 규모가 클수록 슬리피지를 낮추고, 대형 풀이 더 안정적인 가격 실행을 제공합니다.
- P/S 비율과 수수료/TVL은 DeFi 프로토콜의 수익성과 자본 효율성을 평가하는 핵심 밸류에이션 지표입니다.
- DeFi 투자의 고유 리스크(스마트 컨트랙트 취약점, 오라클 조작, 거버넌스 공격)는 수익률 계산에 앞서 반드시 검토해야 합니다.
