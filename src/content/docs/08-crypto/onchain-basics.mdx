## 온체인 분석이란?

온체인 분석(On-Chain Analysis)**블록체인에 공개적으로 기록된 데이터를** 분석하여 자산의 가치와 시장 심리를 파악하는 방법론입니다. 전통 금융 시장과 달리 암호화폐 블록체인은 모든 거래 내역, 지갑 잔액, 네트워크 활동을 누구나 조회할 수 있습니다.

전통적인 기술적 분석이가**격과 거래량만**을 다루는 반면, 온체인 분석은 체인 위에서 실제로 일어나는 자금 흐름, 투자자 행동, 네트워크 건전성을 직접 관찰할 수 있다는 점에서 차별화됩니다.

<CandlestickDemo client:visible dataset="volatile-crypto" title="가상자산 가격 차트" />

| 구분 | 기술적 분석 | 온체인 분석 |
|------|------------|------------|
| 데이터 원천 | 거래소 시세 데이터 | 블록체인 원장 |
| 주요 변수 | 가격, 거래량 | 지갑 활동, 자금 흐름 |
| 투명성 | 거래소마다 상이 | 누구나 검증 가능 |
| 활용 | 단기~중기 타이밍 | 중기~장기 사이클 판단 |

>**핵심**: 온체인 데이터는 위·변조가 불가능한 블록체인에 기록되므로, 시장 조성자가 조작할 수 없는 '원시 데이터'에 가깝습니다 [Glassnode Academy, 2026].

## 네트워크 활동 지표

### 활성 주소 (Active Addresses)

특정 기간(보통 24시간) 동안송**신 또는 수신에 참여한 고유 주소의 **입니다. 블록체인 네트워크의 실제 사용자 규모를 나타내는 기본 지표입니다.

```
활성 주소 증가 → 네트워크 채택 확대 → 장기적 가격 지지 요인
활성 주소 감소 → 사용자 이탈 또는 관심 저하 → 약세 신호 가능성
```

주의할 점은 한 사람이 여러 주소를 사용하거나, 자동화된 봇 트랜잭션도 집계된다는 것입니다. 따라서 단독 지표보다는추**세 변화를** 봐야 합니다.

### 트랜잭션 볼륨 (Transaction Volume)

체인 위에서 실제로 이동한 코인의 총량(USD 환산값 포함)입니다. 거래소 거래량과 달리,**실제 온체인 결제와 이동을** 측정합니다.

```
온체인 볼륨 급증 + 가격 상승 → 실수요 기반 강세
온체인 볼륨 감소 + 가격 상승 → 거래소 내 투기성 랠리 가능성
```

## 거래소 유입/유출 분석

### 거래소 유입 (Exchange Inflow)

코인이개**인 지갑에서 거래소 지갑으로 이동하**는 것을 의미합니다. 거래소로 코인을 옮기는 가장 일반적인 이유는매**도 준비입**니다.

```
거래소 유입 급증 → 잠재 매도 압력 증가 → 단기 가격 하락 가능성
```

### 거래소 유출 (Exchange Outflow)

코인이거**래소에서 개인 지갑(콜드월렛 등)으로 이동하**는 것입니다. 장기 보유 의도를 나타내는 경향이 있습니다.

```
거래소 유출 급증 → 공급 감소 → 중장기 가격 상승 지지
```

### 거래소 보유량 (Exchange Reserve)

거래소 전체 지갑에 보관된 코인 총량입니다. 이 수치가 지속적으로 감소하면 시장에 유통되는 유효 공급이 줄어드는 것을 의미합니다.

| 거래소 보유량 변화 | 해석 |
|-----------------|------|
| 급격히 감소 | 강세 신호 — 장기 보유자 축적 |
| 급격히 증가 | 약세 신호 — 매도 압력 증가 |
| 완만히 감소 | 지속적인 수요 |

## 수익성 기반 지표

### 실현 시가총액 (Realized Cap)

시장 시가총액(현재가 × 총공급량)과 달리, 각 코인이마**지막으로 온체인 이동했을 때의 가격을** 기준으로 계산한 시가총액입니다 [CoinMetrics, 2026].

```
실현 시가총액 = 각 UTXO(코인 단위) × 마지막 이동 시점 가격의 합
```

실현 시가총액은 시장 전체의평**균 원가 기준에** 가까우며, 시장 참여자들이 얼마에 코인을 보유하고 있는지 보여줍니다.

### NUPL (Net Unrealized Profit/Loss)

전체 시장 참여자의미**실현 손익 비율입**니다.

$$NUPL = \frac{\text{시가총액} - \text{실현 시가총액}}{\text{시가총액}}$$

| NUPL 값 | 시장 상태 |
|---------|----------|
| 0.75 초과 | 탐욕(Greed) — 역사적 고점 근방 |
| 0.5 ~ 0.75 | 믿음/낙관(Belief/Optimism) |
| 0.25 ~ 0.5 | 희망(Hope) |
| 0 ~ 0.25 | 항복(Capitulation) 직전 |
| 0 미만 | 공포(Fear) — 역사적 저점 근방 |

NUPL이 0 아래로 내려가면 시장 전체가 평균적으로 손실 상태로, 역사적으로 강력한매**수 구역이**었습니다 [Glassnode Academy, 2026].

### MVRV 비율 (Market Value to Realized Value)

$$MVRV = \frac{\text{시가총액}}{\text{실현 시가총액}}$$

MVRV는 현재 가격이 실현 시가총액(시장 평균 원가) 대비 얼마나 높은지를 나타냅니다.

| MVRV 값 | 해석 |
|---------|------|
| 3.5 초과 | 역사적 과매수 — 고점 경계 |
| 1.5 ~ 3.5 | 정상 상승 구간 |
| 1.0 근처 | 중립 — 평균 원가 근방 거래 |
| 1.0 미만 | 역사적 저점 구역 — 축적 기회 |

>**실전 예시**: 비트코인 MVRV가 3.7을 돌파했던 2021년 11월 전후로 시장은 대형 고점을 형성했습니다. 반대로 2022년 말 MVRV가 0.7까지 하락하며 역사적 저점 구간을 나타냈습니다.

## 장기 보유자 vs 단기 보유자

온체인 분석에서는코**인을 마지막으로 이동한 시점을** 기준으로 보유자를 구분합니다.

-**장기 보유자(Long-Term Holder, LTH**): 155일 이상 코인을 이동하지 않은 주소
-**단기 보유자(Short-Term Holder, STH**): 155일 미만 보유 주소

장기 보유자가 코인을 매도하기 시작하면 상당한 공급 압력이 생깁니다. 반대로 장기 보유자의축**적(accumulation**)패턴은 바닥 형성의 신호로 자주 해석됩니다.

## UTXO 코인 나이 분석 (UTXO Age Analysis)

### UTXO란?

비트코인은**UTXO(Unspent Transaction Output, 미사용 트랜잭션 출력**)모델로 동작합니다. 각 UTXO는 마지막으로 이동한 시점이 블록체인에 기록되어 있으며, 이를 통해 코인이얼**마나 오래 이동하지 않았는지분**석할 수 있습니다.

```
UTXO 나이 예시:
- 2020년 1월에 마지막으로 이동한 100 BTC → 나이: 약 6년
- 2024년 12월에 마지막으로 이동한 0.5 BTC → 나이: 약 3개월
```

### HODL 웨이브 (HODL Waves)

HODL 웨이브는 각 나이 구간의 UTXO 비중을 색상으로 시각화한 차트입니다.

| UTXO 나이 구간 | 해석 |
|--------------|------|
| 1일~1주 | 단기 투기 활동 |
| 1주~1개월 | 단기 트레이더 보유 |
| 1~6개월 | 중기 보유자 |
| 6개월~1년 | 장기 보유 시작 |
| 1년~2년 | 강한 장기 보유자 (Diamond Hands) |
| 2년 이상 | 극장기 보유자, 사실상 유통 불가 물량 |

```
신호 해석:
장기 UTXO 비중 증가 → 코인이 강한 손으로 집중
→ 유통 공급 감소 → 장기 강세 신호

단기 UTXO 비중 급증 → 오래된 코인이 움직임
→ 장기 보유자 매도 가능성 → 주의 신호
```

:::note
오래된 코인이 갑자기 움직이는 현상을**"고래 깨어남(Dormant Whale Awakening)"**이라 부릅니다. 이 신호는 대형 매도 압력의 선행 지표가 될 수 있습니다.
:::

## NVT 비율 (Network Value to Transactions Ratio)

### 개념

NVT(Network Value to Transactions)는 주식의 PER(주가수익비율)과 유사한 개념으로,**네트워크 가치(시가총액)를 실제 거래량(온체인 전송량)으로 나눈 **입니다 [CoinMetrics, 2026].

$$NVT = \frac{\text{시가총액}}{\text{일일 온체인 전송량 (USD 환산)}}$$

```
예시:
BTC 시가총액: 1조 달러
일일 온체인 전송량: 50억 달러
NVT = 1,000,000,000,000 / 5,000,000,000 = 200
```

### NVT 해석

| NVT 수준 | 해석 |
|----------|------|
| 낮음 (50 미만) | 네트워크 활동 대비 저평가 — 잠재적 매수 구간 |
| 보통 (50~150) | 정상 범위 |
| 높음 (150 초과) | 투기적 거품 가능성 — 주의 |
| 매우 높음 (200 초과) | 역사적 과열 구간 |

```
NVT 활용 예시:
2017년 12월 BTC 급등 시 NVT 약 250 → 이후 급락
2018~2019년 베어마켓 NVT 40~60 → 저점 구간 확인
```

:::caution
NVT는 느리게 움직이는 후행 지표입니다. 단기 트레이딩 신호보다는사**이클 과열/저평가 판단에** 활용하고, 다른 온체인 지표와 함께 사용하세요.
:::

### NVT 시그널 (NVT Signal)

NVT를 14일 이동평균으로 부드럽게 처리한 버전으로, 단기 노이즈를 줄여 신호의 신뢰도를 높입니다.

$$NVT\ Signal = \frac{\text{시가총액}}{\text{온체인 전송량의 14일 이동평균}}$$

## 실전 온체인 분석 워크플로우

### 단계별 분석 절차

온체인 데이터를 실전에서 활용하는 체계적인 분석 순서입니다.

```
[Step 1] 시장 사이클 위치 확인 (주간 단위)

도구: NUPL, MVRV
판단:
- NUPL 0 미만 → 약세장 저점 구간 (매수 관심)
- MVRV 1.0 미만 → 역사적 저점 근방
- NUPL 0.75 초과 + MVRV 3.5 초과 → 강세장 고점 경계

[Step 2] 공급 압력 점검 (주간 단위)

도구: 거래소 보유량, 거래소 유입/유출
판단:
- 거래소 보유량 지속 감소 + 유출 급증 → 공급 부족 신호
- 거래소 유입 급증 + NUPL 높음 → 장기 보유자 매도 주의

[Step 3] 장단기 보유자 동향 확인 (주간 단위)

도구: LTH/STH 실현 손익, HODL 웨이브
판단:
- LTH 순실현 이익 증가 → 장기 보유자 차익 실현 시작
- STH 손실 상태 + LTH 축적 → 저점 근방 신호

[Step 4] 네트워크 건전성 확인 (월간 단위)

도구: 활성 주소, 트랜잭션 볼륨, NVT
판단:
- 활성 주소 증가 + NVT 정상 범위 → 실수요 기반 상승
- 가격 상승인데 활성 주소 감소 → 투기적 랠리 경계

[Step 5] 종합 판단 및 포지션 방향 결정

온체인 신호 + 기술적 분석 구조 확인
→ 온체인 강세 + 가격 지지구조 확인 시 매수 우선
→ 온체인 약세 + 저항 구조 시 매도 또는 관망
```

:::tip
무료로 활용할 수 있는 온체인 도구: Glassnode (일부 무료), CryptoQuant (일부 무료), Look Into Bitcoin (무료 주요 지표), Bitcoin Visuals (무료). 유료 플랜 없이도 NUPL, MVRV, 거래소 보유량 등 핵심 지표 대부분을 확인할 수 있습니다.
:::

## 흔한 실수

### 1. 온체인 신호만으로 단기 트레이딩

```
실수: NUPL이 낮다 → 지금 당장 매수
현실: 온체인 지표는 후행성이 강해 수주~수개월 단위 판단에 적합
결과: 저점보다 훨씬 일찍 진입 → 추가 하락에 노출

올바른 활용: 온체인 = 사이클 위치 파악
            가격 구조 = 진입 타이밍 결정
```

### 2. 거래소 유입을 무조건 매도 신호로 해석

```
오해: 거래소 유입 급증 → 즉시 매도
현실: 거래소 유입에는 여러 이유가 있음
  - 실제 매도 준비 (매도 압력)
  - 거래소 내 스테이킹/이자 농사 목적
  - 거래소 간 이전 (내부 이체)
  - 수탁 서비스 정리

대응: 단순 유입량이 아닌 유입 후 실제 매도 여부(거래량 확인)까지 체크
```

### 3. 단일 지표에 과도하게 의존

```
실수: "MVRV가 1 미만이니 반드시 저점"
현실: 2018년과 2022년 모두 MVRV 1 미만에서 추가 하락 발생
교훈: NUPL + MVRV + 거래소 보유량 + 기술적 지지 구조를
      동시에 확인해야 신뢰도 향상
```

### 4. 이더리움에 비트코인 UTXO 분석을 그대로 적용

```
문제: 이더리움은 UTXO 모델이 아닌 계정(Account) 모델 사용
→ 비트코인의 HODL 웨이브, UTXO 나이 분석이 직접 적용 불가
→ 이더리움 온체인 분석은 별도의 지표 체계 사용

대응: 이더리움 분석 시 ETH 전용 지표 사용
      (예: ETH 스테이킹 비율, Gas 사용량, DeFi TVL 등)
```

### 5. 온체인 데이터 제공업체의 방법론 차이 무시

```
문제: Glassnode와 CryptoQuant의 동일 지표가 다른 값 표시
이유: 거래소 주소 식별 방법론, 데이터 수집 시점 차이
결과: 지표 해석의 혼란

대응: 하나의 플랫폼을 주로 사용하고 일관성 유지
      절대값보다 추세 변화에 집중
```

---

## 온체인 분석의 한계

1.**데이터 해석의 모호성**: 거래소 내부 이체, 수탁(custodial) 지갑 정리 등은 매도와 무관하더라도 유입으로 집계될 수 있습니다.
2.**주요 데이터 제공업체 의존**: Glassnode, CoinMetrics 등 유료 서비스가 정교한 지표를 제공하지만, 비용이 발생합니다.
3.**이더리움 등 계정 모델**: 비트코인의 UTXO 기반 분석 방법이 이더리움에 그대로 적용되지 않을 수 있습니다.
4.**지연 효과**: 온체인 지표는 후행성을 가지며, 가격 선행 신호로 맹신해서는 안 됩니다.

## 에지 케이스: ETF 승인 이후 온체인 분석의 변화

비트코인 현물 ETF 승인(2024년 1월) 이후 대규모 BTC가 ETF 수탁 기관(Coinbase Custody 등) 지갑으로 이동하면서 전통적인 "거래소 보유량 감소 = 강세 신호" 해석이 복잡해졌습니다. ETF 자금 유입으로 발생한 거래소 유출이 실제 개인 보유 증가와 동일하지 않기 때문입니다. ETF 데이터(일별 순유입/유출)를 온체인 데이터와 함께 참고하여 보다 정확한 수급 분석을 하십시오.

## 핵심 요약

- 온체인 분석은 블록체인의 공개 데이터를 활용해 가격 외 변수(자금 흐름, 수익성, 네트워크 활동)를 관찰합니다.
- 거래소 유입 급증은 단기 매도 압력 신호이며, 유출은 장기 보유 의도를 나타냅니다.

- MVRV는 현재 가격이 시장 평균 원가 대비 얼마나 과대/과소평가됐는지를 보여줍니다.
- 온체인 지표는 단독 사용보다 기술적 분석과 병행할 때 신뢰도가 높아집니다.
