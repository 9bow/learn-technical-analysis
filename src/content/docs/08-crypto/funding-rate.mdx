## 무기한 선물이란?

암호화폐 시장에서 가장 많이 거래되는 파생상품은무**기한 선물(Perpetual Futures, 영구 계약**)입니다. 전통 선물과의 핵심적인 차이는만**기일이 없다는** 것입니다.

| 구분 | 전통 선물 | 무기한 선물 |
|------|----------|-----------|
| 만기일 | 있음 (월물, 분기물) | 없음 |
| 결제 방식 | 만기 시 현물 또는 현금 정산 | 펀딩비로 지속 조정 |
| 가격 수렴 | 만기 시 현물가로 자동 수렴 | 펀딩비 메커니즘으로 수렴 |
| 주요 시장 | CME(비트코인 선물) | Binance, Bybit, OKX 등 |

만기일이 없기 때문에 선물 가격이 현물가에서 크게 이탈하는 것을 막기 위해펀**딩비(Funding Rate**)라는 독특한 메커니즘이 도입되었습니다 [Binance Academy, 2026].

## 펀딩비 메커니즘

### 기본 원리

펀딩비는롱**(매수) 포지션 보유자와 숏(매도) 포지션 보유자 사이에 주기적으로 교환되는 비용입**니다. 거래소가 수취하는 수수료가 아니라, 트레이더 간에 직접 주고받는 금액입니다.

```
펀딩비 지급 주기: 대부분의 거래소에서 8시간마다 (하루 3회)
지급 대상: 해당 시점에 포지션을 보유하고 있는 트레이더
```

### 펀딩비 방향

**양(+) 펀딩비**: 선물 가격이 현물가보다 높을 때 (콘탱고 상태)
- 롱 보유자가 숏 보유자에게 비용을 지급합니다.
- 결과적으로 롱 포지션 비용 증가 → 선물가 하락 압력 → 현물가로 수렴

**음(-) 펀딩비**: 선물 가격이 현물가보다 낮을 때 (백워데이션 상태)
- 숏 보유자가 롱 보유자에게 비용을 지급합니다.
- 결과적으로 숏 포지션 비용 증가 → 선물가 상승 압력 → 현물가로 수렴

```
예시 계산:
- 포지션 규모: 10,000 USDT (레버리지 10x → 실제 노출 100,000 USDT)
- 펀딩비율: +0.1% (8시간마다)
- 롱 보유자가 지불하는 비용: 100,000 × 0.001 = 100 USDT (8시간마다)
- 연간 환산 시: 0.1% × 3회 × 365일 = 109.5% APR
```

>**핵심**: 펀딩비는 시장 참여자들이 선물 가격을 현물 가격 근처에 유지하도록 경제적으로 유인하는 자기조정 메커니즘입니다 [Bybit Learn, 2026].

## 펀딩비를 매매 신호로 활용

### 정상 범위

대부분의 시장에서 펀딩비는 `+0.01%` 내외를 유지합니다. 많은 거래소가 이 값을 기본(중립) 펀딩비로 설정해 둡니다.

| 펀딩비 범위 | 시장 해석 |
|-----------|----------|
| +0.01% 내외 | 정상, 중립적 시장 |
| +0.05% ~ +0.1% | 과도한 롱 편향, 주의 필요 |
| +0.1% 초과 | 극단적 과매수 — 역추세 신호 |
| -0.05% 미만 | 강한 숏 편향 |
| -0.1% 미만 | 극단적 과매도 — 역추세 신호 |

### 역추세(Contrarian) 신호

펀딩비가 극단적으로 높을 때(예: +0.2% 이상)는 시장이과**도하게 롱에 쏠린상**태를 의미합니다. 이 상황에서:

1. 롱 포지션 유지 비용이 커져 점점 많은 롱이 강제 청산됩니다.
2. 아직 숏을 치지 않은 세력이 이 과열 상태를 역이용해 숏을 잡습니다.
3. 결과적으로 가격이 급락하는롱** 스퀴즈(Long Squeeze**)가 발생하기 쉽습니다.

```
역추세 활용 원칙:
- 펀딩비 극단 + 가격 상승 둔화 → 숏 관심 구간
- 펀딩비 극단 음수 + 가격 하락 둔화 → 롱 관심 구간
- 단독 신호로 진입하지 말 것 — 가격 구조 확인 필수
```

>**실전 예시**: 2021년 4월 비트코인이 64,000달러 고점을 형성할 당시 일부 거래소의 펀딩비는 +0.3%를 초과했습니다. 이후 50% 이상 급락이 발생하며 극단적 펀딩비가 고점 신호였음을 보여주었습니다.

### 누적 펀딩비 (Cumulative Funding Rate)

단일 시점의 펀딩비보다누**적 펀딩비 추이가** 더 신뢰할 수 있는 신호를 제공합니다. 장기간 높은 양의 펀딩비가 누적되면, 시장은 이를 해소하는 방향으로 움직이는 경향이 있습니다.

## 펀딩비 재정거래 (Funding Rate Arbitrage)

### 기본 구조

펀딩비가 일관되게 높을 때, 방향성 리스크 없이 펀딩비 수입만을 목적으로 하는 전략입니다.

```
구성:
1. 현물 매수 (또는 현물 ETF 보유)
2. 무기한 선물 동일 수량 숏 포지션 진입
3. 델타 중립 달성 (가격 변동 리스크 제거)
4. 펀딩비 수익을 숏 측에서 수령
```

이 전략을캐**시 앤 캐리(Cash-and-Carry**)또는델**타 뉴트럴 펀딩 전략이**라 부릅니다.

### 리스크 요인

1.**펀딩비 방향 전환**: 양의 펀딩비가 음으로 바뀌면 오히려 비용을 지불해야 합니다.
2.**청산 리스크**: 레버리지 포지션이 포함되면 가격 급변 시 마진 부족 청산 가능성이 있습니다.
3.**거래 비용**: 진입/청산 수수료와 슬리피지가 수익을 잠식할 수 있습니다.

## 펀딩비 계산 메커니즘 상세

### 펀딩비 구성 요소

대부분의 거래소에서 펀딩비는 두 가지 요소로 구성됩니다.

$$\text{펀딩비} = \text{프리미엄 지수} + \text{이자율 컴포넌트}$$

$$\text{프리미엄 지수} = \frac{\frac{\text{선물 최우선 매수호가} + \text{선물 최우선 매도호가}}{2} - \text{현물 지수 가격}}{\text{현물 지수 가격}}$$

- 이자율 컴포넌트: 대부분의 거래소에서 연 10.95% (하루 0.03%, 8시간 0.01%)를 기본값으로 설정 → USD 차입 비용을 반영

### 바이낸스 펀딩비 계산 예시

```
[실제 계산 예시 — 바이낸스 BTCUSDT 영구 계약]

현물 지수 가격: 50,000 USDT
선물 최우선 매수호가: 50,080 USDT
선물 최우선 매도호가: 50,120 USDT

선물 중간가: (50,080 + 50,120) / 2 = 50,100 USDT

프리미엄 지수:
= (50,100 - 50,000) / 50,000
= 100 / 50,000
= 0.002 (0.2%)

기본 이자율: 0.01% (8시간)

최종 펀딩비:
= 이자율 + 클램핑(프리미엄 - 이자율)
= 0.01% + 클램핑(0.2% - 0.01%)
= 0.01% + 클램핑(0.19%)

클램핑 범위: -0.05% ~ +0.05% (바이낸스 기준)
클램핑 결과: 0.19% → +0.05%로 상한 제한

최종 펀딩비: 0.01% + 0.05% = 0.06% (8시간마다)
```

:::note
클램핑(Clamping)은 극단적인 시장 상황에서 펀딩비가 과도하게 높아지거나 낮아지는 것을 방지하는 장치입니다. 거래소마다 클램핑 범위가 다릅니다 (바이낸스: ±0.05%, Bybit: ±0.375%).
:::

### 펀딩비 지급 금액 계산

```
[포지션별 펀딩비 지급/수령 계산]

시나리오: BTC 가격 50,000 USDT, 펀딩비 +0.06%

포지션 A: 롱 10 BTC (노출 금액 500,000 USDT)
지급 비용: 500,000 × 0.06% = 300 USDT (8시간마다 지급)
일간 비용: 300 × 3회 = 900 USDT
연간 환산: 900 × 365 = 328,500 USDT (65.7% APR)

포지션 B: 숏 10 BTC (노출 금액 500,000 USDT)
수령 금액: 500,000 × 0.06% = 300 USDT (8시간마다 수령)
→ 포지션 A가 포지션 B에게 직접 지급
```

:::caution
레버리지 10배 포지션에서 연 65.7% APR의 펀딩비는 원금 대비 매우 큰 비용입니다. 장기 롱 포지션 보유 시 펀딩비 누적 비용을 반드시 계산에 포함해야 합니다.
:::

## 펀딩비 재정거래 전략 심화

### 캐시 앤 캐리 전략 단계별 실행

```
[전략 구성 예시]

목표: BTC 펀딩비 +0.1% (8시간)를 활용한 무방향 수익

Step 1: 현물 매수
현물 거래소에서 BTC 1개 매수 (50,000 USDT 투입)

Step 2: 선물 숏 진입
무기한 선물에서 BTC 1개 상당 숏 포지션 진입
증거금: 5,000 USDT (레버리지 10배 기준)

Step 3: 델타 확인
현물 +1 BTC + 선물 -1 BTC = 순 포지션 0 (델타 뉴트럴)
→ 가격이 오르든 내리든 BTC 가치 변동에 중립

Step 4: 펀딩비 수익 수령
8시간마다: 50,000 × 0.1% = 50 USDT 수령
일간: 150 USDT
월간: 약 4,500 USDT
연간 수익률: 4,500 × 12 / 55,000 ≈ 98% APR (총 투자금 기준)
```

### 실질 수익률 계산 (비용 차감)

```
총 투자금: 50,000 (현물) + 5,000 (증거금) = 55,000 USDT

월간 수익 계산:
펀딩비 수입: +4,500 USDT
현물 매수 수수료 (0.1%): -50 USDT
선물 진입 수수료 (0.05%): -25 USDT
선물 청산 수수료 (0.05%): -25 USDT (청산 시)
현물 출금 슬리피지: -50 USDT

순 월간 수익 (첫 달): ≈ 4,350 USDT
실질 월간 수익률: 4,350 / 55,000 ≈ 7.9%
```

### 전략 위험 요소 상세

```
위험 1: 펀딩비 역전
+0.1% 펀딩비가 -0.05%로 전환 시
→ 숏 포지션에서 비용 지불 시작
→ 즉시 포지션 해제 필요

위험 2: 현물 거래소와 선물 거래소 가격 괴리
→ 청산 시 양 거래소 가격 차이로 예상 외 손실 발생 가능

위험 3: 선물 마진 콜
가격 급등 시 숏 포지션 마진 부족
→ 현물 BTC를 담보로 추가 마진 공급 필요
→ 현물-선물 거래소가 다르면 이동 시간 지연 위험

위험 4: 거래소 리스크
거래소 해킹, 파산, 출금 제한 시 손실 가능
→ 유명 대형 거래소 선택 필수
```

:::tip
펀딩비가 +0.05% 이상 안정적으로 유지될 때만 전략을 실행합니다. 단기 고펀딩비는 빠르게 정상화되는 경향이 있으므로 최소 48시간 이상 지속된 고펀딩비를 확인 후 진입하세요.
:::

## 실전 적용

### 펀딩비 모니터링 루틴

```
[일간 체크 루틴]

1. CoinGlass에서 BTC/ETH 주요 거래소 펀딩비 확인
   → 정상 범위(+0.01% 내외) vs 이상 범위 파악

2. 펀딩비 추이 확인 (7일 누적)
   → 지속적 상승: 과열 경고
   → 지속적 하락/음수: 과도한 숏 편향

3. 펀딩비 + 미결제약정(OI) 조합 확인
   → 펀딩비 급등 + OI 급증: 레버리지 과부하 → 조정 위험
   → 펀딩비 급등 + OI 감소: 청산 진행 중

4. 가격 구조와 대조
   → 펀딩비 극단 + 저항 구조: 숏 진입 관심
   → 펀딩비 극단 음수 + 지지 구조: 롱 진입 관심
```

### 펀딩비를 활용한 시장 심리 지수 구성

```
시장 심리 판단표 (펀딩비 기반):

+0.05% 이상 + 가격 신고점 = 극단적 탐욕 (숏 준비)
+0.02% ~ +0.05% + 가격 상승 = 과열 주의
+0.005% ~ +0.02% = 정상 강세
-0.005% ~ +0.005% = 중립
-0.005% ~ -0.02% = 정상 약세
-0.02% 이하 + 가격 신저점 = 극단적 공포 (롱 준비)
```

## 흔한 실수

### 1. 펀딩비 하나만 보고 포지션 방향 결정

```
실수: 펀딩비 +0.15% → 즉시 숏 진입
위험: 강한 추세장에서 펀딩비는 장기간 높게 유지 가능
      추세에 역행하는 숏이 더 큰 손실 유발

올바른 접근:
펀딩비 극단 + 가격 구조 저항 + 모멘텀 약화 → 종합 판단 후 진입
```

### 2. 재정거래 시 펀딩비 역전 리스크 미고려

```
실수: +0.1% 펀딩비만 보고 캐리 트레이드 진입
상황: 다음 날 펀딩비 -0.03%로 역전
결과: 수령 대신 지급 → 수익 손실 + 포지션 해제 비용

대응: 진입 전 "펀딩비가 음수로 전환될 경우 즉시 청산" 규칙 설정
      알림 설정으로 펀딩비 방향 전환 즉시 감지
```

### 3. 누적 펀딩비 비용을 무시한 채 롱 포지션 장기 보유

```
상황: BTC 강세 확신으로 10배 레버리지 롱 장기 보유
펀딩비: 평균 +0.03% (8시간)

1개월 누적 비용:
= 0.03% × 3회 × 30일 = 2.7% (포지션 노출 금액 기준)
레버리지 10배 → 원금 대비 27% 비용 발생

결과: 가격이 5% 상승해도 펀딩비로 실질 수익 거의 없음
```

### 4. 거래소 간 펀딩비 차이를 기회로 착각

```
오해: A 거래소 +0.1%, B 거래소 -0.05% → 두 거래소 간 재정거래
현실: 거래소 간 BTC 이동에 수십 분~수 시간 소요
      이동 중 가격 변동 + 출금 수수료로 수익 소멸 가능
      일부 거래소는 출금에 KYC/제한 존재

실현 가능한 재정거래는 알고리즘 봇 운영자들이 이미 소진
```

### 5. 미결제약정(OI) 없이 펀딩비만 분석

```
불완전한 분석: 펀딩비 +0.08% → "과열, 곧 하락"
완전한 분석:
+ OI 동시 증가 → 새로운 레버리지 포지션 유입 (조정 위험 ↑)
+ OI 동시 감소 → 청산 진행 중 (과열 해소 중)

펀딩비는 반드시 OI, 가격 구조, 거래량과 함께 해석해야 합니다.
```

## 에지 케이스: 펀딩비 역전 후 반전 신호

펀딩비가 장기간(수일~수주) 음수(-0.02% 이하)를 유지하다가 갑자기 0 근방으로 복귀하는 시점은 강한 매수 신호일 수 있습니다. 과도한 숏 포지션이 청산되면서 가격이 급등하는 "숏 스퀴즈(Short Squeeze)"가 발생할 수 있기 때문입니다. 반대로 장기 양의 펀딩비가 0 근방으로 급락하는 시점은 롱 포지션의 대규모 청산("롱 스퀴즈")을 예고할 수 있습니다. 펀딩비의 방향 전환 속도와 미결제약정(OI) 변화를 함께 관찰하십시오.

## 핵심 요약

- 무기한 선물의 펀딩비는 선물 가격을 현물 가격 근처에 유지시키는 자동 조정 장치입니다.
- 양의 펀딩비는 롱 보유자 → 숏 보유자로 비용이 흐르며, 음의 펀딩비는 반대 방향입니다.
- 펀딩비가 극단적으로 높거나 낮으면 시장의 과열 또는 과냉을 나타내는 역추세 신호로 활용할 수 있습니다.
- 펀딩비 재정거래(델타 뉴트럴)는 방향성 없이 펀딩 수익을 추구하는 전략이지만, 펀딩비 역전과 청산 리스크를 반드시 관리해야 합니다.
- CoinGlass 등 도구를 활용해 거래소 간 펀딩비를 비교하고 시장 심리를 파악하세요.
